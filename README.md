# <a name="microsoft-powerapps-documentation"></a>Документация по Microsoft PowerApps

Это репозиторий GitHub для технической документации по продукту **PowerApps**. Эта документация опубликована в [https://docs.microsoft.com/powerapps](https://docs.microsoft.com/powerapps).

## <a name="how-to-contribute"></a>Как принять участие

Благодарим за ваш интерес в дополнении публикаций [Docs.microsoft.com](https://docs.microsoft.com/) — сборника технических статей по продуктам и службам Майкрософт.

Чтобы узнать, как опубликовать статью в репозитории, ознакомьтесь с [руководством по публикации документации](https://docs.microsoft.com/contribute). Если вы являетесь сотрудником Майкрософт, просмотрите [внутреннюю версию](https://aka.ms/docsguidescontribute) руководства.
