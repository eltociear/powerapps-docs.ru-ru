# <a name="functional-application-consultant-learning-catalog"></a>Каталог обучения консультантов функционального приложения

Вы являетесь экспертом по внедрению для бизнес-области? 

Следующий каталог организован в порядке от основных знаний до конкретных областей, и от самых базовых до самых сложных. Если содержимое существует в нескольких форматах, мы сообщим вам об этом, чтобы вы могли выбрать формат обучения, который наилучшим образом отвечает вашим потребностям.

Для функциональных консультантов наше исследование анализа задания-задачи показало, что каждый должен понимать основной набор содержимого, а также область домена.
