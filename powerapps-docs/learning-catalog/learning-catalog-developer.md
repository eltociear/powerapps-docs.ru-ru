# <a name="developer-learning-catalog"></a>Каталог обучения для разработчика

Вам нужно написать код для интеграции с другими источниками данных, расширить функциональность основной системы или создать сложное приложение?

Следующий каталог организован в порядке от основных знаний до конкретных областей, и от самых базовых до самых сложных. Если содержимое существует в нескольких форматах, мы сообщим вам об этом, чтобы вы могли выбрать формат обучения, который наилучшим образом отвечает вашим потребностям.
