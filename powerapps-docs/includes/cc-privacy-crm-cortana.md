Cortana не имеет доступ к данным, хранящимся в базе данных Dynamics 365. Cortana лишь принимает голосовые команды. При использовании голосовых команд в Cortana Майкрософт использует собственные технологии, например акустические модели и модели обработки живой речи, для записи и интерпретации запроса пользователя с целью распознавания запроса пользователя и улучшения продуктов и услуг по распознаванию речи от Майкрософт. Голосовые поисковые запросы Bing обрабатываются так же, как другие текстовые поисковые запросы, и могут использоваться для улучшения результатов поиска Bing, однако голосовые команды Dynamics 365, представленные здесь, не используются в целях рекламы. Дополнительные сведения о конфиденциальности и Кортане см. в разделе [Вопросы и ответы по Кортане и конфиденциальности](http://www.windowsphone.com/en-us/how-to/wp8/cortana/cortana-and-my-privacy-faq).  
  
 Если вы являетесь администратором, вы можете управлять доступом предприятия к Cortana вместе с поставщиком услуг конфигурации PolicyManager посредством отдельной службы управления устройствами. Политика для этой службы устанавливается на уровне телефона и не может быть установлена только для приложения Dynamics 365 для телефонов. Дополнительные сведения см. в разделе MSDN [Поставщик услуг конфигурации PolicyManager](https://msdn.microsoft.com/library/dn499739.aspx).
