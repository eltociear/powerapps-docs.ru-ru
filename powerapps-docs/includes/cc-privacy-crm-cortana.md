У Cortana отсутствует доступ к данным, хранящимся в базе данных Dynamics 365. Cortana записывает только ваши голосовые команды. При подаче голосовых команд через Cortana, чтобы и понять запрос, и улучшить продукты и службы Майкрософт, связанные с распознаванием речи, корпорация Майкрософт использует собственные технологии, такие, как, например, модели обработки акустических и естественных языков для записи и интерпретации запросов пользователей. Поисковые запросы Bing, введенные с голоса, обрабатываются как другие, текстовые, поисковые запросы, и могут быть использованы для улучшения результатов поиска Bing. Но поданные голосом команды Dynamics 365, перечисленные ниже, не используются для предоставления рекламы. Дополнительные сведения о конфиденциальности и Cortana см. в разделе [Вопросы и ответы по Cortana и моей конфиденциальности](http://www.windowsphone.com/en-us/how-to/wp8/cortana/cortana-and-my-privacy-faq).  
  
 Будучи администратором, можно управлять корпоративным доступом к Cortana с помощью поставщика службы настройки PolicyManager, используя отдельную службу управления устройством. Соответствующая политика устанавливается на уровне телефона и не может быть установлена только для приложения Dynamics 365 для телефонов. Дополнительные сведения см. в разделе MSDN [Поставщик службы настройки PolicyManager](https://msdn.microsoft.com/library/dn499739.aspx).
