[!INCLUDE[](pn-social-engagement-long.md)] использует Microsoft Cognitive Services для определения языка содержимого в личных сообщениях со страниц [!INCLUDE[](tn-twitter.md)] и [!INCLUDE[](tn-facebook.md)]. Для определения языка в Microsoft Cognitive Services отправляется только заголовок и содержимое личных сообщений. Заголовок и содержимое личных сообщений не сохраняются в Microsoft Cognitive Services. Просмотрите [заявление о конфиденциальности для Microsoft Cognitive Services](https://go.microsoft.com/fwlink/p/?linkid=867081).
