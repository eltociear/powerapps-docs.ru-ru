Чтобы использовать Microsoft Dynamics 365 для Outlook, вам нужно войти с помощью своих учетных данных (адрес электронной почты и пароль). Вы можете сохранить эту информацию локально, чтобы она не запрашивалась при каждом открытии Outlook. Если вы решили сохранить эту информацию локально, Dynamics 365 для Outlook автоматически подключается к Microsoft Dynamics 365 (Online) при каждом открытии Outlook.  
  
 После первого входа и использования Dynamics 365 для Outlook соединение между вашим компьютером и Dynamics 365 (Online) всегда будет установлено постоянно при наличии доступа к Интернету. Вы можете отключить это соединение только с помощью параметра конфигурации, но в этом случае производительность Dynamics 365 для Outlook может снизиться.  
  
 Если вы используете Dynamics 365 для Outlook для отслеживания электронной почты, цепочка сообщений электронной почты будет видна пользователям вашей организации, имеющим разрешение на ее просмотр.  
  
Для каждого получаемого вами сообщения Dynamics 365 для Outlook будет отправлять в Dynamics 365 (Online) адрес электронной почты отправителя, адрес электронной почты получателя и строку темы сообщения. Это позволяет Dynamics 365 (Online) определять, следует ли сохранять конкретное сообщение в службе Dynamics 365 (Online). При отслеживании элемента его копия будет храниться в службе Dynamics 365 и будет видна другим пользователям вашей организации с соответствующими разрешениями. Когда вы отменяете отслеживание элемента, эта копия автоматически удаляется из Dynamics 365 (Online) только в том случае, если вы являетесь владельцем элемента.
