Включив мобильное приложение Project Finder, вы предоставите пользователям в организации возможность прозрачной работы над проектами и регистрации для участия в работе. Пользователи также могут работать с приложением Project Finder, чтобы просматривать и обновлять свои навыки. Мобильное приложение Project Finder включается с помощью параметров службы проектов; установите значения атрибутов "Требования к ресурсам видны ресурсам" и "Разрешить обновление навыков по ресурсам" равными "Да". Изменение значений этих атрибутов на "Нет" отключает эту функцию.  
  
 При использовании этой функции необходимо принять во внимание следующее.  
  
-   Требования к ресурсам видны ресурсам. Установка этого атрибута равным "Да" предоставляет пользователям возможность видеть рабочий проект и дает им следующие сведения о проекте:  
  
    -   имя проекта;  
  
    -   имя роли;  
  
    -   имя клиента;  
  
    -   адрес клиента;  
  
    -   длительность работы.  
  
-   Разрешить обновление навыков по ресурсам. Установка этого атрибута равным "Да" предоставляет пользователям возможность добавить навыки в свои профили и задавать уровень квалификации для имеющихся у них навыков.
