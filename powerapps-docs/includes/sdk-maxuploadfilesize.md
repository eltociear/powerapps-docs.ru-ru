Максимальный размер отправляемых файлов определяется свойством **Organization.MaxUploadFileSize**. Это свойство задается на вкладке **Электронная почта** в области **Системные параметры** приложения Dynamics 365. Этот параметр ограничивает размер файлов, которые можно присоединять к сообщениям электронной почты, заметкам и веб-ресурсам. Значение по умолчанию — 5 МБ.
