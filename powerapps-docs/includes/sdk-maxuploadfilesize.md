Максимальный размер файлов, которые можно отправить, определяется свойством **Organization.MaxUploadFileSize**. Это свойство можно задать на вкладке **Электронная почта** окна **Параметры системы** в приложении Dynamics 365. Этот параметр ограничивает размер файлов, которые можно вложить в сообщения электронной почты, заметки и веб-ресурсы. Значение по умолчанию равно 5 МБ.
