Если вы принимаете условия управления мероприятиями, будет активирована функция интеграции с вебинарами. Функция интеграции с вебинарами позволяет проводить мероприятия и сессии благодаря партнерству с поставщиком вебинаров. Чтобы пользоваться какой-либо из служб поставщиков вебинаров, требуется учетная запись такой службы. В настоящее время готовое решение предлагает только один поставщик вебинаров — ON24. При использовании функции интеграции с вебинарами необходимые для этого данные будут обрабатываться и храниться в [!INCLUDE[pn-azure-service-fabric](../includes/pn-azure-service-fabric.md)], после чего передаваться ON24. К таким данным может относиться информация о регистрации участников, например их имена, адреса электронной почты и названия компаний. Кроме того, ON24 будет передавать в [!INCLUDE[pn-microsoftcrm](../includes/pn-microsoftcrm.md)] через [!INCLUDE[pn-azure-service-fabric](../includes/pn-azure-service-fabric.md)] соответствующие показатели, например информацию о длительности просмотра.

Нет необходимости сначала активировать поддержку вебинаров, чтобы использовать остальные элементы решения по управлению мероприятиями. Администратор может отключить функцию интеграции с вебинарами, удалив учетные данные в конфигурации вебинаров.

Компоненты и службы [!INCLUDE[pn-windows-azure](../includes/pn-windows-azure.md)], используемые функцией интеграции с вебинарами:

- [!INCLUDE[pn_azure_key_vault](../includes/pn_azure_key_vault.md)] ([!INCLUDE[proc-more-information](../includes/proc-more-information.md)] [Что такое Azure Key Vault?](https://docs.microsoft.com/en-us/azure/key-vault/key-vault-whatis))
  - Предоставляет ключ шифрования для шифрования и расшифровки учетных данных ON24 клиента
- [!INCLUDE[pn-azure-service-fabric](../includes/pn-azure-service-fabric.md)] ([!INCLUDE[proc-more-information](../includes/proc-more-information.md)] [Обзор Azure Service Fabric](https://docs.microsoft.com/en-us/azure/service-fabric/service-fabric-overview))
  - Обрабатывает и отправляет регистрационные данные и учетные данные участников вебинаров в ON24
  - Передает показатели вебинаров из ON24 в [!INCLUDE[pn-microsoftcrm](../includes/pn-microsoftcrm.md)]. Сохраняет учетные данные ON24 клиента (с пользовательским шифрованием)
