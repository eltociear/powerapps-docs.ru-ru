Microsoft Social Engagement сохраняет настройки поиска данных клиента и курирования данных в изолированных базах данных (данные клиента). Данные клиентов затем кэшируется на стороне сервера во внутреннем приложении для обеспечения индексирования поиска с целью увеличения производительности решения.   
 Доступ к индексу кэшированных данных (данные клиента) обслуживается исключительно внутренним приложением, которое не разрешает пользователям получать доступ или изменять индекс кэшированных данных в интерактивном режиме во внутреннем приложении. После прекращения действия лицензионного соглашения или соглашения о подписке данные индекса удаляются по истечении 180 дней, в соответствии с нашей политикой хранения данных.
