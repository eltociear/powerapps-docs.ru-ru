Microsoft Social Engagement хранит пользовательские конфигурации поиска и проверки данных в базах данных, разделенных по клиентам (данные клиента). Данные клиента затем кэшируются на стороне сервера во внутреннем приложении, что обеспечивает единое индексирование при извлечении исключительно с целью максимально увеличить производительность решения.   
 Доступ к индексу кэшированных данных (данным клиента) ведется исключительно во внутреннем приложении; пользователям запрещено читать или изменять индекс кэшированных данных интерактивно во внутреннем приложении. По истечении срока действия лицензионного соглашения подписки индекс кэшированных данных клиента удаляется в течение 180 дней в соответствии с нашими политиками хранения данных.
