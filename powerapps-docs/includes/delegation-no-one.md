
При использовании с источником данных эту функцию невозможно делегировать. Будет извлечена только первая часть источника данных, затем будет применена функция. Результат может не отражать ситуацию в целом.  При создании отображается предупреждение, чтобы напомнить об этом ограничении и предложить по возможности переключиться на подлежащие делегированию альтернативы. Дополнительные сведения см. в [обзоре делегирования](../maker/canvas-apps/delegation-overview.md).
