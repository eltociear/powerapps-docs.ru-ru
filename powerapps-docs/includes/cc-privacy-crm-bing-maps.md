При использовании Microsoft Dynamics 365 компонент "Карты Bing" автоматически отправляет адрес через Интернет службе карт Bing для отображения в Dynamics 365 интернет-карты с адресом.  При щелчке карт Bing в Dynamics 365 вы будете перенаправлены на сайт [www.bing.com/maps](http://www.bing.com/maps). Использование карт Bing также регулируется соглашением об использовании [карт Bing для конечных пользователей](http://go.microsoft.com/?linkid=9710837).  
  
 Ваш администратор может включить или отключить функцию карт Bing в разделе **Параметры** > **Администрирование** > **Системные параметры**. Отключение приложения карт Bing приведет к отключению этой функции в Dynamics 365.  
  
 На сведения, отправляемые в карты Bing, распространяется [заявление о конфиденциальности и файлы cookie корпорации Майкрософт](http://go.microsoft.com/fwlink/p/?linkid=521839).
