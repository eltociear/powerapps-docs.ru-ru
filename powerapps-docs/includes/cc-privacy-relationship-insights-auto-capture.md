После включения функции автоматического сбора данные [!INCLUDE[pn_Exchange](pn-exchange.md)] будут автоматически извлекаться [!INCLUDE[pn_dynamics_crm](pn-dynamics-crm.md)], но не сохраняться, для отображения действий с электронной почтой вошедшего в систему пользователя. Пользователи могут явным образом отслеживать эти сообщения, чтобы копировать сообщения [!INCLUDE[pn_Exchange](pn-exchange.md)] в [!INCLUDE[pn_dynamics_crm](pn-dynamics-crm.md)] в качестве действий электронной почты для предоставления к ним доступа определенным пользователям, которые задаются разрешениями [!INCLUDE[pn_dynamics_crm](pn-dynamics-crm.md)].  
  
 Администратор может включать и отключать функцию автоматического сбора. Для этого ему необходимо перейти в раздел **Параметры** > **Конфигурация аналитики**, открыть вкладку **Автоматический сбор** и выбрать соответствующие параметры. Кроме того, пользователь может отключить функцию автоматического сбора на вкладке "Электронная почта" в разделе **Личные параметры**. Для этого ему нужно изменить значение параметра **Показывать электронные сообщения, которые не отслеживаются в Dynamics 365, в списке действий** на **Нет**.
