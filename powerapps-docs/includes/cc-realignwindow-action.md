Отображает размещенный элемент управления в указанном месте монитора. Размещенные элементы управления можно отображать на одном или двух мониторах. Это действие применимо к тем экземплярам размещенных элементов управления, которые настроены на размещение на панелях типов USDFloatingPanel и USDFloatingToolPanel.  
  
|Параметр|Описание|  
|---------------|-----------------|  
|screen|Задает экран, на котором должен отображаться размещенный элемент управления. Допустимые значения — 1 и 2. Если этот параметр не указан, по умолчанию передается значение 1.|  
|left|Задает позицию (в процентах) от левого края целевого монитора, на котором должен отображаться размещенный элемент управления. Допустимые значения — от 0 до 100. Если этот параметр не указан, по умолчанию передается значение 0.|  
|top|Задает позицию (в процентах) от верхнего края целевого монитора, на котором должен отображаться размещенный элемент управления. Допустимые значения — от 0 до 100. Если этот параметр не указан, по умолчанию передается значение 0.|  
|width|Задает ширину (в процентах) окна размещенного элемента управления на целевом мониторе. Допустимые значения — от 1 до 100. Если этот параметр не указан, по умолчанию передается значение 100.|  
|height|Задает высоту (в процентах) окна размещенного элемента управления на целевом мониторе. Допустимые значения — от 1 до 100. Если этот параметр не указан, по умолчанию передается значение 100.|
