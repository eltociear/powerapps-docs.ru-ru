Отображает размещенный элемент управления в заданном расположении на мониторе. Можно отобразить размещенный элемент управления на одном или двух мониторах. Это действие актуально для размещенных экземпляров элементов управления, которые настроены для размещения на панели типа USDFloatingPanel или USDFloatingToolPanel.  
  
|Параметр|Описание|  
|---------------|-----------------|  
|экран|Указывает на экран для отображения размещенного элемента управления. Допустимые значения: 1 или 2. Если этот параметр не задан, по умолчанию передается значение 1.|  
|left|Задает положение (в процентах) от левого края экрана на целевом мониторе, где должен отображаться размещенный элемент управления. Допустимые значения: от 0 до 100. Если этот параметр не задан, по умолчанию передается значение 0.|  
|top|Задает положение (в процентах) от верхнего края экрана на целевом мониторе, где должен отображаться размещенный элемент управления. Допустимые значения: от 0 до 100. Если этот параметр не задан, по умолчанию передается значение 0.|  
|width|Задает ширину (в процентах) окна размещенного элемента управления на целевом мониторе. Допустимые значения: от 1 до 100. Если этот параметр не задан, по умолчанию передается значение 100.|  
|height|Задает высоту (в процентах) окна размещаемого элемента управления на целевом мониторе. Допустимые значения: от 1 до 100. Если этот параметр не задан, по умолчанию передается значение 100.|
