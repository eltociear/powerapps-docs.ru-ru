Microsoft Social Engagement использует файлы cookie сеанса, чтобы пользователи переходили со страницы на страницу без необходимости каждый раз входить в систему. Руководствуясь указаниями пользователя, мы можем использовать файлы cookie для хранения учетных данных, чтобы обеспечить аутентификацию для последующих сеансов.   
 Файлы cookie сеансов — важный фактор работы с Microsoft Social Engagement. Отказаться от их использования нельзя. Но файлы cookie для входа сохраняются только по запросу пользователя.
