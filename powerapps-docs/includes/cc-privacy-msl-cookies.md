Microsoft Social Engagement использует файлы cookie сеансов, чтобы позволить пользователям переходить со страницы на страницу без необходимости повторно выполнять вход на каждой из страниц. По выбору пользователя мы можем также использовать файлы cookie для сохранения учетных данных, упрощая проверку подлинности в последующих сеансах.   
 Файлы cookie сеансов необходимы для работы Microsoft Social Engagement, отключить использование этих файлов cookie невозможно. Однако файлы cookie входа в систему сохраняются лишь по запросу пользователя.
