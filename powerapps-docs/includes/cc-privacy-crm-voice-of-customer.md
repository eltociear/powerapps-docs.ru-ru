Включив решение "Мнение клиента" для Dynamics 365, при публикации опроса в Dynamics 365 определение опроса будет отправлено в Azure и сохранено в службе хранилища Azure. Когда респондент отправляет опрос (предварительно перейдя по ссылке с приглашением пройти опрос, отправленный ему по электронной почте), ответы на опрос временно хранятся на Служебной шине Azure, а затем извлекаются и сохраняются в Dynamics 365. После сохранения ответов в Dynamics 365 они удаляются из Azure.  
  
 Обратите внимание, что при отрисовке опроса для респондента в опрос можно включать данные Dynamics 365, такие как имя клиента, имя продукта, номер обращения и т. д. (внутри элементов опроса, таких как вопросы, ответы и т. д.). Когда создается ссылка на приглашение опроса, данные Dynamics 365 будут отправлены из Dynamics 365 и сохранены в Базе данных SQL Azure в обмен на идентификатор, который используется в ссылке-приглашении на опрос. Этот идентификатор используется для отображения данных Dynamics 365 в рамках опроса после его открытия с использованием ссылки-приглашения. Идентификаторы в ссылке на опрос, которые отправляются по электронной почте респонденту, хранятся в системе электронной почты респондентов.  
  
 Администратор может включить функцию "Мнение клиента" для Dynamics 365, установив ее как решение в организации Dynamics 365. Кроме того, впоследствии администратор может отключить функцию, удалив это решение из организации Dynamics 365.  
  
 Компоненты и службы Azure, используемые с функцией "Мнение клиента" для Dynamics 365, подробно рассматриваются в следующих разделах.  
  
 Примечание. Дополнительные сведения о дополнительных предложениях служб Azure см. в Microsoft Azure центре управления безопасностью ([https://azure.microsoft.com/support/trust-center/](https://azure.microsoft.com/support/trust-center/)).  
  
 **Облачные службы** ([https://azure.microsoft.com/services/cloud-services/](https://azure.microsoft.com/services/cloud-services/))  
  
 **Служба конструктора (веб-роль)**  
  
 Это обеспечивает несколько веб-служб для связи между организацией Dynamics 365 и многопользовательским решением "Мнение клиента" для компонентов Dynamics 365 Azure.  Например, опросы публикуются и хранятся в хранилище BLOB-объектов Azure.  Ответы на опрос извлекаются из очереди Служебной шины Azure и возвращаются для хранения в организацию Dynamics 365.  Все запросы проходят аутентификацию в Azure Active Directory.  
  
 **Среда выполнения опроса (веб-роль)**  
  
 Это веб-приложение, которое предоставляет респондентам опросы.  Представленные ответы опроса временно хранятся в очереди Служебной шины Azure перед обработкой развертывания с помощью асинхронной службы Dynamics 365.  
  
 **Обработчик ответа (рабочая роль)**  
  
 Эта рабочая роль отвечает за обработку необработанных завершенных опросов в действительные ответы опроса, которые могут быть созданы в Dynamics 365.  
  
 **Обработчик push-уведомлений (Рабочая роль)**   Эта Рабочая роль отвечает за обработку допустимых ответов на опросы и обновление записей сущностей Dynamics 365. 
 
 **Azure Key Vault** ([https://azure.microsoft.com/services/key-vault/](https://azure.microsoft.com/services/key-vault/))  
  
 Все службы хранят данные конфигурации в Azure Key Vault.  Данные организации и владельца хранятся в SQL Azure.  
  
 **База данных SQL Azure** ([https://azure.microsoft.com/services/sql-database/](https://azure.microsoft.com/services/sql-database/))  
  
 "Мнение клиента" для Dynamics 365 использует SQL Azure для хранения:  
  
-   перенаправленных данных;  
  
-   метаданных опроса;  
  
-   данных организации (владельца).  
  
 **Хранилище BLOB-объектов Azure** ([https://azure.microsoft.com/services/storage/](https://azure.microsoft.com/services/storage/))  
  
 Определения опроса и частично завершенные (сохраненные) ответы хранятся в хранилище BLOB-объектов Azure.  
  
 **Сеть доставки содержимого Azure (CDN)** ([https://azure.microsoft.com/services/cdn/](https://azure.microsoft.com/services/cdn/))  
  
 Решение "Мнение клиента" для Dynamics 365 использует сеть доставки содержимого Azure для обслуживания статического содержимого в среде выполнения опроса, например изображений (включая загруженные изображения, такие как логотипы клиентов), JavaScript и CSS.  
  
 **Azure Active Directory** ([https://azure.microsoft.com/services/active-directory/](https://azure.microsoft.com/services/active-directory/))  
  
 Решение "Мнение клиента" для Dynamics 365 использует службу Azure Active Directory для аутентификации веб-служб.  
  
 **Служебная шина Azure** ([https://azure.microsoft.com/services/service-bus/](https://azure.microsoft.com/services/service-bus/))  
  
 Сообщения, созданные при отображении или отправке опроса, временно сохраняются в очереди Служебной шины Azure (владельца), пока рабочая роль Azure принудительно не отправит ответы опроса в экземпляр организации Dynamics 365 и не сохранит их в качестве записей сущности Dynamics 365.
