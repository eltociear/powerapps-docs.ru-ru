После включения функции помощника по отношениям определенный объем данных, который включает имя и адрес электронной почты отправителя, а также отрывки текста сообщения электронной почты, будет извлекаться (но не сохраняться в [!INCLUDE[pn_dynamics_crm](pn-dynamics-crm.md)]) в целях отображения соответствующей аналитики вашей электронной почты. Более того, функцию помощника по отношениям можно настроить для извлечения сведений, связанных с новостями, финансами и информацией о полете, отправив запросы во внешние компоненты, такие как MSN Финансы и Bing (которые не считаются основными службами [!INCLUDE[pn_ms_dyn_365](pn-ms-dyn-365.md)]). Администратор может включать и отключать функцию помощника по отношениям. Для этого ему необходимо перейти в раздел **Параметры** > **Relationship Insights**, открыть вкладку **Помощник по отношениями** и выбрать соответствующие параметры.  
  
 Внешние компоненты, необходимые для реализации функций помощника по отношениям, рассматриваются в следующих разделах.  
  
 **[!INCLUDE[pn_bing](pn-bing.md)]**  
  
 Помощник по отношениям использует [!INCLUDE[pn_bing](pn-bing.md)] для поиска новостей, которые будут показываться пользователю, основываясь на названиях организаций в данных [!INCLUDE[pn_dynamics_crm](pn-dynamics-crm.md)] пользователя.  
  
 **[!INCLUDE[pn_ms_MSN_Money](pn-ms-msn-money.md)]**  
  
 Помощник по отношениям использует [!INCLUDE[pn_ms_MSN_Money](pn-ms-msn-money.md)] для показа пользователю информации о биржевых котировках, основываясь на биржевых символах организаций в данных [!INCLUDE[pn_dynamics_crm](pn-dynamics-crm.md)] пользователя.
