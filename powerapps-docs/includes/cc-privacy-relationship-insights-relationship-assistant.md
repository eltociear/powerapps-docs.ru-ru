При включении функции "Помощник по отношениям" некоторые данные, такие как имя отправителя, адрес электронной почты и выдержки из текста сообщения, извлекаются (без сохранения в [!INCLUDE[pn_dynamics_crm](pn-dynamics-crm.md)]) для отображения в сводке по вашему сообщению. Кроме того, функцию "Помощник по отношениям" можно настроить так, чтобы появилась возможность получать новости, финансовые сведения и информацию об авиарейсах путем отправки запросов к внешним компонентам, таким как MSN Money и Bing (не относящимся к основным службам [!INCLUDE[pn_ms_dyn_365](pn-ms-dyn-365.md)]). Администратор может включать и отключать функцию "Помощник по отношениям" на странице **Параметры** > **Конфигурация аналитики**, на вкладке **Помощник по отношениям**, выбрав соответствующий параметр.  
  
 Внешние компоненты, участвующие в работе функции "Помощник по отношениям", описаны в следующих разделах.  
  
 **[!INCLUDE[pn_bing](pn-bing.md)]**  
  
 "Помощник по отношениям" при поиске соответствующих новостей для отображения пользователю использует [!INCLUDE[pn_bing](pn-bing.md)] и имена учетных записей из данных [!INCLUDE[pn_dynamics_crm](pn-dynamics-crm.md)] этого пользователя.  
  
 **[!INCLUDE[pn_ms_MSN_Money](pn-ms-msn-money.md)]**  
  
 "Помощник по отношениям" для отображения пользователю информации об акциях использует [!INCLUDE[pn_ms_MSN_Money](pn-ms-msn-money.md)] и символ тикера учетной записи из данных [!INCLUDE[pn_dynamics_crm](pn-dynamics-crm.md)] этого пользователя.
