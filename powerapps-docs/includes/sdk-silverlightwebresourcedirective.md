Чтобы отобразить веб-ресурс Silverlight за пределами формы сущности или диаграммы, создайте веб-ресурс HTML в качестве страницы для размещения веб-ресурса Silverlight. Затем с помощью директивы **$webresource:** откройте веб-ресурс HTML.
