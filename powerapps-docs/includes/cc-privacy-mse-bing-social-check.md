Для каждой новой или измененной конфигурации раздела поиска мы выполняем первоначальную оценку с участием сторонних поставщиков содержимого из социальных сетей для определения объема сообщений, которые мы получим на основе предоставленной вами конфигурации раздела поиска. Для этой цели ключевые слова, включения, исключения и примененные фильтры языка будут отправлены сторонним поставщикам содержимого из социальных сетей.
