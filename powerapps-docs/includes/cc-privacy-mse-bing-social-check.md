Для каждой новой или измененной конфигурации тем поиска проводится начальная оценка с использованием сторонних поставщиков контента социальных сетей, чтобы определить количество сообщений, которые можно получить с указанной конфигурацией тем поиска. Для этого ваши ключевые слова, включения, исключения и применяемые языковые фильтры будут отправлены сторонним поставщикам контента социальных сетей.
