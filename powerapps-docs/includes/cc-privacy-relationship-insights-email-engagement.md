После включения функции встроенной аналитики "Взаимодействие по электронной почте", если сообщение электронной почты, помеченное параметром **Подписаться**, отправляется из [!INCLUDE[pn_dynamics_crm](pn-dynamics-crm.md)], сведения о взаимодействии получателей с этим сообщением будут собираться и сохраняться в [!INCLUDE[pn_azure_shortest](pn-azure-shortest.md)] для расчета КПЭ действий получателя и взаимодействий по отслеживаемым сообщениям.  
  
 Чтобы включить функцию "Взаимодействие по электронной почте", системный администратор должен подготовить эту функцию на вкладке **Взаимодействие по электронной почте** во встроенной аналитике. Позже администратор может отключить взаимодействие по электронной почте для организации в узле **Конфигурация аналитики** раздела **Параметры**.  
  
 После отключения этой функции сообщения электронной почты, отправляемые из [!INCLUDE[pn_dynamics_crm](pn-dynamics-crm.md)], невозможно будет отслеживать ни через пользовательский интерфейс, ни программным образом. Кроме того, больше не будут собираться данные о взаимодействии получателей с отправленными сообщениям, отмеченными параметром **Подписаться**. Все ранее собранные данные останутся в [!INCLUDE[pn_azure_shortest](pn-azure-shortest.md)] и будут снова доступны после включения этой функции в организации. Данные сохраняются в течение 90 дней после того, как клиент прекратит действие своей подписки Microsoft. Пользователи [!INCLUDE[pn_dynamics_crm](pn-dynamics-crm.md)] также могут отключить функцию встроенной аналитики "Взаимодействие по электронной почте" на уровне отдельных контактов или интересов, изменив параметр **Отслеживание эл. почты** в разделе **Предпочтения контакта**.  
  
 Компоненты и службы [!INCLUDE[pn_azure_shortest](pn-azure-shortest.md)], используемые с функцией взаимодействия по электронной почте, подробно рассматриваются в следующих разделах.  
  
 [!INCLUDE[cc_privacy_note_azure_trust_center](cc-privacy-note-azure-trust-center.md)]  
  
 **[!INCLUDE[pn_azure_storage_account](pn-azure-storage-account.md)]**  
  
 Функция отслеживания электронной почты использует [!INCLUDE[pn_azure_storage_account](pn-azure-storage-account.md)] для временного хранения больших двоичных объектов взаимодействия по электронной почте.
