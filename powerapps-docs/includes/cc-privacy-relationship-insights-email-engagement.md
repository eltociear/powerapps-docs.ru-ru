Если при включенной функции взаимодействия по электронной почте, относящейся ко встроенной аналитике, сообщение электронной почты, помеченное параметром **Подписаться**, отправляет из [!INCLUDE[pn_dynamics_crm](pn-dynamics-crm.md)], информация о взаимодействии с ним получателей собирается и хранится в [!INCLUDE[pn_azure_shortest](pn-azure-shortest.md)] для определения КПЭ активности получателей и взаимодействия с сообщениями, на которые оформлена подписка.  
  
 Системный администратор включает взаимодействие по электронной почте, подготовив компонент на вкладке **Взаимодействие по электронной почте** во встроенной аналитике. Впоследствии администраторы могут отключить взаимодействие по электронной почте в организации на узле **Конфигурация аналитики** в разделы **Параметры**.  
  
 Когда эта функция отключена, на сообщения электронной почты, отправляемые из [!INCLUDE[pn_dynamics_crm](pn-dynamics-crm.md)], невозможно подписаться как из пользовательского интерфейса, так и программным способом. Кроме того, данные о взаимодействии получателей больше не собираются для отправленных сообщений электронной почты, помеченных параметром **Подписаться**. Все собранные ранее данные остаются в [!INCLUDE[pn_azure_shortest](pn-azure-shortest.md)] и будут снова доступны при повторном включении этой функции в организации. Данные хранятся в течение 90 дней после того, как клиент прекращает действие своей подписки в корпорации Майкрософт. Пользователи [!INCLUDE[pn_dynamics_crm](pn-dynamics-crm.md)] также могут отключить функцию взаимодействия по электронной почте во встроенной аналитике для отдельных контактов или интересов, изменив параметр **Контролировать сообщение электронной почты** в разделе **Контактная информация**.  
  
 Компоненты и службы [!INCLUDE[pn_azure_shortest](pn-azure-shortest.md)], используемые с функцией взаимодействия по электронной почте, подробно рассматриваются ниже.  
  
 [!INCLUDE[cc_privacy_note_azure_trust_center](cc-privacy-note-azure-trust-center.md)]  
  
 **[!INCLUDE[pn_azure_storage_account](pn-azure-storage-account.md)]**  
  
 Функция взаимодействия по электронной почте использует [!INCLUDE[pn_azure_storage_account](pn-azure-storage-account.md)] для временного хранения BLOB-объектов взаимодействия по электронной почте.
