Чтобы гарантировать получение пользователями необходимой информации при нажатии кнопки **Справка**, необходимо определить [!INCLUDE[pn_crm_shortest](pn-crm-shortest.md)] для использования настраиваемой справки. Выберите **Параметры** > **Администрирование**, щелкните **Параметры системы**, выберите вкладку **Общие**, а затем **Да** для параметра **Использовать пользовательскую справку для настраиваемых объектов**.
