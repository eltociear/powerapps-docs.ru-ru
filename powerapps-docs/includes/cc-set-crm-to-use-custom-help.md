Чтобы гарантировать, что пользователи получат правильные сведения при нажатии кнопки **Справка**, необходимо настроить [!INCLUDE[pn_crm_shortest](pn-crm-shortest.md)] для использования настраиваемой справки. Перейдите в раздел **Параметры** > **Администрирование**, щелкните **Системные параметры**, перейдите на вкладку **Общие** и выберите значение **Да** для параметра **Использовать настраиваемую справку для настраиваемых сущностей**.
