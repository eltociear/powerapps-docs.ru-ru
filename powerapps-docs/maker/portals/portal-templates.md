# <a name="portal-templates"></a>Шаблоны порталов

Вы можете создать Common Data Service начальный портал или портал Dynamics 365 в PowerApps на основе выбранной среды.

> [!NOTE]
> Существующие порталы и существующие шаблоны портала (сообщество, партнер, самообслуживание сотрудников, самообслуживание клиентов) не полностью поддерживаются в студии портала PowerApps, и некоторые компоненты могут не отображаться. Однако можно продолжить редактирование остальных компонентов как обычно. 

## <a name="environment-with-common-data-service"></a>Среда с Common Data Service

При выборе среды, содержащей Common Data Service, можно создать Common Data Service начальный портал. Для быстрого начала работы портал Common Data Service Starter поставляется с образцами данных. Он также содержит следующие встроенные образцы страниц:

- Шаблон по умолчанию для Studio
- Страница с заголовком
- Страница с дочерними ссылками

## <a name="environment-with-model-driven-apps-in-dynamics-365"></a>Среда с приложениями, управляемыми моделями, в Dynamics 365 

При выборе среды, содержащей приложения, управляемые моделями, в Dynamics 365 (Dynamics 365 Sales, Dynamics 365, служба поддержки клиентов, Dynamics 365 Marketing и Dynamics 365 Project Service Automation) можно создавать следующие порталы. :

- **Портал самообслуживания клиентов**. Портал самообслуживания клиентов позволяет клиентам получать доступ к базе знаний самообслуживания, ресурсам поддержки, просматривать ход выполнения своих обращений и оставлять отзывы.
- **Портал партнера**. портал партнера позволяет всем организациям с торговыми посредниками, дистрибьюторами, поставщиками или партнерами иметь доступ к каждому этапу общих действий.

    > [!NOTE]
    > Для включения соответствующих параметров необходимо установить в Организации Dynamics 365 пакет службы полей и пакетов служб проекта. Дополнительные сведения см. в статье [Интеграция службы автоматизации служб Project](https://docs.microsoft.com/en-us/dynamics365/portals/integrate-project-service-automation) и [интеграции полей](https://docs.microsoft.com/en-us/dynamics365/portals/integrate-field-service).

- **Портал самообслуживания сотрудников**. Портал самообслуживания сотрудников создает эффективные и хорошо обоснованные трудные ресурсы, оптимизируя распространенные задачи и предоставляя каждому сотруднику определенный источник знаний.
- **Портал сообщества**. портал сообщества использует одноранговые взаимодействия между клиентами и экспертами, чтобы полностью расширить каталог доступных знаний из статей базы знаний, форумов и блогов, а также предоставить отзыв через Комментарии и оценки
- **Портал с пустым**: создание веб-сайта для совместного использования данных внешними и внутренними пользователями. Этот шаблон поставляется с примерами страниц, с помощью которых можно быстро начать работу.
