# <a name="portal-templates"></a>Шаблоны портала

В зависимости от выбранной среды в Power Apps, можно создать начальный портал Common Data Service или портал в среде, содержащей приложения на основе модели в Dynamics 365.

## <a name="environment-with-common-data-service"></a>Среда в Common Data Service

Если вы выбираете среды, которая содержит Common Data Service, можно создавать. начальный портал Common Data Service. С начальным порталом Common Data Service поставляются демонстрационные данные для быстрого начала работы. Также он имеет следующие встроенные образцы страниц:

- Шаблон студии по умолчанию
- Страница с заголовком
- Страница с дочерними ссылками

## <a name="environment-with-model-driven-apps-in-dynamics-365"></a>Среда с приложениями на основе модели в Dynamics 365 

Если вы выбираете среду, которая содержит приложения на основе модели в Dynamics 365 (Dynamics 365 Sales, Dynamics 365 Customer Service, Dynamics 365 Field Service, Dynamics 365 Marketing, or Dynamics 365 Project Service Automation), можно создать следующие порталы:

- **Портал самообслуживания клиентов**: портал самообслуживания клиентов позволяет клиентам получать доступ к знаниям по самообслуживанию, ресурсам поддержки, просматривать ход своих дел и предоставлять отзывы.
- **Портал партнеров**: портал партнеров предоставляет любой организации с торговыми посредниками, агентами по продажам, поставщиками и партнерами доступ к каждой стадии совместных действий в реальном времени.

    > [!NOTE]
    > Пакеты Field Service и Project Service необходимо установить в вашу организацию Dynamics 365 для включения соответствующих параметров. Чтобы получить дополнительные сведения, см. разделы [Интеграция Project Service Automation](https://docs.microsoft.com/dynamics365/portals/integrate-project-service-automation) и [Интеграция Field Service](https://docs.microsoft.com/dynamics365/portals/integrate-field-service).

- **Портал самообслуживания сотрудников**: портал самообслуживания сотрудников формирует эффективный и хорошо информированный персонал, оптимизируя типовые задачи и предоставляя каждому работнику доступ к надежному источнику знаний.
- **Портал сообщества**: Портал сообщества поддерживает прямое взаимодействие между клиентами и экспертами, способствуя естественному росту объема накопленных знаний в статьях базы знаний, на форумах и в блогах. Кроме того, портал позволяет оставлять отзывы с помощью комментариев и оценок.
- **Портал с нуля**: Создание веб-сайта с совместным доступом к данным с внешними и внутренними пользователями. Этот шаблон поставляется с примерами страниц, которые помогут вам быстро начать работу.
