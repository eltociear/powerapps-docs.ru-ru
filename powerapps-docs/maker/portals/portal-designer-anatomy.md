# <a name="portal-designer-anatomy"></a>Анатомия конструктора порталов

[!include[cc-beta-prerelease-disclaimer](../../includes/cc-beta-prerelease-disclaimer.md)]

Можно использовать портальный конструктор для создания и для настройки вашего веб-сайта. Он содержит различные параметры для добавления и настройки веб-страниц, компонентов, форм и списков. Анатомия конструктора порталов выглядит следующим образом:

![Анатомия конструктора порталов](media/maker-anatomy.png "Анатомия конструктора порталов")  

| **Аннотация** | **Название**        | **Описание**                                                                              |
|----------------|-----------------|----------------------------------------------------------------------------------------------|
| 1              | Панель команд     | Позволяет создавать веб-страницу, удалить компонент, и просматривать веб-сайт, который вы создаете.  |
| 2              | Toolbelt        | Позволяет вам:<ul><li>Просмотр и управление веб-страницами</li><li>Добавление компонентов</li><li>Изменение шаблонов</li></ul>  |
| 3              | Холст          | Содержит компоненты, которые создают веб-страницу.                                                    |
| 4              | Нижний колонтитул          | Отображает состояние автосохранения и позволяет открыть редактор исходного кода.                         |
| 5              | Область свойств | Отображает свойства веб-страницы и выбранных компонентов и позволяет редактировать их по мере необходимости. |
