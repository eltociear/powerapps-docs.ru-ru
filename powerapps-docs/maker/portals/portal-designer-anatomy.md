# <a name="power-apps-portals-studio-anatomy"></a>Анатомия студии порталов Power Apps

Можно использовать студию порталов Power Apps для создания и для настройки вашего веб-сайта. Он содержит различные параметры для добавления и настройки веб-страниц, компонентов, форм и списков. Анатомия студии порталов Power Apps выглядит следующим образом:

![Анатомия студии порталов Power Apps](media/maker-anatomy.png "Анатомия студии порталов Power Apps")  

| **Аннотация** | **Название**        | **Описание**                                                                              |
|----------------|-----------------|----------------------------------------------------------------------------------------------|
| 1              | Панель команд     | Позволяет создавать веб-страницу, удалить компонент, и просматривать веб-сайт, который вы создаете.  |
| 2              | Toolbelt        | Позволяет вам:<ul><li>Просмотр и управление веб-страницами</li><li>Добавление компонентов</li><li>Изменение шаблонов</li></ul>  |
| 3              | Холст          | Содержит компоненты, которые создают веб-страницу.                                                    |
| 4              | Нижний колонтитул          | Отображает состояние автосохранения и позволяет открыть редактор исходного кода.                         |
| 5              | Область свойств | Отображает свойства веб-страницы и выбранных компонентов и позволяет редактировать их по мере необходимости. |

> [!NOTE]
> Редактирование портала через Power Apps Portals Studio временно приведет к снижению производительности портала из-за множественных фоновых процессов. Например, процесс очистки кэша запускается и перезагружает данные из Common Data Service.
