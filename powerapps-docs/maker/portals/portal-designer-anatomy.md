# <a name="powerapps-portals-studio-anatomy"></a>Портал PowerApps Studio, структура

Для создания и настройки веб-сайта можно использовать портал PowerApps студии. Он содержит различные параметры для добавления и настройки веб-страниц, компонентов, форм и списков. Структура PowerApps Portals Studio выглядит следующим образом:

![Powerapps Portals Studio Анатомия](media/maker-anatomy.png "порталы powerapps")  

| **Заметки** | **Name** (Наименование)        | **Description;**                                                                              |
|----------------|-----------------|----------------------------------------------------------------------------------------------|
| 1              | Панель команд     | Позволяет создать веб-страницу, удалить компонент и просмотреть создаваемый сайт.  |
| 2              | Воле        | Позволяет:<ul><li>Просмотр веб-страниц и управление ими</li><li>Добавить компоненты</li><li>Изменение шаблонов</li></ul>  |
| 3              | На основе холста          | Содержит компоненты, создающие веб-страницу.                                                    |
| 4              | Нижний колонтитул          | Отображает состояние автоматического сохранения и позволяет открывать редактор исходного кода.                         |
| 5              | Панель "Свойства" | Отображает свойства веб-страницы и выбранных компонентов и позволяет при необходимости изменять их. |
