# <a name="behavior-and-format-of-the-date-and-time-field"></a>Поведение и формат поля "Дата и время"

В Common Data Service тип данных "Дата и время" используется во многих полях системных сущностей. Например, можно указать, когда организация последний раз использовалась в маркетинговой кампании, или показать дату и время, когда обращение было передано на более высокий уровень рассмотрения. Можно создавать настраиваемые сущности, которые будут включать поля даты и времени. В зависимости от того, что представляет поле, можно выбрать одно из следующих поведений поля для форм и сеток портала: 
- **Часовой пояс пользователя**: значения полей отображаются в часовом поясе пользователя и форматируются согласно текущему часовому поясу/языку портала. Значения сохраняются в формате часового пояса UTC в Common Data Service. Когда пользователь в Common Data Service (или другой пользователь портала) в другом часовом поясе просматривает это значение, оно отображается согласно его часовому поясу.
- **Только дата**: значения полей содержат только дату и отображаются без преобразования часового пояса. Часть "время" в значении всегда равна 12:00. Значение, введенное одним пользователем, отображается так же для другого пользователя в другом часовом поясе (например, даты рождения).
  
  > [!Note]
  > Поведение этого поля невозможно изменить после его сохранения.
  
- **Независимо от часового пояса**: значения полей содержат дату и время и отображаются без преобразования часового пояса. Значение, введенное одним пользователем, отображается так же для другого пользователя в другом часовом поясе.
  
  > [!Note]
  > Поведение этого поля невозможно изменить после его сохранения.

Можно также переопределить формат даты и времени по умолчанию для использования на порталах, создав следующие параметры сайта:
- DateTime/DateFormat: формат даты, используемый на портале. 
- DateTime/TimeFormat: формат времени, используемый на портале. 
- DateTime/DateTimeFormat: формат полной даты и времени, используемый на портале.

По умолчанию портал использует стандартный формат даты и времени, определенный языковыми параметрами веб-сайта.
Принятые форматы даты и времени указаны [здесь](https://docs.microsoft.com/dotnet/standard/base-types/custom-date-and-time-format-strings).
