# <a name="behavior-and-format-of-the-date-and-time-field"></a>Поведение и формат поля даты и времени

В Common Data Service тип данных даты и времени используется во многих полях системной сущности. Например, можно отобразить время последнего использования учетной записи в маркетинговой кампании или отобразить дату и время эскалации обращения. Можно также создавать пользовательские сущности, включающие поля даты и времени. В зависимости от того, что представляет поле, можно выбрать одно из следующих поведений полей для форм и сеток портала: 
- **Локальная пользователь**: значения полей отображаются в местном времени пользователя и форматируются в соответствии с текущим языком и региональными параметрами портала. Значения хранятся в формате часового пояса UTC в Common Data Service. Когда пользователь в Common Data Service (или на другом пользователе портала) в различных представлениях часового пояса имеет значение, он преобразуется в собственный часовой пояс.
- **Только дата**: значения полей содержат только дату и отображаются без преобразования часовых поясов. Временная часть значения всегда 12:00 AM. Значение, указанное одним пользователем, видят те же значения, что и другие пользователи в разных часовых поясах (например, даты рождения).
  
  > [!Note]
  > Поведение этого поля нельзя изменить после его сохранения.
  
- **Независимо от часового пояса**: значения полей содержат дату и время и отображаются без преобразования часовых поясов. Значение, указанное одним пользователем, видят другие пользователи в разных часовых поясах.
  
  > [!Note]
  > Поведение этого поля нельзя изменить после его сохранения.

Можно также переопределить формат даты и времени по умолчанию для использования на порталах, создав следующие параметры сайта.
- DateTime/DateFormat: формат даты, используемый на портале. 
- DateTime/TimeFormat: формат времени, используемый на портале. 
- DateTime/DateTimeFormat: формат полной даты и времени, используемый на портале.

По умолчанию на портале используются стандартные форматы даты и времени, заданные в параметрах языка веб-сайта.
Допустимые форматы даты и времени указаны [здесь](https://docs.microsoft.com/dotnet/standard/base-types/custom-date-and-time-format-strings).
