# <a name="configure-site-settings-for-portals"></a>Настройка параметров сайта для порталов

Настройка сайта представляет собой конфигурируемое именованное значение, используемое кодом веб-сайта для изменения поведения или визуального стиля портала. Обычно когда разработчик создает код веб-сайта, он ссылается на настройки сайта для различных компонентов, чтобы разрешить конечному пользователю изменить значения параметра для изменения веб-сайта без изменения кода, повторного компилирования и повторного развертывания веб-сайта.

Образец порталов, предоставляемый при установке порталов Power Apps, содержит несколько конфигурируемых настроек сайта для различных стилей, используемых для изменения многих визуальных элементов на сайте, таких как стиль фона, цвет текста и ширина макета.
Можно управлять следующими типами параметров сайта:

- **Глобальные параметры порталов**: эти параметры применяются ко всем порталам, связанным со средой Common Data Service, в которую они добавляются.
- **Параметры сайта портала**: эти параметры применяются к определенным порталам (записям веб-сайтов), связанным со средой Common Data Service, в которую они добавляются.


## <a name="manage-portal-site-settings"></a>Управление параметрами сайта портала

1. Перейдите в [параметры портала](../manage-existing-portals.md#settings) и выберите **Параметры сайта**.

2. Чтобы создать новый параметр, выберите **Создать**.

3. Чтобы отредактировать существующий параметр, выберите **Параметр сайта**, перечисленный в сетке.

4. Укажите значения в указанных полях: 

    - **Имя**: подпись, на которую ссылается код веб-сайта для извлечения соответствующей настройки. Имя должно быть уникальным для связанного веб-сайта, так как код, извлекающий настройку, получит первую найденную запись с соответствующим именем.
    
    - **Веб-сайт**: обязательный связанный веб-сайт. 
    
    - **Значение**: настройка
    
    - **Описание**: назначение настройки или специальные инструкции.

5. Выберите **Сохранить и закрыть**.

> [!NOTE] 
> Интеграция карт Bing не поддерживается в государственном облаке в Германии. При попытке создать параметр Bingmaps/credentials в этой среды отобразится сообщение об ошибке.

## <a name="manage-global-portal-settings"></a>Управление глобальными параметрами порталов

1. Перейдите в [параметры портала](../manage-existing-portals.md#settings) и выберите **Параметры сайта**.

2. Перейдите в раздел **Параметры** &gt; **Параметры**.

3. Чтобы создать новый параметр, выберите **Создать**.

4. Чтобы отредактировать существующий параметр, выберите **Параметр сайта**, перечисленный в сетке.

5. Укажите значения в указанных полях: 

    - **Имя**: уникальное имя, на которое ссылается код для извлечения соответствующего параметра.

    - **Значение**: настройка

    - **Описание**: назначение настройки или специальные инструкции.

6. Выберите **Сохранить и закрыть**.

> [!NOTE] 
> Интеграция карт Bing не поддерживается в государственном облаке в Германии. При попытке создать параметр BinMap/Key или Adxstudio/ProductivityPack/BingMap/Key в этой среде отобразится сообщение об ошибке.
