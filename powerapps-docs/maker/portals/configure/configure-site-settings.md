# <a name="configure-site-settings-for-portals"></a>Настройка параметров сайта для порталов

Параметр сайта — это настраиваемое именованное значение, которое используется кодом веб-сайта для изменения поведения или визуального стиля портала. Как правило, когда разработчик создает код веб-сайта, он будет ссылаться на параметры сайта для различных компонентов, чтобы позволить пользователю изменять значения параметров для изменения веб-сайта без необходимости изменения кода, повторной компиляции и повторного развертывания веб-сайта.

Примеры порталов, поставляемые с установкой порталов PowerApps, содержат несколько настраиваемых параметров сайта для различных стилей, используемых для изменения многих визуальных элементов на сайте, таких как стиль фона, цвет текста и ширина макета.
Можно управлять следующими типами параметров сайта.

- **Глобальные параметры портала**. Эти параметры применяются ко всем порталам, связанным со средой Common Data Service, в которой они добавляются.
- **Параметры сайта портала**. Эти параметры применяются к конкретным порталам (записям веб-сайта), связанным со средой Common Data Service, в которой они добавляются.


## <a name="manage-portal-site-settings"></a>Управление параметрами сайта портала

1. Перейдите в раздел [Параметры портала](../manage-existing-portals.md#settings) и выберите **Параметры сайта**.

2. Чтобы создать новый параметр, нажмите кнопку **создать**.

3. Чтобы изменить существующий параметр, выберите **параметр сайта** , указанный в сетке.

4. Укажите значения для указанных полей: 

    - **Name**: метка, на которую ссылается код веб-сайта, для получения соответствующего параметра. Имя должно быть уникальным для связанного веб-сайта, так как код, извлекающий этот параметр, будет принимать первую найденную запись с соответствующим именем.
    
    - **Веб-сайт**. связанный веб-сайт. 
    
    - **Значение**: параметр
    
    - **Описание**: назначение параметра или специальные инструкции.

5. Выберите **Сохранить и закрыть**.

> [!NOTE] 
> Интеграция карт Bing не поддерживается в Германии независимых Cloud. При попытке создать параметр Bingmaps/Credential в этой среде будет выведено сообщение об ошибке.

## <a name="manage-global-portal-settings"></a>Управление параметрами глобального портала

1. Перейдите в раздел [Параметры портала](../manage-existing-portals.md#settings) и выберите **Параметры сайта**.

2. Последовательно выберите **параметры** &gt; **Параметры**.

3. Чтобы создать новый параметр, нажмите кнопку **создать**.

4. Чтобы изменить существующий параметр, выберите **параметр сайта** , указанный в сетке.

5. Укажите значения для указанных полей: 

    - **Имя**: уникальное имя, на которое ссылается код, для получения соответствующего параметра.

    - **Значение**: параметр

    - **Описание**: назначение параметра или специальные инструкции.

6. Выберите **Сохранить и закрыть**.

> [!NOTE] 
> Интеграция карт Bing не поддерживается в Германии независимых Cloud. Если вы попытаетесь создать параметр Бинмап/Key или Адксстудио/Продуктивитипакк/Бингмап/Key в этой среде, отобразится сообщение об ошибке.
