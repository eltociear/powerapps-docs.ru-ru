# <a name="invite-contacts-to-your-portals"></a>Приглашение контактов на порталы

Используйте функцию приглашения порталов, чтобы приглашать контакты на портале с помощью автоматизированных электронных писем, созданных в Common Data Service. Приглашенные пользователи получают электронное письмо, полностью настраиваемое вами, со ссылкой на портал и кодом приглашения. Этот код можно использовать для получения специального доступа, настроенного вами. С помощью этой функции вы можете:

- Отправка одного приглашения или приглашений группы
-   При необходимости укажите дату окончания срока действия
-   При необходимости укажите контакт пользователя или портала в качестве приглашающего
-   Автоматически назначать приглашенных контактов учетной записи после активации приглашения
-   Автоматически выполнять рабочий процесс после активации приглашения
-   Автоматически назначать приглашаемых контактов веб-ролям при активации

Можно выполнить погашение приглашения с помощью любого из наших способов проверки подлинности. Документацию по проверке подлинности на портале см. в разделе [Настройка удостоверения проверки подлинности для портала](set-authentication-identity.md) и выбор модели, применимой к версии и конфигурации портала. Пользователь будет принимать любые параметры, предоставленные администратором при активации. Для приглашения и контакта будет создано действие "пригласить".

Приглашения отправляются через рабочий процесс **отправки приглашения** . По умолчанию рабочий процесс создает сообщение электронной почты с универсальными сообщениями и отправляет его на основной адрес электронной почты приглашаемого контакта. Рабочий процесс **отправки приглашения** содержит шаблон сообщения электронной почты, который необходимо изменить, чтобы он содержал конкретное сообщение для портала и соответствующую гиперссылку на **страницу активации приглашения**на портале.

Чтобы изменить шаблон сообщения для рабочего процесса **отправки приглашения** , выберите его и отключите. После деактивации измените шаблон сообщения электронной почты, чтобы отправить нужное сообщение, и укажите ссылку на **страницу** "Отправить активацию" на портале.

> [!NOTE]
> Приглашение отправляется только в основную электронную почту (emailaddress1) контакта. Приглашение не будет отправлено на дополнительный адрес электронной почты (emailAddress2) или запасной адрес электронной почты (emailAddress3) записи контакта.

## <a name="create-invitations-from-portal-management-app"></a>Создание приглашений из приложения управления порталом

1.  Откройте [приложение управления портала](configure-portal.md).

2.  Перейдите на **порталы** > **Контакты**.
    Кроме того, можно также открыть страницу **Контакты** из панели [общий доступ](../manage-existing-portals.md#share) . 

3.  Выберите контакт или откройте запись контакта, которая будет приглашена.

4.  На панели команд выберите **создать приглашение**.

5.  На странице **приглашения** введите соответствующие значения в поля. Дополнительные сведения: [атрибуты приглашения](#invitation-attributes)

6.  Щелкните **Сохранить**.

7.  На панели команд выберите **Flow** > **Отправить приглашение**.

    > [!div class="mx-imgBorder"]
    > ![Рабочий процесс отправки приглашения](../media/send-invitation-portal-app.png "Рабочий процесс отправки приглашения")

8.  В окне подтверждения нажмите кнопку **ОК**. Приглашение будет отправлено выбранному контакту.

    > [!div class="mx-imgBorder"]
    > ![Подтверждение отправки приглашения](../media/confirm-invitation-portal-app.png "Подтверждение отправки приглашения")

### <a name="send-multiple-invitations"></a>Отправка нескольких приглашений

Вы можете создать приглашения для контактов, а затем отправить все приглашения одновременно.

1.  Создайте приглашения для нужных контактов, а затем перейдите на **порталы** > **приглашения**.

2.  Выберите созданные приглашения.

3.  На панели команд выберите **Flow** > **Отправить приглашение**.

    > [!div class="mx-imgBorder"]
    > ![Рабочий процесс отправки приглашения](../media/send-invitation-portal-app.png "Рабочий процесс отправки приглашения")

4.  В окне подтверждения нажмите кнопку **ОК**. Приглашения будут отправляться выбранным контактам.

    > [!div class="mx-imgBorder"]
    > ![Подтверждение отправки нескольких приглашений](../media/confirm-multiple-invites-portal-app.png "Подтверждение отправки нескольких приглашений")
