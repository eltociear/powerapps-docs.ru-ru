# <a name="manage-web-links"></a>Управление веб-ссылками

Веб-ссылка может ссылаться на любой URL-адрес или может ссылаться на другую страницу в пределах того же веб-сайта. Если веб-ссылка связана со страницей, ее состояние безопасности и публикации также будет применено к веб-ссылке. Веб-ссылки всегда являются частью набора веб-ссылок. Набор веб-ссылок — это группа ссылок, таких как первичная Навигация или группа ссылок нижнего колонтитула. Наборы веб-ссылок разрешают внутренние, независимо от размещения на карте узла, а также внешние ссылки, сгруппированные и упорядоченные.

## <a name="manage-web-links-in-powerapps-portals"></a>Управление веб-ссылками на порталах PowerApps

После импорта настроек портала в среду Common Data Service можно управлять веб-ссылками из набора веб-ссылок.

1. Откройте [приложение управления портала](configure-portal.md).

2. Перейдите на **порталы** > **наборы веб-ссылок**.

3. Чтобы создать новый набор веб-ссылок, нажмите кнопку **создать**.

4. Чтобы изменить существующий набор веб-ссылок, выберите имя набора веб-ссылок.

5. Введите соответствующие значения в поля.

6. При создании нового набора веб-ссылок выберите **сохранить** , чтобы сохранить запись, чтобы можно было добавить веб-ссылки.

7. Перейдите на вкладку **ссылки** .

8. Чтобы создать новую веб-ссылку, выберите **создать веб-ссылку**.

    ![Добавить веб-ссылку](../media/add-web-link.png "Добавить веб-ссылку")

9. Чтобы изменить существующую веб-ссылку, выберите имя веб-ссылки.

9. Введите соответствующие значения в поля.

6. Сохраните изменения.
