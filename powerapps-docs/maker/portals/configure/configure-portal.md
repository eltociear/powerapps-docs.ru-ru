# <a name="portal-management-app"></a>Приложение управления портала

Приложение управления портала позволяет выполнять дополнительные действия по настройке на портале. Приложение доступно после успешного создания базы данных на Common Data Service.

Чтобы открыть приложение управления портала, перейдите в раздел " **приложения** " на домашней странице PowerApps, найдите приложение управления порталом и выберите его.

> [!div class=mx-imgBorder]
> ![](../media/portal-mgmt.png "Приложение") управления портала приложений Управление порталом

Приложение управления портала открывается в едином интерфейсе. Вы можете настроить портал в соответствии с вашими требованиями.
