# <a name="add-a-conditional-step-type"></a>Добавление типа условного шага

Шаг веб-формы может иметь тип "условие", который указывает, что шаг должен оценить выражение. Если выражение принимает значение "истина", отображается следующий шаг. Если выражение принимает значение "ложь" и если указано значение "Следующий шаг в случае невыполнения условия", отображается этот шаг. Текущая сущность — это целевое значение, используемое для оценки выражения. Источник записи по умолчанию равен источнику записи предыдущего шага.

## <a name="attributes"></a>Атрибуты

| Имя (название)                         | Описание                                                                                                                                                                                                                          |
|------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Условие                    | Оцениваемое условное выражение                                                                                                                                                                                           |
| Следующий шаг в случае невыполнения условия | Тип условного шага, в отличие от всех прочих, имеет два поиска следующего шага. Поиск следующего шага по умолчанию будет производиться, если условие истинно. Это свойство задает следующий шаг, если условие ложно. |

Доступны следующие операнды:

| Операнд(-ы)    | Тип                   |
|---------------|------------------------|
| =, ==         | Равно                 |
| !=            | Не равно             |
| &gt;          | Больше           |
| &lt;          | Меньше              |
| &gt;=         | Больше или равно |
| &lt;=         | Меньше или равно    |
| &             | И                    |
| \|             | Или                     |
| !             | Нет                    |
| =\*, ==\*, -= | Подобно                   |
| !=\*          | Не подобно               |

## <a name="format"></a>Формат

Формат выражения следующий:

\[логическое имя атрибута сущности\] \[операнд\] \[значение\]

Пример:

new\_categorycode = 750101

Условие может иметь несколько выражений. Можно использовать скобки для группирования вложенных выражений, например:

new\_categorycode = 750101 & gendercode = 2

-   new\_categorycode = 750101 & (gendercode = 2 | gendercode = 3)

-   new\_name = Jane Doe

-   new\_twooptionfield = true

-   new\_twooptionfield = false
