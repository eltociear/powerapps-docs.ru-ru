# <a name="add-a-conditional-step-type"></a>Добавление условного типа шага

Шаг веб-формы может быть типом "условие", указывающим на то, что шаг должен вычислять выражение. Если выражение принимает значение true, отображается следующий шаг. Если выражение принимает значение false и если указан параметр "следующий шаг при сбое условия", этот шаг будет отображен. Текущая сущность является целевым объектом, используемым для вычисления выражения. Источник записей по умолчанию имеет источник записей предыдущего шага.

## <a name="attributes"></a>Атрибуты

| Имя                         | Description                                                                                                                                                                                                                          |
|------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Выполняет                    | Условное выражение для вычисления                                                                                                                                                                                           |
| Следующий шаг при сбое условия | Условный тип шага, в отличие от остальных, имеет два следующих этапа поиска. Подстановка следующего шага по умолчанию будет соблюдаться, если условие примет значение true. Это свойство задает следующий шаг, если условие принимает значение false. |

Доступны следующие операнды:

| Операнды    | Тип                   |
|---------------|------------------------|
| =, ==         | Прошлом                 |
| !=            | Не равно             |
| &gt;          | Больше           |
| &lt;          | Меньше              |
| &gt;=         | Больше или равно |
| &lt;=         | Меньше или равно    |
| &             | And                    |
| \|             | Or                     |
| !             | Not                    |
| =\*, = =\*,-= | Например                   |
| ! =\*          | Не нравится               |

## <a name="format"></a>Формат

Выражение имеет следующий формат:

\[логическое имя атрибута сущности\] \[операнд\] \[значение\]

Пример.

New\_категорикоде = 750101

Условие может иметь несколько выражений. Круглые скобки можно использовать для группирования вложенных выражений, например:

New\_категорикоде = 750101 & жендеркоде = 2

-   New\_категорикоде = 750101 & (жендеркоде = 2 | жендеркоде = 3)

-   новое имя\_= Мария Петров

-   New\_твуптионфиелд = true

-   New\_твуптионфиелд = false
