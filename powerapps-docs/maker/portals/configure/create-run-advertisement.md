# <a name="create-and-run-advertisements-on-a-portal"></a>Создание и запуск рекламы на портале

Создайте текстовые или графические рекламные объявления и отображайте их в нескольких местах на сайте. Задайте случайное отображение объявлений или выберите конкретные объявления для конкретных мест. Можно выбрать даты выпуска и завершения срока действия для планируемого содержимого, зависящего от времени. Объявления можно связать ссылками с любым пунктом назначения и открывать в текущем или новом окне. Объявления отображаются на портале через две сущности: сущность размещения рекламных объявлений и связанная сущность рекламного объявления. Объявления можно загружать различными способами: с помощью готовых шаблонов Liquid, доступных в порталах, с помощью шаблонов Liquid/примеров веб-шаблонов или с помощью страницы ASPX через действия MVC.

## <a name="create-a-new-advertisement"></a>Создание нового рекламного объявления

Объявления представляют определенную рекламу или изображение, которые будут отображаться на портале в определенный момент времени. Сущность объявления отображается в месте, указанном размещением объявления. Объявление должно быть связано с размещением объявления, чтобы оно отображалось на портале. Для данной демонстрации объявление "Заполнитель" и размещение объявления "Внизу боковой панели" готового примера будут отображаться на портале компании, чтобы продемонстрировать основную функциональность до создания более сложных объявлений. Вместо портала компании можно использовать любые начальные сайты. Однако обратите внимание, что шаблоны Liquid, использованные для этой демонстрации, вызывают имя размещения объявления "Внизу боковой панели".

1. Откройте [Приложение управления порталом](configure-portal.md).

2. Перейдите в раздел **Порталы** > **Рекламные объявления**.

3. Откройте объявление **Заполнитель**, связанное с веб-сайтом **Портал компании** (это можно сделать с начальным сайтом по вашему выбору, для этого выберите **+Создать** и создайте идентичное объявление под веб-сайтом). 

4. Выберите значок **Сохранить** в правом нижнем углу (или **Сохранить и закрыть** в левом верхнем углу, если создано новое объявление).

В форме "Рекламное объявление" указывается **Имя**, описывающее объявление, **Веб-сайт**, на котором будет отображаться объявление, и **Состояние публикации**. Если требуется, можно указать веб-шаблон и дату выпуска/окончания срока действия. Необходимо указать какой-то сорт данных, чтобы объявление отображалось. Используйте таблицу атрибутов объекта "Объявление" далее в этом руководстве, чтобы задать конкретные требуемые значения для вашего объявления.


## <a name="create-a-new-advertisement-placements"></a>Создание новых мест размещения рекламных объявлений

1. Откройте [Приложение управления порталом](configure-portal.md).

2. Перейдите к пункту **Порталы** > **Размещения рекламных объявлений**.

3. Выберите поле веб-шаблона, чтобы выбрать веб-шаблон. Для целей демонстрации выбран веб-шаблон "Случайная реклама".

4. В самом правом углу таблицы рекламных объявлений выберите **+**, чтобы выбрать объявление, созданное на предыдущем шаге.

5. Выберите значок **Сохранить** в нижнем правом углу

При создании нового размещения рекламных объявлений укажите **Имя**, описывающее размещение объявления, и **Веб-сайт**, где будет отображаться размещение рекламных объявлений. Пример веб-шаблонов, которые позволяют использовать рекламные объявления как готовую функцию, отображается в подстановке поля веб-шаблона в форме. Эти шаблоны также предназначены для использования в качестве источника для создания настраиваемых шаблонов.

## <a name="using-liquid-templates-to-place-advertisements"></a>Использование шаблонов Liquid для размещения рекламных объявлений

Менеджеры содержимого могут использовать Liquid для добавления рекламного объявления в любой область доступного для редактирования содержимого, как описано в разделах [Работа с шаблонами Liquid](../liquid/liquid-overview.md) и, более конкретно, [Рекламные объявления](../liquid/liquid-objects.md#ads).

Этот шаблон выводит рекламное объявление по его имени или выбирает случайное объявление из рекламного размещения. В настоящее время приведенный ниже код не отображает несколько рекламных объявлений в размещении объявления (т. е., сменяющееся рекламное объявление). Для отображения нескольких рекламных объявлений в размещении объявления потребуется сборка API рекламных объявлений Liquid. Дополнительные сведения о встроенных веб-шаблонах см. в разделе [Сохранение содержимого источника с помощью веб-шаблонов](../liquid/store-content-web-templates.md).

```
{% include 'ad' ad_name:'Name' %}
{% include 'ad' ad_placement_name:'Placement Name' %}
1. {% include 'Random Ad' placement:ads.placements[Sidebar Bottom] %}
```
ИЛИ 

```
1. {% include 'Ad Template' ad:ads{Retail Ad - Go Greene] %}
```


## <a name="attributes"></a>Атрибуты

Сущность "Объявление" сущность имеет следующие атрибуты:


|        Полное имя        |                                                                                                                                                                                                                                             Описание                                                                                                                                                                                                                                              |
|--------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
|        Полное имя        |                                                                                                                                                                                                                                    Описательное название рекламного объявления                                                                                                                                                                                                                                     |
|      Веб-сайт       |                                                                                                                                                                                                                                  Связанный веб-сайт. Обязательное.                                                                                                                                                                                                                                   |
|    Веб-шаблон    |                                                                                                                                                Связанные [веб-шаблоны](../liquid/store-content-web-templates.md), которые будут использоваться по умолчанию для отображения рекламного объявления. Это поле не является обязательным; если оно пустое, рекламное объявление отображается с использованием шаблона по умолчанию.                                                                                                                                                |
|    Дата выпуска    |                                                                                      Управляет датой и/или временем, после которых рекламное объявление будет видимым на портале. Если в размещении рекламных объявлений циклически отображаются несколько объявления, еще не выпущенное объявление не отображается. Если нет выпущенных рекламных объявлений, связанных с размещением объявления, ничего не отображается. Это удобно для управления выпуском содержимого, зависящего от времени.                                                                                      |
|  Дата окончания срока действия   |                                                                                                                                                                                                             Управляет датой или временем, до которых рекламное объявление будет видимым на портале.                                                                                                                                                                                                             |
|  Состояние публикации  |                                                                                                                                                                                                                                    Текущие состояния публикации.                                                                                                                                                                                                                                     |
|    URL-адрес перенаправления    |                                                                                                                                                                                При нажатии рекламного объявления пользователь направляется по этому URL-адресу. **Это поле необязательно.** Если значение не задано, переход при нажатии объявления не производится.                                                                                                                                                                                 |
| Открыть в новом окне |                                                                                                                                                                                                             Логическое значение. Если выбрано значение "истина", нажатое рекламное объявление открывается в новом окне браузера.                                                                                                                                                                                                             |
|       Должность        |  Одна строка текста для рекламного объявления, которое может отображаться в портале. Отображается она или нет, определяется свойством в элементе управления AdPlacement. Это в первую очередь удобно для текстовых рекламных объявлений или простых однострочных ссылок, которые требуется разместить на портале с помощью мест размещения объявлений. Если заголовок отображается, по умолчанию он будет представлен как гиперссылка, указывающая на URL-адрес перенаправления. Это поведение может быть изменено с помощью настраиваемого [веб-шаблона](../liquid/store-content-web-templates.md).   |
|        Копировать        |                                                                      Несколько строк текста или другое веб-содержимое, которое будет отображаться в размещении объявления. Это позволяет использовать размещение аналогично фрагментам содержимого, но не рекомендуется использовать их просто в качестве контейнера для содержимого (для этого используйте фрагменты). Вместо этого они наиболее хорошо используются для отображения циклических изображений или текстового содержимого.                                                                      |
|     URL-адрес изображения      | URL-адрес изображения, которое будет отображаться рекламным объявлением. Необязательно; используйте это поле, если требуется, чтобы рекламное объявление отображало статический ресурс или веб-файл. Изображение будет доступно для нажатия и связано с URL-адресом перенаправления, если он задан. Если к рекламному объявлению прикреплено примечание с вложенным файлом изображения, рекламное объявление отображает это изображение. Возможно, это самый простой способ настроить изображения для рекламных объявлений и связать изображения непосредственно с объявлением. В этом случае использовать поле URL-адреса изображения не требуется. |
|    Ширина изображения     |                                                                                                                                                                                    Ширина изображения. Это поле не является обязательным, но рекомендуется, чтобы отображаемое рекламное объявление было допустимым и доступным кодом HTML                                                                                                                                                                                     |
|    Высота изображения    |                                                                                                                                                                                    Высота изображения. Это поле не является обязательным, но рекомендуется, чтобы отображаемое рекламное объявление было допустимым и доступным кодом HTML                                                                                                                                                                                    |
|   Альтернативный текст изображения   |                                                                                                                                                                                  Альтернативный текст для изображения. Это поле не является обязательным, но рекомендуется, чтобы отображаемое рекламное объявление было допустимым и доступным кодом HTML.                                                                                                                                                                                  |
