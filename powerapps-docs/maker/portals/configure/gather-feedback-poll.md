# <a name="gather-feedback-by-using-polls-on-a-portal"></a>Сбор отзывов с помощью опросов на портале

Опросы обеспечивают вашим веб-аудиториям быстрый и удобный способ выразить свое мнение по конкретным темам и сразу же автоматически увидеть результат своего голосования.

Используйте функциональную возможность голосования порталов, чтобы запросить мнение аудитории об интересующих темах и позволить им дать ответ с возможностью выбора одного или нескольких вариантов. В любом случае их отклики немедленно сохраняются и связываются с соответствующей записью контакта для немедленного рассмотрения или сводной отчетности. Можно использовать опросы как простые средства изучения рыночной конъюнктуры и, если обновлять или заменять опросы динамически, ваш веб-сайт будет выглядеть современным и злободневным.

Опросы размещаются на портале с помощью элемента управления PollPlacement. Этот элемент управления работает аналогично элементу управления AdPlacement. Если имеются опросы, связанные с сущностью размещения опроса, которая обрабатывается элементом управления PollPlacement, эти опросы будут отображаться. При наличии нескольких опросов для указанного размещения это размещение будет случайным образом показывать один из указанных опросов.

> [!Note]
> Пользователи могут голосовать анонимно. Повторяющиеся голосование не допускается. Основные сведения об отправке отслеживаются, и для пользователей, которые вошли на веб-сайт, их отправки будут связываться с сущностью контакта, которая отслеживает этого пользователя в Common Data Service.

## <a name="add-a-poll-to-the-page"></a>Добавление опроса на страницу

Диспетчеры содержимого могут использовать [теги шаблонов](../liquid/liquid-overview.md) для добавления опросов в любую редактируемую область содержимого:  

`{% include 'Random Poll' placement:polls.placements[Sidebar] %}`

ИЛИ

`{% include 'Poll Template' ad:ads[Wireframe Development] %}`

> [!Note]
> Образцы веб-шаблонов настроены в начальных веб-сайтах. Вы можете использовать шаблон «Случайный опрос» для отображения случайного опроса из определенной сущности размещения опросов или можно использовать шаблон «Шаблон опроса» для отображения конкретного опроса. Можно редактировать эти шаблоны или на основе этих образцов создавать собственные шаблоны с помощью [Опросов](../liquid/liquid-objects.md#polls). 

## <a name="create-a-poll-placement"></a>Создание размещения опроса

Создание нового региона размещения опроса:

1. Откройте [Приложение управления порталом](configure-portal.md).

2. Перейдите к пункту **Порталы** > **Размещения опросов**.

3. Выберите **Создать**.

4. Выберите связанный **Веб-сайт**, задайте имя размещения и, &mdash; если требуется, &mdash; выберите [веб-шаблоны](../liquid/store-content-web-templates.md), которые будут управлять способом отображения.

5. После создания размещения необходимо связать один или несколько опросов с этим размещением. На вкладке **Опросы** сущности размещения опроса выберите **Добавить существующий опрос**. 

6. В появившемся поле поиска выберите существующую запись опроса или создайте новый опрос, выбрав **Создать опрос**.
  

## <a name="polls"></a>Опросы

Опрос представляет собой простой вопрос типа "да/нет" или вопрос с возможностью выбора нескольких вариантов, который можно отображать на портале с помощью размещений опроса. Разработчикам доступны много настраиваемых параметров для отображения опросов, но для диспетчеров содержимым добавление опросов на ваш веб-сайт — это просто выбор вопроса и набора возможных ответов (варианты опроса). Для работы опроса с ним должны быть связаны варианты, и опрос должен быть связан с размещением опроса, чтобы он отображался на портале.

Новый опрос можно создать двумя способами: 
- Перейдя в раздел **Опросы** в области **Порталы**.
- Выбрав кнопку **Создать** в окне **Поиск в записях** при добавлении опроса в размещение опроса.

## <a name="poll-attributes"></a>Атрибуты опроса

| Полное имя                | Описание                                                                                                                                                                                                                                                                                                                                  |
|---------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Полное имя                | Описательное имя опроса.                                                                                                                                                                                                                                                                                                            |
| Веб-сайт             | Связанные [веб-шаблоны](../liquid/store-content-web-templates.md).                                                                                                                                                                                                                                                                |  
| Веб-шаблон        | Связанные [веб-шаблоны](../liquid/store-content-web-templates.md), которые будут использоваться по умолчанию для отображения опроса. Это поле не является обязательным; если оставить его пустым, опрос будет отображаться с использованием шаблона по умолчанию.                                                                                                                     |  
| Вопрос            | Это фактический вопрос, задаваемый в опросе. Связанные варианты опроса — это возможные ответы, которые могут быть выбраны для этого опроса.                                                                                                                                                                                             |
| Метка кнопки "Отправить" | Текст, который должен использоваться для кнопки отправки.                                                                                                                                                                                                                                                                                       |
| Дата выпуска        | Управляет датой и временем, после которых опрос будет видимым на портале. Если в размещении опроса циклически отображаются несколько опросов, еще не выпущенные опросы не отображаются. Если с размещением опросов не связаны никакие выпущенные опросы, ничего не отображается. Это удобно для управления выпуском содержимого, зависящего от времени.         |
| Дата окончания срока действия     | Управляет датой и временем, до которых опрос будет видимым на портале.                                                                                                                                                                                                                                                                  |
| Дата закрытия голосования   | До этой даты пользователи, которые еще не голосовали в опросе, могут проголосовать в нем.|

> [!Note] 
> - Когда пользователь проголосует в опросе, он увидит сводку текущих результатов для опроса. Результаты также будут отображаться для опроса, дата закрытия которого уже прошла, но в котором пользователь не голосовал. Это позволяет продолжать показывать результаты опросов, когда пользователи уже не могут голосовать в них. 
> - Разница между датой закрытия голосования и датой окончания срока действия заключается в том, что после даты окончания срока действия опрос больше не отображается в размещении опроса (не используется в цикле отображения). Дата закрытия голосования определяет только дату, после которой пользователи не могут проголосовать в опросе.

Теперь, когда опрос создан, необходимо связать один или несколько параметров опроса с этим опросом. На вкладке **Параметры** опроса выберите **Новый параметр опроса**.

> ![Параметры добавления опроса](../media/add-poll-options.png "Параметры добавления опроса")  

## <a name="poll-options"></a>Параметры опроса

Опрос представляет собой вопрос, задаваемый пользователю. Опрос имеет два или несколько возможных ответов, определенные автором содержимого. Эти ответы представляются с помощью параметров опроса, которые должны быть связаны с рассматриваемым опросом. Новый параметр опроса создается с помощью окна **Поиск записей** при добавлении параметров опроса в опрос, как описано выше.

## <a name="poll-option-attributes"></a>Атрибуты параметра опроса

| Имя          | Описание                                                                |
|---------------|----------------------------------------------------------------------------|
| Имя          | Описательное имя для параметра опроса.                                  |
| Опрос          | Опрос, с которым связан параметр опроса.                               |
| Ответ        | Текст для отображения в качестве доступного варианта голосования в опросе.                    |
| Голоса         | Число голосов, полученных параметром опроса (только для чтения).              |
| Порядок отображения | Числовое значение, которое определяет порядок отображения параметров опроса. |

## <a name="poll-submissions"></a>Отправки опроса

Когда пользователь посещает веб-сайт, ему будут предоставлена возможность проголосовать в опросе, отображаемом на странице.

> [!div class=mx-imgBorder]
> ![Отправить опрос](../media/submit-poll.png "Отправить опрос")  

Пользователи могут проголосовать только один раз, затем, если опрос отображается, они видят результаты для данного опроса.

> [!div class=mx-imgBorder]
> ![Голоса опроса](../media/poll-votes.png "Голоса опроса")  

Сведения о результатах голосования по опросу хранятся в Common Data Service в виде записей отправок опроса. Сущность отправки опроса содержит следующие сведения:

| Имя        | Описание                                                                                   |
|-------------|-----------------------------------------------------------------------------------------------|
| Имя        | Показывает имя голосовавшего, если этот пользователь выполнил вход; в противном случае записывается анонимный идентификатор. |
| Опрос        | Связанный опрос.                                                                          |
| Параметр опроса | Параметр опроса, выбранный пользователем.                                                       |
| Контакт     | Связанная запись контакта голосовавшего, если пользователь выполнил вход                           |
| Код посетителя  | Анонимный идентификатор голосовавшего, если пользователь анонимный.                                       |
