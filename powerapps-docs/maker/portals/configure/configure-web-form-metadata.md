# <a name="configure-web-form-metadata-for-portals"></a>Настройка метаданных веб-формы для порталов

Метаданные веб-формы содержат дополнительную логику изменения поведения для расширения или переопределения функциональных возможностей полей формы, которые в противном случае невозможны при использовании возможностей редактирования в форме сущностей.

## <a name="add-a-new-record"></a>Добавить новую запись

1. На шаге веб-формы с полями, которые вы хотите изменить, перейдите к **связанным** > **метаданные** . 

2. Выберите **создать метаданные веб-формы**.

## <a name="web-form-metadata-properties"></a>Свойства метаданных веб-формы

Следующие атрибуты предоставляют дополнительные возможности стилей и возможностей для элементов в форме.

| Имя          | Description                                                                                                                                                                                                                                                                                                                                          |
|---------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Шаг веб-формы | Шаг веб-формы, связанный с записью метаданных веб-формы.                                                                                                                                                                                                                                                                                      |
| Тип          | Доступны следующие параметры:<ul><li>Версию</li><li>Раздел</li><li>TAB</li></ul>При выборе атрибута в качестве значения типа отображаются соответствующие параметры для изменения полей в текущей форме, отображаемой для связанного шага. При выборе раздела в качестве значения типа отображаются параметры, доступные для изменения раздела в форме. При выборе вкладки в качестве значения типа отображаются параметры, доступные для изменения вкладки в форме.  |

## <a name="web-form-metadata-type--attribute"></a>Тип метаданных веб-формы = атрибут

Если выбран тип "атрибут", отображаются следующие свойства.


|          Имя          |                                                                                                                                                    Description                                                                                                                                                     |
|------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Логическое имя атрибута |                                                                                                                              Логическое имя изменяемого поля атрибута.                                                                                                                               |
|         Метка          | Заменяет метку по умолчанию, назначенную атрибуту сущности, текстом, указанным во входных данных. Для каждого установленного и включенного языкового пакета для среды Common Data Service поле будет доступно для ввода сообщения на соответствующем языке. |

|         Имя         |                                                                                                                                                                                                                                                                Description                                                                                                                                                                                                                                                                 |
|----------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Игнорировать значение по умолчанию |                                                                              Игнорирует значение указанного поля атрибута по умолчанию. Полезен для атрибутов, которые являются двумя полями параметров, отображаемыми как «да» и «нет» переключателей. Так как значение Да или нет автоматически назначается по умолчанию, этот параметр позволяет отображать ответы "да/нет" без предопределенного ответа.                                                                               |

Для выбора контакта текущего пользователя требуется **атрибут from** , который является атрибутом в сущности Contact, который будет извлечен из записи контакта текущего пользователя и задан в указанном поле атрибута. |

|    From, атрибут    |                                                                                                                                                                                             Атрибут сущности Contact, который будет получен из записи текущего пользователя портала и назначен полю при загрузке формы.                                                                                                                                                                                             |

### <a name="validation"></a>Проверка

В следующем разделе содержатся свойства, изменяющие различные параметры проверки и сообщения об ошибках.

Для каждого языкового пакета, установленного и включенного для среды Common Data Service, будет доступно поле для ввода сообщения на соответствующем языке.

| Имя                                        | Description                                                                                                                                                                                                                                                      |
|---------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Сообщение об ошибке проверки                    | Переопределяет сообщение об ошибке проверки по умолчанию для поля.                                                                                                                                                                                                    |
| Регулярное выражение                          | Регулярное выражение, добавляемое для проверки поля.                                                                                                                                                                                                          |
| Сообщение об ошибке проверки регулярного выражения | Сообщение об ошибке проверки, отображаемое при сбое проверки регулярного выражения.                                                                                                                                                                               |
| Поле является обязательным                           | Убедитесь, что поле атрибута должно содержать значение.                                                                                                                                                                                                   |
| Сообщение об ошибке проверки обязательного поля     | Переопределяет сообщение об ошибке обязательного поля по умолчанию, если поле не содержит значения.                                                                                                                                                                        |
| Сообщение об ошибке проверки диапазона              | Переопределяет сообщение об ошибке проверки диапазона по умолчанию, отображаемое, если значение поля выходит за пределы допустимого минимального и максимального значений, указанных в атрибуте сущности, который имеет тип целое число, десятичное число, число с плавающей запятой или валюта. |
| Сообщение об ошибке средства проверки географического расположения         | Применимо, если атрибут является одной строкой текста, а заданный стиль элемента управления представляет собой однострочный текст в качестве средства проверки географического поиска, после чего будет переопределено сообщение об ошибке по умолчанию, отображаемое в случае сбоя проверки ввода. |
| Сообщение об ошибке проверки суммы константы       | Применимо, если атрибут является целым числом, а указанный стиль элемента управления является целым числом группы в виде постоянной суммы, то при неудачной проверке входных данных будет переопределено сообщение об ошибке по умолчанию.                    |
| Сообщение об ошибке проверки нескольких вариантов    | Применимо, если атрибут имеет два типа параметров, и указанный стиль элемента управления имеет несколько вариантов, то при сбое проверки ввода будет переопределено сообщение об ошибке по умолчанию.                                         |
| Сообщение об ошибке при проверке порядка ранжирования | Применимо, если атрибут является целым числом, а заданный стиль элемента управления является целым числом групп в виде порядка ранжирования без связей, а это переопределяет сообщение об ошибке по умолчанию, отображаемое при сбое проверки ввода.              |
