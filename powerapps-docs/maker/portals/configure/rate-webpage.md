# <a name="rate-or-vote-on-a-webpage-on-a-portal"></a>Оценка или голосование на веб-странице или на портале

Оценки предоставляют пользователям возможность выставлять оценки или голосовать на веб-странице. Рейтинги можно также включить для комментариев на страницах. По умолчанию эта функция отключена, но ее можно включать отдельно для каждой страницы.

Рейтинги — это настраиваемые действия, которые могут использоваться таким же образом, как и любые другие действия, такие как сообщения электронной почты, телефонные звонки и т. д. Поскольку рейтинги являются действиями, с помощью настройки можно задать, чтобы рейтинги отображались для любого выбранного объекта, который отображается или обрабатывается на портале, включая настраиваемые сущности.

## <a name="enable-ratings-for-pages"></a>Включение оценок для страницы

1. Откройте [Приложение управления порталом](configure-portal.md).

2. Откройте **Порталы** > **Веб-страницы**.

3. Выберите **веб-страницу**, на которой необходимо разрешить оценки.

4. На вкладке **Общее** для **Параметры страницы** задайте **Включить оценки** как **Да**.

5. Выберите **Сохранить и закрыть**.

## <a name="use-ratings"></a>Использование оценок

Веб-страницы, для которых включена оценка страниц и разработчик применил элемент управления к шаблону, пользователи могут оценивать страницу с помощью шкалы рейтинга или голосования, в зависимости от типа, выбранного при добавлении элемента управления в шаблон страницы.

## <a name="manage-ratings"></a>Управление рейтингами

Рейтинги для веб-страниц можно просматривать, изменять или удалять на порталах Power Apps.

1. Откройте [Приложение управления порталом](configure-portal.md).

2. Перейдите **Веб-страница**, для которой необходимо просматривать оценки.

3. На вкладке **Связано** выберите **Действия**. В связанном представлении перечисляются оценки для выбранной веб-страницы. В этом представлении пользователи могут изменять или удалять существующие рейтинги.
