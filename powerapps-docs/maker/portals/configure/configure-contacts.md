# <a name="configure-a-contact-for-use-on-a-portal"></a>Настройка контакта для использования на портале

После заполнения основных сведений для контакта (или после того, как пользователь заполнит предложенную форму регистрации на портале) необходимо перейти на вкладку аутентификации в форме контакта на портале, чтобы настроить контакт с использованием локальной аутентификации. Дополнительные сведения о параметрах федеративной проверки подлинности см. в разделе [Задание удостоверения аутентификации для портала](set-authentication-identity.md). Для настройки контакта для порталов с использованием локальной аутентификации следуйте инструкциям ниже.  

1.  Введите **имя пользователя**.
2.  На ленте команд перейдите к **Дополнительные команды** &gt; **Изменить пароль**.

Выполните процедуру изменения пароля, после чего необходимые поля будут настроены автоматически. После этого ваш контакт будет настроен для ваших порталов.

## <a name="change-password-for-a-contact-from-portal-management-app"></a>Изменение пароля для контакта из приложения управления порталами

1.  Откройте [Приложение управления порталом](configure-portal.md).

2.  Выберите **Порталы** > **Контакты** и откройте контакт, для которой требуется изменить пароль.
    Можно также открывать страницу **Контакты** из панели [Общий доступ](../manage-existing-portals.md#share). 

3.  Выберите **Поток задач** на панели инструментов вверху.

    > [!div class="mx-imgBorder"]
    > ![Значок потока задач](../media/task-flow.png "Значок потока задач")

4.  Выберите поток задач **Изменить пароль для контакта портала**.

5.  В области **Изменить пароль для контакта портала** выберите или создайте контакт, для которого требуется изменить пароль, затем выберите **Далее**.

    > [!div class="mx-imgBorder"]
    > ![Выбор контакта для изменения пароля](../media/change-password-select-contact.png "Выбор контакта для изменения пароля")

6.  В поле **Новый пароль** введите новый пароль, затем выберите **Далее**.

    > [!div class="mx-imgBorder"]
    > ![Ввод нового пароля для контакта](../media/change-password-new-password.png "Ввод нового пароля для контакта")

    Если вы не введете пароль и выберите **Далее**, будет выведен запрос, требуется ли удалить пароль для выбранного контакта.

    > [!div class="mx-imgBorder"]
    > ![Удаление пароля для контакта](../media/change-password-remove-password.png "Удаление пароля для контакта")

7.  После внесения изменений выберите **Готово**.
