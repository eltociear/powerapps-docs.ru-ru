# <a name="customize-content-by-using-content-snippets"></a>Настройка содержимого с помощью фрагментов содержимого

Фрагменты содержимого — это небольшие фрагменты редактируемого содержимого, которые могут быть помещены разработчиком на шаблон страницы, что позволяет легко заполнять любую часть макета страницы с помощью настраиваемого содержимого. Элементы управления фрагментами кода, которые отвечают за визуализацию содержимого фрагментов кода на веб-портале, размещаются разработчиками в шаблоне страницы.

## <a name="edit-snippets"></a>Изменить фрагменты кода

Фрагменты можно изменять либо с помощью приложения управления порталом. Основной степенью фрагмента кода является тот факт, что вы можете выделить немного содержимого (отличного от основной копии страницы) и изменить его отдельно, что позволяет всем статическим содержимым на сайте полностью управлять содержимым и редактировать его.

1. Откройте [приложение управления портала](configure-portal.md).

2.  Перейдите на **порталы** > **фрагменты содержимого**.

3.  Чтобы создать новый фрагмент, выберите **создать**.

4.  Чтобы изменить существующий фрагмент, выберите существующий **фрагмент содержимого** в сетке.

Введите значения для следующих полей:

| Имя    | Description                                                                                                   |
|---------|---------------------------------------------------------------------------------------------------------------|
| Имя    | Имя может использоваться разработчиком для помещения значения фрагмента в шаблон страницы в коде портала. |
| Веб | Веб-сайт, связанный с фрагментом.                                                              |
| Value   | Содержимое фрагмента кода, отображаемого на портале. Он может содержать обычный текст или разметку HTML.         |
