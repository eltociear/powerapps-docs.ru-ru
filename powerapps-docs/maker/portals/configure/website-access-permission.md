# <a name="create-website-access-permissions"></a>Создание разрешений на доступ к веб-сайту

Разрешения на доступ к веб-сайту — это набор разрешений, связанный с [веб-ролью](create-web-roles.md), который позволяет изменять различные элементы управления содержимым на портале, отличном от веб-страниц. Параметры разрешений определяют, какие компоненты можно управлять на портале.

| Имя                         | Description                                                                                      |
|------------------------------|--------------------------------------------------------------------------------------------------|
| Управление фрагментами содержимого      | Позволяет изменять элементы управления фрагментами кода.                                                          |
| Управление маркерами сайта          | Позволяет изменять гиперссылки, использующие маркеры сайта.                                           |
| Управление наборами веб-ссылок         | Позволяет изменять [наборы веб-ссылок](manage-web-links.md), включая добавление удаления веб-ссылок из набора веб-ссылок. |
| Предварительный просмотр неопубликованных сущностей | Позволяет просматривать доступные на портале сущности с состоянием публикации черновик.             |
|||

Чтобы создать разрешение на доступ к веб-сайту и добавить его в веб-роль, выполните следующие действия.

1. Откройте [приложение управления портала](configure-portal.md).

2. Перейдите на **порталы** > **разрешения на доступ к веб-сайту**.

3. Нажмите кнопку **Создать**.

4. В разделе **Общие**введите имя, веб-сайт и выберите необходимые разрешения.

5. В разделе **веб-роли**выберите и добавьте веб-роль, чтобы связать разрешение с.

6. Сохраните изменения.
