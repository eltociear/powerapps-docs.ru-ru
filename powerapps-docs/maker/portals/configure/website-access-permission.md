# <a name="create-website-access-permissions"></a>Создание разрешений доступа к веб-сайтам

Разрешения на доступ к веб-сайту представляют собой набор разрешений, связанный с [веб-ролью](create-web-roles.md), который разрешает клиентское редактирование на портале различных управляемых элементов содержимого, отличных от просто веб-страниц. Параметры разрешений определяют, какими компонентами можно управлять на портале.

| Имя (название)                         | Описание                                                                                      |
|------------------------------|--------------------------------------------------------------------------------------------------|
| Управление фрагментами содержимого      | Разрешает редактирование элементов управления фрагментами.                                                            |
| Управление маркерами сайта          | Разрешает редактирование гиперссылок, которые используют маркеры сайтов                                           |
| Управление наборами веб-ссылок         | Разрешает редактирование [наборов веб-ссылок](manage-web-links.md), включая добавление и удаление веб-ссылок из набора веб-ссылок. |
| Предварительный просмотр неопубликованных сущностей | Разрешает просмотр сущностей, доступных на портале, которые имеют состояние публикации Черновик.             |
|||

Для создания разрешения на доступ к веб-сайту и добавления его в веб-роль:

1. Откройте [Приложение управления порталом](configure-portal.md).

2. Выберите **Порталы** > **Разрешения на доступ к веб-сайтам**.

3. Выберите **Создать**.

4. В разделе **Общие** введите имя, веб-сайт и выберите требуемые разрешения.

5. В разделе **Веб-роли** выберите и добавьте веб-роль, с которой требуется связать разрешение.

6. Сохраните изменения.
