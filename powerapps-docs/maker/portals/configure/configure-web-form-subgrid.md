# <a name="configure-web-form-subgrids-for-portals"></a>Настройка вложенных сеток веб-формы для порталов

Вложенные сетки веб-форм настраиваются точно так же, как подсетки формы сущностей: сначала создайте запись метаданных для шага веб-формы с вложенной сеткой, а затем добавьте метаданные конфигурации.

Добавление вложенных сеток в управляемые формы на портале несложно — просто добавьте вложенную сетку в форму, которую вы управляете, используя готовый конструктор форм, и все готово. Сетка будет использовать представление, указанное в Common Data Service конструкторе форм, показать только связанные записи, если выбран этот параметр, при необходимости Показать строку поиска и даже учитывать [разрешения сущностей для порталов](assign-entity-permissions.md). Отображение списка записей, доступного только для чтения, не дает более простого представления. Чтобы включить действия для сетки — создание, обновление, удаление и т. д., необходимо настроить эти действия с помощью конфигурации метаданных.

## <a name="add-subgrid-metadata-to-your-form"></a>Добавление метаданных вложенной сетки в форму

Чтобы добавить метаданные вложенной сетки в форму сущности, перейдите к **метаданным формы сущности** , используя либо верхний раскрывающийся список, либо вложенную сетку в основной форме записи, с которой вы работаете. Дополнительные сведения: [Определение форм сущностей](entity-forms.md).

Чтобы добавить новую запись, выберите **Добавить новые метаданные формы сущности**.

Чтобы изменить существующую запись, выберите запись в сетке. При выборе вложенной **сетки** в качестве значения **типа** отображается другой атрибут, **имя подсетки**.


|     Имя     |                                                       Description                                                        |
|--------------|--------------------------------------------------------------------------------------------------------------------------|
| Имя вложенной сетки | Уникальное имя вложенной сетки в связанной форме сущности. |
|              |                                                                                                                          |

При выборе вложенной сетки в редакторе форм отображается окно свойств. Он содержит поле **имени** , которое следует использовать для назначения полю имени вложенной **сетки** в записи метаданных формы сущности.

![Добавить метаданные вложенной сетки](../media/add-subgrid-metadata.png "Добавить метаданные вложенной сетки")  

При указании допустимого имени вложенной сетки будут отображены параметры конфигурации вложенной сетки. По умолчанию отображаются только **Основные параметры** . Выберите **Дополнительные параметры** , чтобы отобразить дополнительные параметры.

По умолчанию большинство параметров отображаются свернутыми, чтобы сэкономить место. Выберите * * * *, чтобы развернуть раздел и просмотреть дополнительные параметры. Выберите * * * *, чтобы свернуть раздел.
