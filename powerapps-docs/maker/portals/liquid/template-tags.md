# <a name="template-tags"></a>Теги шаблонов

Теги шаблона позволяют управлять выходными данными шаблона различными способами и разрешать сочетание нескольких шаблонов в один выход.

## <a name="include"></a>относится

Включает содержимое одного шаблона в другой, по имени. В порталах PowerApps источником этого другого шаблона обычно будет [веб-шаблон](store-content-web-templates.md). Это позволяет многократно использовать фрагменты общих шаблонов в нескольких местах.  

Если шаблон включен в другой, включенный шаблон будет иметь доступ к любым переменным, определенным в родительском шаблоне.

`{% include 'My Template' %}`

Можно также передать любое количество именованных параметров в тег include. Затем они будут определены как переменные в включаемом шаблоне.

`{% include 'My Template' a:x, b:y %}`

## <a name="block"></a>блок

Используется в сочетании с расширениями для предоставления наследования шаблона. См. раздел расширения для использования.

## <a name="extends"></a>предоставляет

Используется в сочетании с тегом блока, предоставляет наследование шаблона. Это позволяет нескольким шаблонам использовать общий макет при переопределении определенных областей родительского макета.

В порталах PowerApps имя родительского шаблона, предоставленное тегу, обычно будет ссылаться на имя [веб-шаблона](store-content-web-templates.md).  

Когда используется расширение, оно должно быть первым содержимым в шаблоне, за которым может следовать только один или несколько тегов Block.

Если блок, определенный в родительском шаблоне, не переопределяется, то его содержимое в родительском шаблоне (если таковое имеется) будет подготовлено к просмотру.

## <a name="comment"></a>comment

Позволяет оставить невизуализированный код внутри шаблона ликвидности. Любое содержимое в блоке не будет визуализировано, а любой код жидкости внутри не будет выполняться.

**Приведен**

`Hello{% comment %}, {{ user.fullname }}{% endcomment %}. My name is Charles.`

**Проверки**

`Hello. My name is Charles.`

## <a name="raw"></a>Чтен

Позволяет выводить код жидкости на странице без его синтаксического анализа и выполнения.

**Проверки**

`Hello, {{ user.fullname }}. My name is Charles.`
