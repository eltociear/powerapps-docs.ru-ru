# <a name="work-with-liquid-templates"></a>Работа с шаблонами Liquid

Жидкость — это язык шаблона с открытым исходным кодом, интегрированный в порталы. Его можно использовать для добавления динамического содержимого на страницы, а также для создания широкого спектра пользовательских шаблонов. С помощью жидкостей можно:

- Добавление динамического содержимого непосредственно в поле **копирования** веб-страницы или содержимого [фрагмента содержимого](https://docs.microsoft.com/en-us/dynamics365/customer-engagement/portals/customize-content-snippets).  

- [Храните исходное содержимое с помощью веб-шаблонов](store-content-web-templates.md), полностью с помощью конфигурации в PowerApps, для использования в системе управления содержимым порталов powerapps.  

- [Выводит заголовок веб-сайта и основную панель навигации](render-site-header-primary-navigation.md)полностью с помощью настройки в PowerApps.
