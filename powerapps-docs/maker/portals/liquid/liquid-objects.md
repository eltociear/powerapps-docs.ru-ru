# <a name="available-liquid-objects"></a>Доступные объекты Liquid

Объекты жидкости содержат атрибуты для вывода динамического содержимого на страницу. Например, объект Page имеет атрибут Title, который может использоваться для вывода заголовка текущей страницы.

Чтобы получить доступ к атрибуту объекта по имени, используйте точку. Чтобы отобразить атрибут объекта в шаблоне, заключите его в {{и}}.

```
{{ page.title }}
```
Доступ к атрибутам объекта также можно получить с помощью имени строки и \[\]. Это полезно в тех случаях, когда требуемый атрибут определяется динамически, или имя атрибута содержит символы, пробелы, Специальные символы и т. д., которые будут недействительны при использовании. Syntax.

```
{{ page[title] }}

{% assign attribute_name = Name with spaces %}

{{ object[attribute_name] }}
```

### <a name="eventoccurences-object"></a>Объект евентоккуренцес

Объект евентоккурренцес позволяет получить доступ к коллекции объектов экземпляров событий. Можно упорядочить экземпляры событий и указать диапазон дат для извлекаемых вхождений, а также выполнить разбивку на страницы с помощью фильтров жидкости.

## <a name="forloop"></a>ForLoop

Содержит свойства, полезные в блоке цикла [for](iteration-tags.md#for) .  

> [!Note]
> ForLoop можно использовать только в теге [for](iteration-tags.md#for) .

**Приведен**

```
{% for child in page.children %}

{% if forloop.first %}

This is the first child page!

{% else %}

This is child page number {{ forloop.index }}.

{% endif %}

{% endfor %}
```

**Проверки**

```
This is the first child page!

This is child page number 2.

This is child page number 3.
```

| Недопустим  | Возвращает число итераций для цикла, ߝ количество элементов в коллекции, по которым выполняется итерация. |
