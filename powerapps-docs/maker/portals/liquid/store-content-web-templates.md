# <a name="store-source-content-by-using-web-templates"></a>Хранение исходного содержимого с помощью веб-шаблонов

Веб-шаблон — это сущность PowerApps (ADX\_Web Template), которая входит в состав порталов PowerApps, которая используется для хранения исходного содержимого шаблона. Веб-шаблон обычно содержит ликвидность для визуализации динамического содержимого и является центральной сущностью, используемой для интеграции шаблонов жидкостей с остальной частью системы порталов PowerApps.

Веб-шаблоны могут включаться в другое содержимое или сочетаться с другими шаблонами с помощью тегов шаблона, и на них имеются ссылки в этих тегах по атрибуту **Name** . Они также могут использоваться для создания всех пользовательских шаблонов страниц или для создания пользовательских верхних и нижних колонтитулов на веб-сайте портала.

## <a name="web-template-attributes"></a>Атрибуты веб-шаблона

|           |                                                                                                                                                                                                                                                                                 |
|-----------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
|   Имя    |                                                                         Имя шаблона. Используется для ссылки на этот шаблон, если он включен в другое содержимое или расширен другими шаблонами.                                                                         |
|  Источника   |                                  Исходное содержимое шаблона. В PowerApps для этого поля предоставляется редактор исходного кода с выделением синтаксиса и другими функциями редактирования кода.                                  |
| Тип MIME | Дополнительно предоставляет тип MIME для содержимого шаблона. Если значение не указано, предполагается тип Text/HTML. Это значение будет использоваться только в тех случаях, когда шаблон связан с шаблоном страницы, и управляет отображением всего содержимого для этого шаблона. |
|           |                                                                                                                                                                                                                                                                                 |

## <a name="web-templates-as-page-templates"></a>Веб-шаблоны в качестве шаблонов страниц

Веб-шаблоны можно использовать вместе с шаблонами страниц для создания новых шаблонов для системы управления содержимым порталов PowerApps. Это можно сделать полностью в PowerApps, не требуя написания кода .NET или повторного развертывания приложения портала.

Чтобы создать новый шаблон страницы на основе веб-шаблона, выберите **тип** веб-шаблона при создании новой записи шаблона страницы. Затем выберите **веб-шаблон**.

Обратите внимание, что параметр **использовать верхний и нижний колонтитулы веб-сайта** (установлен по умолчанию). Если этот флажок установлен, веб-шаблон будет управлять отображением всего содержимого страницы между верхним и нижним колонтитулами глобального веб-сайта. Если этот параметр не установлен, ваш веб-шаблон будет отвечать за отображение всего ответа в случае, если вы отправите HTML-код, это означает, что все из DOCTYPE в корневой &lt;HTML-теги&gt; и все в между ними.

Хотя наиболее распространенными вариантами использования веб-шаблонов является визуализация HTML, отрисовка всего ответа (путем отменяя выбор **верхнего и нижнего колонтитулов веб-сайта**) дает возможность отобразить любой выбранный текстовый формат. Именно здесь атрибут **типа MIME** веб-шаблона становится релевантным. При отображении шаблона страницы, который не использует верхний и нижний колонтитулы веб-сайта, в качестве заголовка Content-Type для типа MIME задается соответствующий веб-шаблон. (если тип MIME не указан, будет использоваться текст или HTML.) Это предоставляет широкий спектр возможностей для отображения содержимого в формате, отличном от HTML, с использованием жидкостей. Распространенным вариантом использования является визуализация [RSS](https://en.wikipedia.org/wiki/RSS) -канала путем задания типа MIME application/RSS + XML.  

## <a name="web-templates-as-website-headers-and-footers"></a>Веб-шаблоны как верхние и нижние колонтитулы веб-сайта

Веб-шаблоны также можно использовать для переопределения глобального верхнего и нижнего колонтитула, используемого порталом PowerApps. Для этого в поле шаблон **шаблона** или **нижнего колонтитула** веб-сайта задайте нужный веб-шаблон. Обратите внимание, что при переопределении **заголовка веб-сайта**выбранный шаблон принимает ответственность за подготовку к просмотру основных переходов, ссылок входа и выхода, интерфейса поиска и т. д. для элементов интерфейса сайта, которые обычно обрабатываются по умолчанию. шаблон заголовка.
