# <a name="what-is-powerapps-portals"></a>Что такое порталы PowerApps?

Теперь руководители PowerApps могут создать мощный новый тип взаимодействия: внешние веб-сайты, которые позволяют пользователям за пределами Организации входить в систему с использованием широкого спектра удостоверений, создавать и просматривать данные в Common Data Service или даже просматривать содержимое анонимно. Все возможности порталов Dynamics 365, которые ранее предлагались только в качестве надстройки для приложений, управляемых моделью в Dynamics 365, теперь доступны в PowerApps полностью автономно.  

Эти возможности позволяют быстро создавать веб-сайты и настраивать их с помощью страниц, макетов и содержимого. Изготовители могут повторно использовать макеты страниц с помощью шаблонов, добавлять формы и представления для отображения ключевых данных из Common Data Service и публиковать их для пользователей.

> [!NOTE]
> Некоторые возможности могут иметь известные проблемы. Эти проблемы описаны в разделе « [Известные проблемы](known-issues.md) » далее в этом документе.
