# <a name="change-the-base-url-of-a-portal"></a>Изменение базового URL-адреса портала

Можно изменить базовый URL-адрес портала после его подготовки. Например, если выбрать `contosocommunity.microsoftcrmportals.com` в качестве основного URL-адреса при подготовке портала, позже можно изменить его на `contosocommunityportal.microsoftcrmportals.com` для удовлетворения собственных требований.

> [!NOTE]
> Как только базовый URL-адрес портала будет изменен, доступ по старому URL-адресу будет прекращен, и адрес будет доступен другим клиентам для использования для их порталов.

1.  Открытие [Центра администрирования портала Power Apps](admin-overview.md).

2.  Выберите **Действия портала** > **Изменить базовый URL-адрес**. 

    > [!div class=mx-imgBorder]
    > ![Изменение базового URL-адреса портала](../media/change-base-url-action.png "Изменение базового URL-адреса портала")

3.  В окне изменения базового URL-адреса введите новый базовый URL-адрес для портала.

    > [!div class=mx-imgBorder]
    > ![Указание нового базового URL-адреса портала](../media/change-base-url.png "Указание нового базового URL-адреса портала")

4.  Выберите **Изменить URL-адрес** в окне подтверждения.

## <a name="troubleshooting"></a>Устранение неполадок

В этом разделе представлена информация о проблемах устранения неполадок при изменении базового адреса портала.

### <a name="changing-the-base-url-fails"></a>Не удалось изменить базовый URL-адрес

Если при изменении базового URL-адреса портала возник сбой, отображается ошибка, как показано на следующем изображении:

> [!div class=mx-imgBorder]
> ![Ошибка при изменении базового URL-адреса портала](../media/change-base-url-error.png "Ошибка при изменении базового URL-адреса портала")

Как правило, это временные ошибки, и необходимо выбрать **Изменить базовый URL-адрес**, чтобы снова попробовать изменить базовый URL-адрес. Если проблема сохранится, обратитесь в службу поддержки клиентов Майкрософт.
