# <a name="portal-details"></a>Сведения о портале

[!include[cc-beta-prerelease-disclaimer](../../../includes/cc-beta-prerelease-disclaimer.md)]

После того, как вы создадите портал, он начнет подготовку, и вы будете уведомлены о состоянии посредством уведомления. Как только портал подготовлен, вы будете перенаправлены в конструктор порталов. Можно использовать портальный конструктор для создания и для настройки вашего веб-сайта.

Если вам необходимо просмотреть более подробную информацию о подготовленном портале, например, тип портала, идентификатор приложения и т.д., можно открыть [центре администрирования порталов PowerApps](admin-overview.md) и перейти на вкладку **Сведения о портале**.
