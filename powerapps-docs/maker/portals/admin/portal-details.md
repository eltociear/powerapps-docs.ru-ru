# <a name="portal-details"></a>Сведения о портале

После создания портала он начнет подготовку, и вы получите уведомление о состоянии с помощью уведомлений. После подготовки портала вы будете перенаправлены в студию PowerApps Portals. Для создания и настройки веб-сайта можно использовать портал PowerApps студии.

Если необходимо просмотреть дополнительные сведения о подготовленном портале, например, тип портала, идентификатор приложения, владелец портала и т. д., можно открыть [центр администрирования порталов PowerApps](admin-overview.md) и перейти на вкладку **сведения о портале** . Пользователь, создавший портал, является владельцем портала.
