# <a name="portal-details"></a>Сведения о портале

После того, как вы создадите портал, он начнет подготовку, и вы будете уведомлены о состоянии посредством уведомления. Как только портал подготовлен, вы будете перенаправлены в студию порталов PowerApps. Можно использовать студию порталов PowerApps для создания и для настройки вашего веб-сайта.

Если необходимо просмотреть больше сведений о подготовленном портале, такие как тип портала, идентификатор приложения, владелец портала и т. д., можно открыть [центр администрирования порталов PowerApps](admin-overview.md) и перейти на вкладку **Сведения о портале**. Пользователь, который создал портал, является владельцем портала.
