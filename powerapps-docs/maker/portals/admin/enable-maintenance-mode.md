# <a name="maintenance-mode-for-a-portal"></a>Режим обслуживания для портала

Возможны случаи, когда ваш веб-сайт находится в состоянии запланированного обслуживания или не работает из-за временного сбоя. Когда клиент обращается к веб-сайту во время обслуживания, непредсказуемое поведение и периодическое недоступность могут быть опытными. 

Администратор портала может настроить на портале отображение правильного сообщения клиентам при каждом выполнении действия по обслуживанию (например, "пакеты решений обновляются"). Эту возможность можно использовать, включив режим обслуживания на портале. Если режим обслуживания включен, отображается сообщение, и клиенты не смогут просматривать веб-страницы, кроме страницы `<portal URL>/_services/about`.

> [!div class=mx-imgBorder]
> Страница режима ![обслуживания по]умолчанию на(../media/default-maint-page.png "странице режима обслуживания")

## <a name="enable-maintenance-mode"></a>Включение режима обслуживания

Можно включить режим обслуживания на портале, чтобы обеспечить единообразное сообщение, а не выполнять непредсказуемое поведение, если веб-сайт находится под планом обслуживания. Это обеспечит лучшую работу для пользователей портала.

1. Откройте [центр администрирования порталов PowerApps](admin-overview.md).

3. Перейдите к **действиям портала** > **включить режим обслуживания**.

    > [!div class=mx-imgBorder]
    > ![Включение режима обслуживания](../media/enable-maint-mode-button.png "Включение режима обслуживания")

4. В окне **Включение режима обслуживания** введите следующие значения:
    - **Выберите страницу, которая будет использоваться при включенном режиме обслуживания**: выберите одно из следующих значений:

        - **Страница по умолчанию**: выберите это значение, если необходимо, чтобы страница по умолчанию отображалась при включенном режиме обслуживания. По умолчанию этот параметр выбран.

        - **Настраиваемая страница**. Выберите это значение, если требуется, чтобы пользовательская HTML-страница отображалась при включенном режиме обслуживания.

    - **URL-адрес настраиваемой страницы**: это поле доступно только в том случае, если выбран параметр для вывода НАСТРАИВАЕМой HTML-страницы. Необходимо убедиться, что предоставленный URL-адрес страницы является общедоступным. Если указанная страница HTML недоступна, страница по умолчанию отображается с заметкой "Администраторы".

5. Выберите **включить**. При включенном режиме обслуживания портал перезапускается и недоступен в течение нескольких минут. 

    > [!div class=mx-imgBorder]
    > ![Включение параметров режима обслуживания](../media/enable-maint-mode.png "Включение параметров режима обслуживания")

## <a name="configure-or-disable-maintenance-mode"></a>Настройка или отключение режима обслуживания

После включения режима обслуживания для портала можно обновить параметры режима обслуживания и выбрать другую страницу.

Можно также отключить режим обслуживания на портале после завершения запланированного обслуживания веб-сайта. Теперь пользователи портала могут просматривать все веб-страницы и получать к ним доступ как обычно.

1. Откройте [центр администрирования порталов PowerApps](admin-overview.md).

2. Перейдите к **действиям портала** > **Настройка или отключение режима обслуживания**.

    > [!div class=mx-imgBorder]
    > ![Настройка режима обслуживания](../media/configure-maint-mode-button.png "Настройка режима обслуживания")

3. Измените необходимые параметры и выберите **Обновить**. Например, вы можете отобразить страницу по умолчанию, если ранее вы выбрали отображение пользовательской страницы.

4. Чтобы отключить режим обслуживания, выберите **Отключить**. При обновлении или отключении режима обслуживания портал перезапускается и недоступен в течение нескольких минут.

    > [!div class=mx-imgBorder]
    > ![Обновление параметров режима обслуживания](../media/configure-maint-mode.png "Обновление параметров режима обслуживания")
