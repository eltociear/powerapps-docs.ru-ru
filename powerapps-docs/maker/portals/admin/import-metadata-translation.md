# <a name="import-metadata-translation"></a>Импорт перевода метаданных

При подготовке портала связанные с порталом решения устанавливаются в организацию. Во время установки решений переводы метаданных решений (например, имя поля, имя формы и имя представления) устанавливаются только для языков, которые в данный момент активированы в организации. Если в будущем вы активируете новый язык, метаданные не устанавливаются автоматически для нового активированного языка. Чтобы получить перевод метаданных для нового активированного языка, необходимо импортировать перевод метаданных из центра администрирования порталов PowerApps.

## <a name="to-import-metadata-translation"></a>Чтобы импортировать перевод метаданных

1.  Открытие [Центра администрирования портала PowerApps](admin-overview.md).

2.  Перейдите в раздел **Действия портала** > **Получить последние переводы метаданных**. Откроется окно подтверждения с вопросом, следует ли обновить портальные решения.

3.  Выберите **Обновить**. Портальные решения будут обновлены с учетом последнего перевода метаданных.

> [!Note]
> - Если доступна последняя версия пакета портала, она не обновится. Портальные решения обновляются в той же версии. Чтобы обновить портальные решения на основе последних доступных пакетов, необходимо получить доступ к центру администрирования решения.
> - Если пользователь изменил какие-либо данные в Common Data Service, существующие данные не перезаписываются во время обновления.
> - Если устанавливаются портальные решения, невозможно активировать обновление решения.
