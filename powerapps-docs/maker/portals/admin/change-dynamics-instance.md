# <a name="change-the-dynamics-365-instance-audience-or-type-of-portal"></a>Изменение экземпляра, аудитории или типа портала Dynamics 365

После создания и настройки портала можно изменить сведения об экземпляре и портале Dynamics 365.

1. Перейдите на страницу **Центр администрирования Dynamics 365**, затем выберите вкладку **Приложения**.

2. Выберите имя портала, которое требуется изменить, затем выберите **Управление**.

3. Выберите вкладку **Управление экземпляром Dynamics 365**. На этой странице можно просмотреть экземпляр Dynamics 365, который в настоящее время связан с порталом.

4. Выберите **Обновить экземпляр Dynamics 365**. В диалоговом окне используйте имеющиеся поля, чтобы изменить свой экземпляр Dynamics 365, состояние разработки портала, язык портала или администратора портала. Можно также сохранить этот же экземпляр Dynamics 365, но изменить аудиторию портала или тип портала.

5. Выберите кнопку ![Действие подтверждения](../media/confirm-action-icon.png "Действие подтверждения"), чтобы подтвердить изменения.
