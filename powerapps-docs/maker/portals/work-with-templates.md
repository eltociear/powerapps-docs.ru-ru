# <a name="work-with-templates"></a>Работа с шаблонами

Встроенные шаблоны доступны в соответствии с подготовленным порталом. Шаблоны можно изменять с помощью редактора кода. Например, при подготовке Common Data Service начального портала доступны следующие встроенные шаблоны:

- Шаблон по умолчанию для Studio
- Страница с заголовком
- Страница с дочерними ссылками


> [!NOTE]
> Не рекомендуется изменять шаблон, **профиль** **и шаблоны в** **студии по умолчанию**.

Чтобы открыть шаблон в редакторе кода, выполните следующие действия.

1.  [Измените портал](manage-existing-portals.md#edit) , чтобы открыть его в PowerApps Portals Studio.  

2.  Щелкните ![значок шаблоны](media/templates-icon.png "Значок "Шаблоны"") шаблонов в воле в левой части экрана. Отобразятся доступные шаблоны.  

    > [!div class=mx-imgBorder]
    > ![панель "Шаблоны"](media/templates-pane.png "Панель "Шаблоны"")  

3.  Выберите требуемый шаблон, чтобы открыть его в редакторе кода.

4.  Измените код и сохраните изменения.

> [!NOTE]
> - Для расширенной конфигурации можно также добавить теги жидкостей в редакторе исходного кода. Дополнительные сведения: [Работа с шаблонами жидкостей](liquid/liquid-overview.md)
> - Шаблоны страниц, создаваемые с помощью [приложения управления порталом](configure/configure-portal.md) , также отображаются в области **шаблоны** .
