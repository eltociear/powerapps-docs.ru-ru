# <a name="work-with-templates"></a>Работа с шаблонами

Вы можете редактировать встроенные шаблоны с помощью редактора кода. Следующие шаблоны доступны для редактирования:

- Полностью пустая страница
- Пустая страница
- Пустая страница с заголовком
- Целевая страница
- Страница с дочерними ссылками
- Страница с навигацией сбоку

> [!NOTE]
> Рекомендуется не редактировать шаблоны **Профиль** и **Поиск**.

Чтобы открыть шаблон в редакторе кода:

1.  [Изменение портал](manage-existing-portals.md#edit) чтобы открыть его в конструкторе портала.  

2.  Выберите **Шаблоны** ![значок "Шаблоны"](media/templates-icon.png "значок \"Шаблоны\"") из панели инструментов в левой части экрана. На экране появятся доступные шаблоны.  

    > [!div class=mx-imgBorder]
    > ![область шаблонов](media/templates-pane.png "Область шаблонов")  

3.  Выберите нужный шаблон, чтобы открыть его в редакторе кода.

4.  Отредактируйте код и сохраните изменения.

> [!NOTE]
> Можно также добавлять теги Liquid в редакторе исходного кода для расширенной конфигурации. Дополнительные сведения см. в разделе [Работа с шаблонами Liquid](https://docs.microsoft.com/en-us/dynamics365/customer-engagement/portals/custom-templates-dynamic-content)
