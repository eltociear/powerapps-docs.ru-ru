# <a name="work-with-templates"></a>Работа с шаблонами

Встроенные шаблоны доступны согласно порталу, который вы подготавливаете. Вы можете редактировать шаблоны с помощью редактора кода. Например, доступны следующие встроенные шаблоны при подготовке начального портала Common Data Service:

- Шаблон студии по умолчанию
- Страница с заголовком
- Страница с дочерними ссылками


> [!NOTE]
> Рекомендуется не редактировать шаблоны **Шаблон студии по умолчанию**, **Профиль** и **Поиск**.

Чтобы открыть шаблон в редакторе кода:

1.  [Изменить портал](manage-existing-portals.md#edit), чтобы открыть его в студии порталов Power Apps.  

2.  Выберите **Шаблоны** ![значок "Шаблоны"](media/templates-icon.png "Значок "Шаблоны"") из панели инструментов в левой части экрана. На экране появятся доступные шаблоны.  

    > [!div class=mx-imgBorder]
    > ![область шаблонов](media/templates-pane.png "Область шаблонов")  

3.  Выберите нужный шаблон, чтобы открыть его в редакторе кода.

4.  Отредактируйте код и сохраните изменения.

> [!NOTE]
> - Можно также добавлять теги Liquid в редакторе исходного кода для расширенной конфигурации. Дополнительные сведения см. в разделе [Работа с шаблонами Liquid](liquid/liquid-overview.md)
> - Шаблоны страниц, созданные с помощью [приложения управления порталами](configure/configure-portal.md), также отображаются в области **Шаблоны**.
