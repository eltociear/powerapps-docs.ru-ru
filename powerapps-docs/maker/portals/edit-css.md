# <a name="edit-css"></a>Редактирование CSS

Каскадные таблицы стилей (CSS) позволяет управлять форматированием веб-сайта. По умолчанию файлы начальной загрузки. min. CSS и Theme. CSS доступны. Можно изменить существующие файлы CSS и отправить новые файлы CSS. При отправке нового CSS-файла он будет доступен в виде файла в приложении управления портала.

Чтобы открыть CSS в редакторе кода, выполните следующие действия.

1.  [Измените портал](manage-existing-portals.md#edit) , чтобы открыть его в PowerApps Portals Studio.  

2.  В левой части **экрана выберите значок тема темы** ![значок](media/theme-icon.png "темы") из воле. Отобразятся доступные темы.  

    > [!div class=mx-imgBorder]
    > ![](media/theme-pane.png "Область") темы области темы  

3.  Выберите нужный CSS, чтобы открыть его в редакторе кода.

4.  Измените код и сохраните изменения.

Чтобы отправить новый CSS-файл, выполните следующие действия.

1.  [Измените портал](manage-existing-portals.md#edit) , чтобы открыть его в PowerApps Portals Studio.  

2.  В левой части **экрана выберите значок тема темы** ![значок](media/theme-icon.png "темы") из воле. Отобразятся доступные темы.  

3. Выберите **Отправить настраиваемую таблицу CSS**.

    > [!div class=mx-imgBorder]
    > ![](media/upload-css.png "Область") темы области темы  

4. Найдите и выберите файл CSS для отправки.
