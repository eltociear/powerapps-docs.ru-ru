# <a name="keyboard-shortcuts-for-canvas-apps"></a>Сочетания клавиш для приложений Canvas

> [!NOTE]
> Сочетания клавиш могут различаться в зависимости от раскладки клавиатуры.

## <a name="file"></a>File

| Сочетания | Поддержки |
|--|--|
| Ctrl + O (или ALT + F) | Откройте файл. |
| Ctrl + Shift + S (или Alt + P) | Сохраните приложение с другим именем. |
| Ctrl + S | Сохраните приложение с тем же именем или в первый раз. |
| F12 | Скачайте файл приложения (. MSApp). |
| ALT + F | Откройте меню **файл** . |

## <a name="ribbon"></a>Ленте

| Сочетания | Поддержки |
|--|--|
| ВВОД | Выполните выбранную команду. |
| TAB | Перемещение между командами на выбранной вкладке, а затем на следующую вкладку. |
| Ctrl + F6 | Перейдите к следующему ориентиру. |
| Ctrl + Shift + F6 | Перейдите к предыдущему ориентиру. |
| Alt + I | Перейдите на вкладку **Вставка** . |

## <a name="editing"></a>Ввода

| Сочетания | Поддержки |
|--|--|
| Ctrl + A | Выбрать все. |
| Ctrl + X | Бавьте. |
| Ctrl + C | Копии. |
| Ctrl + V | Авить. |
| Ctrl + Z | Команда "отменить". |
| Ctrl + Y | Команда повтора. |
| Ctrl + M | Добавьте экран. |
| CTRL + = или CTRL + SHIFT + = | Увеличить масштаб. |
| CTRL +-или CTRL + SHIFT +- | Уменьшить. |
| Ctrl + 0 | Подогнать полотно к странице. |
| Shift + Ввод | Разбиение строки в формуле. |

## <a name="preview"></a>Предварительный просмотр

| Сочетания | Поддержки |
|--|--|
| ЖМЕМ | Откройте режим предварительного просмотра. |
| ESC | Закрытие режима предварительного просмотра, диалогового окна или раскрывающейся панели.|

## <a name="canvas"></a>На основе холста

| Сочетания | Поддержки |
|--|--|
| TAB | Выберите следующий элемент управления. |
| Ctrl + щелчок или Shift + щелчок | Одновременное выделение нескольких объектов. |
| Стрелка вправо | Сдвинуть выбранный элемент управления вправо. |
| Стрелка влево | Сдвинуть выбранный элемент управления влево. |
| Стрелка вверх | Сдвиг выбранного элемента управления вверх. |
| Стрелка вниз | Сдвиг выбранного элемента управления вниз. |

## <a name="tree-view"></a>Представление в виде дерева

> [!NOTE]
> Для этих сочетаний клавиш требуется, чтобы панель **древовидного представления** была в фокусе.

| Сочетания | Поддержки |
|--|--|
| F2 | Переименование элемента управления. |
| ESC | Отмена переименования элемента управления. |
| Ctrl + G | Группирование и разгруппирование элементов управления. |
| Ctrl +] | Перенесите элемент управления вперед. |
| Ctrl + [ | Отправка элемента управления назад. |
| Ctrl + Shift +] | Переместить на передний план. |
| Ctrl + Shift + [ | Переместить на задний план. |

## <a name="resize"></a>Изменить размер

| Сочетания | Поддержки |
|--|--|
| Shift + стрелка влево | Уменьшить ширину. |
| Ctrl + Shift + стрелка влево | Немного уменьшите ширину. |
| Shift + стрелка вниз | Уменьшить высоту. |
| Ctrl + Shift + стрелка вниз | Немного уменьшить высоту. |
| Shift + стрелка вправо | Увеличить ширину. |
| Ctrl + Shift + стрелка вправо | Немного увеличьте ширину. |
| Shift + стрелка вверх | Увеличить высоту. |
| Ctrl + Shift + стрелка вверх | Немного увеличить высоту. |

## <a name="text-format"></a>Формат текста

| Сочетания | Поддержки |
|--|--|
| Ctrl + B  | Пройдите по уровням полужирный. |
| Ctrl + I | Включить или выключить курсив. |
| Ctrl + U | Добавление или удаление подчеркивания. |

## <a name="other"></a>Иной

| Сочетания | Поддержки |
|--|--|
| СПРАВОЧ | Откройте документацию. |
| Shift + F10 | Откройте контекстное меню в, например в **древовидном представлении**. |
| Alt или CTRL + SHIFT | Скройте элементы дизайна, чтобы можно было взаимодействовать с элементами управления, как если бы приложение было запущено. |
