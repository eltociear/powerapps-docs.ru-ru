# <a name="keyboard-shortcuts-for-canvas-apps"></a>Сочетания клавиш для приложений на основе холста

> [!NOTE]
> Сочетания клавиш может отличаться в зависимости от раскладки клавиатуры.

## <a name="file"></a>Файл

| Ярлык | Действие |
|--|--|
| CTRL + O (или Alt + F) | Откройте файл. |
| Ctrl + Shift + S (или Alt + P) | Сохраните приложение с другим именем. |
| CTRL + S | Сохраните приложение с таким именем или в первый раз. |
| F12 | Загрузите файл приложения (.msapp). |
| ALT + F | Откройте **файл** меню. |

## <a name="ribbon"></a>ленты

| Ярлык | Действие |
|--|--|
| ВВОД | Запустите выбранную команду. |
| TAB | Перемещение между командами, на вкладке "выбранные", а затем на вкладку "Далее". |
| CTRL + F6 | Перейдите к следующей черт лица. |
| Ctrl + Shift + F6 | Перейти к предыдущей черт лица. |
| ALT + I | Выберите **вставить** вкладки. |

## <a name="editing"></a>Редактирование

| Ярлык | Действие |
|--|--|
| CTRL + A | Выберите все. |
| Ctrl+X | Вырежьте. |
| CTRL + C | Скопируйте. |
| CTRL + V | Вставить. |
| CTRL + Z | Команда Undo. |
| CTRL + Y | Отменить команду. |
| CTRL + M | Добавление экрана. |
| CTRL += или Ctrl + Shift += | Увеличьте масштаб. |
| CTRL +- или Ctrl + Shift +- | Уменьшить масштаб. |
| CTRL + 0 | Автоподбор размера полотна страницы. |
| SHIFT + ВВОД | Разбиение строки в формуле. |

## <a name="preview"></a>Предварительный просмотр

| Ярлык | Действие |
|--|--|
| F5 | Откройте режим предварительного просмотра. |
| ESC | Закройте режим предварительного просмотра, диалоговое окно или в раскрывающейся области.|

## <a name="canvas"></a>На основе холста

| Ярлык | Действие |
|--|--|
| TAB | Выберите следующий элемент управления. |
| CTRL + щелчок или Shift + щелчок | Выберите несколько объектов одновременно. |
| Стрелка вправо | : Сдвиг выбранного элемента управления вправо. |
| Стрелка влево | : Сдвиг выбранного элемента управления влево. |
| Стрелка вверх | Сдвинуть выбранный элемент управления вверх. |
| Стрелка вниз | Сдвинуть выбранный элемент управления вниз. |

## <a name="tree-view"></a>Представление в виде дерева

> [!NOTE]
> Следующие сочетания клавиш требуют **представление в виде дерева** панели, чтобы иметь фокус.

| Ярлык | Действие |
|--|--|
| F2 | Переименуйте элемент управления. |
| ESC | Отмените Переименование элемента управления. |
| CTRL + G | Группы/Разгруппируйте элементы управления. |
| CTRL +] | Переведите элемент управления вперед. |
| CTRL + [ | Переместить элемент управления назад. |
| Ctrl + Shift +] | Переведите на передний план. |
| Ctrl + Shift +[ | Переместить на задний план. |

## <a name="resize"></a>Изменение размера

| Ярлык | Действие |
|--|--|
| Shift + Стрелка влево | Уменьшите ширину. |
| Ctrl + Shift + Стрелка влево | Немного уменьшите ширину. |
| Shift + Стрелка вниз | Уменьшение высоты. |
| Ctrl + Shift + Стрелка вниз | Немного уменьшите высоту. |
| Shift + Стрелка вправо | Увеличьте ширину. |
| Ctrl + Shift + Стрелка вправо | Немного увеличьте ширину. |
| Shift + Стрелка вверх | Увеличьте высоту. |
| Ctrl + Shift + Стрелка вверх | Немного увеличьте высоту. |

## <a name="text-format"></a>Текстовый формат

| Ярлык | Действие |
|--|--|
| CTRL + B  | Циклическое переключение между уровнями полужирного шрифта. |
| CTRL + I | Выключать курсивом. |
| CTRL + U | Добавить или удалить подчеркивание. |

## <a name="other"></a>Другие

| Ярлык | Действие |
|--|--|
| F1 | Откройте документацию. |
| SHIFT + F10 | Например, откройте контекстное меню, **представление в виде дерева**. |
| ALT или Ctrl + Shift | Скрыть элементы проектирования, чтобы вы могли взаимодействовать с элементами управления так, как если бы приложение запущено. |
