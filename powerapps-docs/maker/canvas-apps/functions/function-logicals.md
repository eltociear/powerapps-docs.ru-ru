# <a name="and-or-and-not-functions-in-powerapps"></a>Функции And, Or и Not в PowerApps

Логические булевы функции, широко используемые для обработки результатов сравнений и проверок.

## <a name="description"></a>Описание

Функция **And** возвращает **true** (истина), если все ее аргументы принимают значение **true**.

Функция **Or** возвращает **true** (истина), если хотя бы один из ее аргументов принимает значение **true**.

Функция **Not** возвращает **true** (истина), если ее аргумент принимает значение **false** (ложь), и возвращает **false**, если ее аргумент принимает значение **true**.

Эти функции работают так же, как в Excel. Можно также использовать [операторы](operators.md) в эти же операции, используя синтаксис Visual Basic или JavaScript:

| Функция нотации | Нотация оператора Visual Basic | Нотация оператора JavaScript |
| -------------|------------|--------|
| **И (x, y)** | **x и y** | **x & & y** |
| **Or( x, y )** | **x или y** | **x &#124;&#124; y** |
| **Not( x )** | **Не x** | **! x** |

Эти функции принимают логические значения. Нельзя передавать их число или строка непосредственно. Вместо этого необходимо сделать сравнение или теста. Например, это логическую формулу **x > 1** результатом которого является логическое значение **true** Если **x** больше, чем **1**. Если **x** — меньше, чем **1**, формула, результатом которого является **false**.

## <a name="syntax"></a>Синтаксис

**And**( *логическая_формула_1*, *логическая_формула_2* [, *логическая_формула_3*, ...])<br>
**Or**( *LogicalFormula1*, *LogicalFormula2* [, *LogicalFormula3*, ... ] )<br>
**Not**(*логическая_формула*)

- *логическая_формула_x* — обязательный аргумент.  Это логические формулы, которые возвращают логические значения и с которыми выполняет операцию соответствующая функция.

## <a name="examples"></a>Примеры

В примерах этого раздела используются эти глобальные переменные:

- **a** = *false*
- **b** = *true*
- **x** = 10
- **y** = 100
- **s** = «Hello World»

Чтобы создать эти глобальные переменные в приложении, вставьте [ **кнопку** ](../controls/control-button.md) и задать его **OnSelect** следующую формулу:

```powerapps-dot
Set( a, false ); Set( b, true ); Set( x, 10 ); Set( y, 100 ); Set( s, "Hello World" )
```

Нажмите кнопку (щелкнув его во время удерживайте нажатой клавишу Alt), а затем установите **текст** свойство [ **метка** ](../controls/control-text-box.md) управления формулу в первом столбце следующей таблицы.

| Формула | Описание | Возвращаемый результат |
|---------|-------------|--------|
| **И (a, b)** | Проверяет значения **a** и **b**.  Один из аргументов является *false*, поэтому функция возвращает *false*. | *false* |
| **и b** | То же, как в предыдущем примере, используя нотацию Visual Basic. | *false* |
| **& & b** | То же, как в предыдущем примере, используя нотацию JavaScript. | *false* |
| **Or( a, b )** | Проверяет значения **a** и **b**. Один из аргументов является *true*, поэтому функция возвращает *true*. | *true* |
| **Or b** | То же, как в предыдущем примере, используя нотацию Visual Basic. | *true* |
| **a &#124;&#124; b** | То же, как в предыдущем примере, используя нотацию JavaScript. | *true* |
| **Не (a)** | Проверяет значение **a**. Аргумент является *false*, поэтому функция возвращает противоположный результат. | *true* |
| **Не** | То же, как в предыдущем примере, используя нотацию Visual Basic. | *true* |
| **! Это** | То же, как в предыдущем примере, используя нотацию JavaScript. | *true* |
| **LEN (&nbsp;s&nbsp;)&nbsp;<&nbsp;20 и&nbsp;не&nbsp;IsBlank (&nbsp;s&nbsp;)** | Тесты ли длина **s** меньше 20 и ли не **пустой** значение. Длина — меньше 20, а значение не пустое. Таким образом, результатом является *true*. | *true* |
| **Or(&nbsp;Len(&nbsp;s&nbsp;)&nbsp;<&nbsp;10, x&nbsp;<&nbsp;100, y&nbsp;<&nbsp;100&nbsp;)** | Тесты ли длина **s** является менее чем 10 ли **x** меньше 100 и был ли **y** меньше 100. Первый и третий аргументы имеют значение false, но второй — true. Таким образом, функция возвращает *true*. | *true* |
| **Не IsBlank (&nbsp;s&nbsp;)** | Тесты ли **s** — *пустой*, который возвращает *false*. **Не** возвращает отрицание этот результат, который является *true*. | *true* |
