# <a name="concat-and-concatenate-functions-in-powerapps"></a>Функции Concat и Concatenate в PowerApps

Объединяют отдельные строки текста и строки в [таблицах](../working-with-tables.md).

## <a name="description"></a>Описание

Функция **Concatenate** объединяет сочетание отдельных строк и таблицу из одного столбца со строками. При использовании этой функции с отдельными строками, это эквивалентно использованию **&** [оператор](operators.md).

Функция **Concat** объединяет результат формулы, примененной ко всем [записям](../working-with-tables.md#records) таблицы, в результате чего получается одна строка. Используйте эту функцию для объединения строк таблицы, как функция **[Sum](function-aggregates.md)** делает с числами.

[!INCLUDE [record-scope](../../../includes/record-scope.md)]

Используйте [ **разбиения** ](function-split.md) или [ **MatchAll** ](function-ismatch.md) функцию для разбиения строки на таблицу с подстроками.

## <a name="syntax"></a>Синтаксис

**Concat**( *Table*, *Formula* )

- *Table* — обязательный аргумент.  Таблица, для которой выполняется операция.
- *Formula* — обязательный аргумент.  Формула, которую необходимо применить к записям таблицы.

**Concatenate**( *Строка1* [, *Строка2*, ...] )

- *Строка* — обязательные аргументы.  Сочетание отдельных строк или таблица из одного столбца со строками.

## <a name="examples"></a>Примеры

В примерах этого раздела используются эти глобальные переменные:

- **FirstName** = «Jane»
- **LastName** = «Doe»
- **Продукты** = ![таблицы с двумя столбцами и четырьмя строками](media/function-concatenate/products.png)

Чтобы создать эти глобальные переменные в приложении, вставьте [ **кнопку** ](../controls/control-button.md) и задать его **OnSelect** следующую формулу:

```powerapps-dot
Set( FirstName, "Jane" ); Set( LastName, "Doe" );
Set( Products,
    Table(
        { Name: "Violin", Type: "String" },
        { Name: "Cello", Type: "String" },
        { Name: "Trumpet", Type: "Wind" }
    )
)
```

Нажмите кнопку (щелкнув его удерживая нажатой клавишу Alt).

### <a name="trimming-the-end"></a>Обрезки окончания

Две последние примеры включают дополнительный «,» в конце результат. Функция добавляет запятую и пробел **имя** значение каждой записи в таблице, включая последнюю запись.

В некоторых случаях эти дополнительные символы не имеет значения. Например непробельного разделитель не отображается, если для отображения результатов в метке. Если вы хотите удалить эти дополнительные символы, используйте [ **слева** ](function-left-mid-right.md) или [ **соответствия** ](function-ismatch.md) функции.

В этих примерах значение **текст** свойства элемента управления label к формуле в первом столбце следующей таблицы.

| Формула | Описание | Возвращаемый результат |
|---------|-------------|--------|
| **Left (Concat (&nbsp;продуктов,&nbsp;имя&nbsp;&&nbsp;«,&nbsp;"&nbsp;), Len (&nbsp;Concat (&nbsp;продуктов,&nbsp;имя&nbsp; & &nbsp;«,&nbsp;"&nbsp;)&nbsp;)&nbsp;-&nbsp;2)** | Возвращает результат **Concat** , но удаляет последние два символа, которые формируют лишний разделитель. | «Скрипка,&nbsp;Cello,&nbsp;Trumpet» |
| **Match (Concat (&nbsp;продуктов,&nbsp;имя&nbsp;&&nbsp;«,&nbsp;"&nbsp;), «^ (?&lt; Обрезать&gt;. *),&nbsp;$») .trim** | Возвращает символы **Concat** с начала текстовой строки (^) и заканчивая ($), но не включает нежелательных запятой и пробелом в конце. | «Скрипка,&nbsp;Cello,&nbsp;Trumpet» |
