# <a name="replace-and-substitute-functions-in-power-apps"></a>Замена и замена функций в Power Apps
Замена части текстовой строки другой строкой.

## <a name="description"></a>Description
В функции **Replace** текст, который требуется заменить, определяется начальной позицией и длиной.  

Функция **Substitute** ищет текст, который нужно заменить, сравнивая его с определенной строкой. Если найдено несколько совпадений, можно заменить их или указать одно для замены.

Если передать одну строку, функция возвращает ее измененную версию. Если передать содержащую строки [таблицу](../working-with-tables.md) из одного столбца, функция возвращает таблицу из одного столбца с измененными строками. Таблицу с несколькими столбцами можно преобразовать в таблицу с одним столбцом, как описано в статье об [использовании таблиц](../working-with-tables.md).

## <a name="syntax"></a>Синтаксис
**Replace**(*строка*, *начальная_позиция*, *количество_символов*, *новая_строка*)

* *Строка* — обязательный аргумент. Это строка, с которой выполняется преобразование.
* *начальная_позиция* — обязательный аргумент. Позиция символа, с которого нужно начать замену. Первый символ аргумента *строка* находится в позиции 1.
* *Количество_символов* — обязательный аргумент. Число символов, которые нужно заменить в аргументе *строка*.
* *новая_строка* — обязательный аргумент. Строка, которой нужно заменить текст в исходной строке. Количество символов в этом аргументе может отличаться от значения аргумента *количество_символов*.

**Substitute**(*строка*, *старая_строка*, *новая_строка*[, *номер_вхождения*])

* *Строка* — обязательный аргумент. Это строка, с которой выполняется преобразование.
* *старая_строка* — обязательный аргумент. Строка, которую требуется заменить.
* *новая_строка* — обязательный аргумент. Строка, которой нужно заменить текст в исходной строке. У аргументов *старая_строка* и *новая_строка* может быть разная длина.
* *номер_вхождения* — необязательный аргумент. Используйте этот аргумент, чтобы указать, какой экземпляр *олдстринг* следует заменить, если *строка* содержит более одного экземпляра. Если этот аргумент не указан, все экземпляры будут заменены.

**Replace**(*таблица_с_одним_столбцом*, *начальная_позиция*, *количество_символов*, *новая_строка*)

* *SingleColumnTable* — обязательный аргумент. Таблица с одним столбцом, для строк в котором нужно выполнить преобразование.
* *начальная_позиция* — обязательный аргумент. Позиция символа, с которого нужно начать замену.  Первый символ каждой строки из таблицы находится в позиции 1.
* *Количество_символов* — обязательный аргумент. Число символов, которые нужно заменить в каждой из строк.
* *новая_строка* — обязательный аргумент.  Строка, которой нужно заменить текст в исходной строке. Количество символов в этом аргументе может отличаться от значения аргумента *количество_символов*.

**Substitute**(*таблица_с_одним_столбцом*, *старая_строка*, *новая_строка*[, *номер_вхождения*])

* *SingleColumnTable* — обязательный аргумент. Таблица с одним столбцом, для строк в котором нужно выполнить преобразование.
* *старая_строка* — обязательный аргумент.  Строка, которую требуется заменить.
* *новая_строка* — обязательный аргумент.  Строка, которой нужно заменить текст в исходной строке. У аргументов *старая_строка* и *новая_строка* может быть разная длина.
* *номер_вхождения* — необязательный аргумент. Используйте этот аргумент, чтобы указать, какой экземпляр *олдстринг* следует заменить, если *строка* содержит более одного экземпляра. Если этот аргумент не указан, все экземпляры будут заменены.

## <a name="examples"></a>Примеры

| Формула | Description | Возвращаемый результат |
|---------|-------------|--------|
| **Replace ("абкдефгхижк",&nbsp;6,&nbsp;5,&nbsp;"*")** | Заменяет пять символов в "абкдефгхижк" одним символом "*", начиная с шестого символа ("f"). | "abcd * k" |
| **Replace (&nbsp;"2019",&nbsp;3,&nbsp;2,&nbsp;"20"&nbsp;)** | Заменяет последние два символа "2019" на "20". | "2020" |
| **Replace (&nbsp;"123456",&nbsp;1,&nbsp;3,&nbsp;"_"&nbsp;)** | Заменяет первые три символа "123456" одним символом "_". | "_456" | 
| **Замените (&nbsp;"данные продаж&nbsp;",&nbsp;"продажи",&nbsp;"стоимость"&nbsp;)** | Заменяет строку "Cost" на "Sales". | "Данные о затратах" | 
| **Замените ("Quarter&nbsp;1,&nbsp;2018", "1", "2", 1)** | Заменяет только первый экземпляр "1" на "2", поскольку четвертый аргумент (*инстанценумбер*) предоставляется с 1. |  «Квартал 2, 2018» |
| **Замените ("Quarter&nbsp;1,&nbsp;2011", "1", "2", 3)** | Заменяет только третий экземпляр "1" на "2", так как четвертый аргумент (*инстанценумбер*) предоставляется с 3. | «Квартал 1, 2012» |
| **Замените ("Quarter&nbsp;1,&nbsp;2011", "1", "2")** | Заменяет все экземпляры "1" на "2", поскольку не указан четвертый аргумент (*инстанценумбер*). | «Квартал 2, 2022» |
| **Replace (<br>[&nbsp;"квартал&nbsp;1,&nbsp;2018",<br>"квартал&nbsp;2,&nbsp;2011",<br>"квартал&nbsp;4,&nbsp;2019"],<br>9, 1, "3")** | Заменяет девятый символ в каждой записи таблицы с одним столбцом на «3». | [&nbsp;"квартал&nbsp;3,&nbsp;2018",<br>"Quarter&nbsp;3,&nbsp;2011",<br>"Quarter&nbsp;3,&nbsp;2019"&nbsp;] |
| **Замените (<br>[&nbsp;"квартал&nbsp;1,&nbsp;2018",<br>"квартал&nbsp;1,&nbsp;2011",<br>"Q1,&nbsp;2019"&nbsp;],<br>"1", "3", 1)** | Так как четвертый аргумент (*инстанценумбер*) предоставляется со значением 1, заменяет в каждой записи таблицы с одним столбцом "3" только первый экземпляр "1". | [&nbsp;"квартал&nbsp;3,&nbsp;2018",<br>"Quarter&nbsp;3,&nbsp;2011",<br>"Q3,&nbsp;2019"&nbsp;] |
| **Замените (<br>[&nbsp;"квартал&nbsp;1,&nbsp;2018",<br>"квартал&nbsp;1,&nbsp;2011",<br>"Q1,&nbsp;2019"&nbsp;],<br>"1", "3")** | Поскольку четвертый аргумент (*инстанценумбер*) не указан, заменяет все экземпляры "1" в каждой записи таблицы с одним столбцом на "3". | [&nbsp;"квартал&nbsp;3,&nbsp;2038",<br>"Quarter&nbsp;3,&nbsp;2033",<br>"Q3,&nbsp;2039"&nbsp;] |
