# <a name="text-function-in-powerapps"></a>Функция Text в PowerApps
Преобразует любое значение и форматирует значение числа или даты и времени в строку текста.

## <a name="description"></a>Описание
Функция **Text** форматирует число или значение даты и времени на основе аргументов следующих типов.

* Стандартный формат даты и времени, который указывается с помощью перечисления **DateTimeFormat**. Для дат и времени этот подход является предпочтительным, так как он автоматически учитывает язык и регион для каждого пользователя.
* Пользовательский формат, который состоит из строки заполнители, которые определяют, к примеру, ли нумерация показывает десятичного разделителя и даты Показывать полное имя месяца сокращенного названия месяца и месяц как число. В PowerApps используется тот же набор заполнителей, что и в Microsoft Excel. В данной строке язык указывает язык, на котором для интерпретации других заполнители. Если настраиваемого формата содержит точку, к примеру, заполнитель языковый формат указывает, является ли период десятичного разделителя (ja-JP) или тысяч разделителя (es-ES).

Подробнее см. статью о [работе с датами и временем](../show-text-dates-times.md).

**Текст** функции также можно преобразовать любой тип данных с использованием формата по умолчанию в текстовом представлении. Используется для передачи нетекстовые значения в текстовых функции, такие как [ **Len**](function-len.md), [ **справа**](function-left-mid-right.md), и [  **IsMatch**](function-ismatch.md).

### <a name="number-placeholders"></a>Заполнители для чисел

| Заполнитель | Описание |
| --- | --- |
| **0** (*ноль*) |Отображает незначащие нули, если число имеет меньше разрядов, чем количество нулей в строке форматирования. Например, формат **#.00** позволяет отобразить значение **8,9** как **8,90**. |
| **#** |Работает так же, как **0** (ноль). Но в этом случае функция **Text** не возвращает дополнительные нули, если число имеет слева или справа от десятичного разделителя меньше цифр, чем количество символов # в строке форматирования. Например, число **8,9** при использовании формата **#.##** будет отображаться как **8,9**. |
| **.** (*точка*) |Отображает символ десятичного разделителя. Зависит от языка пользовательского формата; см. в разделе [глобальные приложения](#global-apps) для получения дополнительных сведений. |
| **,** (*запятая*) |Отображает символ разделителя разрядов, обычно отделяющий значения тысяч. Функция **Text** разделяет число запятыми, если в строке форматирования есть запятая между символами номера ( **#** ) или нулями. Зависит от языка пользовательского формата; см. в разделе [глобальные приложения](#global-apps) для получения дополнительных сведений. |

Если число имеет больше разрядов справа от десятичного разделителя, чем количество заполнителей в строке форматирования, число округляется до стольких десятичных разрядов, сколько указано заполнителей. Если число имеет больше разрядов слева от десятичного разделителя, чем количество заполнителей в строке форматирования, отображаются все дополнительные цифры. Если в строке форматирования слева от десятичной запятой указаны только символы номера (#), числа меньше 1 начинаются с десятичного разделителя (например, **.47**).

## <a name="global-apps"></a>Глобальные приложения
Функция **Text** учитывает региональные форматы. Во многих языках она используется для правильного оформления дат, времени, валют и чисел. Чтобы сделать все правильно, ей нужны два блока информации.

* **Язык пользовательского формата:** Для разработчиков как следует интерпретировать пользовательский формат? Знаки разделителей ( **.** и **,** ) имеют разные значения в разных языках. Если указать настраиваемый формат, можно включить заполнитель с указанием языка, или значение по умолчанию, который отражает язык, задаваемое для устройства. Даже проще, можно использовать один из [стандартных форматов даты и времени](#predefined-datetime-formats), которой зависят от языка.
* **Язык результата:** Для пользователей на каком языке с результатом функции элементы: Имена месяцев и дней недели должны быть на соответствующем языке для пользователя приложения, который можно задать, добавив третий необязательный аргумент для **текст** функции. 

В обоих случаях пользователь указывает язык, с помощью [тег языка](function-language.md#language-tags). Чтобы просмотреть список поддерживаемых языков, введите **Text (1234, «»,)** в строке формул или **Дополнительно** вкладке панели справа, а затем прокрутите список языковых стандартов, предлагаемые для третьего аргумента.

### <a name="language-placeholder"></a>Заполнитель с указанием языка
Чтобы указать язык пользовательского формата, используйте следующий синтаксис.

| Заполнитель | Описание |
| --- | --- |
| **[$-*LanguageTag*]** |Здесь *LanguageTag* обозначает тег языка, возвращаемый функцией **Language**. Его можно указать только на языке (такие как **[$-en]** для английского языка), или его также можно указать этот регион (такие как **[$-en-GB]** для указания Соединенное Королевство). |

Заполнитель с указанием языка может находиться в любом месте пользовательской строки форматирования, но должен встречаться только один раз.

Если задать пользовательский формат без заполнитель с указанием языка и формата является неоднозначным с точки зрения глобальных, автоматически вставляется тег языка для текущего языка.  

**[$-en US]**  предполагается, что если этот заполнитель не присутствует, при запуске приложения. 

> [!NOTE]
> В будущих версиях синтаксис этого заполнителя может быть изменен во избежание путаницы с похожие, но отличающиеся заполнитель, который поддерживает Excel.

### <a name="result-language-tag"></a>Тег для языка результата
Результат **текст** включает переведенные строки для месяцев, дней недели и обозначения AM/PM, а также соответствующую группу и десятичные разделители.

По умолчанию функция **Text** использует язык пользователя, запустившего приложение. Функция **Language** возвращает тег языка для текущего пользователя. Это значение по умолчанию можно переопределить, указав тег языка для третьего аргумента **текст**.

## <a name="syntax"></a>Синтаксис
**Текст**( *NumberOrDateTime*, *DateTimeFormatEnum* [, *ResultLanguageTag* ])

* *NumberOrDateTime* — обязательный. Числовое значение или значение даты и времени, которое нужно отформатировать.
* *DateTimeFormat* — обязательный аргумент.  Значение из списка **DateTimeFormat**.
* *ResultLanguageTag* — необязательный аргумент. Тег языка для форматирования результата. По умолчанию используется язык текущего пользователя.

**Текст**( *NumberOrDateTime*, *CustomFormat* [, *ResultLanguageTag* ])

* *Number* — обязательный аргумент. Числовое значение или значение даты и времени, которое нужно отформатировать.
* *CustomFormat* — обязательный аргумент. Один или несколько заполнителей, заключенные в двойные кавычки.
* *ResultLanguageTag* — необязательный аргумент. Тег языка для форматирования результата. По умолчанию используется язык текущего пользователя.

**Текст**( *AnyValue* )

* *AnyValue* — обязательный. Значение, которое необходимо преобразовать в текстовом виде. Используется формат по умолчанию.
