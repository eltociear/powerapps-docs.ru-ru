# <a name="preview-embed-canvas-apps-in-your-applications"></a>Предварительная версия: внедрение приложений Canvas в приложения

[!INCLUDE[cc-beta-prerelease-disclaimer](../../includes/cc-beta-prerelease-disclaimer.md)]

Power Apps — это высокопроизводительная платформа разработки с низким кодом, которая предоставляет разработчикам приложения и позволяет любому пользователю создавать обширные возможности. Внедрение приложений Canvas в приложение позволяет быстро и легко преобразовывать клиентов в приложения. Новые Создатели приложений могут расширять ваши приложения, создавая простые пользовательские формы для добавления многофункциональных экранов, а также все, что между ними. Они могут подключаться к данным, создавать бизнес-логику и управлять рабочими процессами в приложении. Благодаря тому, что клиенты смогут работать в качестве руководителей приложений, ваши приложения и службы могут, наконец, пересекать эту последнюю миля и полностью подоходили в рабочих процессах клиентов, что делает их еще более ценными и незаменимыми.

Пакет SDK для новых приложений Canvas позволяет внедрять приложения Canvas в приложения.
