# <a name="preview-embed-canvas-apps-in-your-applications"></a>Предварительный просмотр: Внедрение приложения на основе холста в приложениях

[!INCLUDE[cc-beta-prerelease-disclaimer](../../includes/cc-beta-prerelease-disclaimer.md)]

PowerApps — высокая производительность, это платформа разработки малокодовому принципу, который расширяет возможности разработчиков приложений и дает возможность всем пользователям создавать полнофункциональные. Путем внедрения приложений на основе холста в приложении, можно быстро и легко преобразовать клиентов в разработчиков приложений. Ваш новый создатели приложений могут расширить возможности приложений от создания простых пользовательских форм для добавления многофункциональных экраны компонентов и многого другого. Их можно подключиться к данным, создания бизнес-логики и координировать процессы из приложения. Включив своими клиентами, чтобы выступать в роли разработчиков приложений, приложений и служб можно наконец cross, последней мили и отлично подходят в рабочие процессы ваших заказчиков, что делает их еще более ценным и необходимыми компонентами.

Внедрение пакета SDK позволяет внедрять новые приложения на основе холста приложения в приложениях на основе холста.

> - Это предварительная версия функции.

Внедрение пакета SDK для нового приложения на основе холста имеет функцию предварительного просмотра, которая требует, чтобы зарегистрироваться как изнутри, прежде чем его можно использовать. Чтобы зарегистрироваться для компонента и просмотра документации:

1. Объединения как изнутри, посетив <https://experience.dynamics.com/insider>.
2. Поиск программы: **Внедрение пакета SDK для предварительной версии PowerApps**.
