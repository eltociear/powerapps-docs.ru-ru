# <a name="gallery-control-in-canvas-apps"></a>Элемент управления "Галерея" в приложениях Canvas

Элемент управления, содержащий другие элементы управления и отображающий набор данных.

## <a name="description"></a>Описание

Элемент управления **Коллекция** отображает несколько записей из источника данных, и каждая запись может содержать данные нескольких типов. Например, используйте элемент управления " **коллекция** " для отображения нескольких контактов с каждым элементом, в котором отображаются контактные данные, включающие имя, адрес и номер телефона для каждого контакта.

Каждое поле данных отображается в отдельном элементе управления в **коллекции** . Эти элементы управления можно настроить в шаблоне. Шаблон отображается как первый элемент в коллекции:

- На левой границе элемента управления **галереи** в горизонтальной или альбомной ориентации.
- И в верхней части элемента управления " **Галерея** " в вертикальной и книжной ориентации. 

Все внесенные в шаблон изменения отражаются во всем элементе управления **Коллекция**.

Доступны предопределенные шаблоны для отображения изображений и текста в галерее, а коллекции для элементов с переменной высотой.

## <a name="limitations"></a>Ограничения

Если пользователь прокручивает элемент управления галереи **гибкой высоты** до загрузки всех элементов, то элемент, который в данный момент находится в представлении, может быть отправлен и выведен из представления после завершения загрузки данных. Чтобы избежать этой проблемы, используйте стандартный элемент управления **галереи** вместо варианта **гибкой высоты** .

## <a name="key-properties"></a>Основные свойства

[По умолчанию](properties-core.md) — элемент или запись из источника данных, который будет выбран в коллекции при запуске приложения.

[Items](properties-core.md) — источник данных, отображаемых в элементе управления, такой как коллекция, список или диаграмма.

**Selected** — выбранный элемент.

## <a name="additional-properties"></a>Дополнительные свойства

[Акцессиблелабел](properties-accessibility.md) — метка коллекции (а не содержащихся в ней элементов) для средств чтения с экрана. Должна описывать список элементов.

**AllItems** — все элементы в коллекции, включая дополнительные значения элементов управления, которые являются частью шаблона коллекции.

[BorderColor](properties-color-border.md) — цвет границы элемента управления.

[BorderStyle](properties-color-border.md) — стиль границы элемента управления: **Solid**, **Dashed**, **Dotted** или **None**.

[BorderThickness](properties-color-border.md) — толщина границы элемента управления.

[DisplayMode](properties-core.md) — указывает, допускает ли элемент управления ввод данных пользователем (**изменить**), отображает только данные (**представление**) или отключено (**отключено**).

[Fill](properties-color-border.md) — цвет фона элемента управления.

[Height](properties-size-location.md) — расстояние между верхним и нижним краями элемента управления.

**Итемакцессиблелабел** — метка каждого элемента коллекции для средств чтения с экрана. Должен описывать каждый элемент.

**NavigationStep** — определяет шаг прокрутки коллекции, если ее свойству **ShowNavigation** присвоено значение **true** и пользователь выбрал стрелку навигации на любом конце этой коллекции.

Возможность **выбора** — можно ли выбирать элементы коллекции. Если задано **значение true**, средства чтения с экрана обозначают коллекцию как выбираемый список. Выберите элемент, выбрав его. Если задано **значение false**, средства чтения с экрана обозначают коллекцию как обычный список, а выбор элемента не выделяет его.

**Шовнавигатион** — отображается ли стрелка на каждом конце коллекции, чтобы пользователь мог прокручивать элементы коллекции, выбирая стрелку.

**ShowScrollbar** — определяет появление полосы прокрутки при наведении пользователем указателя на коллекцию.

**Snap** — при прокрутке пользователем коллекции определяет ее автоматическую привязку, которая позволяет отображать следующий элемент целиком.

**TemplateFill** — цвет фона коллекции.

**TemplatePadding** — расстояние между элементами в коллекции.

**Темплатесизе** — высота шаблона для коллекции в вертикальной или книжной ориентации. Или ширину шаблона для коллекции в горизонтальной или альбомной ориентации.

**Transition** — визуальный эффект (**Pop**, **Push** или **None**) при наведении пользователем указателя на элемент коллекции.

[Visible](properties-core.md) — определяет, отображается элемент управления или он скрыт.

[Width](properties-size-location.md) — расстояние между левым и правым краями элемента управления.

**WrapCount** — количество отображаемых элементов в строке (горизонтальный макет) или столбце (вертикальный макет).

[X](properties-size-location.md) — расстояние между левым краями элемента управления и левой границей родительского контейнера или экрана.

[Y](properties-size-location.md) — расстояние между верхним краями элемента управления и верхней границей родительского контейнера или экрана.

## <a name="accessibility-guidelines"></a>Руководство по настройке специальных возможностей

### <a name="color-contrast"></a>Контрастность

Так как при щелчке в любом месте элемента коллекции должен происходить его выбор, следует установить достаточную контрастность между следующими элементами:

- [BorderColor](properties-color-border.md) и цвет вне коллекции (при наличии границы).
- [Заливка](properties-color-border.md) и цвет вне коллекции (если граница отсутствует).

### <a name="screen-reader-support"></a>Поддержка средства чтения с экрана

- [Акцессиблелабел](properties-accessibility.md) должен присутствовать.

    > [!NOTE]
    > Средства чтения с экрана будут сообщать об изменении элементов в коллекции. **AccessibleLabel** также упоминается. Так обеспечивается контекст оповещения. Это еще важнее при наличии нескольких коллекций на экране.

- Если элемент коллекции содержит несколько элементов управления, используйте **итемакцессиблелабел** для отображения содержимого элементов коллекции.

- Присвойте параметру **SELECT** значение **true** , если нужно, чтобы пользователи могли выбрать элемент коллекции. В противном случае присвойте этому параметру значение **false**.

- Если элемент коллекции содержит несколько элементов управления, используйте **итемакцессиблелабел** для предоставления сводки содержимого элемента коллекции.

- Возможность **выбора** должна быть задана соответствующим образом в зависимости от того, предназначены ли пользователи для выбора элемента коллекции.

### <a name="keyboard-support"></a>Поддержка клавиатуры

- Установите для параметра **ShowScrollbar** значение **true**. На большинстве устройств с сенсорным экраном полоса прокрутки не будет отображаться, пока не начнется переход.
- Так как при щелчке в любом месте элемента коллекции должен происходить его выбор, поэтому для клавиатуры также должны быть предусмотрены аналогичные действия. Например, можно добавить [кнопку](control-button.md) , для которой свойство **OnSelect** имеет значение **SELECT (Parent)** .

    > [!NOTE]
  > Элементы управления вне коллекции не учитываются в порядке навигации с клавиатуры внутри коллекции. Области элементов управления [TabIndex](properties-accessibility.md) внутри галереи ограничены. Дополнительные сведения см. в разделе [свойства специальных возможностей](properties-accessibility.md).
