# <a name="show-edit-or-add-a-record-in-a-canvas-app"></a>Отображение, изменение или Добавление записи в приложении Canvas

В приложении Canvas добавьте и настройте элемент управления формы **[отображения](controls/control-form-detail.md)** для отображения всех полей в записи, можно также добавить и настроить элемент управления формы **[редактирования](controls/control-form-detail.md)** , чтобы изменить любое поле в записи, добавить запись и сохранить изменения в источнике данных.

## <a name="prerequisites"></a>Технические условия

- Узнайте, как [Добавить и настроить элемент управления](add-configure-controls.md) в Power Apps.
- Скачайте [этот файл Excel](https://az787822.vo.msecnd.net/documentation/get-started-from-data/FlooringEstimates.xlsx), содержащий образец данных для этого руководства.
- Отправьте файл Excel в [учетную запись облачного хранилища](connections/cloud-storage-blob-connections.md), например OneDrive для бизнеса.
- Создайте или откройте приложение для телефонов, [Добавьте подключение](add-data-connection.md) к таблице **Флуринжестиматес** в файле Excel.

    Форму можно добавить в приложение для планшетов, но она не будет соответствовать этому разделу, так как по умолчанию в форме будут сопоставлены три столбца.

- При открытии существующего приложения добавьте на него [экран](add-screen-context-variables.md) .

## <a name="add-a-form-and-show-data"></a>Добавление формы и отображение данных
1. На пустом экране добавьте элемент управления "раскрывающийся **[список](controls/control-drop-down.md)** " и назовите его **чусепродукт**.

    > [!NOTE]
   > Если вы не знаете, как добавить и переименовать элемент управления или установить для него свойство, см. статью о [добавлении и настройке элементов управления](add-configure-controls.md).

1. На вкладке **Свойства** в правой области задайте для **элементов** **значение** `FlooringEstimates` и `Name`.

    ![Задание свойства элементов формы](./media/add-form/items-property.png)

    В списке отображаются названия товаров из категории напольных покрытий, полученные из источника данных.

1. Добавьте элемент управления формы **редактирования** , переместите его под **чусепродукт**, а затем измените размер формы, чтобы охватить большую часть экрана.

    ![Добавление формы](./media/add-form/add-a-form.png)

    > [!NOTE]
   > В этом разделе описывается элемент управления формы **редактирования** , но аналогичные принципы применяются к элементу **управления формы.**

1. Задайте для свойства **[DataSource](controls/control-form-detail.md)** формы значение **флуринжестиматес** , а для свойства **[Item](controls/control-form-detail.md)** — следующую формулу:

    `First(Filter(FlooringEstimates; Name=ChooseProduct.Selected.Value))`

   Формула настроена таким образом, что после завершения настройки формы в ней будет отображаться запись, выбранная пользователем в списке **ChooseProduct**.

1. На вкладке **Свойства** правой панели выберите **изменить поля**.

    ![Изменить поля](./media/add-form/edit-fields.png)

1. В области **Поля** выберите **Добавить поле**, установите флажок для каждого поля и выберите **Добавить**.

    ![Добавить поля](./media/add-form/add-fields.png)

1. Нажмите кнопку с многоточием (...) рядом с **полем Добавить**, выберите **Свернуть все**и перетащите **имя** в начало списка.

    ![Переместить поле](./media/add-form/move-field.png)

    Изменение будет отражено в элементе управления формы **редактирования** .

    ![Отобразить форму](./media/add-form/show-form1.png)

## <a name="set-the-card-type-for-a-field"></a>Выбор типа карточки для поля
1. В области **поля** разверните поле Price ( **Цена** ), щелкнув стрелку вниз.

1. Откройте список **тип элемента управления** и выберите **изменить ползунок**.

    ![Изменить ползунок](./media/add-form/edit-slider.png)

    В форме в поле " **Цена** " отображается элемент управления " **ползунок** " вместо элемента управления **вводом текста** .

1. используемых Выполните те же действия, чтобы изменить элемент управления для поля " **Обзор** " на " **Изменение многострочного текстового** элемента управления".

## <a name="edit-form-only-save-changes"></a>Только для формы редактирования: сохранение изменений

1. Переименуйте форму **EditForm**.

1. Добавьте элемент управления **[Кнопка](controls/control-button.md)** и задайте следующую формулу в качестве значения свойства **[OnSelect](controls/properties-core.md)** :

   `SubmitForm(EditForm)`

1. Нажмите клавишу F5, чтобы открыть окно предварительного просмотра, изменить имя продукта, а затем нажать созданную кнопку.

    Функция **[субмитформ](functions/function-form.md)** сохраняет изменения в источнике данных.

1. используемых Закройте окно предварительного просмотра, нажав клавишу ESC (или щелкнув значок Закрыть в правом верхнем углу).
