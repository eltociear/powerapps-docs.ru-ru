# <a name="show-edit-or-add-a-record-in-a-canvas-app"></a>Отображение, редактирование и Добавление записи в приложение на основе холста

В приложение на основе холста, добавьте и настройте **[отображения](controls/control-form-detail.md)** элемент управления, чтобы отобразить все поля в записи, можно также добавить и настроить формы **[изменить](controls/control-form-detail.md)** элемент управления формы для редактирования любого поля в записи, добавьте запись и сохраните изменения в источнике данных.

## <a name="prerequisites"></a>Технические условия

- Узнайте, как [добавить и настроить элемент управления](add-configure-controls.md) в PowerApps.
- Скачайте [этот файл Excel](https://az787822.vo.msecnd.net/documentation/get-started-from-data/FlooringEstimates.xlsx), содержащий образец данных для этого руководства.
- Отправьте файл Excel в [учетную запись облачного хранилища](connections/cloud-storage-blob-connections.md), например OneDrive для бизнеса.
- Создайте или откройте приложения для телефонов, [добавить подключение](add-data-connection.md) для **FlooringEstimates** таблицы в файле Excel.

    Формы можно добавить в приложение для планшета, но она не будет соответствовать в этом разделе, так как форма будет иметь три столбца по умолчанию.

- Если открыть существующее приложение, [добавьте экран](add-screen-context-variables.md) к нему.

## <a name="add-a-form-and-show-data"></a>Добавление формы и отображение данных
1. Пустой экран, добавьте **[раскрывающийся список](controls/control-drop-down.md)** управления и назовите его **ChooseProduct**.

    > [!NOTE]
   > Если вы не знаете, как добавить и переименовать элемент управления или установить для него свойство, см. статью о [добавлении и настройке элементов управления](add-configure-controls.md).

1. На **свойства** вкладку на панели справа задайте **элементы** для `FlooringEstimates` и **значение** для `Name`.

    ![Задайте свойства элементов формы](./media/add-form/items-property.png)

    В списке отображаются названия товаров из категории напольных покрытий, полученные из источника данных.

1. Добавить **изменить** элемент управления формы, переместите его под **ChooseProduct**и затем растяните форму на основную часть экрана.

    ![Добавление формы](./media/add-form/add-a-form.png)

    > [!NOTE]
   > В этом разделе описывается **изменить** элемента управления формы, но подобные принципы применяются к **отображения** элемент управления формы.

1. Формы задайте **[DataSource](controls/control-form-detail.md)** свойства **FlooringEstimates** и его **[элемент](controls/control-form-detail.md)** свойства Эта формула:

    `First(Filter(FlooringEstimates, Name=ChooseProduct.Selected.Value))`

   Формула настроена таким образом, что после завершения настройки формы в ней будет отображаться запись, выбранная пользователем в списке **ChooseProduct**.

1. На **свойства** вкладку на правой панели, выберите **изменить поля**.

    ![Изменение полей](./media/add-form/edit-fields.png)

1. В области **Поля** выберите **Добавить поле**, установите флажок для каждого поля и выберите **Добавить**.

    ![Добавление полей](./media/add-form/add-fields.png)

1. Нажмите кнопку с многоточием (...) рядом с полем **добавить поле**выберите **Свернуть все**, а затем перетащите **имя** в верхней части списка.

    ![Переместить поле](./media/add-form/move-field.png)

    **Изменить** ваши изменения отразятся в элемент управления формы.

    ![Просмотр формы](./media/add-form/show-form1.png)

## <a name="set-the-card-type-for-a-field"></a>Выбор типа карточки для поля
1. В **поля** панели разверните **цена** поле, выбрав стрелку вниз.

1. Откройте **типа элемента управления** , а затем выберите **изменить ползунок**.

    ![Изменить ползунок](./media/add-form/edit-slider.png)

    В форме **цена** поле показано **ползунок** вместо элемента **ввода текста** элемента управления.

1. (необязательно) Примените ту же процедуру, чтобы изменить элемент управления для **Обзор** поле **изменить многострочный текст** элемента управления.

## <a name="edit-form-only-save-changes"></a>Только для формы редактирования: сохранение изменений

1. Измените имя формы **EditForm**.

1. Добавьте элемент управления **[Кнопка](controls/control-button.md)** и задайте следующую формулу в качестве значения свойства **[OnSelect](controls/properties-core.md)**:

   `SubmitForm(EditForm)`

1. Нажмите клавишу F5, чтобы открыть предварительный просмотр, измените имя продукта, а затем выберите созданную кнопку.

    **[SubmitForm](functions/function-form.md)** функция сохраняет изменения в источнике данных.

1. (необязательно) Закройте режим предварительного просмотра, нажав клавишу Esc (либо нажав значок закрытия в правом верхнем углу).
