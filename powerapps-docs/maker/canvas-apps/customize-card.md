# <a name="customize-a-card-in-a-canvas-app"></a>Настройка карточки в приложение на основе холста

Выполните основную настройку (без разблокировки карточки), например измените элемент управления. Выполните дополнительную настройку, разблокировав карточку и, например, добавив элемент управления, недоступный по умолчанию для этой карточки.

Общие сведения см. в статье [Общие сведения о карточках данных](working-with-cards.md).

## <a name="prerequisites"></a>Технические условия

- Узнайте, как [добавлять и настраивать элементы управления](add-configure-controls.md).
- Можно просмотреть в этом разделе описаны только общие принципы, или для выполнения его шаг за шагом выполните процедуры, описанные в следующих разделах:

    1. [Создание приложения](data-platform-create-app.md)
    1. [Настройка коллекции](customize-layout-sharepoint.md)
    1. [Настройка форм](customize-forms-sharepoint.md)

## <a name="customize-a-locked-card"></a>Настройка заблокированной карточки

В этой процедуре вы замените **[текстовое поле](controls/control-text-input.md)** управления **["ползунок"] (элементы управления, элемент управления slider.md** без разблокирования карточки.

1. В приложении, созданное и настроенное выберите **EditForm1** на левой навигационной панели, а затем выберите **изменить поля** на **свойства** вкладку на панели справа.

1. В списке полей, щелкните стрелку вниз для **количество сотрудников**, а затем откройте список в разделе **типа элемента управления**.

    > [!div class="mx-imgBorder"]
    > ![Стрелку раскрывающегося списка параметров для числовой карточки](./media/customize-card/card-selector.png)

1. Выберите **изменить ползунок**.

    Изменения отразятся на экране.

    > [!div class="mx-imgBorder"]
    > ![EditForm1 с элементом управления "ползунок"](./media/customize-card/add-slider.png)

## <a name="unlock-and-customize-a-card"></a>Разблокировка и настройка карточки

В этой процедуре вы карточка разблокируется и обновить **Max** свойство **ползунок** элемент управления, который вы только что добавили.

1. В **EditForm1**выберите **ползунок** контролировать **количество сотрудников** карты.

    > [!div class="mx-imgBorder"]
    > ![Установите ползунок](./media/customize-card/select-slider.png)

1. На **Дополнительно** вкладку на панели справа выберите значок замка, чтобы разблокировать карточку.

    > [!div class="mx-imgBorder"]
    > ![Карточка разблокируется](./media/customize-card/lock-icon.png)

1. Задайте **Max** свойство **ползунок** управления до 10 000.

    > [!div class="mx-imgBorder"]
    > ![Свойство максимальной на вкладке "Дополнительно"](./media/customize-card/max-property.png)

    **Ползунок** элемент управления отображает более точного значения.

    > [!div class="mx-imgBorder"]
    > ![Диапазон "ползунок": 0-10,000](./media/customize-card/final-slider.png)
