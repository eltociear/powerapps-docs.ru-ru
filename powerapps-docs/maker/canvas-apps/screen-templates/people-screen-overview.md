# <a name="overview-of-the-people-screen-template-for-canvas-apps"></a>Обзор шаблонов экрана пользователей для приложений на основе холста

В приложение на основе холста добавьте экран людей, позволяет пользователям находить людей внутри организаций. Пользователи могут поиска, выберите и добавить к коллекции пользователей. Можно изменить типы, которые отображаются в коллекции результатов поиска, использовать выбранные параметры пользователей для отправки сообщения электронной почты и выполните другие настройки.

Вы также можете добавить другие экраны на базе шаблона, такие как отображение различных данных из Office 365, [электронной почты](email-screen-overview.md), пользователя [календаря](calendar-screen-overview.md), и [доступности](meeting-screen-overview.md) людей может пользователей нужно пригласить на собрание.

В этом обзоре показано:
> [!div class="checklist"]
> * Инструкции по использованию экрана людей по умолчанию.
> * Как изменить на экране.
> * Как интегрировать в приложения на экране.

Подробные сведения о функции по умолчанию этот экран, см. в разделе [людей экранах](people-screen-reference.md).

## <a name="prerequisite"></a>Необходимое условие

Знакомство с тем, как добавить и настроить экраны и другие элементы управления, как вы [Создание приложения в PowerApps](../data-platform-create-app-scratch.md).

## <a name="default-functionality"></a>Стандартная функциональность

Добавление экрана людей на основе шаблона:

1. [Войдите в](http://web.powerapps.com?utm_source=padocs&utm_medium=linkinadoc&utm_campaign=referralsfromdoc) в PowerApps, для создания приложения или откройте существующее приложение в PowerApps Studio.

    В этом разделе показано приложение для телефона, но те же принципы применимы к приложение для планшета.

1. На **Главная** вкладке на ленте выберите **новый экран** > **людей**.

    По умолчанию экран выглядит примерно так:

    ![Состояние экрана начальной людей](media/people-screen/people-screen-empty.png)

1. Для начала поиска для пользователей, выберите поле ввода текста в верхней и начните вводить имя коллег. Результаты поиска отображаются под текстовым полем ввода:

    ![состояние поиска людей экрана](media/people-screen/people-browse-gall-full.png)

1. При выборе отдельных пользователей в результатах поиска, они добавляются к **MyPeople** коллекции. Поиск панели входное значение сбрасывается, отображая коллекции людей, которые вы выбрали:

    ![результаты коллекции экрана людей](media/people-screen/people-people-gall-full.png)

## <a name="modify-the-screen"></a>Изменение экрана

Можно изменять функциональность по умолчанию этот экран с [показаны различные данные для людей](people-screen-overview.md#show-different-data-for-people).

Если вы хотите изменить дальнейшей экрана, используйте [людей экранах](./people-screen-reference.md) по.

### <a name="show-different-data-for-people"></a>Отображение различных данных для пользователей

Использует этот экран [Office365Users.SearchUser](https://docs.microsoft.com/connectors/office365users/#searchuser) операции для поиска пользователей в вашей организации. Он предоставляет дополнительные поля для каждого события, за что отображается в **UserBrowseGallery** элемента управления. Добавление или изменение поля в коллекции представляет собой простой процесс.

1. В **UserBrowseGallery**, выберите метку, изменить (или добавьте его и оставить его включенным).

1. С его **текст** свойства, выбранного в строке формул Замените содержимое с `ThisItem.`

    IntelliSense отображает список полей, которые можно выбрать.

1. Выберите нужное поле.

    **Текст** свойство обновится и будет `ThisItem.{FieldSelection}`.

## <a name="integrate-the-screen-into-an-app"></a>Интеграция экрана в приложение

На экране для людей представляет собой мощный набор элементов управления в сам по себе, но обычно выполняет лучшие как часть большего размера, более универсальное приложение. Этот экран можно интегрировать в приложение большего размера несколькими способами, в том числе [с помощью вашей кэшированный список людей](people-screen-overview.md#use-your-cached-list-of-people).

### <a name="use-your-cached-list-of-people"></a>Использовать кэшированный список людей

На экране людей кэширует выбранных людей в **MyPeople** коллекции. Следует ли вызывать бизнес-сценария для уточняющего запроса пользователя, необходимо знать, как использовать эту коллекцию. Здесь вы пошагово рассмотрим способы подключения этот экран на экран элементарные электронной почты и отправки сообщений электронной почты пользователей в **MyPeople** коллекции. Вы также получите представление о том, как [электронной почты экрана](./email-screen-overview.md) работает.

1. Добавление источника данных Office 365 Outlook в приложение, выбрав **представление** вкладке, выбрав **источников данных** > **добавить источник данных**и поиск Office 365 outlook connector. Может потребоваться выбор **новое подключение** его найти.
1. После вставки людей, вставьте новый пустой экран. В этом экране добавьте значок со стрелкой назад, два поля ввода текста и значка отправки.
1. Назовите экран **EmailScreen**, значок со стрелкой назад, чтобы **BackIcon**, одно поле для ввода текста для **SubjectLine**, другой на **MessageBody**и значок отправки для **SendIcon**.
1. Задайте **OnSelect** свойство **BackIcon** для `Back()`.
1. Задайте **OnSelect** свойство **SendIcon** следующую формулу:

    ```powerapps-dot
    Office365.SendEmail( 
        Concat( MyPeople, UserPrincipalName & ";" ), 
        SubjectLine.Text, 
        MessageBody.Text 
    )
    ```
    
    Здесь вы используете соединитель Outlook для отправки сообщения электронной почты. Он передается `Concat(MyPeople, UserPrincipalName & ";")` как список получателей. Эта формула выполняет объединение все адреса электронной почты в **MyPeople** коллекции в одну строку с разделив их запятой. Это ничем не отличается от написания строку адресов электронной почты, разделенных точкой с запятой в строке «Кому» любимый адрес электронной почты клиента.
    * Передается `SubjectLine.Text` как тему сообщения, и `MessageBody.Text` в теле сообщения.
1. На экране "люди" в правом верхнем углу, вставить **Mail** значок.
   Измените цвет значка, чтобы все, что вам подходит.
1. Задайте **OnSelect** свойство **SendIcon** для `Navigate( EmailScreen, None )`.

    Теперь у вас есть приложение с двумя экранами, в котором можно выбрать пользователей, создать сообщение электронной почты, чтобы их и отправьте его. Вы можете протестировать его, но Будьте внимательны, так как приложение отправляет отправляет по электронной почте всем пользователям можно добавить в **MyPeople** коллекции.
