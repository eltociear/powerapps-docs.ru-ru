# <a name="create-a-canvas-app-from-within-a-solution"></a>Создание приложения Canvas из решения

Создайте приложение в рамках решения, если, например, требуется развернуть приложение в другой среде. Решения могут содержать не только приложения, но и настраиваемые сущности, наборы параметров и другие компоненты. Вы можете быстро настроить среду различными способами, создав приложения и другие компоненты из решения, экспортировав решение, а затем импортировав его в другую среду.

Дополнительные сведения о решениях см. в разделе [Общие сведения о решениях](../common-data-service/solutions-overview.md).

## <a name="prerequisite"></a>Необходимое условие

Чтобы выполнить действия, описанные в этом разделе, необходимо переключиться на среду, содержащую Common Data Service базу данных.

## <a name="create-a-solution"></a>Создание решения

Эту процедуру можно пропустить, если у вас уже есть решение, в котором вы хотите создать приложение или связать его с приложением.

1. [Войдите](https://web.powerapps.com?utm_source=padocs&utm_medium=linkinadoc&utm_campaign=referralsfromdoc) в PowerApps, а затем (при необходимости) переключитесь в соответствующую среду:

    - Если вы хотите создать приложение из решения, переключитесь в любую среду, содержащую Common Data Service базу данных.
    - Если требуется связать существующее приложение с решением, переключитесь в среду, содержащую это приложение.

1. На панели навигации слева выберите **решения**.

    > [!div class="mx-imgBorder"]
    > ![Параметр Solutions (решения) в левой](./media/add-app-solution/left-nav.png "панели навигации") слева

1. В заголовке окна щелкните **новое решение**.

    > [!div class="mx-imgBorder"]
    > ![Новый параметр-Solution в объявлении](./media/add-app-solution/banner-new-solution.png "нового решения в баннере")

1. В появившемся окне укажите отображаемое имя, издателя и версию решения.

    > [!div class="mx-imgBorder"]
    > ![Параметры нового решения](./media/add-app-solution/configure-new-solution.png "для нового решения")

    Имя (без пробелов) будет создано автоматически на основе указанного отображаемого имени, но при необходимости можно настроить созданное имя. Можно указать издателя по умолчанию для среды и **1,0** для версии, если в этих областях нет особых нужд.

1. В левом верхнем углу щелкните **сохранить и закрыть**.

    > [!div class="mx-imgBorder"]
    > ![Сохранение нового]решения(./media/add-app-solution/save-new-solution.png "Сохранение нового решения")

## <a name="create-a-canvas-app-in-a-solution"></a>Создание приложения Canvas в решении

В решении можно создать пустое приложение Canvas. Вы не можете автоматически создать приложение с тремя экранами или настроить шаблон или пример приложения из решения.

1. [Войдите](https://web.powerapps.com?utm_source=padocs&utm_medium=linkinadoc&utm_campaign=referralsfromdoc) в PowerApps.

1. При необходимости переключитесь на среду, содержащую решение, в котором нужно создать приложение Canvas.

1. На панели навигации слева выберите **решения**.

    > [!div class="mx-imgBorder"]
    > ![Параметр Solutions (решения) в левой](./media/add-app-solution/left-nav.png "панели навигации") слева

1. В списке решений выберите решение, в котором нужно создать приложение Canvas.

1. В заголовке окна щелкните **новое**приложение  > **app** > **Canvas**, а затем выберите форму (телефон или планшет) приложения, которое требуется создать.

    > [!div class="mx-imgBorder"]
    > ![Параметры для создания приложения в](./media/add-app-solution/new-option.png "вариантах решения для создания приложения в решении")

    PowerApps Studio открывается с пустым холстом на другой вкладке браузера.

1. Создайте приложение (или сделайте хотя бы одно изменение), а затем сохраните изменения.

1. На вкладке браузер, где выбрано решение, нажмите кнопку **Готово** , чтобы обновить список компонентов в решении.

    > [!div class="mx-imgBorder"]
    > ![](./media/add-app-solution/done-button.png "Кнопка") "Готово"

    Новое приложение появится в списке компонентов для этого решения. Если сохранить изменения в приложении, они будут отражены в версии, которая находится в решении.

## <a name="link-an-existing-canvas-app-to-a-solution"></a>Связывание существующего приложения Canvas с решением

Если вы хотите связать приложение с решением, оба должны находиться в той же среде, и приложение должно быть создано из решения.

1. [Войдите](https://web.powerapps.com?utm_source=padocs&utm_medium=linkinadoc&utm_campaign=referralsfromdoc) в PowerApps.

1. При необходимости переключитесь на среду, содержащую решение, с которым необходимо связать приложение.

1. На панели навигации слева выберите **решения**.

    > [!div class="mx-imgBorder"]
    > ![Параметр Solutions (решения) в левой](./media/add-app-solution/left-nav.png "панели навигации") слева

1. В списке решений выберите решение, с которым необходимо связать приложение.

1. В заголовке окна выберите **добавить существующее**приложение  > **приложения** **холста** > .

    > [!div class="mx-imgBorder"]
    > ![Параметры баннера для связывания существующих](./media/add-app-solution/add-existing.png "параметров баннера приложения с связыванием существующего приложения")

    Откроется список приложений Canvas, созданных в решении в этой среде.

1. В списке приложений выберите приложение, которое нужно связать с решением, а затем нажмите кнопку **Добавить**.

    > [!div class="mx-imgBorder"]
    > ![Добавить]кнопку "(./media/add-app-solution/add-button.png "Добавить") кнопку"
