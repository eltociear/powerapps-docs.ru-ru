# <a name="create-a-canvas-app-from-within-a-solution"></a>Создание из приложения на основе холста в решении

Создание приложения в решении, если, например, вы хотите развернуть приложение в другую среду. Решения могут содержать не только приложений, но также настраиваемые сущности, наборы параметров и других компонентов. Могут быстро настроить среду, в различные виды путем создания приложений и других компонентов из в решении, экспорта решения и затем импортировать его в другой среде.

Решения создаются на той же платформе, как Dynamics 365 для взаимодействия с клиентами. Дополнительные сведения см. в разделе [Обзор решений](../common-data-service/solutions-overview.md).

## <a name="prerequisite"></a>Необходимое условие

Чтобы выполнить действия, описанные в этом разделе, переключитесь на среду, которая содержит базу данных Common Data Service.

## <a name="create-a-solution"></a>Создание решения

Эту процедуру можно пропустить, если у вас уже есть решение, в которой вы хотите создать приложение, или к которым вам нужно связать приложение.

1. [Войдите в](https://web.powerapps.com?utm_source=padocs&utm_medium=linkinadoc&utm_campaign=referralsfromdoc) в PowerApps, а затем (при необходимости) перейдите в соответствующую среду:

    - Если вы хотите создать приложение в решении, переключитесь в любой среде, которая содержит базу данных Common Data Service.
    - Если вы хотите связать существующее приложение в решение, переключитесь на среду, содержащий это приложение.

1. На панели навигации слева, выберите **решения**.

    > [!div class="mx-imgBorder"]
    > ![Параметр решения на панели навигации слева](./media/add-app-solution/left-nav.png "параметр решения на панели навигации слева")

1. В баннере под строкой заголовка, выберите **новое решение**.

    > [!div class="mx-imgBorder"]
    > ![Параметр New решения на баннере](./media/add-app-solution/banner-new-solution.png "параметр New решения в заголовке")

1. В появившемся окне укажите отображаемое имя, издатель и версия для вашего решения.

    > [!div class="mx-imgBorder"]
    > ![Параметры для нового решения](./media/add-app-solution/configure-new-solution.png "параметры для нового решения")

    Имя (с без пробелов) создается автоматически на основе отображаемого имени вами, но можно настроить автоматически созданное имя, если требуется. Можно указать издателя по умолчанию для вашей среды и **1.0** для версии, если у вас нет конкретных потребностей в этих областях.

1. В левом верхнем углу выберите **сохраните и закройте**.

    > [!div class="mx-imgBorder"]
    > ![Сохранить новое решение](./media/add-app-solution/save-new-solution.png "сохранить новое решение")

## <a name="create-a-canvas-app-in-a-solution"></a>Создайте приложение на основе холста в решении

Можно создать пустой холст в ней приложение из решения. Невозможно автоматически создайте приложение с тремя экранами или настроить шаблон или пример приложения из внутри решения.

1. [Войдите в](https://web.powerapps.com?utm_source=padocs&utm_medium=linkinadoc&utm_campaign=referralsfromdoc) в PowerApps.

1. При необходимости переключитесь на среду, содержащий решение, в котором вы хотите создать приложение на основе холста.

1. На панели навигации слева, выберите **решения**.

    > [!div class="mx-imgBorder"]
    > ![Параметр решения на панели навигации слева](./media/add-app-solution/left-nav.png "параметр решения на панели навигации слева")

1. В списке решений выберите решение, в котором вы хотите создать приложение на основе холста.

1. В баннере под строкой заголовка, выберите **New** > **приложения** > **приложения на основе холста**, а затем выберите форм-фактор (телефон или планшет) приложения, Вы хотите создать.

    > [!div class="mx-imgBorder"]
    > ![Параметры для создания приложения в решении](./media/add-app-solution/new-option.png "параметры для создания приложения в решении")

    PowerApps Studio откроется пустой холст в новой вкладке браузера.

1. Создание нового приложения (или сделать по крайней мере одно изменение), а затем сохраните изменения.

1. На вкладке браузера, если вы выбрали решения, выберите **сделать** , чтобы обновить список компонентов в решении.

    > [!div class="mx-imgBorder"]
    > ![Кнопка "Готово"](./media/add-app-solution/done-button.png "кнопка \"Готово\"")

    Новое приложение появится в списке компонентов для этого решения. Если сохранить изменения в приложение, они будут отражены в версии, который находится в решении.

## <a name="link-an-existing-canvas-app-to-a-solution"></a>Связывание существующего приложения canvas в решение

Если вы хотите связать приложение с решением, оба должны быть в той же среде, и приложения должна быть создана из внутри решения.

1. [Войдите в](https://web.powerapps.com?utm_source=padocs&utm_medium=linkinadoc&utm_campaign=referralsfromdoc) в PowerApps.

1. При необходимости переключитесь на среду, содержащий решение, к которому вы хотите связать приложение.

1. На панели навигации слева, выберите **решения**.

    > [!div class="mx-imgBorder"]
    > ![Параметр решения на панели навигации слева](./media/add-app-solution/left-nav.png "параметр решения на панели навигации слева")

1. В списке решений выберите решение, к которому вы хотите связать приложение.

1. В баннере под строкой заголовка, выберите **добавить существующий** > **приложения** > **приложения на основе холста**.

    > [!div class="mx-imgBorder"]
    > ![Баннер параметры, чтобы связать существующее приложение](./media/add-app-solution/add-existing.png "баннер параметры, чтобы связать существующее приложение")

    Появится список приложений на основе холста, которые были созданы в рамках решения в этой среде.

1. В списке приложений выберите приложение, которое вы хотите связать решение, а затем выберите **добавить**.

    > [!div class="mx-imgBorder"]
    > ![Кнопка "Добавить"](./media/add-app-solution/add-button.png "кнопка \"Добавить\"")
