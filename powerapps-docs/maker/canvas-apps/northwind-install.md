# <a name="install-northwind-traders-database-and-apps"></a>Установка базы данных Northwind Traders и приложений

Выполнив действия, описанные в [этой серии документов](northwind-orders-canvas-part1.md), вы узнаете основные понятия о реляционных данных, реализованное в образце базы данных в Common Data Service. Можно также изучить пример бизнес-приложений, как на основе холста и на основе модели, для управления данными и получите практический опыт, создавая такое приложение. В этом первом разделе объясняется, как установить базу данных Northwind Traders в своей среде и получить доступ к примеров приложений, которые можно открыть для редактирования, чтобы показать, как они были созданы.

Компания Northwind Traders занимается вымышленной организации, который управляет заказы, продуктами, клиентами, поставщиками и многие другие аспекты малого бизнеса. В этом примере в первых версиях Microsoft Access, а должен по-прежнему доступна как шаблон доступа.

## <a name="prerequisites"></a>Технические условия

- Лицензия PowerApps, Common Data Service поддерживает. Вы можете [использования бесплатной пробной лицензии](../signup-for-powerapps.md) в течение 30 дней.
- Среды с базой данных Common Data Service. Вы можете [создание такой среды](https://docs.microsoft.com/power-platform/admin/create-environment) при наличии соответствующих разрешений.

## <a name="download-the-solution"></a>Загрузка решения

> [!div class="nextstepaction"]
> [Загрузите файл решения Northwind Traders](https://pwrappssamples.blob.core.windows.net/samples/NorthwindTraders_1_0_0_6.zip)

Это [решение](../../developer/common-data-service/introduction-solutions.md) файл (ZIP-файл) содержит определения сущностей, наборы параметров и бизнес-процессов; canvas и приложений на основе модели; и другие элементы, которые используются совместно.

## <a name="install-the-solution"></a>Установка решения

1. Войдите в [PowerApps](https://web.powerapps.com?utm_source=padocs&utm_medium=linkinadoc&utm_campaign=referralsfromdoc), а затем убедитесь, что вы работаете в среде, которая содержит базу данных Common Data Service.

1. В области навигации слева выберите **решения**, а затем выберите **импорта** на панели инструментов в верхней части экрана:

    > [!div class="mx-imgBorder"]
    > ![Точка входа решений представление и импорта решения](media/northwind-install/solution-import.png)

1. В **выберите пакет решения** выберите **Обзор**.

    > [!div class="mx-imgBorder"]
    > ![Выберите страницу пакета решения до выбора пакета](media/northwind-install/select-solution2.png)

1. Найти файл, который был загружен и затем выберите **откройте**.

    Если вы выбрали другое расположение, файл будет находиться в папке загрузок.

1. Если у вас есть правильный файл (номер версии может отличаться от), выберите **Далее**:

    > [!div class="mx-imgBorder"]
    > ![После выбора пакета, выберите страницу пакета решения](media/northwind-install/confirm-solution2.png)

1. В **информацию о решениях** выберите **Далее** если верны имя решения и издателем.

    > [!div class="mx-imgBorder"]
    > ![Страница сведений о решении](media/northwind-install/confirm-publisher.png)

1. В **параметры импорта** выберите **импорта** для подтверждения обработку сообщения SDK, что для примера требуется:

    > [!div class="mx-imgBorder"]
    > ![Страница "Параметры импорта"](media/northwind-install/confirm-sdk.png)

    Другая страница появляется и отображает ход выполнения, как решение устанавливается на следующих нескольких минут:

    > [!div class="mx-imgBorder"]
    > ![Индикатор хода выполнения](media/northwind-install/solution-progress.png)

    По завершении установки исходной странице показан результат:

    > [!div class="mx-imgBorder"]
    > ![Страница "Импорт решения"](media/northwind-install/solution-success.png)

1. Нажмите **Закрыть**.

## <a name="load-the-sample-data"></a>Загрузка образца данных

1. Выберите **приложений**, а затем выберите **образец данных "Борей"** .

    Подождите несколько минут, если приложения Northwind не отображаются сразу после установки решения:

    > [!div class="mx-imgBorder"]
    > ![База данных "Борей" в списке приложений](media/northwind-install/sample-data-app.png)

1. Когда приложение запрашивает разрешения на взаимодействие с Common Data Service, выберите **Разрешить**:

    > [!div class="mx-imgBorder"]
    > ![Диалоговое окно согласия для Common Data Service](media/northwind-install/sample-data-permission.png)

1. Приложение загружает и показывает, что пример сущности содержат ни одной записи, установите флажок **загрузки данных** для заполнения сущностей:

    > [!div class="mx-imgBorder"]
    > ![Кнопка загрузки данных в диспетчере образца данных](media/northwind-install/sample-data-load.png)

    Как приложение загружает данные, в верхней части приложения и записей увеличения числа март точек.

    Сущности загружаются в определенном порядке, чтобы связи могут устанавливаться между записями. К примеру **Order Details** сущность имеет отношение "многие к одному" с **заказы** и **заказ продуктов** сущности, которые загружались первыми.

    Процесс можно отменить в любое время, выбрав **отменить**, и данные можно удалить в любое время, щелкнув **удаления данных**:

    > [!div class="mx-imgBorder"]
    > ![Пример диспетчера данных, что при загрузке данных](media/northwind-install/sample-data-progress.png)

    После загрузки последней строки данных (**много, чтобы много связей**) показывает **сделать**и **загрузки данных** и **удаления данных** снова становятся активными кнопки:

    > [!div class="mx-imgBorder"]
    > ![Образец Data Manager после загрузки данных](media/northwind-install/sample-data-complete.png)

## <a name="sample-apps"></a>Примеры приложений

Решение "Борей" включает в себя эти приложения для взаимодействия с этими данными:

- Northwind Orders (холст)
- Northwind Orders (на основе модели)

Откройте эти приложения так же, что вы открыли приложение в предыдущей процедуре.

### <a name="canvas"></a>На основе холста

Это приложение с одним экраном предлагает простое представление «основной-подробности» **заказы** сущности, где можно просмотреть и изменить сводка заказа и каждой позиции строки заказа. Отображается список заказов у левого края, а стрелка можно выбрать в этом списке, чтобы отобразить сводные данные и подробности данного заказа. Дополнительные сведения: [Обзор приложения canvas для Northwind Traders](northwind-orders-canvas-overview.md).

> [!div class="mx-imgBorder"]
> ![Список заказов и сведений о в Northwind приложения на основе холста](media/northwind-install/orders-canvas.png)

Сведения о заказе отображается на отдельной форме:

> [!div class="mx-imgBorder"]
> ![сведения о заказе в приложении на основе модели](media/northwind-install/orders-model-2.png)

Если вы прокрутите страницу вниз, как приложение на основе холста отобразится те же элементы строки:

> [!div class="mx-imgBorder"]
