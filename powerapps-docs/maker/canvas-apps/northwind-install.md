# <a name="install-northwind-traders-database-and-apps"></a>Установка базы данных и приложений Northwind Traders

Выполнив действия, описанные в [этой серии разделов](northwind-orders-canvas-part1.md), можно найти основные понятия о реляционных данных, реализованные в образце базы данных в Common Data Service. Вы также можете исследовать примеры бизнес-приложений, как на холста, так и на основе моделей, для управления этими данными и получения практических впечатления от создания такого приложения. В этом первом разделе объясняется, как установить базу данных Northwind Traders в своей среде и получить доступ к примерам приложений, которые можно открыть для редактирования, чтобы увидеть, как они были созданы.

Northwind Trader — это вымышленная организация, которая управляет заказами, продуктами, клиентами, поставщиками и многими другими аспектами малого бизнеса. Этот пример появился в первой версии Microsoft Access и по-прежнему доступен в качестве шаблона доступа.

## <a name="prerequisites"></a>Технические условия

- Лицензия PowerApps, которая поддерживает Common Data Service. [Бесплатную пробную лицензию можно использовать в](../signup-for-powerapps.md) течение 30 дней.
- Среда с Common Data Serviceной базой данных. Вы можете [создать такую среду](https://docs.microsoft.com/power-platform/admin/create-environment) , если у вас есть соответствующие разрешения.

## <a name="download-the-solution"></a>Скачивание решения

> [!div class="nextstepaction"]
> [Скачайте файл решения Northwind Traders](https://pwrappssamples.blob.core.windows.net/samples/NorthwindTraders_1_0_0_6.zip)

## <a name="install-the-solution"></a>Установка решения

1. Войдите в [PowerApps](https://web.powerapps.com?utm_source=padocs&utm_medium=linkinadoc&utm_campaign=referralsfromdoc)и убедитесь, что вы работаете в среде, содержащей базу данных Common Data Service.

1. В левой области навигации выберите **решения**, а затем щелкните **Импорт** на панели инструментов в верхней части экрана:

    > [!div class="mx-imgBorder"]
    > ![Solutions Просмотр и импорт-решение точка входа @ no__t-1

1. На странице **Выбор пакета решения** нажмите кнопку **Обзор**.

    > [!div class="mx-imgBorder"]
    > Страница пакета решения ![Select до выбора пакета @ no__t-1

1. Найдите скачанный файл и нажмите кнопку **Открыть**.

    Если вы не выбрали другое расположение, файл будет находиться в папке загрузки.

1. Если у вас есть правильный файл (номер версии может отличаться), нажмите кнопку **Далее**.

    > [!div class="mx-imgBorder"]
    > Страница пакета решения ![Select после выбора пакета @ no__t-1

1. На странице **сведения о решении** выберите **Далее** , если имя решения и издатель указаны правильно.

    > [!div class="mx-imgBorder"]
    > Страница сведений о @no__t 0Solution @ no__t-1

1. На странице **Параметры импорта** выберите **Импорт** , чтобы подтвердить обработку сообщения SDK, для которого требуется образец:

    > [!div class="mx-imgBorder"]
    > Страница параметров @no__t 0Import @ no__t-1

    Появится другая страница с ходом выполнения установки решения в течение следующих нескольких минут:

    > [!div class="mx-imgBorder"]
    > @no__t 0progress Bar @ no__t-1

    По завершении установки на исходной странице отобразится результат:

    > [!div class="mx-imgBorder"]
    > страница решения @no__t 0Importing @ no__t-1

1. Нажмите **Закрыть**.

## <a name="load-the-sample-data"></a>Загрузка образца данных

1. Выберите **приложения**, а затем выберите **учебные данные Northwind**.

    Подождите несколько минут, если приложения Northwind не будут отображаться сразу после установки решения:

    > [!div class="mx-imgBorder"]
    > @no__t база данных 0Northwind в списке приложений @ no__t-1

1. Когда приложение запрашивает разрешение на взаимодействие с Common Data Service, выберите **Разрешить**:

    > [!div class="mx-imgBorder"]
    > диалоговое окно «![Consent» для Common Data Service @ no__t-1

1. После того как приложение загрузится и покажет, что образцы сущностей не содержат записей, выберите **загрузить данные** для заполнения сущностей.

    > [!div class="mx-imgBorder"]
    > Кнопка данных ![Load в образце Диспетчер данных @ no__t-1

    По мере того, как приложение загрузит данные, в течение марта в верхней части приложения и число записей увеличится.

    Сущности загружаются в определенном порядке, что позволяет устанавливать связи между записями. Например, сущность **Order Details** имеет связь «многие к одному» с сущностями **Orders** и **Order Products** , которые загружаются первыми.

    Вы можете отменить процесс в любое время, нажав **кнопку Отмена**, и вы можете удалить данные в любое время, выбрав **удалить данные**:

    > [!div class="mx-imgBorder"]
    > ![Sample Диспетчер данных по мере загрузки данных @ no__t-1

    Когда данные загружаются **, выводятся**последняя строка (**многие ко многим**), а кнопки **загрузить данные** и **удалить данные** снова включены.

    > [!div class="mx-imgBorder"]
    > ![Sample Диспетчер данных после загрузки данных @ no__t-1

- Заказы "Борей" (холст)

Вы открываете эти приложения так же, как вы открыли приложение в предыдущей процедуре.

Это одноэкранное приложение предлагает простое представление "основной/подробности" сущности " **заказы** ", где можно просматривать и редактировать сводку заказа и каждого элемента строки для заказа. Рядом с левой границей отображается список заказов, и можно выбрать стрелку в этом списке, чтобы отобразить сводку и подробные сведения об этом заказе. Дополнительные сведения: [Общие сведения о приложении Canvas для Northwind Traders](northwind-orders-canvas-overview.md).

## <a name="do-it-yourself"></a>Самостоятельное

Вы можете выполнить пошаговые инструкции, чтобы создать приложение Canvas, показанное ранее в этом разделе.  Инструкции делятся на три части:

1. [Создайте коллекцию заказов](northwind-orders-canvas-part1.md).
1. [Создайте сводную форму](northwind-orders-canvas-part2.md).
1. [Создание коллекции сведений](northwind-orders-canvas-part3.md).

Если вы хотите пропустить, решение содержит приложение начальной точки для каждой части.  В списке приложений найдите **заказы Northwind (Canvas), начиная с первой части** и т. д.

В этом [обзоре приложения](northwind-orders-canvas-overview.md) объясняется пользовательский интерфейс, источники данных и принципы использования связей.
