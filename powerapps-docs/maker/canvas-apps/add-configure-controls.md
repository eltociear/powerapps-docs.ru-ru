# <a name="add-and-configure-a-canvas-app-control-in-powerapps"></a>Добавление и настройка элемента управления приложения на основе холста в PowerApps

Вы можете добавлять в приложение на основе холста различные элементы пользовательского интерфейса и настраивать их внешний вид и поведение непосредственно на панели инструментов, вкладке **Свойства** или в строке формул. Эти элементы пользовательского интерфейса называются элементами управления, а настраиваемые параметры — свойствами.

## <a name="prerequisites"></a>Технические условия

1. Если у вас еще нет лицензии на Power Apps, [Зарегистрируйтесь](../signup-for-powerapps.md), а затем выполните [Вход](https://make.powerapps.com?utm_source=padocs&utm_medium=linkinadoc&utm_campaign=referralsfromdoc).
1. В разделе **Создание собственного приложения**наведите указатель мыши на **приложение Canvas с пустого**поля, а затем выберите **сделать это приложение**.
1. Если появится запрос на вводный обзор, нажмите кнопку **Далее** , чтобы ознакомиться с основными областями интерфейса Power Apps (или выберите **пропустить**).

    Вы всегда можете просмотреть этот курс позже, щелкнув значок вопросительного знака в правом верхнем углу экрана, а затем выбрав пункт взять вводный **Обзор**.

## <a name="add-and-select-a-control"></a>Добавление и выбор элемента управления

На вкладке **Вставка** выполните одно из следующих действий.

- Выберите **метку** или **кнопку** , чтобы добавить один из этих типов элементов управления.
- Выберите категорию элементов управления, а затем выберите тип элемента управления, который требуется добавить.

Например, выберите **создать экран**и нажмите кнопку **пусто** , чтобы добавить в приложение пустой экран. (Экраны — это тип элемента управления, который может содержать другие типы элементов управления.)

![Добавить экран](./media/add-configure-controls/add-screen.png)

Новый экран называется **Screen2** и отображается в левой области навигации. На этой панели показан иерархический список элементов управления в приложении, что позволяет легко находить и выбирать каждый элемент управления.

![Screen2 в списке](./media/add-configure-controls/list-screen2.png)

Чтобы продемонстрировать работу этого списка, выберите **Метка** на вкладке **Вставка** . Новый элемент управления появится в разделе **Screen2** в иерархическом списке.

![Screen2 в списке](./media/add-configure-controls/add-label.png)

На экране в поле с шестью маркерами по умолчанию окружается метка. Этот тип прямоугольника окружает любой элемент управления. Если вы выберете экран, щелкнув его (но вне метки), поле исчезнет из метки. Чтобы снова выбрать метку, щелкните ее или коснитесь мышью или коснитесь ее имени в иерархическом списке элементов управления.

> [!IMPORTANT]
> Необходимо всегда выбирать элемент управления, прежде чем его можно будет настроить.

## <a name="rename-a-control"></a>Переименование элемента управления

В иерархическом списке элементов управления наведите указатель мыши на элемент управления, который требуется переименовать, нажмите появившуюся кнопку, а затем выберите **Переименовать**. Затем можно ввести уникальное, запоминающее имя, чтобы упростить создание приложения.

![Переименование элемента управления](./media/add-configure-controls/rename-control.png)

## <a name="delete-a-control"></a>Удаление элемента управления

В иерархическом списке элементов управления наведите указатель мыши на элемент управления, который необходимо удалить, нажмите появившуюся кнопку, а затем выберите **Удалить**. Чтобы удалить элемент управления, который не является экраном, можно также выбрать элемент управления на холсте и нажать клавишу DELETE.

![Удалить элемент управления](./media/add-configure-controls/delete-control.png)

## <a name="reorder-screens"></a>Переупорядочить экраны

В иерархическом списке элементов управления наведите указатель мыши на экран, который требуется переместить вверх или вниз, нажмите появившуюся кнопку, а затем выберите **вверх или вниз** **.**

![Переупорядочить экран](./media/add-configure-controls/reorder-screen.png)

> [!NOTE]
> Когда приложение открывается, экран в верхней части иерархического списка элементов управления обычно отображается первым. Но можно указать другой экран, задав для свойства **[OnStart](controls/control-screen.md)** формулу, включающую функцию **[navigate](functions/function-navigate.md)** .

## <a name="move-and-resize-a-control"></a>Перемещение элемента управления и изменение его размера

Чтобы переместить элемент управления, выделите его, наведите указатель мыши на его центр, чтобы появилась четырехконечная стрелка, а затем перетащите элемент управления в другое место.

![Перемещение элемента управления](./media/add-configure-controls/move-control.png)

Чтобы изменить размер элемента управления, выделите его, наведите указатель мыши на любой маркер в поле выбора, чтобы появилась двунаправленная стрелка, а затем перетащите маркер.

![Перемещение элемента управления](./media/add-configure-controls/resize-control.png)

> [!NOTE]
> Как описано в этом разделе ниже, можно также перемещать элемент управления и изменять его размер, изменяя любое сочетание свойств **[X](controls/properties-size-location.md)** , **[Y](controls/properties-size-location.md)** , **[высоты](controls/properties-size-location.md)** и **[ширины](controls/properties-size-location.md)** в строке формул.

## <a name="change-the-text-of-a-label-or-a-button"></a>Изменение текста метки или кнопки

Выберите метку или кнопку, дважды щелкните текст, отображаемый в элементе управления, и введите нужный текст.

![Изменить текст](./media/add-configure-controls/change-text.png)

> [!NOTE]
> Как описано в этом разделе ниже, этот текст также можно изменить, изменив его свойство **[Text](controls/properties-core.md)** в строке формул.

## <a name="configure-a-control-from-the-toolbar"></a>Настройка элемента управления на панели инструментов

При настройке элемента управления на панели инструментов вы можете указать большее количество параметров по сравнению с его непосредственной настройкой.

Например, можно выбрать метку, перейти на вкладку **Главная** , а затем изменить шрифт текста в метке.

![Изменить шрифт](./media/add-configure-controls/change-font.png)

## <a name="configure-a-control-from-the-properties-tab"></a>Настройка элемента управления на вкладке "Свойства"

С помощью вкладки **Свойства** можно указать более широкий набор параметров, чем можно, настроив элемент управления с панели инструментов.

Например, можно выбрать элемент управления, а затем показать или скрыть его, изменив его свойство **Visible** .

![Задать видимость](./media/add-configure-controls/set-visibility.png)

## <a name="configure-a-control-in-the-formula-bar"></a>Настройка элемента управления в строке формул

Вместо непосредственной настройки элемента управления на панели инструментов или на вкладке **Свойства** можно настроить элемент управления, выбрав свойство в списке свойств, а затем указав значение в строке формул. Свойства сортируются по алфавиту. Используя этот подход, вы можете указать больше типов значений.

Например, можно выбрать метку, а затем настроить ее следующим образом:

- Переместите его, выбрав **X** или **Y** в списке свойства, а затем указав другое число в строке формул.

    ![Задать свойство X](./media/add-configure-controls/x-property.png)

- Измените размер, выбрав **высоту** или **ширину** в списке свойств, а затем указав другое число в строке формул.

    ![Задание свойства высоты](./media/add-configure-controls/height-property.png)

- Измените текст, выбрав **текст** в списке свойств, а затем указав любое сочетание литеральной строки, выражения или формулы в строке формул.

    - Литеральная строка заключена в кавычки и отображается в точности так, как если бы она была введена. **"Hello, World"** — строка литерала.

        ![Задайте для свойства Text литеральную строку](./media/add-configure-controls/literal-string.png)

    - Выражение не включает функцию и часто основано на свойстве другого элемента управления. **Screen1. Height** — это выражение, показывающее высоту **Screen1**.

        ![Задание для свойства Text значения выражения](./media/add-configure-controls/expression.png)

    - Формула включает одну или несколько функций. Функция **Now** возвращает текущую дату и время в местном часовом поясе, а **текстовая** функция форматирует такие значения, как даты, время и валюта.

        ![Задание для свойства Text значения формулы](./media/add-configure-controls/formula.png)

        Формулы обычно гораздо сложнее, чем этот пример, чтобы они могли обновлять данные, сортировать их, фильтровать и выполнять другие операции. Дополнительные сведения см. в [справочнике по формулам](formula-reference.md).
