# <a name="add-and-configure-a-canvas-app-control-in-powerapps"></a>Добавление и настройка элемента управления приложения на основе холста в PowerApps

Вы можете добавлять в приложение на основе холста различные элементы пользовательского интерфейса и настраивать их внешний вид и поведение непосредственно на панели инструментов, вкладке **Свойства** или в строке формул. Эти элементы пользовательского интерфейса называются элементами управления, а настраиваемые параметры — свойствами.

## <a name="prerequisites"></a>Технические условия

1. Если у вас нет лицензии PowerApps, [зарегистрироваться](../signup-for-powerapps.md), а затем [вход](https://web.powerapps.com?utm_source=padocs&utm_medium=linkinadoc&utm_campaign=referralsfromdoc).
1. В разделе **создавайте собственное приложение**, наведите указатель мыши **приложения на основе холста с нуля**, а затем выберите **сделать это приложение**.
1. Если будет предложено ознакомьтесь с кратким обзором, выберите **Далее** ознакомиться с основных областей интерфейса PowerApps (или выберите **Skip**).

    Можно всегда сделать это позже, щелкнув значок вопросительного знака в правом верхнем углу экрана, а затем выбрав **ознакомьтесь с кратким обзором**.

## <a name="add-and-select-a-control"></a>Добавить и выберите элемент управления

На **вставить** вкладке, выполните одно из следующих действий:

- Выберите **метка** или **кнопку** для добавления одного из этих типов элементов управления.
- Выберите категорию элементов управления и затем выберите тип элемента управления, который вы хотите добавить.

К примеру, выберите **новый экран**, а затем выберите **пустой** для добавления пустой экран в приложение. (Экраны — это тип элемента управления, который может содержать другие типы элементов управления.)

![Добавление экрана](./media/add-configure-controls/add-screen.png)

Новый экран называется **Screen2** и отображается в левой области навигации. Эта панель показывает иерархический список элементов управления в приложение, вы можете легко найти и выделить каждый элемент управления.

![Screen2 в списке](./media/add-configure-controls/list-screen2.png)

Чтобы продемонстрировать, как работает этот список, выберите **метка** на **вставить** вкладки. Новый элемент управления отображается в разделе **Screen2** в иерархическом списке.

![Screen2 в списке](./media/add-configure-controls/add-label.png)

На экране коробки шесть дескрипторы окружает метка по умолчанию. Такого рода поле окружает выбран любой элемент управления. Если вы выберите экран, щелкнув или коснувшись его (но за пределами метки), поле исчезнет из метки. Выбрать метку, щелкните или коснитесь его, или щелкните или коснитесь **Label2** в иерархическом списке элементов управления.

> [!IMPORTANT]
> Всегда необходимо выбрать элемент управления, прежде чем его можно настроить.

## <a name="rename-a-control"></a>Переименование элемента управления

В иерархическом списке элементов управления, при наведении указателя мыши на элементе управления, который требуется переименовать, выберите кнопку с многоточием, которая отображается, а затем выберите **Переименовать**. Можно, введите уникальное и запоминающееся имя для создание приложения проще.

![Переименование элемента управления](./media/add-configure-controls/rename-control.png)

## <a name="delete-a-control"></a>Удалить элемент управления

В иерархическом списке элементов управления, наведите курсор на элемент управления, который требуется удалить, выберите кнопку с многоточием, которая отображается, а затем выберите **удалить**. Чтобы удалить элемент управления, который не экран, можно также выбрать элемент управления на холсте и нажмите клавишу Delete.

![Удалить элемент управления](./media/add-configure-controls/delete-control.png)

## <a name="reorder-screens"></a>Изменить порядок экранов

В иерархический список элементов управления, наведите указатель мыши на экран, на котором вы хотите переместить вверх или вниз, выберите кнопку с многоточием, который отображается, а затем выберите **вверх** или **вниз**.

![Изменение порядка экране](./media/add-configure-controls/reorder-screen.png)

> [!NOTE]
> При открытии приложения экрана в верхней части иерархического списка элементов управления обычно отображаются первыми. Но можно указать другой экран, задав **[OnStart](controls/control-screen.md)** формулу, которая включает **[Navigate](functions/function-navigate.md)** функции.

## <a name="move-and-resize-a-control"></a>Перемещение и изменение размеров элемента управления

Чтобы переместить элемент управления, выберите его, наведите указатель мыши ее центра, четырехстороннюю стрелку и перетащите элемент управления в другое место.

![Переместить элемент управления](./media/add-configure-controls/move-control.png)

Чтобы изменить размер элемента управления, выберите его, при наведении поверх любого дескриптора в поле выбора, двунаправленную стрелку и перетащите маркер.

![Переместить элемент управления](./media/add-configure-controls/resize-control.png)

> [!NOTE]
> Как описано далее в этом разделе, можно также перемещать и изменение размеров элемента управления, изменив любое сочетание его  **[X](controls/properties-size-location.md)**,  **[Y](controls/properties-size-location.md)**,  **[Высота](controls/properties-size-location.md)**, и **[ширины](controls/properties-size-location.md)** свойства в строке формул.

## <a name="change-the-text-of-a-label-or-a-button"></a>Изменение текста метки или кнопки

Выберите метку или кнопку, дважды щелкните текст, отображаемый в элементе управления и введите текст, который вы хотите.

![Изменить текст](./media/add-configure-controls/change-text.png)

> [!NOTE]
> Как описано далее в этом разделе, можно также изменить этот текст, изменив его **[текст](controls/properties-core.md)** свойства в строке формул.

## <a name="configure-a-control-from-the-toolbar"></a>Настройка элемента управления на панели инструментов

При настройке элемента управления на панели инструментов вы можете указать большее количество параметров по сравнению с его непосредственной настройкой.

Например, можно выбрать метку, выберите **Главная** вкладку, а затем изменить шрифт текста в метке.

![Изменить шрифт](./media/add-configure-controls/change-font.png)

## <a name="configure-a-control-from-the-properties-tab"></a>Настройка элемента управления на вкладке "Свойства"

С помощью **свойства** вкладке можно указать большее количество параметров, чем при настройке элемента управления панели инструментов.

Например, можно выбрать элемент управления и затем отобразить или скрыть его, изменив его **Visible** свойство.

![Настройка видимости](./media/add-configure-controls/set-visibility.png)

## <a name="configure-a-control-in-the-formula-bar"></a>Настройка элемента управления в строке формул

Вместо того чтобы настраивать элемент управления напрямую, на панели инструментов или в **свойства** вкладке вы можете настроить элемент управления, выбрав свойство в списке свойств и указав значение в строке формул. Свойства сортируются по алфавиту. Используя этот подход, вы можете указать больше типов значений.

Например можно выбрать метку и настройте его следующим образом:

- Переместить его, выбрав **X** или **Y** в списке свойств, а затем указать другой номер в строке формул.

    ![Значение X-свойство](./media/add-configure-controls/x-property.png)

- Измените его размер, выбрав **высота** или **ширины** в списке свойств, а затем указать другой номер в строке формул.

    ![Установите свойство Height](./media/add-configure-controls/height-property.png)

- Измените его текст, выбрав **текст** в списке свойств, а затем указать любое сочетание строковый литерал, выражение или формулу в строке формул.

    - Строковый литерал, заключенных в кавычки и отображается так, как вы его вводите. **«Hello, world»** является строковым литералом.

        ![Значение свойства Text строковый литерал](./media/add-configure-controls/literal-string.png)

    - Выражение не включает функцию и часто зависит от свойства другого элемента управления. **Screen1.Height** выражение, которое высота **Screen1**.

        ![Задать выражение свойства Text](./media/add-configure-controls/expression.png)

    - Формула содержит одну или несколько функций. **Теперь** функция возвращает текущую дату и время в локальном часовом поясе и **текст** функция форматирует значения, такие как даты, времени и валюты.

        ![Значение свойства Text формулу](./media/add-configure-controls/formula.png)

        Формулы, обычно гораздо сложнее, чем в этом примере, чтобы их можно обновлять данные, сортируют их, отфильтровать их и выполнить другие операции. Дополнительные сведения см. в разделе [справочнике по формулам](formula-reference.md).
