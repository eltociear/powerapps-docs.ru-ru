# <a name="set-up-and-learn-about-the-meeting-capture-sample-template-in-power-apps"></a>Настройка и изучение шаблона примера записи собраний в Power Apps

## <a name="overview"></a>Обзор

 В Power Apps шаблон образца записи собраний — это средство, предназначенное для сбора информации о собраниях по мере их возникновения. Запустив приложение, можно просмотреть сведения о встрече, заметки, фотографировать интерактивные доски, рисовать и назначать задачи. Также можно экспортировать все это, отправлять всем участникам собрания и быстро и легко планировать следующие встречи.

[![Meeting Capture](media/sample-meeting-capture/MeetingCapture.png)](https://aka.ms/previewmeetingcapture)

## <a name="where-can-i-run-the-app"></a>Где можно запускать приложение?

Этот пример приложения можно запускать в браузере на планшете или другом устройстве, имеющем аналогичный форм-фактор.
