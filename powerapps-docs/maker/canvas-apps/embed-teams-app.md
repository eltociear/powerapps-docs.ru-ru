# <a name="embed-a-powerapps-app-in-teams"></a>Внедрение приложения PowerApps в команды 

Вы можете поделиться созданной вами PowerApps, внедрив ее непосредственно в Microsoft Teams. По завершении пользователи могут выбрать **+** , чтобы добавить приложение в любой канал **команды или** беседы в группе, в которой вы являетесь. Приложение отображается в виде плитки в разделе **вкладок команды**. 

Администратор может отправить приложение, чтобы оно выводилось для **всех** команд в вашем клиенте в **разделе все вкладки**. См. раздел [совместное использование приложения в Microsoft Teams](https://docs.microsoft.com/en-us/power-platform/admin/embed-app-teams).

> [!NOTE]
> Политики настраиваемых приложений команды должны быть настроены, чтобы разрешить отправку пользовательских приложений. Если не удается внедрить приложение в группы, обратитесь к администратору, чтобы узнать, не настроены ли [Параметры настраиваемого приложения](https://docs.microsoft.com/MicrosoftTeams/teams-custom-app-policies-and-settings#custom-app-policy-and-settings). 

## <a name="prerequisites"></a>Технические условия

- [Наличие лицензии PowerApps](https://docs.microsoft.com/power-platform/admin/pricing-billing-skus)
- Создано приложение Canvas

## <a name="locate-your-powerapps-guid"></a>Определение идентификатора GUID PowerApp

Найдите и запишите идентификатор GUID PowerApp, который будет использоваться на следующем шаге.

1. Войдите в [https://web.powerapps.com](https://web.powerapps.com), а затем выберите **приложения** в меню.

   > [!div class="mx-imgBorder"] 
   > ![Отобразить список приложений](./media/embed-teams-app/file-apps2.png "Открытие списка приложений")

2. Выберите **другие команды** (...) для приложения, которое вы хотите использовать в командах, а затем щелкните **сведения**.

   > [!div class="mx-imgBorder"] 
   > ![Сведения о приложении](./media/embed-teams-app/app-details.png "Сведения о приложении")


3. Запишите **идентификатор приложения** для последующего использования.

   > [!div class="mx-imgBorder"] 
   > ![Сведения о приложении](./media/embed-teams-app/app-details2.png "Сведения о приложении")

## <a name="install-app-studio"></a>Установка App Studio

Вы можете пропустить эти шаги, если приложение App Studio уже установлено. 

1. В меню команды выберите **приложения** в левом нижнем углу окна группы (![значок приложения](./media/embed-teams-app/apps-icon.png "Значок "приложения"")).

2. Выполните поиск по запросу "App Studio" в поле поиска, а затем выберите его.

   > [!div class="mx-imgBorder"] 
   > ![App Studio](./media/embed-teams-app/store-app-studio.png "App Studio")

3. Нажмите кнопку **установить**. 

   > [!div class="mx-imgBorder"] 
   > ![Установка App Studio](./media/embed-teams-app/install-app-studio.png "Установка App Studio")

4. Выберите **Открыть** для функции приложения.

   > [!div class="mx-imgBorder"] 
   > ![Открыть App Studio](./media/embed-teams-app/open-app-studio.png "Открыть App Studio")

## <a name="create-a-teams-app-for-your-powerapp"></a>Создание приложения группы для PowerApp

1. В командах Откройте App Studio.

   > [!div class="mx-imgBorder"] 
   > ![Открыть App Studio](./media/embed-teams-app/open-app-studio2.png "Открыть App Studio")

2. Перейдите на вкладку **редактор манифестов** и выберите **создать новое приложение** в разделе Добро пожаловать.

   > [!div class="mx-imgBorder"] 
   > ![Создать новое приложение](./media/embed-teams-app/create-new-app.png "Создание приложения")

3. Заполните сведения о приложении на странице **сведений о приложении** .  Для идентификатора GUID приложения следует использовать записанный ранее идентификатор GUID идентификатора приложения PowerApp.  Это позволит избежать дублирования приложений групп для определенного PowerApp.
 
   > [!div class="mx-imgBorder"] 
   > ![Заполнение сведений](./media/embed-teams-app/fill-in-info-about-app.png "Заполнение сведений")

   |Поля  |Description  |
   |---------|---------|
   |**Имена приложений** |    |
   |Короткое имя     | Обязательно. Краткое отображаемое имя приложения. ограничение в 30 символов.        |
   |Длинное имя     | Полное имя приложения, используемое, если полное имя приложения превышает 30 символов.       | 
   |**Идентификатор**     |         |
   |Идентификатор приложения     | Обязательно. Уникальный идентификатор, созданный корпорацией Майкрософт для этого приложения.        |
   |Имя пакета     | Обязательно. Уникальный идентификатор для этого приложения. Рекомендуется использовать обратную нотацию домена. Например, com. example. <AppName>.       |
   |Версия     | Обязательно. Версия конкретного приложения. Если вы обновляете что-либо в манифесте, необходимо также увеличить версию.     |
   |**Описания**    |     |
   | Краткое описание    | Обязательно. Краткое описание возможностей приложения, используемое при ограниченном пространстве. ограничение в 80 символов.   |
   | Длинное описание    | Обязательно. Полное описание приложения.     |
   | **Сведения для разработчиков**    |     |
   | Имя    | Обязательно. Отображаемое имя компании или разработчика.     |
   | Веб    | Обязательно. URL-адрес https://веб-сайта для вашего приложения через powerapps.com. Когда пользователь устанавливает приложение, появится страница "о приложении". Он должен ссылаться на веб-версию приложения на powerapps.com.   |
   | **URL-адреса приложений**    | Эти ссылки будут отображаться на странице **About** вместе с URL-адресом веб-сайта.     |
   | Заявление о конфиденциальности    | Обязательно. URL-адрес https://политики конфиденциальности разработчика. [Пример](https://go.microsoft.com/fwlink/p/?LinkID=698505).   |
   | Условия использования    | Обязательно. URL-адрес https://для условий использования разработчика.  [Пример](https://go.microsoft.com/fwlink/p/?LinkID=698507).  |
   | **Фирменной символики**    |     |
   | Полный цвет    | Относительный путь к файлу для полного цветового значка 192x192 PNG.    |
   | Прозрачный контур    |Относительный путь к файлу прозрачного значка в формате 32x32 PNG.     |
   | Контрастный цвет    | Цвет, используемый в сочетании с и в качестве фона для значков структуры.     |

Дополнительные сведения см. в разделе [редактор манифестов](https://docs.microsoft.com/microsoftteams/platform/get-started/get-started-app-studio#manifest-editor) и [схема манифеста](https://docs.microsoft.com/microsoftteams/platform/resources/schema/manifest-schema).

4. Прокрутите вниз до раздела фирменной символики и добавьте свои логотипы и желаемый цвет для вашего приложения.  Это логотипы, которые будут отображаться для вашего приложения в командах. 

   > [!div class="mx-imgBorder"] 
   > ![Фирменная символика и вкладки](./media/embed-teams-app/branding-tabs.png "Фирменная символика и вкладки")

5. В разделе **возможности**выберите **вкладки**.

6. На **вкладке Команда** выберите **Добавить**.

   > [!div class="mx-imgBorder"] 
   > ![Добавление вкладки команды](./media/embed-teams-app/team-tab-add.png "Добавление вкладки команды")

7. Добавьте URL-адрес конфигурации приложения в поле ввода "URL-адрес конфигурации", используя следующий формат: `https://apps.powerapps.com/teams/settings/<PowerApp ID>`

   Замените `<PowerApp ID>` ИДЕНТИФИКАТОРом идентификатора приложения, записанным выше.

   Выберите [область](https://docs.microsoft.com/microsoftteams/platform/concepts/tabs/tabs-overview#tab-scope) , в которой будет отображаться приложение. Убедитесь, что установлен флажок **может обновить конфигурацию** , а затем нажмите кнопку **сохранить**.

   > [!div class="mx-imgBorder"] 
   > ![URL-адрес конфигурации](./media/embed-teams-app/configuration-url.png "URL-адрес конфигурации")

8. В разделе **Готово**выберите **Допустимые домены**. Добавьте **Apps.powerapps.com** и **Apps.Preview.powerapps.com** в качестве допустимых доменов для командного приложения.

   > [!div class="mx-imgBorder"] 
   > ![Добавить допустимые домены](./media/embed-teams-app/add-valid-domains.png "Добавить допустимые домены")

9. В разделе **Готово**выберите **тест и распространить**. В разделе **Установка**выберите **установить**.

   > [!div class="mx-imgBorder"] 
   > ![Выбор установки](./media/embed-teams-app/test-distribute-app.png "Выбор установки")

10. Выберите группу, в которой нужно установить приложение, а затем нажмите кнопку **установить**.

    > [!div class="mx-imgBorder"] 
    > ![Добавить в командную установку](./media/embed-teams-app/new-app-add-to-team.png "Добавить в командную установку")
11. Если вы хотите добавить экземпляр этого приложения в канал сразу же, выберите канал, в котором вы хотите использовать приложение, и нажмите кнопку **настроить**.

    > [!div class="mx-imgBorder"] 
    > ![Выберите настроить](./media/embed-teams-app/app-now-available.png "Выберите настроить")

12. Щелкните **Сохранить**.

## <a name="add-the-app-as-a-tab"></a>Добавить приложение как вкладку

Чтобы добавить приложение в качестве вкладки в любой канал или беседу, выберите **+** , а затем в разделе **вкладки команды** выберите свое приложение. 

> [!div class="mx-imgBorder"] 
> ![Добавить вкладку "приложение как"](./media/embed-teams-app/add-app-as-tab.png "Добавить вкладку "приложение как"")

Теперь приложение отображается как вкладка.

> [!div class="mx-imgBorder"] 
> ![Приложение как вкладку](./media/embed-teams-app/app-as-tab.png "Приложение как вкладку")
