# <a name="solution-publisher-overview"></a>Обзор издателя решения

Каждое создаваемое вами приложения или настройка является частью решения. Каждое решение имеет издателя. Вы указываете издателя, когда создаете решение. 

> [!div class="mx-imgBorder"] 
> <img src="media/solution-publisher-select.png" alt="Select solution publisher" height="731" width="416">

Издатель решения указывает, кто разработал приложение. Префикс может предоставить быстрый способ понять, какое решение добавило элемент. По этим причинам вам следует создать издателя решения и указать значимый префикс. Это особенно важно при просмотре элементов метаданных, включенных из импортированных решений. Например, решение, которое содержит пример приложения для сбора средств, использует *sample* в качестве префикса издателя. 

> [!div class="mx-imgBorder"] 
> ![Префикс издателя решения сбора средств](media/fundraiser-sample-app-prefix.png)

> [!NOTE]
> При изменении префикса издателя решения это необходимо сделать перед созданием каких-либо новых приложений или элементов метаданных. Имена элементов метаданных невозможно изменить. 

## <a name="common-data-services-default-solution"></a>Решение по умолчанию Common Data Service
Решение по умолчанию в Power Apps —это решение по умолчанию Common Data Service, которое связано с издателем по умолчанию Common Data Service. Префикс издателя по умолчанию будет случайно назначен для этого издателя, например это может быть *cr8a3*. Это означает, что имя каждого нового элемента метаданных, созданных в решении по умолчанию, будет иметь этот префикс рядом с именами, используемыми для уникального определения элементов. При создании новой сущности с именем *Animal* уникальное имя, используемое Common Data Service, будет *cr8a3_animal*. То же самое верно для всех новых полей (атрибутов), отношений или параметров набора параметров. Если вы будете настраивать решение по умолчанию, рассмотрите возможность изменения префикса издателя. 

## <a name="create-a-solution-publisher"></a>Создание издателя решения
1.  На портале Power Apps выберите значок **Параметры** (шестеренка), затем выберите **Дополнительные параметры**. 
2.  Выберите **Параметры** > **Настройки** > **Издатели**. 
3.  На панели команд **Главное представление издателя** выберите **Создать**, 
4.  Введите необходимые сведения: 
   - **Отображаемое имя**. Введите отображаемое имя для издателя. 
   - **Имя**. Введите уникальное имя для издателя. 
   - **Префикс**. Введите требуемый префикс издателя. 
   -    **Префикс значения параметра**. Это поле генерирует число на основе префикса издателя. Это число используется при добавлении параметров в наборы параметров и указывает, какое именно решение использовалось для добавления параметра. 
   - **Сведения о контакте**. При желании вы можете добавить контактную и адресную информацию.
5. Выберите **Сохранить и закрыть**.

## <a name="change-a-solution-publisher"></a>Изменение издателя решения
Вы можете изменить издателя решения для неуправляемого решения, выполнив следующие действия:
1.  На портале Power Apps выберите **Решения**, выберите **...** рядом с требуемым решением, затем выберите **Параметры**. 
2.  На панели **Настройки решения** выберите **Изменить издателя**. 
3.  Измените поля **Отображаемое имя** и **Префикс** на требуемые значения. Поле **Префикс значения параметра** генерирует число на основе префикса издателя. Это число используется при добавлении параметров в наборы параметров и указывает, какое именно решение использовалось для добавления параметра. 
4.  В дополнение к префиксу вы также можете изменить отображаемое имя издателя решения, контактную информацию и адрес в разделе **Сведения о контакте**. 
5.  Выберите **Сохранить и закрыть**.
