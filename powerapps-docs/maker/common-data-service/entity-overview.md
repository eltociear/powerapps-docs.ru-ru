# <a name="entity-overview"></a>Обзор сущностей

Сущности используются для моделирования бизнес-данных и управление ими. При разработке приложения можно использовать стандартные сущности, настраиваемые сущности или оба варианта. Common Data Service предоставляет стандартные сущности по умолчанию. Они разработаны в соответствии с наилучшими практиками для реализации самых распространенных концепций и сценариев в организации.
