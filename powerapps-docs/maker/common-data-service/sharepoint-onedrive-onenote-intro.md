# <a name="sharepoint-onenote-and-onedrive-integration-with-common-data-service"></a>Интеграция SharePoint, OneNote и OneDrive с Common Data Service

Common Data Service обеспечивает поддержку интеграции SharePoint, OneDrive и OneNote. Для интеграции с этими сервисами требуется сначала включить интеграцию SharePoint.   


|Сервис Office 365  |Описание  | Дополнительные сведения  |
|---------|---------|---------|---------|
|SharePoint     |  Позволяет пользователям управлять распространенными типами документов, такими как Word, Excel, PowerPoint, OneNote, и создавать папки для сохранения и для управления этими документами, которые удобно сохраняются в SharePoint из приложений Common Data Service.     |  [Управление документами с помощью SharePoint](/dynamics365/customer-engagement/admin/manage-documents-using-sharepoint) <br /> <br /> [Настройка интеграции SharePoint](/dynamics365/customer-engagement/admin/set-up-sharepoint-integration)     |
|OneDrive для бизнеса     |  Пользователи приложения могут создавать и управлять личными документами, доступ к которым можно получить из приложений Common Data Service.      |  [Включить OneDrive для бизнеса](/dynamics365/customer-engagement/admin/enable-onedrive-for-business)     |
|OneNote     | Пользователи приложения могут использоваться OneNote для создания и просмотра заметок из записей Common Data Service.      |   [Настройка интеграции OneNote](/dynamics365/customer-engagement/admin/set-up-onenote-integration-in-dynamics-365)   |
