# <a name="edit-a-canvas-app-embedded-on-a-model-driven-form"></a>Редактирование приложения на основе холста, внедренного в управляемую моделью форму
В этом разделе рассматривается порядок редактирования приложения на основе холста, внедренного в управляемую моделью форму.

## <a name="edit-the-canvas-app-directly"></a>Изменение приложения на основе холста непосредственно
Можно изменить приложение на основе холста, внедренное в управляемую моделью форму, как и любое другое приложение на основе холста. Шаги по редактированию приложения на основе холста см. в разделе: [Редактирование приложения на основе холста](../canvas-apps/edit-app.md)

## <a name="edit-the-canvas-app-via-the-host-model-driven-form"></a>Изменение приложения на основе холста через управляемую моделью хост-форму
Другой вариант — изменить внедренное приложение на основе холста через управляемую моделью хост-форму.

Представьте, что требуется изменить приложение на основе холста, внедренное на форму с именем "Основная форма организации" для сущности "Организация". Для этого выполните следующие шаги. 

1.  Выполните вход в [Power Apps](https://make.powerapps.com/?utm_source=padocs&utm_medium=linkinadoc&utm_campaign=referralsfromdoc).
2.  [Измените форму](create-and-edit-forms.md) с именем "Основная форма организации" для сущности "Организация". 
3.  На панели команд выберите **Перейти в классический режим**, чтобы открыть форму в классическом конструкторе формы.
4.  В классическом конструкторе форм выберите поле, которое настроено для отображения внедренного приложения на основе холста.
5.  При выбранном поле на вкладке **Домашняя страница** в группе **Правка** выберите **Изменить свойства**.
6.  В диалоговом окне **Свойства поля** выберите вкладку **Элементы управления**.
7.  В диалоговом окне **Свойства поля** в списке элементов управления выберите **Приложение на основе холста**.
8.  В разделе под списком элементов управления, выберите **Настроить** для редактирования приложения на основе холста. Откроется приложение на основе холста для редактирования, в Power Apps Studio, на новой вкладке.
       > [!NOTE]
       > Если открытие Power Apps Studio заблокировано из-за блокировки всплывающих окон веб-браузера, необходимо вручную включить сайт make.powerapps.com или временно отключить средство блокирования всплывающих окон и затем снова выбрать **Настройка**.
9. После завершения внесения изменений выберите вкладку **Файл**, затем выберите **Сохранить**.
10. Чтобы сделать изменения доступными для конечных пользователей, выберите **Опубликовать**, затем выберите **Опубликовать эту версию**.
