# <a name="customize-dynamics-365-app-for-outlook"></a>Настройка Dynamics 365 App for Outlook

Оптимально настройте Dynamics 365 App for Outlook в соответствии с потребностями организации. 

Подробные сведения о том, как настраивать App for Outlook, см. в статье [Настройка App for Outlook](https://docs.microsoft.com/dynamics365/customer-engagement/outlook-app/customizing-the-app)

> [!NOTE]
> Перед тем как можно будет настраивать App for Outlook, необходимо установить решение, затем развернуть его в вашей среде. 

1. Чтобы установить решение, перейдите в раздел [Установка Microsoft Dynamics 365 App for Outlook](https://appsource.microsoft.com/product/dynamics-365/mscrm.fa50aa98-e8bb-4757-83ce-6d607959b985?tab=Overview), затем выберите **ПОЛУЧИТЬ** и выполните шаги для выбора среду, в которую требуется выполнить установку.

   > [!div class="mx-imgBorder"]
   > ![Установка приложения Outlook](media/appsource.png "Установка приложения Outlook")
   
2. Для развертывания приложения см. раздел [Развертывание Dynamics 365 App for Outlook](https://docs.microsoft.com/dynamics365/customer-engagement/outlook-app/deploy-dynamics-365-app-for-outlook).
