# <a name="create-or-edit-model-driven-app-dashboards"></a>Создание или изменение панелей мониторинга управляемых моделью приложений

Существует два основных типа панелей мониторинга: системные панели мониторинга и пользовательские панели мониторинга. Пользователь приложения может создать панель мониторинга, видимую только ему в областях приложения, на доступ к которым у него есть привилегии. Системные панели мониторинга создаются и настраиваются администратором или настройщиком. После публикации они доступны всем пользователям приложения. Пользователь может сделать свою пользовательскую панель мониторинга панелью по умолчанию и переопределить системную панель мониторинга.   

Панели мониторинга могут быть стандартными или интерактивными. Стандартные панели мониторинга поддерживают добавление одного или нескольких не связанных компонентов, таких как диаграммы или списки. Интерактивные панели мониторинга предоставляют пользователям возможность работать с определенной записью непосредственно из панели мониторинга. В этом разделе рассматриваются стандартные системные панели мониторинга. Дополнительные сведения об интерактивные панели мониторинга, см. в разделе [Настройка панелей мониторинга интерактивного взаимодействия для приложения на основе модели](configure-interactive-experience-dashboards.md).
  
<a name="BKMK_createdashboard"></a>   
## <a name="create-a-new-standard-dashboard"></a>Создание новой стандартной панели мониторинга  
  
1.  Выполните вход в [Power Apps](https://make.powerapps.com/?utm_source=padocs&utm_medium=linkinadoc&utm_campaign=referralsfromdoc).
  
2. Выберите **Решения**, а затем откройте нужное решение.

3. На панели инструментов выберите **Создать**, выберите **Панель мониторинга**, а затем выберите макет с 2, 3 или 4 столбцами.  
  
4.  На странице **Панель мониторинга: создать** введите имя для панели мониторинга.  
  
5.  Выберите одну из областей компонентов, затем выберите значок диаграммы или списка.  
  
     Панель мониторинга может содержать до 6 компонентов.  
  
6.  Например, чтобы добавить диаграмму, выберите значок диаграммы на плитке холста панели мониторинга, где вы хотите, чтобы диаграмма отображалась. Затем, в диалоговом окне **Добавить компонент** выберите значения в полях **Тип записей**, **Представление** и **Диаграмма**, а затем нажмите **Добавить**, чтобы добавить диаграмму на панель мониторинга. Дополнительные сведения о том, как создать диаграмму, см. [Создание системной диаграммы приложения на основе модели](create-edit-system-chart.md).
  
7.  Закончив добавлять компоненты на панель мониторинга, выберите **Сохранить** и **Закрыть**.  

8. На панели инструментов решения выберите **Опубликовать**. 
  
<a name="BKMK_editdashboard"></a>   
## <a name="edit-an-existing-dashboard"></a>Изменение существующей панели мониторинга  
  
1. Выполните вход в [Power Apps](https://make.powerapps.com/?utm_source=padocs&utm_medium=linkinadoc&utm_campaign=referralsfromdoc).

2. Выберите **Решения**, а затем откройте нужное решение.  

3. В списке компонентов решенияоткройте панель мониторинга, выберите одну из областей компонентов, затем на панели инструментов выберите **Изменить компонент**.  
  
4.  В диалоговом окне **Задать свойства** можно внести изменения в диаграмму или список, например изменить сущность, представление по умолчанию, добавить селектор диаграммы или сделать панель мониторинга доступной в мобильных приложениях. По завершении нажмите кнопку **OK**.  
  
     Для получения дополнительных сведений о свойствах компонентов панелей мониторинга см. раздел [Установка свойств диаграммы или списка, включенных на панель мониторинга](set-properties-chart-list-included-dashboard.md).  
  
5.  Когда вы закончите свои изменения на панели инструментов, выберите **Сохранить**, а затем выберите **Закрыть**. 

6. На панели инструментов решения выберите **Опубликовать**.  
  
Дополнительные задачи системных панелей мониторинга:  
  
-   Удаление списка или диаграммы из панели мониторинга  

-   Добавление диаграммы или списка на панель мониторинга  

-   Назначение панели мониторинга по умолчанию  

-   Использование ролей безопасности для отображения панелей мониторинга только для некоторых ролей.
