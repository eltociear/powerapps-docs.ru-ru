# <a name="create-or-edit-filters-in-model-driven-app-views"></a>Создание или изменение фильтров в представлениях управляемого моделью приложения

<a name="BKMK_CreateOrEditViewFilters"></a>   

Фильтры в представлении Power Apps важны для значения, предоставляемого представлением. Применяемые фильтры определяют, как записи отображаются в списке по умолчанию. Вы можете добавить или изменить фильтр для столбцов, которые вы включаете в представление, выбрав столбец и выбрав **Фильтровать по**. Вы также можете использовать построитель выражений в конструкторе представлений. Используйте построитель выражений для добавления или редактирования фильтров для любых полей сущности в текущем представлении или любых полей в связанной сущности. 

В данной теме вы создадите или измените фильтры, выполнив следующие задачи:

-   [Изменение или удаление условия фильтра](create-edit-view-filters.md#edit-or-remove-a-filter-condition)

-   [Открытие построителя выражений](create-edit-view-filters.md#open-the-expression-builder)

-   [Добавление условий в фильтр](create-edit-view-filters.md#add-conditions-to-a-filter)

-   [Добавление условия группы в фильтр](create-edit-view-filters.md#add-a-group-condition-to-a-filter)

-   [Добавление связанной сущности в условие](create-edit-view-filters.md#add-a-related-entity-to-a-condition)

-   [Групповые условия фильтра](create-edit-view-filters.md#group-conditions-of-a-filter)

## <a name="edit-or-remove-a-filter-condition"></a>Изменение или удаление условия фильтра

1. Выполните вход в [Power Apps](https://make.powerapps.com/?utm_source=padocs&utm_medium=linkinadoc&utm_campaign=referralsfromdoc).  

2. Разверните **Данные**, выберите **Сущности**, выберите требуемую сущность, затем выберите вкладку **Представления**.

3. Выберите представление, чтобы открыть его. Панель свойств представления отображает существующие фильтры.

    > [!div class="mx-imgBorder"] 
    > ![Фильтры на панели представления](media/views-panel-filters.png "Фильтры на панели представления")

4. На панели свойств представления выберите условие фильтра.

    > [!div class="mx-imgBorder"] 
    > ![Изменить фильтры](media/edit-filter-viewpanel.png "Изменить фильтры")

5. Выберите условный оператор, который требуется использовать.

6. Введите или выберите значение сравнения для условия.

7. Выберите **Применить**.

8. Чтобы удалить условие, выберите **Закрыть**. Условие удаляется без запроса подтверждения.

### <a name="open-the-expression-builder"></a>Открытие построителя выражений

- На панели свойств представления выберите **Изменить фильтры**.

    > [!div class="mx-imgBorder"] 
    > ![Построитель выражений](media/edit-create-filters.png "Построитель выражений")

### <a name="add-conditions-to-a-filter"></a>Добавление условий в фильтр

1. В построителе выражений выберите **Добавить** > **Добавить строку**.

2. Выберите поле для этого условия.

3. Выберите условный оператор.

4. Выберите значения сравнения.  

    Некоторые условия фильтра не требуют значения сравнения для условия. Например, оператор **Содержит данные** не требует значения сравнения. С другими условиями фильтра вы выбираете значение сравнения из набора параметров. Например, поле **Состояние** имеет набор параметров, который содержит значения **Активно** и **Неактивно**.

    > [!div class="mx-imgBorder"] 
    > ![Условие фильтра](media/add-condition-filter.png "Условие фильтра")

5. Нажмите **ОК**.

### <a name="add-a-group-condition-to-a-filter"></a>Добавление условия группы в фильтр

1. В построителе выражений выберите **Добавить** > **Добавить группу**.

2. Выберите оператор отношения **Или** для группы. **И** — это оператор отношения по умолчанию.

3. Укажите первое предложение сгруппированного условия. Выберите поле, условный оператор и значение сравнения.

4. Выберите **Добавить** > **Добавить группу**

5. Укажите второе предложение сгруппированного условия.

    > [!div class="mx-imgBorder"] 
    > ![Сгруппированное условие фильтра](media/add-group-filter.png "Сгруппированное условие фильтра")

    Вы можете выбрать **Свернуть**, чтобы отобразить группу как условное выражение.

### <a name="add-a-related-entity-to-a-condition"></a>Добавление связанной сущности в условие

1. В построителе выражений выберите **Добавить** > **Добавить связанную сущность**.

2. Выберите поле из текущей сущности, которая связана с другой сущностью. Сущность, связанная с полем, показана в скобках. Вы можете выбрать поля, которые имеют отношение «многие к одному», «один ко многим» или «многие ко многим» со связанной сущностью.

3. Выберите поле связанной сущности для условия.

4. Выберите условный оператор.

5. Выберите или введите значение для сравнения.

    > [!div class="mx-imgBorder"] 
    > ![Фильтр связанной сущности](media/add-relatedentity-filter.png "Фильтр связанной сущности")

### <a name="group-conditions-of-a-filter"></a>Групповые условия фильтра

1. В построителе выражений установите флажок для условий, которые вы хотите сгруппировать.

2. Выберите **Дополнительные команды** (...) для одного из условий, затем выберите **Создать группу**.

3. Чтобы разгруппировать группу, выберите **Дополнительные команды** (...) для группы, затем выберите **Разгруппировать**

    > [!div class="mx-imgBorder"] 
    > ![Сгруппированное условие фильтра](media/group-conditions-filter.png "Сгруппированное условие фильтра")
