# <a name="add-move-or-delete-sections-on-a-form"></a>Добавление, перемещение или удаление разделов в форме 
[!INCLUDE [cc-beta-prerelease-disclaimer](../../includes/cc-beta-prerelease-disclaimer.md)]

Добавление, перемещение или удаление разделов в форме с помощью конструктора форм. 

> [!NOTE]
> При добавлении или перемещении разделов перетаскиванием помните, что предварительный просмотр формы реагирует и может отображать несколько столбцов вкладок друг над другом. Чтобы обеспечить, чтобы добавляемый или перемещаемый раздел находился в столбце нужной вкладки, перетаскивание привязано к другому разделу, который уже находится в столбце этой вкладки.

## <a name="add-sections-to-a-form"></a>Добавление разделов в форму
Для добавления разделов в форму используйте область **Макеты**. 

> [!div class="mx-imgBorder"] 
> ![](media/layouts-pane.png "Область макетов")

  > [!NOTE]
  >   Разделы можно добавить только в основные формы и экспресс-формы. Дополнительные сведения: [Типы форм](types-forms.md)

### <a name="add-sections-to-a-form-using-drag-and-drop"></a>Добавление разделов в форму с помощью перетаскивания

1. Откройте конструктор форм для создания или редактирования формы. Дополнительные сведения: [Создание формы](create-and-edit-forms.md#create-a-form) или [Редактирование формы](create-and-edit-forms.md#edit-a-form)
2. На панели команд выберите **Добавить элемент управления** или в левой панели выберите **Макеты**. 
3. В области **Макеты** выберите элемент управления раздела и перетащите его на предварительный просмотр формы. При перетаскивании раздела на предварительный просмотр формы отображаются цели перетаскивания, куда можно добавить раздел. 
4. Перетащите раздел в требуемое местоположение. Обратите внимание на следующее: 
    - Разделы можно перетаскивать в положение до или после любого существующего раздела.
    - Разделы можно также перетаскивать в пустую область на вкладке. В этом случает раздел будет добавлен в доступное пространство, чтобы равномерно распределить разделы между столбцами вкладки.
    - При наведении указателя мыши на заголовок вкладки во время перетаскивания раздела изменяется текущая выбранная вкладка, позволяя добавить раздел на другую вкладку.   
5. Повторите шаги 3–4 выше, если требуется добавить несколько разделов.
6. На панели команд выберите **Сохранить** для сохранения формы или выберите **Опубликовать**, если требуется сохранить изменения и сделать их видимыми пользователям. 

### <a name="add-sections-to-a-form-using-selection"></a>Добавление разделов в форму с помощью выбора 

1. Откройте конструктор форм для создания или редактирования формы. Дополнительные сведения: [Создание формы](create-and-edit-forms.md#create-a-form) или [Редактирование формы](create-and-edit-forms.md#edit-a-form)
2. В режиме предварительного просмотра формы выберите другой раздел или вкладку. Обратите внимание на следующее:
    - При выборе существующего раздела новый раздел добавляется после существующего раздела. 
    - При выборе вкладки новый раздел добавляется в доступное пространство, чтобы равномерно распределить разделы по столбцам вкладки. 
3. На панели команд выберите **Добавить элемент управления** или в левой панели выберите **Макеты**.  
4. В области **Планировки** выберите элемент управления раздела для добавления в форму. Можно также выбрать **...** рядом с требуемым элементом управления раздела, затем выберите **Добавить в выбранную вкладку**. 
5. Повторите шаги 2–4 выше, если требуется добавить несколько разделов.
6. На панели команд выберите **Сохранить** для сохранения формы или выберите **Опубликовать**, если требуется сохранить изменения и сделать их видимыми пользователям. 

## <a name="move-sections-on-a-form"></a>Перемещение разделов в форме

### <a name="move-sections-on-a-form-using-drag-and-drop"></a>Перемещение разделов в форме с помощью перетаскивания

1. Откройте конструктор форм для создания или редактирования формы. Дополнительные сведения: [Создание формы](create-and-edit-forms.md#create-a-form) или [Редактирование формы](create-and-edit-forms.md#edit-a-form)
2. В режиме предварительного просмотра формы выберите подпись раздела или пустое место в разделе, который требуется переместить, и начните перетаскивание. При перетаскивании раздела на предварительный просмотр формы отображаются цели перетаскивания, куда можно переместить раздел. 
3. Перетащите раздел в требуемое местоположение. Обратите внимание на следующее: 
    - Разделы можно перетаскивать в положение до или после любого существующего раздела.
    - Разделы можно также перетаскивать в пустую область на вкладке. В этом случает раздел будет добавлен в доступное пространство, чтобы равномерно распределить разделы между столбцами вкладки.
    - При наведении указателя мыши на заголовок вкладки во время перетаскивания раздела изменяется текущая выбранная вкладка, позволяя добавить раздел на другую вкладку.   
4. Повторите шаги 2–3 выше, если требуется переместить несколько разделов.
5. На панели команд выберите **Сохранить** для сохранения формы или выберите **Опубликовать**, если требуется сохранить изменения и сделать их видимыми пользователям. 

### <a name="move-sections-on-a-form-using-cut-and-paste"></a>Перемещение разделов в форме с помощью вырезания и вставки

1. Откройте конструктор форм для создания или редактирования формы. Дополнительные сведения: [Создание формы](create-and-edit-forms.md#create-a-form) или [Редактирование формы](create-and-edit-forms.md#edit-a-form)
2. В режиме предварительного просмотра формы выберите раздел, который требуется переместить.
3. На панели команд выберите **Вырезать**.
4. В режиме предварительного просмотра формы выберите существующий раздел или вкладку. Вы также можете переключиться на другую вкладку при необходимости.
5. На панели команд выберите **Вставить** или выберите шеврон, затем выберите **Вставить перед**. Обратите внимание на следующее: 
    - Если вы выбираете **Вставить**, перемещаемый раздел вставляется после существующего раздела. 
    - Если вы выбираете **Вставить перед**, перемещаемый раздел вставляется перед существующим разделом.
    - При выборе вкладки перемещаемый раздел добавляется в доступное пространство, чтобы равномерно распределить разделы по столбцам вкладки. Действие **Вставить перед** не применимо и поэтому не доступно в этом случае.
6. Повторите шаги 2–5 выше, если требуется переместить несколько разделов.
7. На панели команд выберите **Сохранить** для сохранения формы или выберите **Опубликовать**, если требуется сохранить изменения и сделать их видимыми пользователям. 

## <a name="delete-sections-on-a-form"></a>Удаление разделов в форме
1. Откройте конструктор форм для создания или редактирования формы. Дополнительные сведения: [Создание формы](create-and-edit-forms.md#create-a-form) или [Редактирование формы](create-and-edit-forms.md#edit-a-form)
2. В режиме предварительного просмотра формы выберите раздел, который нужно удалить из формы. 
3. На панели команд выберите **Удалить**.
4. Повторите шаги 2–3 выше, если требуется удалить несколько разделов.
4. На панели команд выберите **Сохранить** для сохранения формы или выберите **Опубликовать**, если требуется сохранить изменения и сделать их видимыми пользователям. 

    > [!NOTE]
    >   - Разделы можно удалять только из основные формы и экспресс-форм. Дополнительные сведения: [Типы форм](types-forms.md)
    >   - Если раздел удален по ошибке, на панели команд выберите **Отменить** для возврата формы в предыдущее состояние. 
    >   - Невозможно удалить раздел, содержащий обязательное или заблокированное поле. 
    >   - Невозможно удалить заблокированный раздел. 
    >   - Вкладка должна иметь минимум один раздел в каждом столбце вкладки. При удалении последнего раздела из столбца вкладки новый раздел будет автоматически добавлен.
