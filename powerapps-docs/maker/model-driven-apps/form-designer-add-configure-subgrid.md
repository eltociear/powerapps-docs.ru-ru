# <a name="add-and-configure-a-sub-grid-component-on-a-form"></a>Добавление или настройка компонента вложенной сетки в форме  
Форма в которой отображаются сведения записи может использовать компонент вложенной сетки для отображения списка связанных или самостоятельных записей в формате таблиц. Создатели может добавить и настроить компонент вложенной сетки с использованием конструктора формы.

## <a name="add-a-sub-grid-component"></a>Добавление компонента вложенной сетки
Добавление компонента вложенной сетки так же, как вы добавляете любой другой компонент. Дополнительные сведения: [Добавление, настройка, перемещение или удаление компонентов в форме](add-move-configure-or-delete-components-on-form.md)

## <a name="configure-a-sub-grid-component"></a>Настройка компонента вложенной сетки
Это свойства, доступные для настройки при использовании компонента вложенной сетки в форме с помощью конструктора форм.


|Диаграмма с областями   |Имя  |Описание  |
|---------|---------|---------|
| **Параметры отображения** | **Подпись** | Локализуемая подпись для вложенной сетки, видимая для пользователей. <br /><br />Это обязательное свойство.|
| **Параметры отображения** |  **Название** |  Уникальное имя вложенной сетки, используемое при ссылке на нее в скриптах. Оно может содержать только алфавитно-цифровые знаки и символы подчеркивания. <br /><br />Это обязательное свойство. |
| **Параметры отображения** | **Скрыть на телефоне** |  Вложенная решетка может быть скрыта для отображения сжатой версии формы на экранах телефонов. |
| **Параметры отображения** | **Показать связанные записи** |  Если выбран этот параметр, во вложенной сетке отображаются только записи, связанные с текущей записью, отображаемой в форме. <br /><br />Раскрывающийся список **Сущность** также фильтруется для отображения только тех сущностей, которые связаны с текущей сущностью. |
| **Параметры отображения** | **Сущность** |  Сущность, записи которой вы хотите отобразить во вложенной сетке. <br /><br />При выборе **Показать связанные записи** список сущностей фильтруется для отображения только тех сущностей, которые связаны с текущей сущностью. В дополнение к имени сущности, имя поля поиска также отображается в скобках. |
| **Параметры отображения** | **Представление по умолчанию** |  Представление сущности, выбранной в свойстве **Сущность**, которое будет использоваться для получения и отображения списка записей во вложенной сетке. |
| **Параметры отображения** | **Позволять пользователям изменить представление** |  При выборе этого параметра пользователи приложения могут перейти из представления **Представление по умолчанию** в другое представление сущности, выбранной в свойстве **Сущность**. |
| **Параметры отображения** | **Показать все представления** |  При выборе этого параметра пользователи приложения могут перейти из представления **Представление по умолчанию** ко всем другим представлениям сущности, выбранной в свойстве **Сущность**. <br /><br />Данное свойство доступно только при выборе **Разрешить пользователям изменять представление**. |
| **Параметры отображения** | **Выбранные представления** |  Список представлений сущности, выбранной в свойстве **Сущность**, которую пользователи приложения могут изменить из **Представление по умолчанию**. <br /><br />Данное свойство доступно только при выборе **Разрешить пользователям изменять представление** и **Показать все представления** снято. |
