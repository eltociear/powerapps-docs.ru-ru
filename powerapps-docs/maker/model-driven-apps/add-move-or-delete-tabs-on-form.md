# <a name="add-move-or-delete-tabs-on-a-form"></a>Добавление, перемещение или удаление вкладок в форме  
[!INCLUDE [cc-beta-prerelease-disclaimer](../../includes/cc-beta-prerelease-disclaimer.md)]

Добавление, перемещение или удаление вкладок в форме с помощью конструктора форм.

## <a name="add-tabs-to-a-form"></a>Добавление вкладок в форму
Для добавления вкладок в форму используйте область **Макеты**.  

> [!div class="mx-imgBorder"] 
> ![](media/layouts-pane.png "Область макетов")
   
  > [!NOTE]
  >  Вкладки можно добавить лишь на основные формы. Дополнительные сведения: [Типы форм](types-forms.md)

### <a name="add-tabs-to-a-form-using-drag-and-drop"></a>Добавление вкладок в форму с помощью перетаскивания

1. Откройте конструктор форм для создания или редактирования формы. Дополнительные сведения: [Создание формы](create-and-edit-forms.md#create-a-form) или [Редактирование формы](create-and-edit-forms.md#edit-a-form)
2. На панели команд выберите **Добавить элемент управления** или в левой панели выберите **Макеты**. 
3. В области **Макеты** выберите элемент управления вкладки и перетащите его на предварительный просмотр формы. При перетаскивании вкладки на предварительный просмотр формы отображаются цели перетаскивания, куда можно добавить вкладку. 
4. Перетащите вкладку в требуемое местоположение. Обратите внимание на следующее: 
    - Вкладки можно перетаскивать в положение до или после любых существующих вкладок, наводя указатель мыши на заголовки вкладок.
    - Вкладки также можно перетащить в положение до или после текущей вкладки, наведя указатель мыши на левый или правый край текущей вкладки.
5. Повторите шаги 3–4 выше, если требуется добавить несколько вкладок.
6. На панели команд выберите **Сохранить** для сохранения формы или выберите **Опубликовать**, если требуется сохранить изменения и сделать их видимыми пользователям. 

### <a name="add-tabs-to-a-form-using-selection"></a>Добавление вкладок в форму с помощью выбора 

1. Откройте конструктор форм для создания или редактирования формы. Дополнительные сведения: [Создание формы](create-and-edit-forms.md#create-a-form) или [Редактирование формы](create-and-edit-forms.md#edit-a-form)
2. В режиме предварительного просмотра формы выберите другую существующую вкладку или форму. Обратите внимание на следующее:
    - Если вы выберите существующую вкладку, новая вкладка будет добавлена после существующей вкладки. 
    - Если вы выберите форму, новая вкладка будет добавлена как последняя вкладка формы. 
3. На панели команд выберите **Добавить элемент управления** или в левой панели выберите **Макеты**.  
4. В области **Планировки** выберите элемент управления вкладки для добавления в форму. Можно также выбрать **...** рядом с элементом управления вкладки, затем выбрать **Добавить в форму**. 
5. Повторите шаги 2–4 выше, если требуется добавить несколько вкладок.
6. На панели команд выберите **Сохранить** для сохранения формы или выберите **Опубликовать**, если требуется сохранить изменения и сделать их видимыми пользователям. 

## <a name="move-tabs-on-a-form"></a>Перемещение вкладок в форме

### <a name="move-tabs-on-a-form-using-drag-and-drop"></a>Перемещение вкладок в форме с помощью перетаскивания

1. Откройте конструктор форм для создания или редактирования формы. Дополнительные сведения: [Создание формы](create-and-edit-forms.md#create-a-form) или [Редактирование формы](create-and-edit-forms.md#edit-a-form)
2. В режиме предварительного просмотра формы заголовок вкладки, которую требуется переместить, и начинайте действие перетаскивания. При перетаскивании вкладки на предварительный просмотр формы отображаются цели перетаскивания, куда можно переместить вкладку.  
3. Перетащите вкладку в требуемое местоположение. Обратите внимание на следующее:
    - Вкладки можно перетаскивать в положение до или после любых существующих вкладок, наводя указатель мыши на заголовки вкладок.
    - Вкладки также можно перетащить в положение до или после текущей вкладки, наведя указатель мыши на левый или правый край текущей вкладки.
4. Повторите шаги 2–3 выше, если требуется переместить несколько вкладок.
5. На панели команд выберите **Сохранить** для сохранения формы или выберите **Опубликовать**, если требуется сохранить изменения и сделать их видимыми пользователям. 

### <a name="move-tabs-on-a-form-using-cut-and-paste"></a>Перемещение вкладок в форме с помощью вырезания и вставки

1. Откройте конструктор форм для создания или редактирования формы. Дополнительные сведения: [Создание формы](create-and-edit-forms.md#create-a-form) или [Редактирование формы](create-and-edit-forms.md#edit-a-form)
2. В режиме предварительного просмотра формы выберите вкладку, которую требуется переместить.
3. На панели команд выберите **Вырезать**.
4. В режиме предварительного просмотра формы выберите другую существующую вкладку или форму.
5. На панели команд выберите **Вставить** или выберите шеврон, затем выберите **Вставить перед**. Обратите внимание на следующее: 
    - Если вы выбираете **Вставить**, перемещаемая вкладка вставляется после существующей вкладки. 
    - Если вы выбираете **Вставить перед**, перемещаемая вкладка вставляется перед существующей вкладкой.
    - Если вы выберите форму, перемещаемая вкладка будет добавлена как последняя вкладка формы. Действие **Вставить перед** не применимо и поэтому не доступно в этом случае.
6. Повторите шаги 2–5 выше, если требуется переместить несколько вкладок.
7. На панели команд выберите **Сохранить** для сохранения формы или выберите **Опубликовать**, если требуется сохранить изменения и сделать их видимыми пользователям. 

## <a name="delete-tabs-on-a-form"></a>Удаление вкладок с формы
1. Откройте конструктор форм для создания или редактирования формы. Дополнительные сведения: [Создание формы](create-and-edit-forms.md#create-a-form) или [Редактирование формы](create-and-edit-forms.md#edit-a-form)
2. В режиме предварительного просмотра формы выберите вкладку, которую нужно удалить из формы. 
3. На панели команд выберите **Удалить**.
4. Повторите шаги 2–3 выше, если требуется удалить несколько вкладок.
4. На панели команд выберите **Сохранить** для сохранения формы или выберите **Опубликовать**, если требуется сохранить изменения и сделать их видимыми пользователям. 

    > [!NOTE]
    >   - Вкладки можно удалять лишь с основных форм. Дополнительные сведения: [Типы форм](types-forms.md)
    >   - Если вкладка удалена по ошибке, на панели команд выберите **Отменить** для возврата формы в предыдущее состояние. 
    >   - Невозможно удалить вкладку, содержащую разделы с обязательными или заблокированными полями. 
    >   - Невозможно удалить вкладку с заблокированными разделами. 
    >   - Форма должна содержать по крайней мере одну вкладку. Нельзя удалить последнюю вкладку формы.
