# <a name="unified-interface-playbook"></a>Сборник схем единого интерфейса

Этот сборник схем предназначен для помощи клиентам, партнерам и выездным ролям Майкрософт планировать и выполнять переход со старого веб-клиента на единый интерфейс.

В целом документы сборника схем единого интерфейса помогут вам:
- Понять единый интерфейс. 
- Понять процесс перехода.
- Разработать стратегию для обсуждения и перехода.
- Обеспечить плавный переход.
- Эффективно управлять запросами и потенциальными возражениями.
- Найти ресурсы, которое поддерживают процесс.
