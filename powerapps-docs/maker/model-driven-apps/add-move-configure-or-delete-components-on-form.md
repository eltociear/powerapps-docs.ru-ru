# <a name="add-configure-move-or-delete-components-on-a-form"></a>Добавление, настройка, перемещение или удаление компонентов в форме  
Используя конструктор формы, разработчики могут легко добавить и настроить популярные компоненты, такие как [вложенная сетка](form-designer-add-configure-subgrid.md), [экспресс-форма](form-designer-add-configure-quickview.md), дуговая ручка, линейный ползунок, и т.д.

## <a name="add-components-to-a-form"></a>Добавление компонентов в форму
Для добавления компонентов в форму используйте область **Компоненты**. Область **Компоненты** также позволяет выполнять поиск для быстрого поиска компонентов.  

> [!div class="mx-imgBorder"] 
> ![](media/FormDesignerComponentsPane.PNG "Components pane")

### <a name="add-components-to-a-form-using-drag-and-drop"></a>Добавление компонентов в форму с помощью перетаскивания
> [!NOTE]
> При добавлении или перемещении компонентов перетаскиванием помните, что предварительный просмотр формы реагирует и может отображать несколько столбцов разделов друг над другом. Чтобы обеспечить, чтобы добавляемое или перемещаемое компонент находилось в столбце нужного раздела, перетаскивание привязано к другому полю или компоненту, которое уже находится в столбце этого раздела.
1. Откройте конструктор форм для создания или редактирования формы. Дополнительные сведения: [Создание формы](create-and-edit-forms.md#create-a-form) или [Редактирование формы](create-and-edit-forms.md#edit-a-form)
2. На панели команд выберите **Добавьте компонент**, в левой области выберите **Компоненты**, чтобы просмотреть список доступных компонентов. Вы также можете навести курсор на компонент в списке, чтобы увидеть изображение предварительного просмотра, описание и другие сведения об этом компоненте.
3. Для области **Компоненты**, найдите или прокрутите список, чтобы найти компонент, который вы хотите добавить, и затем выберите его.
4. Перетащите компонента в предварительный просмотр формы. При перетаскивании компонента на предварительный просмотр формы отображаются цели перетаскивания, куда можно добавить компонент. 

   Обратите внимание на следующее: 
    - Компоненты можно перетаскивать в положение до или после любого существующего компонента или поля.
    - Компоненты можно также перетащить в пустую область в разделе. В этом случае компонент будет добавлено в доступное пространство, чтобы равномерно распределить поля и компоненты по столбцам раздела.
    - При наведении указателя мыши на заголовок вкладки во время перетаскивания компонента изменяется текущая выбранная вкладка, позволяя добавить компонент на другую вкладку.
    - Когда вы перетаскиваете компонент, в большинстве случаев вы увидите диалоговое окно для настройки свойств компонента. Убедитесь, что вы настроили все обязательные свойства компонента. 
5. В диалоговом окне для настройки свойств компонента в разделе **Показать компонент**, **Интернет**, **Мобильное устройство** и **Планшет** параметры выбираются по умолчанию, чтобы обеспечить использование компонента при отображении формы в Интернете, мобильном приложении и приложении для планшета. В зависимости от ваших требованиях можно отменить некоторые из этих параметров, чтобы ограничить использование компонента. Выберите **Готово**.
6. Повторите шаги 3-5 выше для добавления других компонентов.
7. На панели команд выберите **Сохранить** для сохранения формы или выберите **Опубликовать**, если требуется сохранить изменения и сделать их видимыми пользователям. 

### <a name="add-components-for-a-field-on-the-form"></a>Добавление компонентов для поля в форме
1. Откройте конструктор форм для создания или редактирования формы. Дополнительные сведения: [Создание формы](create-and-edit-forms.md#create-a-form) или [Редактирование формы](create-and-edit-forms.md#edit-a-form)
2. В режиме предварительного просмотра формы выберите существующее поле.
3. В области свойств, в разделе **Компоненты** выберите , выберите **+ компонент**.
4. Диалоговое окно **Добавить компонент** отображается список компонентов доступные для текущего типа полей. Вы можете навести курсор на компонент в списке, чтобы увидеть изображение предварительного просмотра, описание и другие сведения об этом компоненте.
5. Для диалоговом окне **Добавить компонент**, найдите или прокрутите список, чтобы найти компонент, который вы хотите добавить, и затем выберите его.
   В большинстве случаев отображается диалоговое окно, позволяющее настроить свойства компонента. Убедитесь, что вы настроили все обязательные свойства компонента.
6. В диалоговом окне для настройки свойств компонента в разделе **Показать компонент**, **Интернет**, **Мобильное устройство** и **Планшет** параметры выбираются по умолчанию, чтобы обеспечить использование компонента при отображении формы в Интернете, мобильном приложении и приложении для планшета. В зависимости от ваших требованиях можно отменить некоторые из этих параметров, чтобы ограничить использование компонента. Выберите **Готово**.
7. Повторите шаги 2-6 выше если необходимо добавить дополнительные компоненты в то же или другое поле.
8. На панели команд выберите **Сохранить** для сохранения формы или выберите **Опубликовать**, если требуется сохранить изменения и сделать их видимыми пользователям.

## <a name="configure-components-on-a-form"></a>Настройка компонентов в форме
1. Откройте конструктор форм для создания или редактирования формы. Дополнительные сведения: [Создание формы](create-and-edit-forms.md#create-a-form) или [Редактирование формы](create-and-edit-forms.md#edit-a-form)
2. В режиме предварительного просмотра формы выберите существующее поле.
3. В области свойств, в разделе **Компоненты** выберите компонент, который требуется настроить.
4. Может появиться диалоговое окно для настройки свойств компонента. Измените свойства компонента по мере необходимости и выберите **Готово**.
5. Повторите шаги 2-4, чтобы настроить дополнительные компоненты в одном и том же или другом поле.
6. На панели команд выберите **Сохранить** для сохранения формы или выберите **Опубликовать**, если требуется сохранить изменения и сделать их видимыми пользователям.

## <a name="move-components-on-a-form"></a>Перемещение компонентов в форме
Можно переместить компоненты в форме или переместить или вырезать и вставить действия. 

### <a name="move-components-on-a-form-using-drag-and-drop"></a>Перемещение компонентов в форме с помощью перетаскивания
1. Откройте конструктор форм для создания или редактирования формы. Дополнительные сведения: [Создание формы](create-and-edit-forms.md#create-a-form) или [Редактирование формы](create-and-edit-forms.md#edit-a-form)
2. В режиме предварительного просмотра формы выберите компонент, который требуется переместить, и перетащите его. При перетаскивании компонента в окне предварительного просмотра формы отображаются цели перетаскивания, куда можно переместить компонент.     

   Обратите внимание на следующее: 
    - Компоненты можно перетаскивать в положение до или после любого существующего компонента или поля.
    - Компоненты можно также перетащить в пустую область в разделе. В этом случае компонент будет добавлен в доступное пространство, чтобы равномерно распределить компоненты и поля по столбцам раздела.
    - При наведении указателя мыши на заголовок вкладки во время перетаскивания компонента изменяется текущая выбранная вкладка, позволяя добавить компонент на другую вкладку.   
4. Повторите шаги 2-3 выше для перемещения других компонентов.
5. На панели команд выберите **Сохранить** для сохранения формы или выберите **Опубликовать**, если требуется сохранить изменения и сделать их видимыми пользователям. 

### <a name="move-components-on-a-form-using-cut-and-paste"></a>Перемещение компонентов в форме с помощью вырезания и вставки
1. Откройте конструктор форм для создания или редактирования формы. Дополнительные сведения: [Создание формы](create-and-edit-forms.md#create-a-form) или [Редактирование формы](create-and-edit-forms.md#edit-a-form)
2. В режиме предварительного просмотра формы выберите компонент, который требуется переместить.
3. В панели команд выберите **Вырезать**.
4. В режиме предварительного просмотра формы выберите другое существующее компонент, поле или раздел. Если требуется, можно переключиться на другую вкладку.
5. В панели команд выберите **Вставить** или выберите шеврон, затем выберите **Вставить перед**.     

   Обратите внимание на следующее:
    - Если вы выбираете **Вставить**, перемещаемое компонент вставляется после существующего компонента или поля. 
    - Если вы выбираете **Вставить перед**, перемещаемое компонент вставляется перед существующим компонентом или поля.
    - При выборе раздела перемещаемое компонент добавляется в доступное пространство, чтобы равномерно распределить компоненты и поля по столбцам раздела. Действие **Вставить перед** не применимо и поэтому не доступно в этом случае.
6. Повторите шаги 2–5 выше, если требуется переместить несколько компонентов.
7. На панели команд выберите **Сохранить** для сохранения формы или выберите **Опубликовать**, если требуется сохранить изменения и сделать их видимыми пользователям. 

## <a name="delete-components-on-a-form"></a>Удаление компонентов в форме
1. Откройте конструктор форм для создания или редактирования формы. Дополнительные сведения: [Создание формы](create-and-edit-forms.md#create-a-form) или [Редактирование формы](create-and-edit-forms.md#edit-a-form)
2. В режиме предварительного просмотра формы выберите компонент, который требуется удалить из формы, а затем на панели команд, выберите **Удалить**. 
3. Повторите шаг 2, если нужно удалить другие компоненты.
4. На панели команд выберите **Сохранить** для сохранения формы или выберите **Опубликовать**, если требуется сохранить изменения и сделать их видимыми пользователям. 

     > [!NOTE]
     >   -  Если компонент удален по ошибке, в панели команд выберите **Отменить** для возврата формы в предыдущее состояние. 
     >   -  Невозможно удалить компонент, который заблокирован или использует обязательное поле, которого больше нет в форме.
