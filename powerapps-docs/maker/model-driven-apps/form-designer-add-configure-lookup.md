# <a name="configure-a-lookup-component-on-a-form"></a>Настройка компонента поиска в форме  
Поле поиска может использоваться для ссылки на запись в другой сущности. Компонент поиска автоматически используется при добавлении поля поиска в форму. Разработчики могут настроить компонент поиска с использованием конструктора формы.

## <a name="configure-a-lookup-component"></a>Настройка компонента поиска
Это свойства, доступные для настройки при использовании компонента поиска в форме с помощью конструктора форм.


<!--from editor: "Drop-down" should only be an adjective. In the following table, is it a list? A menu? -->


|Диаграмма с областями  |Имя  |Описание  |
|---------|---------|---------|
| **Параметры отображения** | **Сущность** |  Связанная сущность, к которому подключается поле поиска. |
| **Параметры отображения** | **Представление по умолчанию** |  Представление сущности, выбранной в свойстве **Сущность**, которое будет использоваться для получения и отображения списка записей, который пользователи приложения могут выбрать в раскрывающемся списке. |
| **Параметры отображения** | **Позволять пользователям изменить представление** |  При выборе этого параметра пользователи приложения могут перейти из представления **Представление по умолчанию** в другое представление сущности, выбранной в свойстве **Сущность**. |
| **Параметры отображения** | **Показать все представления** |  При выборе этого параметра пользователи приложения могут перейти из представления **Представление по умолчанию** ко всем другим представлениям сущности, выбранной в свойстве **Сущность**. <br /><br />Данное свойство доступно только при выборе **Разрешить пользователям изменять представление**. |
| **Параметры отображения** | **Выбранные представления** |  Список представлений сущности, выбранной в свойстве **Сущность**, которую пользователи приложения могут изменить из **Представление по умолчанию**. <br /><br />Данное свойство доступно только при выборе **Разрешить пользователям изменять представление** и **Показать все представления** отменено. |
