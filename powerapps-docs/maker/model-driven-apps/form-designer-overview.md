# <a name="overview-of-the-model-driven-form-designer"></a>Обзор конструктора управляемых моделью форм
Новый конструктор управляемых моделью форм предоставляет современное WYSIWYG-взаимодействие разработки при работе с управляемыми моделью формами. Он содержит несколько усовершенствований, недоступных в классическом конструкторе управляемых моделью форм. 

В конструкторе отображается предварительный просмотр WYSIWYG в режиме реального времени (только единый интерфейс) во время разработки формы. Изменения в форме сразу же отражаются в представлении предварительного просмотра, позволяя точно увидеть, как форма будет отображаться пользователям после публикации. Всегда доступная область свойств делает общую задачу обновления свойств быстрой и простой. Обновления свойств также мгновенно отражаются в форме предварительного просмотра. Область полей с возможностями поиска и фильтрации помогает разработчикам быстро находить и добавлять поля в форму. Область компонентов позволяет легко находить и использовать компоненты и обеспечивает богатый опыт работы для конечных пользователей. Древовидное представление помогает визуализировать иерархию полей и компонентов формы и быстро находить и выбирать конкретное поле или компонент в форме.

Интерфейс конструктора форм имеет следующие области: 
- Панель команд — отображаются доступные действия, такие как **Сохранить**, **Опубликовать**, **Отменить** и **Восстановить**. 
- Панели — отображаются панели с конкретными функциями
  - Для добавления полей в форму используйте область **Поля**.
  - Для добавления компонентов в форму используйте область **Компоненты**.
  - **Представление в виде дерева** для просмотра иерархии компонентов в форме и их выбора.
- Предварительный просмотр формы — в режиме реального времени отображается предварительный просмотр формы в том виде, в котором она будет отображаться пользователям после публикации. 
- Панель свойств — отображаются свойства выбранного элемента, и также позволяет вносить изменения.
- Переключатель размера предварительного просмотра — изменяет размер предварительного просмотра формы, помогая увидеть, как форма будет выглядеть на экранах разных размеров.
- Ползунок масштабирования — позволяет увеличивать или уменьшать масштаб предварительного просмотра формы, чтобы вы могли ближе рассмотреть
- По ширине — быстрое действие, чтобы подогнать форму предварительного просмотра к доступной ширине.
