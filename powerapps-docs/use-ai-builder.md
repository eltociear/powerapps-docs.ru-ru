# <a name="use-ai-builder-in-power-apps"></a>Использование AI Builder в Power Apps

AI Builder — это новая возможность Power Platform для экспертов в бизнесе, которая позволяет легко автоматизировать процессы и прогнозировать результаты для повышения производительности бизнеса. AI Builder — это готовое решение, объединяющее возможности ИИ Microsoft с интерактивным взаимодействием, которое напрямую интегрируется в Power Apps и Power Automate. Дополнительные сведения: [Что такое AI Builder?](/ai-builder/)

Модели искусственного интеллекта можно создавать с помощью новой опции **AI Builder** в левой области навигации [make.powerapps.com](https://make.powerapps.com). Дополнительные сведения: [Типы моделей AI](/ai-builder/model-types)

![AI Builder в Power Apps](media/ai-builder.png "AI Builder в Power Apps")

Вы можете использовать модели ИИ, созданные с помощью AI Builder, в приложениях на основе холста и модели, чтобы добавить возможности ИИ в приложения. Дополнительные сведения: [Использование AI Builder в Power Apps](/ai-builder/use-in-powerapps-overview)
