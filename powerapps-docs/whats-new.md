# <a name="whats-new-in-power-apps"></a>Новые возможности в Power Apps

Этот раздел содержит ресурсы, где можно узнать о новых недавно выпущенных возможностях, новых возможностях, которые будут выпущены в течение нескольких следующих месяцев, и известных ограничениях.

## <a name="weekly-releases"></a>Еженедельные выпуски

Сведения о новых функциях, исправлениях и улучшениях, выпущенных за последние несколько недель, см. в разделе [выпущенные версии для Microsoft Power Apps](https://docs.microsoft.com/business-applications-release-notes/powerplatform/released-versions/powerapps).

Подробные сведения о том, как просматривать сведения в еженедельных заметках о выпуске, см. в этой [записи блога](https://powerapps.microsoft.com/blog/stay-tuned-with-the-latest-features-and-fixes-through-powerapps-weekly-release-notes/).

> [!NOTE]
> Выпуски развертываются в течение нескольких дней. Новые и обновленные функции могут отображаться не сразу.

## <a name="release-plan"></a>План выпуска

Сведения о новых функциях, выпускаемых в течение следующих нескольких месяцев, которые можно использовать для планирования, см. в статье [2019. выпуск 2 плана](https://docs.microsoft.com/power-platform-release-plan/2019wave2/microsoft-powerapps/planned-features)

## <a name="known-limitations"></a>Известные ограничения

Сведения об известных ограничениях см. в статье [Типичные проблемы и решения](common-issues-and-resolutions.md).
