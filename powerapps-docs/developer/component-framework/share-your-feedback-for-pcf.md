# <a name="share-your-feedback"></a>Поделитесь своими отзывами

Чтение и публикация на [форуме по инфраструктуре компонентов PowerApps](https://powerusers.microsoft.com/t5/PowerApps-Component-Framework/bd-p/pa_component_framework). Это отличный ресурс, в котором любой пользователь, который использует платформу компонентов PowerApps, может читать и обсуждать вопросы с другими пользователями. Прежде чем публиковать новый вопрос, выполните поиск на форуме обсуждений, чтобы узнать, не был ли ответ на ваш вопрос.
