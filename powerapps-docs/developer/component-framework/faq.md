# <a name="faq"></a>Вопросы и ответы

[!INCLUDE[cc-beta-prerelease-disclaimer](../../includes/cc-beta-prerelease-disclaimer.md)]

## <a name="multiple-components-in-single-manifest-file"></a>Несколько компонентов в одном файле манифеста

Невозможно определить несколько компонентов в одном файле манифеста. 

## <a name="calling-processesactions"></a>Вызов процессов или действий

Это не поддерживается. На сегодняшний день можно вызывать только диалоговые окна с помощью метода [навигации](reference/navigation.md) .

## <a name="calling-components-within-another-component"></a>Вызов компонентов в другом компоненте

Это не поддерживается.

## <a name="font-resource"></a>Шрифтовый ресурс

В настоящее время ресурс Font (. ТФФ) не поддерживается в платформе компонентов PowerApps.
