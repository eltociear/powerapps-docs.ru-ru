|Value|Description|
|--|--|
|`Currency`|Денежные значения в диапазоне от-922 337 203 685 477 до 922 337 203 685 477 могут быть в этом поле. Можно задать уровень точности или выбрать точность на основе определенной валюты или одной стандартной точности, используемой Организацией.|
|`DateAndTime.DateAndTime`|Отображает дату и время.|
|`DateAndTime.DateOnly`|Отображает только дату.|
|`Decimal`|В этом поле можно использовать до 10 десятичных точек точности для значений между-100 000 000 000 и-100 000 000 000. Можно указать уровень точности и максимальное и минимальное значения.|
|`Enum`|Перечисляемый тип данных.|
|`FP`|В этом поле можно использовать до 5 десятичных точек точности для значений между-100 000 000 000 и-100 000 000 000. Можно указать уровень точности и максимальное и минимальное значения. |
|`Multiple`|Это поле может содержать до 1 048 576 текстовых символов. Можно задать максимальную длину, которая будет меньше этой. При добавлении этого поля в форму можно указать размер поля.|
|`OptionSet`|Это поле предоставляет набор параметров. Каждый параметр имеет числовое значение и метку. При добавлении в форму в этом поле отображается элемент управления для выбора только одного параметра. |
|`SingleLine.Email`|Текст содержит ссылку mailto, чтобы открыть приложение электронной почты пользователя.|
|`SingleLine.Phone`|В веб-приложении поля будут включены, чтобы инициировать вызовы с помощью Skype или Lync, если на компьютере установлен клиент для любой из установленных систем. Выбранный поставщик услуг телефонии находится в нижней части вкладки "Общие" окна "Параметры системы". для Dynamics 365 для планшетов Skype является единственным доступным поставщиком телефонии.|
|`SingleLine.Text`|Этот параметр просто отображает текст.|
|`SingleLine.TextArea`|Этот параметр формата можно использовать для вывода нескольких строк текста. Но с ограничением в 4000 символов, несколько строк текстового поля лучше выбрать, если ожидается большой объем текста.|
|`SingleLine.Ticker`|Для большинства языков текст будет включен в качестве ссылки для открытия веб-сайта MSN Money для отображения сведений о стоимости акций, представленной символом-тикером. Для определенных восточно-азиатских языков окно будет открывать результаты поиска Bing для символа-разделителя.|
|`SingleLine.URL`|Текст содержит гиперссылку для открытия указанной страницы. Любой текст, который не начинается с допустимого протокола, будет иметь "http://" в начале. В этом поле разрешены только протоколы HTTP, HTTPS, FTP, FTPS, ONENOTE и TEL.|
|`TwoOptions`|Это поле предоставляет два варианта. Каждый параметр имеет числовое значение 0 или 1, соответствующее значению false или true. Каждый параметр также имеет метку, чтобы значения true или false можно было представить как "Yes" и "No", "горячий" и "холодный", "on" и "OFF", а также любую пару меток, которые нужно отобразить.|
|`Whole.None`|Этот параметр просто отображает число.|

## <a name="value-elements-that-are-not-supported"></a>Неподдерживаемые элементы value

Следующие значения `of-type` атрибутов в настоящее время не поддерживаются:

|Value|Description|
|-----|------|
|`Whole.Duration`|Этот параметр формата можно использовать для вывода списка параметров длительности. Но данные, хранящиеся в базе данных, всегда имеют несколько минут. Поле выглядит как раскрывающийся список и предоставляет предлагаемые параметры, такие как 1 минута, 15 минут, 30 минут до 3 дней. Пользователи могут выбрать эти параметры. Тем не менее, люди могут просто ввести несколько минут, и это разрешается в этот период времени.|
|`Whole.Timezone`|Этот параметр отображает список выбранных часовых поясов, например (GMT-12:00) Международный регион "Западная линия даты" и (GMT-08:00) Тихоокеанское время (США & Канада). Каждая из этих зон хранится в виде числа. Например, для часового пояса (GMT-08:00) Тихоокеанское время (США & Канада) значение Тимезонекоде равно 4. Дополнительные сведения: [класс тимезонекоде (сборка пакета SDK)](https://docs.microsoft.com/en-us/previous-versions/dynamics-crm4/developers-guide/bb959779(v=msdn.10))|
|`Whole.Language`|Этот параметр отображает список языков, подготовленных для вашей организации. Значения отображаются в виде раскрывающегося списка имен языков, но данные хранятся в виде числа с помощью кодов LCID. Код языка представляет собой четырехзначный или пятизначный идентификатор языкового стандарта. Допустимые значения идентификаторов языковых стандартов можно найти на [диаграмме языковых стандартов](https://docs.microsoft.com/en-us/previous-versions/windows/embedded/ms912047(v=winembedded.10)).|
|`Lookup.Simple`|Позволяет использовать одну ссылку на конкретную сущность. Все пользовательские уточняющие запросы относятся к этому типу.|
|`Lookup.Customer`|Позволяет использовать одну ссылку на запись либо в учетной записи, либо в записи контакта. Эти подстановки доступны для сущностей "Возможная сделка", "обращение", "предложение", "заказ" и "счет". У этих сущностей также есть отдельные учетные записи и поиск контактов, которые можно использовать, если клиенты всегда имеют один тип. Можно также включить и, и вместо использования поиска клиента.|
|`Lookup.Owner`|Позволяет использовать одну ссылку на группу или запись пользователя. Все объекты, принадлежащие команде или пользователю, имеют один из них.|
|`Lookup.PartyList`|Позволяет создавать несколько ссылок на несколько сущностей. Эти уточняющие запросы **находятся в полях "сущность** электронной почты **" и "копия"** . Они также используются в сущностях «телефон» и «встреча».|
|`Lookup.Regarding`|Позволяет использовать одну ссылку на несколько сущностей. Эти уточняющие запросы находятся в поле "в отношении", используемом в действиях.|
|`MultiSelectOptionSet`|Можно настроить формы (главное, быстрое создание и быстрый просмотр) и шаблоны электронной почты, добавив поля множественного выбора. При добавлении многовариантного поля набора можно указать несколько значений, которые будут доступны пользователям для выбора. Когда пользователи заполняют форму, они могут выбрать одно, несколько или все значения, отображаемые в раскрывающемся списке.|
