# <a name="pickfileoptions"></a>пиккфилеоптионс

## <a name="available-for"></a>Доступно для 

Приложения на основе модели

## <a name="properties"></a>Свойства

### <a name="accept"></a>гласит

Тип файла изображения для выбора. Допустимые значения: "Audio", "Video" или "Image".

**Тип**: `string`

### <a name="allowmultiplefiles"></a>алловмултиплефилес

Разрешить ли выбор нескольких файлов.

**Тип**: `boolean`

### <a name="maximumallowedfilesize"></a>максимумалловедфилесизе

Должен быть выбран максимальный размер файлов ro.

**Тип**: `number`
