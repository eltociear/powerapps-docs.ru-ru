# <a name="lookupoptions"></a>LookupOptions

Предоставляет все сведения о параметрах, используемых при открытии диалогового окна поиска.

## <a name="available-for"></a>Доступно для 

Приложения на основе модели

## <a name="properties"></a>Свойства

### <a name="allowmultiselect"></a>алловмултиселект

Позволяет ли подстановка выбираться более одного элемента.

**Тип**: `boolean`

### <a name="defaultentitytype"></a>дефаултентититипе

Тип сущности по умолчанию.

**Тип**: `string`

### <a name="defaultviewid"></a>дефаултвиевид

Представление по умолчанию для использования.

**Тип**: `string`

### <a name="entitytypes"></a>entityTypes

Типы сущностей для вывода.

**Тип**: `string[]`

### <a name="viewids"></a>виевидс

Представления, доступные в средстве выбора представлений. Поддерживаются только системные представления (не пользовательские представления).

**Тип**: `string[]`
