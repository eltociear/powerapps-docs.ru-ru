# <a name="isoffline"></a>isOffline

[!INCLUDE [isoffline-description](includes/isoffline-description.md)]

## <a name="syntax"></a>Синтаксис

`context.client.isOffline()`

## <a name="available-for"></a>Доступно для

Приложения на основе модели

## <a name="return-value"></a>Возвращаемое значение

Тип: `Boolean` Description: **true** , если сервер находится в автономном режиме; в противном случае — **значение false** .
