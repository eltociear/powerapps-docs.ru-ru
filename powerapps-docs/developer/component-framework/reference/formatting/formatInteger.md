# <a name="formatinteger"></a>formatInteger

[!INCLUDE [formatinteger-description](includes/formatinteger-description.md)]

## <a name="syntax"></a>Синтаксис

`context.formatting.formatInteger(value);`

## <a name="available-for"></a>Доступно для 

Приложения на основе моделей и приложения Canvas (экспериментальная Предварительная версия)

## <a name="parameters"></a>Вход

| Имя параметра|Тип|Требуется|Description|
| ------------- |----|--------|-----------|
|value|`number`|Да|Числовой примитив, который необходимо отформатировать.|

## <a name="return-value"></a>Возвращаемое значение

Тип: `string`
