# <a name="getstring"></a>getString

[!INCLUDE [getstring-description](includes/getstring-description.md)]

## <a name="available-for"></a>Доступно для 

Приложения на основе модели

## <a name="syntax"></a>Синтаксис

`context.resources.getString(id)`

## <a name="parameters"></a>Вход

| Имя параметра|Тип|Требуется|Description|
| ------------- |----|--------|-----------|
|Id|`String`|Да|Имя ресурса в манифесте компонента.|

## <a name="return-value"></a>Возвращаемое значение

Тип: `String`
