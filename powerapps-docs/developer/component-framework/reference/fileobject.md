# <a name="fileobject"></a>Закрыт

Предоставляет доступ ко всем свойствам файла.

## <a name="available-for"></a>Доступно для

Приложения на основе модели

## <a name="properties"></a>Свойства

### <a name="filecontent"></a>филеконтент

Содержимое файла.

**Тип**: `String`

### <a name="filename"></a>Файлов

Имя файла.

**Тип**: `String`

### <a name="filesize"></a>Размер файла

Размер файла в КБ.

**Тип**: `Number`

### <a name="mimetype"></a>Успешности

Тип MIME файла.

**Тип**: `String`
