# <a name="column"></a>Столбец

[!INCLUDE [column-description](includes/column-description.md)]

## <a name="available-for"></a>Доступно для 

Приложения на основе моделей и приложения Canvas (экспериментальная Предварительная версия)

## <a name="properties"></a>Свойства

### <a name="alias"></a>Псевдоним

Псевдоним столбца.

**Тип**: `string`

### <a name="datatype"></a>Заданий

Тип данных значений столбца.

**Тип**: `string`

### <a name="disablesorting"></a>дисаблесортинг

Не позволяет пользовательскому интерфейсу сделать столбец несортированным.

**Тип**: `boolean`<br />

### <a name="displayname"></a>DisplayName

Локализованное отображаемое имя для столбца

**Тип**: `string`

### <a name="ishidden"></a>isHidden

Состояние видимости столбца.

**Тип**: `boolean`<br />

### <a name="isprimary"></a>isPrimary

Является конкретным столбцом первичного атрибута сущности представления.

**Тип**: `boolean`<br />

### <a name="name"></a>Безымян

Имя столбца, уникальное в наборе данных.

**Тип**: `string`

### <a name="order"></a>Порядок

Порядок столбцов для макета.

**Тип**: `number`

### <a name="visualsizefactor"></a>висуалсизефактор

Настроенные коэффициенты ширины столбцов. 

**Тип**: `number`
