# <a name="retrievemultiplerecords"></a>retrieveMultipleRecords

[!INCLUDE [retrievemultiplerecords-description](includes/retrievemultiplerecords-description.md)]

## <a name="available-for"></a>Доступно для 

Приложения на основе модели

## <a name="syntax"></a>Синтаксис

`context.webAPI.retrieveMultipleRecords(entityLogicalName, options, maxPageSize).then(successCallback, errorCallback);`

## <a name="return-value"></a>Возвращаемое значение

Тип: <RetrieveMultipleResponse> [Promise](https://developer.mozilla.org/docs/Web/JavaScript/reference/Global_Objects/Promise)

Описание. `RetrieveMultipleResponse` возвращает обещание, содержащее массив объектов JSON, содержащих извлеченные записи сущностей, и атрибут **nextLink** с URL-адресом, указывающим на следующий набор записей в случае постраничного просмотра (`maxPageSize`) указан в запросе, а элемент возвращенное число записей превышает значение подкачки. Он имеет следующие параметры:

|Параметр|Возвращаемое значение|Description|
|----|------|-------|
|Объектах|`Entity[]`|Массив объектов JSON, где каждый объект представляет извлеченную запись сущности, содержащую атрибуты и их значения.|
|nextLink|`string`|Если количество извлекаемых записей превышает значение, указанное в параметре "Макспажесизе" в запросе, этот атрибут возвращает URL-адрес для возврата следующего набора записей.|
