## <a name="properties"></a>Свойства

### <a name="etn"></a>етн

Имя типа сущности. Только для чтения.

**Тип**: `string`

### <a name="id"></a>Id

Идентификатор записи. Только для чтения.

**Тип**: `object`

Объект `id` содержит следующее свойство:

|Имя|Тип|Description|
|--|--|--|
|`guid`|`string`|00000000-0000-0000-0000-000000000000|

### <a name="name"></a>Безымян

Имя ссылки на сущность. Только для чтения.

**Тип**: `string`
