# <a name="powerapps-component-framework-api-reference"></a>Справочник по API платформы компонентов PowerApps

Этот раздел содержит справочную документацию по созданию компонентов кода с помощью платформы компонентов PowerApps.

> [!IMPORTANT]
> Вкладка **доступно для** показывает, какие API поддерживаются приложениями и приложениями Canvas, управляемыми моделью (экспериментальная Предварительная версия). Рекомендуется проверить раздел **доступно для** каждого метода API, если он поддерживается. Например, **клиентский** API поддерживается как для приложений, основанных на моделях, так и для приложений Canvas, но метод «в **сети** » **клиентского** API поддерживает только приложения, управляемые моделями.
