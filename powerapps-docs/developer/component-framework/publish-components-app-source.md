# <a name="publish-code-components-on-appsource"></a>Публикация компонентов кода в AppSource

После тестирования компонентов кода и необходимости его распространения другим пользователям для его использования необходимо опубликовать его на Microsoft AppSource.

Подробные сведения о публикации компонентов кода в AppSource см. в статье [Публикация приложения в AppSource](/powerapps/developer/common-data-service/publish-app-appsource).
