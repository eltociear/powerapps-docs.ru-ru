# <a name="use-power-automate-to-automate-processes"></a>Автоматизация процессов с помощью Power автоматизиру

Power автоматизиру позволяет создавать автоматизированные процессы между избранными приложениями и службами. В приложении вы можете запустить Flow на одну или несколько записей. 

Кроме того, вы можете открыть Power автоматизиру из приложения, чтобы просматривать, изменять или создавать новые потоки.  Дополнительные сведения: [Начало работы с Power автоматизиру](https://docs.microsoft.com/flow/getting-started)

## <a name="manage-your-flows"></a>Управление расходами 
Подключитесь к Power автоматизиру из приложения, чтобы управлять потоками.
  
> [!NOTE]
> Для управления и запуска Power автоматизиру из приложения необходимо включить автоматизацию включения в параметрах системы для вашей организации. Дополнительные сведения: [потоки в Организации](https://docs.microsoft.com/flow/organization-q-and-a) 
  
1. На панели команд выберите **Flow**.  
  
   > [!div class="mx-imgBorder"]
   > ![Меню Power автоматизиру](media/flow.png "Меню Power автоматизиру") 
  
2. Появятся следующие параметры.  
  
   -   Выберите **создать последовательность** , чтобы открыть новую вкладку браузера и направить вас на сайт Power автоматизиру. На странице Power Автоматизируйте выберите **продолжить** , чтобы использовать существующий шаблон или **X** для закрытия, выбора или создания другого.  
  
   -   Выберите **Просмотреть** последовательности, чтобы открыть новую вкладку браузера и направить вас на сайт Power автоматизиру, где отображаются потоки в среде потока.  
  
   -   Выберите **Вход для потока** , чтобы подключиться к Power автоматизиру и отобразить доступные последовательности для выполнения. Отображается только в том случае, если вы не вошли в Power автоматизировать.   

    Дополнительные сведения о создании последовательности см. в статье [Приступая к работе с Power автоматизиру](https://docs.microsoft.com/powerapps/maker/canvas-apps/using-logic-flows#create-a-flow) .
