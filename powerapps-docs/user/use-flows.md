# <a name="use-power-automate-to-automate-processes"></a>Использование Power Automate для автоматизации процессов

Power Automate позволяет создавать автоматические рабочие процессы между избранными приложениями и службами. В приложении вы можете запустить Flow на одну или несколько записей. 

Вы также можете открыть Power Automate из приложения, чтобы просмотреть, изменить или создать новые потоки.  Дополнительные сведения: [Начало работы с Power Automate](https://docs.microsoft.com/flow/getting-started)

## <a name="manage-your-flows"></a>Управление потоками 
Подключайтесь с помощью Power Automate из приложения, чтобы управлять потоками.
  
> [!NOTE]
> Для управления и запуска Power Automate из приложения необходимо разрешить Power Automate в параметрах системы вашей организации. Дополнительные сведения:  [Вопросы и ответы об использовании Flow в организации](https://docs.microsoft.com/flow/organization-q-and-a) 
  
1. На панели команд выберите **Поток**.  
  
   > [!div class="mx-imgBorder"]
   > ![Меню Power Automate](media/flow.png "Меню Power Automate") 
  
2. Появятся следующие параметры.  
  
   -   Выберите **Создать поток**, чтобы открыть веб-сайт Power Automate в новой вкладке. На странице Power Automate выберите **Продолжить** для использования существующего шаблона или **X**, чтобы закрыть, выбрать или создать новый.  
  
   -   Выберите **Показать потоки**, чтобы открыть веб-сайт Power Automate в новой вкладке, где будут отображаться все потоки в среде потоков.  
  
   -   Выберите **Вход в поток** для соединения с Power Automate и отображения доступных для выполнения потоков. Этот параметр отображается, только если вы не выполнили вход в Power Automate.   

    Сведения о создании потоков см. в статье [Начало работы с Power Automate](https://docs.microsoft.com/powerapps/maker/canvas-apps/using-logic-flows#create-a-flow)
