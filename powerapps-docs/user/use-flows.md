# <a name="use-microsoft-flow-to-automate-processes"></a>Автоматизация процессов с помощью Microsoft Flow

Microsoft Flow позволяет создавать автоматические рабоче процессы между избранными приложениями и службами. В приложении вы можете запустить Flow на одну или несколько записей. 

Вы также можете открыть Microsoft Flow из приложения, чтобы просмотреть, изменить или создать новые потоки.  Дополнительные сведения: [Начало работы с Microsoft Flow](https://docs.microsoft.com/flow/getting-started)

## <a name="manage-your-flows"></a>Управление расходами 
Подключайтесь с помощью Microsoft Flow из приложения, чтобы управлять потоками.
  
> [!NOTE]
> Для управления и запуска Microsoft Flow из приложения, необходимо разрешить Microsoft Flow в параметрах системы вашей организации. Дополнительные сведения:  [Вопросы и ответы об использовании Flow в организации](https://docs.microsoft.com/flow/organization-q-and-a) 
  
1. Выберите **Потоки** на панели инструментов.  
  
   > [!div class="mx-imgBorder"]
   > ![Меню Microsoft Flow](media/flow.png "Меню Microsoft Flow") 
  
2. Появятся следующие параметры.  
  
   -   Выберите **Создать поток**, чтобы открыть веб-сайт Microsoft Flow в новой вкладке. На странице Microsoft Flow выберите **Продолжить**, для использования существующего шаблона, или **X**, чтобы закрыть, выбрать или создать новый.  
  
   -   Выберите **Показать потоки**, чтобы открыть веб-сайт Microsoft Flow в новой вкладке, где будут отображаться все потоки в среде Flow.  
  
   -   Выберите **Вход в поток** для соединения с Microsoft Flow и отображения доступных для выполнения потоков. Этот параметр отображается только если вы только что не выполнили вход в Microsoft Flow.  
   
    Сведения о создании потоков, см. в статье [Начало работы с Microsoft Flow](/flow/getting-started)
