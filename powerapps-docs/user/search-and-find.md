# <a name="compare-search-and-find-options-in-common-data-service"></a>Сравнение параметров поиска и поиска в Common Data Service

Существует четыре способа поиска данных в Common Data Service.

-   Поиск с сортировкой по релевантности  
  
-   Полнотекстовый быстрый поиск (одна сущность или несколько сущностей)  
  
-   Быстрый поиск (с одной сущностью или с несколькими сущностями)  

-   Расширенный поиск

> [!NOTE]
> Быстрый поиск с несколькими сущностями также называется поиском по категориям.
