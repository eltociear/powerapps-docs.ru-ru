# <a name="export-to-an-excel-dynamic-worksheet"></a>Экспорт в динамический лист Excel

Экспортируйте данные приложения в лист Office Excel, чтобы пользователи могли получить последние сведения. Представьте себе Генерального директора компании, чтобы получить необходимую вам важную информацию, не переходя в приложение, а просто открыв ссылку Excel на своем настольном компьютере. В каждый момент времени можно экспортировать до 100 000 записей.    
  
## <a name="export-data-to-an-excel-dynamic-worksheet"></a>Экспорт данных в динамический лист Excel  

Вы не можете экспортировать данные в динамический лист в Excel для всех типов записей. Если этот параметр не отображается, он недоступен для этой записи.  
  
1. Откройте список записей в приложении и щелкните стрелку справа от параметра **Экспорт в Excel**. 
  
2. Выберите **динамический лист**.  
  
3. В диалоговом окне **Выбор столбцов для динамического Excel** выберите параметры столбца и щелкните **Экспорт**.  
  
4. Выберите **сохранить** , а затем сохраните XLSX файл. Запишите место, где был сохранен файл.  
  
   > [!NOTE]
   > Если вы собираетесь изменить файл данных позже, рекомендуется сохранить файл перед его открытием. В противном случае может появиться следующее сообщение об ошибке: **Excel не может открыть или сохранить больше документов из-за нехватки свободной памяти или места на диске.**  
   > 
   > Чтобы устранить эту проблему, выполните следующие действия.  
   > 
   >    1. Откройте Excel и выберите **файл** > **Параметры** > **центр управления безопасностью** центр **настройки** > защищенное**представление**.  
   >    2. В **режиме защищенного просмотра**очистите все три элемента.  
   >    3. Нажмите кнопку **ОК** > **ОК**.  
   >     
   >    Мы настоятельно рекомендуем сохранить и открыть файл данных, а не отключать защищенное представление, что может поставить под угрозу компьютер.  
  
5. Откройте Excel, а затем откройте XLSX файл, сохраненный на предыдущем шаге.  
  
6. Если отображается предупреждение безопасности подключения к **внешним данным отключены**, выберите **включить содержимое**.  
  
7. Чтобы обновить данные в файле, на вкладке **данные** выберите **Обновить из PowerApps**.  
  
   > [!NOTE]
   > Если у вас есть номер телефона, начинающийся **+** с или **–** (например, + 1-123-456-7890), при обновлении динамического листа номер в поле номера телефона будет отображаться неправильно.   
   >
   > Чтобы избежать проблемы, используйте пробел или круглые скобки **()** , например: + 1 123-456-7890 или + 1 (123)-456-7890.  
  
## <a name="tips"></a>Советы  
  
- Можно отправить по электронной почте динамический файл Excel или сохранить его как общий файл, если получатели находятся в том же домене, что и вы. Когда получатели открывают динамический файл, они увидят данные, которые у них есть разрешение на просмотр в приложении, поэтому данные, которые они видят, могут отличаться от отображаемых.  
  
- Некоторые системные представления, например учетные записи: Без действий кампании за последние 3 месяца их можно экспортировать только на статический лист Excel.  
  
- В PowerApps значения валют экспортируются в Excel как числа. Чтобы отформатировать данные как денежную единицу после завершения экспорта, см. раздел справки Excel под названием "отобразить числа как денежные".

- Значения даты и времени, отображаемые в приложении, отображаются только как Дата при экспорте файла в Excel, но в ячейке фактически отображается дата и время.  
  
- Если вы собираетесь внести изменения и импортировать файл данных обратно в приложение, помните, что защищенные, вычисляемые и составные поля (такие как полное имя) доступны только для чтения и не могут быть импортированы в приложение. Вы сможете изменить эти поля в Excel, но при импорте данных обратно в приложение эти поля не будут обновлены. Если вы хотите обновить эти поля, например имя контакта, рекомендуется использовать это представление для экспорта данных, обновления в Excel и импорта их обратно в приложение для внесения изменений.
