# <a name="use-model-driven-apps-in-power-apps"></a>Использование приложения на основе модели в Power Apps

Приложение на основе модели содержит множество компонентов, включая панели мониторинга, формы, представления, диаграммы и бизнес-процессы, сочетание которых позволяет существенно упростить работу. В этом разделе содержатся сведения о том, как перемещаться по приложению на основе модели, работать с записями, импортировать и экспортировать данные и автоматизировать процессы с помощью Power Automate. 

> [!IMPORTANT]
> Так как приложения на основе холста дают широкие возможности управления взаимодействиями с пользователем и могут представлять собой что угодно, об *использовании приложений на основе холста* существует мало справочной документации. Если вам нужна помощь, обратитесь к автору приложения или администратору Power Apps.
