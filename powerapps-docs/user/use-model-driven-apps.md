# <a name="use-model-driven-apps-in-power-apps"></a>Использование приложений, управляемых моделью, в Power Apps

Приложение, управляемое моделью, содержит несколько компонентов, включая панели мониторинга, формы, представления, диаграммы и бизнес-процессы, которые совместно помогают сделать приложение простым в использовании. В этом разделе содержатся сведения о том, как перемещаться по приложениям, управляемым моделью, работать с записями, импортировать и экспортировать данные и автоматизировать процессы с помощью Power автоматизиру. 

> [!IMPORTANT]
> Поскольку приложения Canvas обеспечивают большой контроль над взаимодействиями с пользователем и, по сути, могут быть любыми, справочная документация по *использованию приложения Canvas*не слишком велика. Если вам нужна помощь с помощью приложения Canvas, обратитесь к администратору приложения или в службу Power Apps.
