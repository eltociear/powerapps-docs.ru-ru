# <a name="use-model-driven-apps-in-powerapps"></a>Использование приложений, управляемых моделью, в PowerApps

Приложение, управляемое моделью, содержит несколько компонентов, включая панели мониторинга, формы, представления, диаграммы и бизнес-процессы, которые совместно помогают сделать приложение простым в использовании. В этом разделе содержатся сведения о том, как перемещаться по приложениям, управляемым моделью, работать с записями, импортировать и экспортировать данные и автоматизировать процессы с помощью Microsoft Flow. 

> [!IMPORTANT]
> Поскольку приложения Canvas обеспечивают большой контроль над взаимодействиями с пользователем и, по сути, могут быть любыми, справочная документация по *использованию приложения Canvas*не слишком велика. Если вам нужна по ним помощь, обратитесь к автору приложения или администратору PowerApps.
