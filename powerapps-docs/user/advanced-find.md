# <a name="create-edit-or-save-an-advanced-find-search"></a>Создание, изменение и сохранение запросов расширенного поиска

Найдите нужные записи в приложении с помощью команды "Расширенный поиск". Расширенный поиск также может использоваться для подготовки данных для экспорта в Office Excel с целью анализа, суммирования или агрегирования данных или создания сводных таблиц для просмотра данных с разных точек зрения.  
  
1. На панели команд нажмите кнопку **Расширенный поиск**.
  
2. Укажите, что искать.  
  
   1.  В списке **Искать** выберите тип записи.  
  
   2.  Щелкните **Выбрать**, чтобы определить условия поиска: поле (например, имя учетной записи или город), реляционный оператор запроса (часть выражения, например "равно" или "содержит", определяющая, как заданный атрибут должен сравниваться со значением) и значения для поиска (например, "Сиэтл" или "Электронная почта").  
  
       Можно выбрать поля из типа текущей записи или из связанных записей. Например, у учетной записи может быть много связанных записей контактов.  
  
       В нижней части списка **Выбор** в разделе **Связанные** отображаются связанные типы записей. Например, большинство типов записей имеют связанный тип записи "Заметки". Чтобы выбрать поля связанного типа записи, выберите поле; отобразится новая ссылка **Выбрать**.  

3. Укажите столбцы, которые необходимо включить в результаты поиска.  
  
   1.  Выберите команду **Изменить столбцы**, а затем — **Добавить столбцы**.  
  
   2.  Выберите тип записи, включающий столбцы, которые необходимо добавить.  
  
   3.  Выберите столбцы, которые нужно добавить, а затем щелкните **ОК**.  
  
4. Укажите порядок сортировки.  
  
   1.  Выберите команду **Изменить столбцы**.  
  
   2.  Выберите команду **Настроить сортировку**.  
  
   3.  Укажите столбец для сортировки, порядок сортировки, а затем выберите **ОК**.  
  
   4.  Выберите **ОК**.  
  
5. Выберите **Результаты**.
