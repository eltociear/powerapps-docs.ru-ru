# <a name="track-your-progress-with-dashboards-and-charts"></a>Отслеживание хода выполнения с помощью панелей мониторинга и диаграмм

Панели мониторинга принимают набор данных приложений и предоставляют аналитические сведения для отображения ключевых показателей эффективности и других важных данных в удобной для чтения интерактивных диаграммах и графиках. Панели мониторинга доступны для всех типов записей.

> [!div class="mx-imgBorder"]
> ![Панель мониторинга](media/Dashboard.png "Панель мониторинга") 

-  Чтобы просмотреть другую структуру панели мониторинга, щелкните стрелку вниз рядом с именем панели мониторинга, а затем выберите нужную структуру.
-  Чтобы выбрать панель мониторинга по умолчанию, отобразите нужную панель мониторинга, а затем в верхней части экрана выберите **задать в качестве значения по умолчанию** .

   > [!div class="mx-imgBorder"]
   > ![Добавление или изменение панели мониторинга](media/add_dashboard.png "Добавление или изменение панели мониторинга") 

## <a name="create-a-new-dashboard"></a>Создать новую панель мониторинга

1. Чтобы создать новую панель мониторинга, выберите **создать панель мониторинга Dynamics 365**. 

   > [!div class="mx-imgBorder"]
   > ![Добавить новую панель мониторинга](media/new_dashboard.png "Добавить новую панель мониторинга")
   
2. Выберите макет панели мониторинга и щелкните **создать**.  

   > [!div class="mx-imgBorder"]
   > ![Создание панели мониторинга](media/create_dashboard.png "Создание панели мониторинга")
 
3. Введите имя панели мониторинга. 
4. Добавьте необходимые объекты в каждую область панели мониторинга. Например, добавим диаграмму. 

   > [!div class="mx-imgBorder"]
   > ![Добавить диаграмму](media/add_chart.png "Добавить диаграмму")
 
 5. Выберите **тип записи** для диаграммы.
 6. Выберите **представление** , в котором будут отображаться данные на диаграмме.
 7. Выберите диаграмму и нажмите кнопку **Добавить**.
 8. Продолжайте добавлять компоненты на панель мониторинга и по завершении нажмите кнопку **сохранить**. 
 
Созданная панель мониторинга появится в раскрывающемся меню доступных панелей мониторинга.

## <a name="use-charts"></a>Использование диаграмм 

Диаграммы дают быстрый обзор того, как вы отслеживаете цели. Они также интерактивны, поэтому можно щелкнуть область диаграммы, чтобы получить дополнительные сведения.

-   Наведите указатель мыши на диаграмму, чтобы увидеть подсказку, которая предоставляет быстрые сведения об этой области диаграммы.
-   Щелкните область диаграммы, чтобы увидеть представление сетки с дополнительными сведениями о данных в диаграмме.
-   Чтобы развернуть диаграмму, нажмите кнопку **развернуть диаграмму**развернуть![представление диаграммы]развернуть(media/expandviewbutton.png "пункт представление диаграммы") .
-   Чтобы просмотреть записи на диаграмме или обновить диаграмму, выберите ![другие команды](media/MoreButton.png "Дополнительные команды") , а затем выберите действие: **Обновление** или **Просмотр записей**.
     
     > [!div class="mx-imgBorder"]
     > ![Представление диаграмм в PowerApps](media/ViewOfCharts.png "Представление диаграмм в PowerApps")  
       

**Изменение представления диаграммы**
 
При изменении представления диаграммы вы узнаете о различных разбиениях данных, например о возможностях, открытых в течение определенного периода времени. Можно изменить представление диаграммы, щелкнув селектор представлений на странице сетки.

Например, выберите "все возможные сделки", а затем выберите другое представление, после чего будут обновлены и диаграмма, и сетка.

> [!div class="mx-imgBorder"]
> ![Изменение представления диаграммы в PowerApps](media/ChangeChartView.png "Изменение представления диаграммы в PowerApps")
