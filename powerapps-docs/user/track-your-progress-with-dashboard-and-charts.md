# <a name="track-your-progress-with-dashboards-and-charts"></a>Отслеживание хода выполнения с помощью панелей мониторинга и диаграмм

Панели мониторинга принимают набор данных приложений и предоставляют аналитические сведения для отображения ключевых показателей эффективности и других важных данных в удобной для чтения форме на интерактивных диаграммах и графиках. Панели мониторинга доступны для всех типов записей.

> [!div class="mx-imgBorder"]
> ![Панель мониторинга](media/Dashboard.png "Панель мониторинга") 

-  Чтобы просмотреть другую структуру панели мониторинга, щелкните стрелку вниз рядом с именем панели мониторинга, а затем выберите нужную структуру.
-  Чтобы выбрать панель мониторинга по умолчанию, откройте нужную панель мониторинга, а затем в верхней части экрана выберите команду **Установить по умолчанию**.

   > [!div class="mx-imgBorder"]
   > ![Добавление или изменение панели мониторинга](media/add_dashboard.png "Добавление или изменение панели мониторинга") 

## <a name="create-a-new-dashboard"></a>Создание панели мониторинга

1. Чтобы создать панель мониторинга, выберите команду **Создать панель мониторинга Dynamics 365**. 

   > [!div class="mx-imgBorder"]
   > ![Добавление новой панели мониторинга](media/new_dashboard.png "Добавление новой панели мониторинга")
   
2. Выберите макет панели мониторинга и нажмите кнопку **Создать**.  

   > [!div class="mx-imgBorder"]
   > ![Создание панели мониторинга](media/create_dashboard.png "Создание панели мониторинга")
 
3. Введите имя для панели мониторинга. 
4. Добавьте необходимые объекты в каждую область панели мониторинга. Можно, например, добавить диаграмму. 

   > [!div class="mx-imgBorder"]
   > ![Добавление диаграммы](media/add_chart.png "Добавление диаграммы").
 
 5. Выберите **тип записи** для диаграммы.
 6. Выберите **представление**, чтобы показать данные на диаграмме.
 7. Выберите диаграмму, а затем нажмите кнопку **Создать**.
 8. Продолжайте добавлять компоненты на панель мониторинга. Закончив, выберите команду **Сохранить**. 
 
Созданная вами панель мониторинга появится в раскрывающемся меню доступных панелей мониторинга.

## <a name="use-charts"></a>Использование диаграмм 

Диаграммы дают быстрый обзор вашего прогресса на пути к цели. Они также интерактивны: можно щелкнуть область диаграммы, чтобы получить дополнительные сведения.

-   Наведите указатель мыши на диаграмму, чтобы увидеть подсказку с краткими сведениями об этой области диаграммы.
-   Щелкните область диаграммы, чтобы увидеть представление сетки с дополнительными сведениями о данных в диаграмме.
-   Чтобы развернуть диаграмму, нажмите кнопку **Развернуть диаграмму**![Развернуть представление диаграммы](media/expandviewbutton.png "Развертывание представления диаграммы").
-   Чтобы просмотреть записи на диаграмме или обновить диаграмму, выберите ![Другие команды](media/MoreButton.png "Дополнительные команды"), а затем выберите действие: **Обновить** или **Просмотр записей**.
     
     > [!div class="mx-imgBorder"]
     > ![Представление диаграмм в Power Apps](media/ViewOfCharts.png "Представление диаграмм в Power Apps")  
       

**Изменение представления диаграммы**
 
Изменяя представление диаграммы, вы сможете использовать различные разбиения данных, чтобы, например, узнать о возможных сделках, открытых за определенный период времени. Можно изменить представление диаграммы, щелкнув селектор представлений на странице сетки.

Например, выберите "Все возможные сделки", а затем выберите другое представление, после чего будут обновлены и диаграмма и сетка.

> [!div class="mx-imgBorder"]
> ![Изменение представления диаграммы в Power Apps](media/ChangeChartView.png "Изменение представления диаграммы в Power Apps")

## <a name="known-issues"></a>Известные проблемы  
В конструкторе диаграмм добавление заказа по определенным вычисляемым полям не поддерживается и приведет к ошибке.  Вычисляемые поля, которые к этому приводят, используют другие вычисляемые поля, связанное поле сущности или локальное поле сущности.
