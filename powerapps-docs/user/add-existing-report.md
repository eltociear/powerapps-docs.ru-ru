# <a name="add-a-report-from-outside-power-apps"></a>Добавление отчета из внешних приложений Power Apps

Если пользовательский отчет создан за пределами системы, его можно легко добавить в Power Apps.

Сведения о создании настраиваемого отчета см. в разделе [руководство по отчетам и аналитике](https://docs.microsoft.com/dynamics365/customer-engagement/analytics/get-started-writing-reports).

1. В области навигации слева выберите область отчеты. 
2. На панели команд выберите **создать**.
  
   **Добавление файла, созданного в другом приложении**  
  
   1. В разделе **источник** в поле **тип отчета** выберите **существующий файл**.  
   
     > [!div class="mx-imgBorder"]
     > ![Добавить существующий отчет](media/add_existing_report.png "Добавить существующий отчет")
  
   2. В поле **расположение файла** введите путь и имя добавляемого файла или нажмите кнопку **Обзор** , чтобы найти файл. 
   
      Можно передать многие другие типы файлов, такие как файл Excel, но для запуска в качестве отчета SQL Server Reporting Services отчет или мастер отчетов, файл должен быть. RDL-файл. Дополнительные сведения см. в статье [о среде записи отчетов с помощью SQL Server Data Tools](https://docs.microsoft.com/dynamics365/customer-engagement/analytics/report-writing-environment-using-sql-server-data-tools).
  
      — ИЛИ  
  
   **Добавление ссылки на веб-страницу**  
  
   1.  В разделе **источник** в поле **тип отчета** выберите **ссылку на веб-страницу**.  
  
   2.  В поле **URL-адрес веб-страницы** введите URL-адрес веб-страницы.  
  
3. Укажите свойства отчета.
  
   1.  В разделе **сведения** укажите понятное имя и описание отчета.  
  
   2.  В текстовом поле **родительский отчет** отображается родительский отчет текущего отчета, если он существует.  
  
   3. **Категории**. Нажмите кнопку **выбрать или измените значения для этого поля** ![кнопка с многоточием](media/ellipsis-button.png "Кнопка с многоточием") , а затем укажите категории для включения в этот отчет.  
  
   4. **Связанные типы записей**. Чтобы отчет отображался в списке отчетов на странице для конкретных типов записей, ![нажмите кнопку](media/ellipsis-button.png "Кнопка с многоточием") **выбрать или изменить значения для этого поля** , а затем выберите типы записей.  
  
   5. **Отобразить в**. Чтобы указать, где должны отображаться отчеты, нажмите кнопку **выбрать или изменить значения для этого поля** ![кнопка с многоточием](media/ellipsis-button.png "Кнопка с многоточием") , а затем выберите один или несколько параметров.  
  
        Если значения не выбраны, отчет не будет виден конечным пользователям.  
  
4. Выберите **сохранить** или **сохранить и закрыть**.
