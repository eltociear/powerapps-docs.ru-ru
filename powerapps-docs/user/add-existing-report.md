# <a name="add-a-report-from-outside-power-apps"></a>Добавление отчета извне Power Apps

Если пользовательский отчет создан за пределами системы, его можно легко добавить в Power Apps.

Сведения о создании пользовательского отчета см. в [руководстве по отчетам и аналитике](https://docs.microsoft.com/dynamics365/customer-engagement/analytics/get-started-writing-reports).

1. На панели навигации слева выберите область отчетов. 
2. На панели команд нажмите **Создать**.
  
   **Добавление файла, созданного в другом приложении**  
  
   1. В разделе **Источник** в поле **Тип отчета** выберите **Существующий файл**.  
   
     > [!div class="mx-imgBorder"]
     > ![Добавление существующего отчета](media/add_existing_report.png "Добавление существующего отчета")
  
   2. В поле **Расположение файла** введите путь и имя добавляемого файла или нажмите кнопку **Обзор**, чтобы найти файл. 
   
      Можно передавать разнообразные типы файлов, такие как файлы Excel, но для запуска аналогично отчету SQL Server Reporting Services или отчету, созданному в мастере отчетов, необходим RDL-файл. Дополнительные сведения см. в разделе [Среда для создания отчетов с помощью SQL Server Data Tools](https://docs.microsoft.com/dynamics365/customer-engagement/analytics/report-writing-environment-using-sql-server-data-tools).
  
      \- ИЛИ  
  
   **Добавление ссылки на веб-страницу**  
  
   1.  В разделе **Источник** в поле **Тип отчета** выберите пункт **Ссылка на веб-страницу**.  
  
   2.  В поле **URL-адрес веб-страницы** введите URL-адрес веб-страницы.  
  
3. Укажите свойства отчета.
  
   1.  В разделе **Сведения** укажите понятное имя и описание отчета.  
  
   2.  В текстовом поле **Родительский отчет** отображается родительский отчет текущего отчета, если он существует.  
  
   3. **Категории**. Нажмите кнопку **Выбрать или изменить значения для этого поля** ![кнопка с многоточием](media/ellipsis-button.png "Кнопка с многоточием"), а затем укажите категории для включения в этот отчет.  
  
   4. **Связанные типы записей**. Чтобы отчет отображался в списке отчетов на странице для конкретных типов записей, нажмите кнопку **Выбрать или изменить значения для этого поля** ![кнопка с многоточием](media/ellipsis-button.png "Кнопка с многоточием"), а затем выберите типы записей.  
  
   5. **Отображение**. Чтобы указать, где должны отображаться отчеты, нажмите кнопку **Выбрать или изменить значения для этого поля** ![кнопка с многоточием](media/ellipsis-button.png "Кнопка с многоточием"), а затем выберите один параметр или несколько.  
  
        Если значения не выбраны, отчет не будет виден конечным пользователям.  
  
4. Выберите команду **Сохранить** или **Сохранить и закрыть**.
