# <a name="find-your-administrator-or-support-person"></a>Поиск администратора или специалиста службы поддержки 

Иногда вы можете столкнуться с проблемами, требующими консультаций с администратором. Если вы не знакомы с администратором, можно воспользоваться расширенным поиском, чтобы найти администратора.  
  
1. Откройте свое приложение.  
  
2. На панели команд нажмите кнопку **Расширенный поиск** .
  
3. На странице Расширенный поиск в списке **Искать** выберите **Пользователи**.  
  
4. Наведите указатель мыши на **пункт выбрать**и выберите **роли безопасности** из списка.  
  
5. В разделе **поля**выберите **имена**.  
  
6. В поле введите **системный администратор** введите **текст**. На этом этапе экран должен выглядеть следующим образом:  
     
   > [!div class="mx-imgBorder"]
   > ![Диалоговое окно «Расширенный поиск»](media/find-admnistrator-advanced-find-complete.png "Диалоговое окно «Расширенный поиск»")  
  
7. Нажмите кнопку **результаты** на ленте.  
  
   Вы увидите список одного или нескольких системных администраторов.
