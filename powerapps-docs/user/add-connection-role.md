# <a name="add-a-connection-role-to-link-records-to-each-other"></a>Добавление роли подключения для связывания записей друг с другом

Подключения позволяют легко связывать друг с другом пользователей, контакты, предложения, заказы на продажу и многие другие записи сущностей. Записям в привязке можно назначить определенные роли, которые помогают определить назначение связи.

Это быстрый способ создания нескольких подключений и ролей для конкретной записи. Например, контакт может иметь много отношений с другими контактами, учетными записями или контрактами. В каждой связи контакт может играть разные роли.

Роли подключения напрямую связаны с подключением. Чтобы использовать роль подключения, необходимо сначала добавить подключение к записи.

Прежде чем можно будет добавить роли подключения, администратор должен включить эту функцию. Дополнительные сведения см. в разделе [Настройка ролей подключения](https://docs.microsoft.com/powerapps/maker/common-data-service/configure-connection-roles).

1. Чтобы добавить подключения или управлять ими, выберите запись, которой вы хотите управлять, например возможность.  
2. Перейдите на вкладку **Связанные** и выберите **Подключение**. Откроется сетка подключений со списком подключений для записи.

    > [!div class="mx-imgBorder"]
    > ![Добавление новой роли подключения](media/connection1.png "Добавление новой роли подключения") 

3. Выберите **Подключить**, а затем выберите **К другому** или **Ко мне**.

    > [!div class="mx-imgBorder"]
    > ![Выбор типа подключения](media/connection2.png "Выбор типа подключения") 
  
4. В поле **Имя** введите или найдите имя записи для подключения.

5. В поле **В качестве этой роли** нажмите на значок поиска и выберите **Новая роль подключения**. Или используйте поиск, чтобы найти существующую роль, которую необходимо связать с подключением, а затем нажмите **Сохранить**.

    > [!div class="mx-imgBorder"]
    > ![Выбор новой роли подключения](media/connection3.png "Выбор новой роли подключения")  

    > [!NOTE]
    > Если вы указали сведения перед созданием новой роли подключения, отобразится диалоговое окно с предупреждением и вопросом, хотите ли вы отменить и продолжить работу с подключением или выйти из текущей записи, над которой вы работаете.

6. Чтобы создать новую роль подключения, на экране **Новая роль подключения** введите имя, а затем выберите **Категория роли подключения**.

    > [!div class="mx-imgBorder"]
    >  ![Добавление категории роли подключения](media/connection4.png "Добавление категории роли подключения") 

7. Когда все будет готово, выберите **Сохранить и закрыть**.

  
## <a name="manage-connection-roles"></a>Управление ролями подключения

Чтобы управлять ролью подключения, выберите роль подключения из сущности подключения. Откроется запись сущности "Роль подключения".  Можно изменить имя, выбрать категорию роли подключения и добавить описание.


   > [!div class="mx-imgBorder"]
   > ![Изменение роли подключения](media/connection7.png "Изменение роли подключения") 
  
Можно также управлять типами ролей подключения, которые должны быть связаны с ролью подключения.

1. Откройте роль подключения и выберите **Управление типом записи** на панели команд. 

    > [!div class="mx-imgBorder"]
    > ![Изменение роли подключения](media/connection5.png "Изменение роли подключения") 
  

2. Откроется список типов ролей подключения, которые можно добавить или удалить для этой роли подключения.

    > [!div class="mx-imgBorder"]
    > ![Управление типом записи](media/connection6.png "Управление типом записи")
