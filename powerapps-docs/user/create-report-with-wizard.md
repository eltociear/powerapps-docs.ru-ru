# <a name="create-a-report-using-the-report-wizard"></a>Создание отчета с помощью мастера отчетов


Используйте мастер отчетов для создания отчетов с диаграммами и таблицами, которые позволяют легко анализировать данные. 

Все отчеты, созданные с помощью мастера отчетов, — это отчеты на основе выборки. Обратите внимание, что все отчеты, созданные с помощью мастера отчетов, печатаются в альбомном режиме.

## <a name="create-a-new-report"></a>Создать новый отчет

1. В области навигации слева выберите область отчеты.  
2. На панели команд выберите **создать**.

    > [!div class="mx-imgBorder"]
    > ![Создание нового отчета](media/newreport.png "Создать новый отчет")
  
3. Появится экран **отчета: новый отчет** . Для элемента **тип отчета** оставьте значение по умолчанию, **отчет мастера** отчетов и нажмите кнопку **Мастер отчетов** . 

    > [!div class="mx-imgBorder"]
    > ![Мастер отчетов](media/report_wizard.png "Экран мастера отчетов")
  
4. На следующем экране оставьте значения по умолчанию и нажмите кнопку **Далее**.
 
    > [!div class="mx-imgBorder"]
    > ![Мастер отчетов](media/report_wizard_1.png "Экран мастера отчетов")
   
4. На экране **Свойства отчета** введите имя отчета, а затем выберите запись, которую необходимо включить в отчет, а затем нажмите кнопку **Далее**.
 
    > [!div class="mx-imgBorder"]
    > ![Экран свойств отчета](media/report_wizard_2.png "Экран свойств отчета")
  
5.  На экране **Выбор записей для включения в отчет** выберите Фильтры, чтобы определить, какие записи включаются в отчет. Например, если вы хотите просмотреть результаты только для записей, измененных за последние 60 дней, можно установить этот фильтр на этом экране. Если вы не хотите, чтобы данные отфильтрованы, выберите **очистить**.

    > [!div class="mx-imgBorder"]
    > ![Выберите записи для включения в отчет *](media/report_wizard_3.png "Выберите записи для включения в отчет")
  
6. На экране макет **полей** выберите макет отчета. Выберите **щелкните здесь, чтобы добавить группирование** , и выберите способ группировки данных.

    > [!div class="mx-imgBorder"]
    > ![Поля макета](media/report_wizard_4.png "Поля макета")

7. Выберите **тип записи** и **столбец** для данных, которые должны быть сгруппированы в отчете. Завершив выбор параметров, нажмите кнопку **ОК**.

    > [!div class="mx-imgBorder"]
    > ![экран группирования DD](media/report_wizard_5.png "Добавить экран группирования")
  
8. Выберите **щелкните здесь, чтобы добавить столбец** к столбцам с данными, связанными с типом записи, выбранным на предыдущем шаге.  

    > [!div class="mx-imgBorder"]
    > ![Добавить экран группирования](media/report_wizard_6.png "Добавить экран группирования")

9. На экране **Добавление столбца** выберите данные, которые должны отображаться для столбца, а затем нажмите кнопку **ОК**. 

    > [!div class="mx-imgBorder"]
    > ![экран добавления столбца](media/report_wizard_7.png "Экран добавления столбца")
  
10. Повторите предыдущий шаг для всех дополнительных столбцов, которые необходимо добавить. Когда все будет готово, на экране « **макет полей** » выберите **Далее**.
 
    > [!div class="mx-imgBorder"]
    > ![экран добавления столбца](media/report_wizard_8.png "Экран добавления столбца")
  
11. На экране **Формат отчета** выберите способ форматирования отчета и нажмите кнопку **Далее**.
 
    > [!div class="mx-imgBorder"]
    > ![Формат отчета](media/report_wizard_9.png "Формат экрана отчета")

12. Просмотрите сводку отчета и нажмите кнопку **Далее** , а затем кнопку **Готово**. Теперь этот отчет можно просмотреть в списке отчетов в системе.

    > [!div class="mx-imgBorder"]
    > ![Просмотреть отчет](media/report_wizard_10.png "Просмотр отчета")
