# <a name="create-a-report-using-the-report-wizard"></a>Создание отчета с помощью мастера отчетов


Используйте мастер отчетов для создания отчетов с диаграммами и таблицами, которые позволяют легко анализировать данные. 

Все отчеты, созданные с помощью мастера отчетов, — это отчеты на основе выборки. Обратите внимание, что все отчеты, созданные с помощью мастера отчетов, выводятся в альбомном режиме.

## <a name="create-a-new-report"></a>Создание нового отчета

1. На панели навигации слева выберите область отчетов.  
2. На панели команд нажмите **Создать**.

    > [!div class="mx-imgBorder"]
    > ![Создать отчет](media/newreport.png "Создание нового отчета")
  
3. Откроется экран **Отчет: Создать отчет**. В качестве **типа отчета** оставьте значение по умолчанию **Отчет мастера отчетов** и нажмите кнопку **Мастер отчетов**. 

    > [!div class="mx-imgBorder"]
    > ![Мастер отчетов](media/report_wizard.png "Экран мастера отчетов")
  
4. На следующем экране оставьте значения по умолчанию и нажмите **Далее**.
 
    > [!div class="mx-imgBorder"]
    > ![Мастер отчетов](media/report_wizard_1.png "Экран мастера отчетов")
   
4. На экране **Свойства отчета** введите имя отчета, а затем выберите запись, которую необходимо включить в отчет, и нажмите **Далее**.
 
    > [!div class="mx-imgBorder"]
    > ![Экран "Свойства отчета"](media/report_wizard_2.png "Экран "Свойства отчета"")
  
5.  На экране **Выберите записи для включения в отчет** выберите фильтры, чтобы определить, какие записи будут включены в отчет. Например, если вы хотите просмотреть результаты только для записей, измененных за последние 60 дней, можно установить этот фильтр на этом экране. Если вы не хотите фильтровать данные, выберите **Очистить**.

    > [!div class="mx-imgBorder"]
    > ![Выберите записи для включения в отчет *](media/report_wizard_3.png "Выберите записи для включения в отчет")
  
6. На экране **Поля макета** выберите макет отчета. Выберите **Щелкните здесь, чтобы добавить группирование** и выберите способ группировки данных.

    > [!div class="mx-imgBorder"]
    > ![Поля макета](media/report_wizard_4.png "Поля макета")

7. Выберите **Тип записи** и **Столбец** для данных, которые должны быть сгруппированы в отчете. Завершив выбор параметров, нажмите **ОК**.

    > [!div class="mx-imgBorder"]
    > ![экран добавления группирования](media/report_wizard_5.png "Экран добавления группирования")
  
8. Выберите **Щелкните здесь, чтобы добавить столбец** для столбцов данных, связанных с типом записи, выбранным на предыдущем шаге.  

    > [!div class="mx-imgBorder"]
    > ![экран добавления группирования](media/report_wizard_6.png "Экран добавления группирования")

9. На экране **Добавить столбец** выберите данные, которые должны отображаться для столбца, а затем нажмите кнопку **ОК**. 

    > [!div class="mx-imgBorder"]
    > ![экран добавления столбца](media/report_wizard_7.png "Экран добавления столбца")
  
10. Повторите предыдущий шаг для всех дополнительных столбцов, которые необходимо добавить. Завершив работу, на экране **Поля макета** нажмите **Далее**.
 
    > [!div class="mx-imgBorder"]
    > ![экран добавления дополнительных столбцов](media/report_wizard_8.png "Экран добавления дополнительных столбцов")
  
11. На экране **Формат отчета** выберите способ форматирования отчета и нажмите кнопку **Далее**.
 
    > [!div class="mx-imgBorder"]
    > ![Формат отчета](media/report_wizard_9.png "Экран "Формат отчета"")

12. Просмотрите сводку отчета и выберите **Далее**, а затем нажмите **Завершить**. Теперь этот отчет отображается в списке отчетов в системе.

    > [!div class="mx-imgBorder"]
    > ![Просмотреть отчет](media/report_wizard_10.png "Просмотр отчета")
