# <a name="use-keyboard-shortcuts-in-power-apps"></a>Использование сочетаний клавиш в Power Apps

Сочетания клавиш позволяют выполнять стандартные задачи вместо использования мыши или касания пальцем. Они помогают легко перемещаться по интерфейсу. Следующие сочетания клавиш применяются в разных приложениях. 

> [!NOTE]
> Сочетания клавиш, описанные здесь, относятся к раскладке клавиатуры США. Клавиши на других раскладках клавиатуры могут не совпадать с клавишами на клавиатуре США.

## <a name="navigating-submenus"></a>Переходы по подменю
Если вы используете средство чтения с экрана, выполните следующие действия, чтобы получить доступ к любым подменю, которые отображаются при наведении указателя мыши на ссылку навигации.

1. Переход между параметрами меню с помощью стандартной навигации по ссылкам.
2. Активируйте ссылку, нажав клавишу ВВОД или пробел, как обычно. Отобразится список подменю.
3. Чтобы получить доступ к этим параметрам подменю, воспользуйтесь сочетанием клавиш вспомогательной технологии (AT) для перехода к следующей кнопке. Например, при использовании средства чтения с экрана NVDA используется сочетание клавиш "b" (NVDA + b). Чтобы просмотреть все параметры, повторно выберите ключ.

> [!IMPORTANT]
> Несколько областей в пользовательском интерфейсе имеют автоматическое свертывание встроенных элементов для меню. Это включает в себя панель навигации и навигацию по форме. Оба расширения, если они выбраны, но после 30 секунд бездействия элементы свернуты.

## <a name="navigating-through-stages-of-a-process"></a>Навигация по этапам процесса
Если вы находитесь в записи (например, руководитель), в этом разделе показаны сочетания клавиш для перемещения по различным этапам процесса продаж для квалификации интереса и выбора на этих этапах. Например, если вы хотите легко перейти с подготовки к этапу закрытия, попробуйте выполнить следующие действия с клавиатуры:
- После выбора этапа перейдите к полям, щелкнув стрелку вниз (↓).
- Если этап свернут, нажмите клавишу ВВОД, чтобы развернуть его.

  > [!NOTE]
  > Имя процесса — это значок в нижней части страницы, а не вверху.

- **Сочетания клавиш в Windows**. Если используется операционная система Windows, выберите следующую ссылку и выберите номер версии из раскрывающегося списка. Можно просмотреть все сочетания клавиш, применимые к среде Windows. См. раздел [сочетания клавиш Windows](https://support.microsoft.com/help/12445/windows-keyboard-shortcuts). 
- **Сочетания клавиш в Mac**. Если вы используете операционную систему Mac, перейдите по следующей ссылке, чтобы просмотреть все ярлыки, применимые к среде Mac. См. раздел [сочетания клавиш для Mac](https://support.apple.com/HT201236).
