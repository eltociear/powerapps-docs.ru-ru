# <a name="use-keyboard-shortcuts-in-power-apps"></a>Использование сочетаний клавиш в Power Apps

Сочетания клавиш предоставляют альтернативный способ выполнять стандартные задачи вместо использования мыши или касания пальцем. Они помогают легко перемещаться по интерфейсу. Следующие сочетания клавиш применяются в приложениях. 

> [!NOTE]
> Сочетания клавиш, описанные здесь, относятся к американской раскладке клавиатуры. Клавиши в других раскладках могут не соответствовать клавишам в американской раскладке.

## <a name="navigating-submenus"></a>Навигация по подменю
Если вы используете средство чтения с экрана, выполните следующие действия, чтобы получить доступ к любым подменю, которые отображаются при наведении указателя мыши на ссылку навигации.

1. Переходите между параметрами меню с помощью стандартной навигации по ссылкам.
2. Активируйте ссылку, нажимая клавиши ВВОД или ПРОБЕЛ, как обычно. Отобразится список подменю.
3. Чтобы получить доступ к этим параметрам подменю, воспользуйтесь сочетанием клавиш специальных возможностей для перехода к следующей кнопке. Например, при использовании средства чтения с экрана NVDA используется сочетание клавиш с клавишей "b" (NVDA + b). Чтобы просмотреть все параметры, нажмите клавишу несколько раз.

> [!IMPORTANT]
> Несколько областей в пользовательском интерфейсе имеют автоматическое свертывание для меню. Сюда входит панель навигации и навигация по форме. Оба элемента разворачиваются при выборе, но после 30 секунд сворачиваются обратно.

## <a name="navigating-through-stages-of-a-process"></a>Навигация по этапам процесса
Если вы находитесь в записи (например, потенциального клиента), в этом разделе показаны сочетания клавиш для перемещения по различным этапам процесса продаж для работы с потенциальным клиентом, а также описано, как делать выбор на этих этапах. Например, если вы хотите легко перейти с этапа отбора на этап закрытия сделки, выполните следующие действия с клавиатуры.
- После выбора этапа перейдите к полям, нажимая стрелку вниз (↓).
- Если этап свернут, нажмите клавишу ВВОД, чтобы развернуть его.

  > [!NOTE]
  > Имя процесса — это значок в нижней части страницы, а не сверху.

- **Сочетания клавиш в Windows**. Если вы используете операционную систему Windows, нажмите на следующую ссылку и выберите номер версии из раскрывающегося списка. Можно просмотреть все сочетания клавиш, применимые в среде Windows. См. [Сочетания клавиш в Windows](https://support.microsoft.com/help/12445/windows-keyboard-shortcuts). 
- **Сочетания клавиш в Mac**. Если вы используете операционную систему Mac, перейдите по следующей ссылке, чтобы просмотреть все сочетания клавиш, применимые в среде Mac. См. [Сочетания клавиш в Mac](https://support.apple.com/HT201236).
