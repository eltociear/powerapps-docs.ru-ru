# <a name="preview-view-and-create-email-through-the-activities-grid"></a>Предварительная версия: Просмотр и создание сообщения электронной почты с помощью сетки действий

Просмотр и создание сообщения электронной почты с помощью сетки действий — функция раннего доступа. Вы можете включить эти функции в своей среде. Это позволит вам протестировать эти функции, а затем принять их в своих средах. Сведения о том, как включить эти функции, см. в разделе [Согласие на обновления первой волны выпуска 2020](https://docs.microsoft.com/power-platform/admin/opt-in-early-access-updates).

Приложения на основе моделей в Dynamics 365 позволяют взаимодействовать с клиентами по электронной почте. Функция электронной почты позволяет делать следующее.

- Просматривать сообщения электронной почты и отвечать на них. 

- Использовать общие функции панели инструментов электронной почты и элементы управления редактора форматированного текста. 

- Просматривать и вставлять изображения во встроенном режиме с помощью функций перетаскивания или копирования и вставки. 

- Создавать сообщения электронной почты во всплывающем окне.  

- Включать предварительный просмотр шаблонов перед их применением. 



## <a name="view-your-email"></a>Просмотр электронной почты

Для просмотра электронной почты выполните следующие действия.

1. В карте сайта приложения на основе модели выберите **Действия**. 

2. Выберите раскрывающийся список **Все действия**, а затем выберите **Полученные мной сообщения электронной почты**.

    ![просмотр почты](media/view-email.png "Отображение полученных сообщений электронной почты")

3. Выберите сообщение электронной почты, которое вы хотите просмотреть, чтобы открыть его. Откроется сообщение электронной почты, где можно будет ответить отправителю и получателям или переслать его.

## <a name="create-email"></a>Создание сообщения электронной почты

Ниже описано, как создать сообщение электронной почты.

1. В карте сайта приложения на основе модели выберите **Действия**.

2. На панели команд выберите **Почта**. Откроется окно нового сообщения электронной почты.

    ![создание сообщения](media/create-email.png "Создание нового сообщения")

    Поле **От** заполняется автоматически на основе сведений текущего вошедшего в систему пользователя.

3. Напишите сообщение непосредственно на редакторе или выберите команду **Вставить шаблон**, чтобы найти и применить шаблон. Дополнительные сведения о вставке шаблона сообщения электронной почты см. в разделе [Вставка шаблона сообщения электронной почты](insert-email-template.md).

4. Чтобы создать сообщение электронной почты в полноэкранном режиме, щелкните значок "Развернуть".

    ![развернуть окно сообщения](media/email-expand-window.png "Развертывание окна электронной почты")

    Окно сообщения содержит редактор форматированного текста, который позволяет создавать форматированное содержимое для сообщений электронной почты со смысловым выделением. Редактор предоставляет общие функции текстового процессора. 

    - Форматирование текста
    - Шрифт и размер
    - Полужирное и курсивное начертание, подчеркивание
    - Цвет фона текста и цвет самого текста
    - Маркированные и нумерованные списки
    - Уменьшение и увеличение отступа
    - Цитаты
    - Выравнивание текста (слева, по центру и справа)
    - Ссылки и отмена привязки
    - Зачеркивание
    - Изображение
    - Направление текста справа налево и слева направо
    - Отмена и повторение
    - Сброс форматирования
    - Таблица

    ![электронная почта — панель инструментов](media/email-toolbar.png "Использование функций редактора форматированного текста")

5. После завершения выберите команду **Отправить**.
