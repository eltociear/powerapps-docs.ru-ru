# <a name="preview-view-and-create-email-through-the-activities-grid"></a>Предварительный просмотр: Просмотр и создание сообщения электронной почты с помощью сетки действий

Просмотр и создание электронной почты с помощью сетки действий — это функция раннего доступа. Вы можете включить эти функции в своей среде заранее. Это позволит вам протестировать эти функции и затем принять их в своих средах. Сведения о том, как включить эти функции, см. в статье [согласие на обновление 2020 выпуска 1](https://docs.microsoft.com/power-platform/admin/opt-in-early-access-updates).

Приложения, управляемые моделями Dynamics 365, позволяют взаимодействовать с клиентами по электронной почте. Функция электронной почты позволяет:

- Просматривайте сообщения электронной почты и отвечайте на них. 

- Используйте общие функции панели инструментов электронной почты и элементы управления редактора форматированного текста. 

- Просматривайте и вставляйте изображения во встроенном режиме с помощью функций перетаскивания или копирования и вставки. 

- Создание сообщения электронной почты во всплывающем окне.  

- Предварительный просмотр шаблонов перед их применением. 



## <a name="view-your-email"></a>Просмотр электронной почты

Чтобы просмотреть электронную почту, сделайте следующее:

1. На портале карты на основе модели выберите **действия**. 

2. Выберите раскрывающийся **список все действия** , а затем выберите **Мои полученные сообщения электронной почты**.

    ![представление-электронная почта](media/view-email.png "Отображение полученных сообщений электронной почты")

3. Выберите адрес электронной почты, который вы хотите просмотреть, чтобы открыть его. Откроется сообщение электронной почты, где можно будет ответить отправителю и получателям или переслать его.

## <a name="create-email"></a>Создание сообщения электронной почты

Ниже описано, как создать сообщение электронной почты.

1. На портале карты на основе модели выберите **действия**.

2. На панели команд выберите **Электронная почта**. Откроется новое окно электронной почты.

    ![Создание сообщения электронной почты](media/create-email.png "Создать новое сообщение электронной почты")

    Поле " **от** " заполняется автоматически на основе текущего вошедшего в систему пользователя.

3. Напишите сообщение непосредственно на композиторе или выберите **Вставить шаблон** , чтобы найти и применить шаблон. Дополнительные сведения о вставке шаблона сообщения электронной почты см. в разделе [Вставка шаблона сообщения электронной почты](insert-email-template.md).

4. Чтобы создать сообщение электронной почты в полноэкранном окне, щелкните значок развертывания.

    ![Электронная почта-развернуть окно](media/email-expand-window.png "Развернуть окно электронной почты")

    Окно сообщения имеет редактор форматированного текста, который позволяет создавать форматированное и хорошо форматированное содержимое для сообщений электронной почты с выделением. Редактор предоставляет такие общие функции текстового процессора, как: 

    - Копировать форматирование
    - Шрифт и размер
    - Полужирный, курсив и подчеркнутый
    - Цвет фона для цвета текста и текста
    - Маркированный и нумерованный список
    - Уменьшить и увеличить отступ
    - Заблокировать цитату
    - Выравнивание текста (выравнивание влево, по центру и справа)
    - Связывание и отмена связи
    - Перечеркивание текста
    - Изображение
    - Направление текста справа налево и слева направо
    - Отмена и повтор
    - Удалить формат
    - Table

    ![Электронная почта — панель инструментов](media/email-toolbar.png "Использование функций редактора форматированного текста")

5. Когда все будет готово, нажмите кнопку **Отправить**.
