# <a name="run-a-canvas-app-from-a-sharepoint-page"></a>Запуск приложения на основе холста со страницы SharePoint

Приложение холста Power Apps можно добавить на страницу SharePoint Online с помощью веб-части Power Apps. После этого пользователи смогут запускать такие приложения со страницы SharePoint Online. Дополнительные сведения: [Office: использование веб-части Power Apps](https://support.office.com/article/use-the-powerapps-web-part-6285f05e-e441-408a-99d7-aa688195cd1c?ui=en-US&rs=en-US&ad=US)
