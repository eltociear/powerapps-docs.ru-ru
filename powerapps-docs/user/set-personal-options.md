# <a name="set-personal-options"></a>Настройка дополнительных параметров




Измените параметры экрана в приложении в соответствии с вашими предпочтениями. Например, можно выбрать страницу, которая будет отображаться сразу после входа в приложение. Вы также можете персонализировать множество других параметров, таких как язык, валюта и часовой пояс.  
    
## <a name="to-set-personal-options"></a>Настройка личных параметров  
  
1.  Нажмите кнопку **Параметры** ![кнопка "Параметры" для Power Apps](media/options-button.png "Кнопка "Параметры" для Power Apps") в правом верхнем углу экрана.  
  
2.  Выберите **Параметры персонализации**.  
  
3.  При необходимости укажите сведения.   
  
4.  Закончив, нажмите кнопку **OK**.
