# <a name="edit-the-default-filter-of-a-report"></a>Изменение фильтра отчета по умолчанию

Если это отчет SQL Server Reporting Services, который поддерживает предварительную фильтрацию и имеет фильтр по умолчанию, вы можете изменить фильтр по умолчанию, чтобы отобразить данные, которые должны отображаться в отчете. Этот фильтр используется каждый раз, когда пользователь запускает отчет.

1. На панели навигации слева выберите область отчетов
2. Выберите отчет и на панели команд выберите **Изменить фильтр по умолчанию**.

     > [!div class="mx-imgBorder"]
     > ![Изменить фильтр отчета по умолчанию](media/edit_filter.png "Изменить фильтр отчета по умолчанию")
  
3. Изменение условий фильтра.  
  
   Критерии группируются по типам записей, которые можно использовать в фильтре, например **Учетные записи** или **Контакты**.  
  
   ### <a name="to-edit-an-existing-row"></a>Изменение имеющейся строки
   1. Выберите реляционный оператор запроса и выберите оператор или выберите подчеркнутое значение и введите новое значение.  
  
   2. Выберите реляционный оператор запроса и выберите оператор.  
  
   Чтобы добавить строку условий, выполните следующие действия.  

   1.  Нажмите **Выбрать** и укажите поле для фильтрации.  

   2.  Выберите реляционный оператор запроса и выберите оператор.  

   3.  Выберите **Ввести значение** и введите значение для фильтрации. Для некоторых значений можно нажать кнопку **Выбрать или изменить значения для этого поля** ![кнопку с многоточием](media/ellipsis-button.png "кнопку с многоточием"), чтобы открыть диалоговое окно **Выбрать значения** и выбрать нужное значение.  

   ### <a name="to-group-criteria"></a>Группирование критериев
   Необходимо выбрать две строки или более для одного типа записей. Однако строки со значениями полей из различных типов записей, таких как **Учетная запись** и **Контакт**, нельзя сгруппировать.  

   1.  Для каждой строки, которую необходимо сгруппировать, в режиме подробных сведений нажмите кнопку **меню параметров** для этой строки, а затем выберите **Выбрать строку**.  

   2.  На панели инструментов фильтра выберите **Группа И** или **Группа ИЛИ**.  

   3.  Чтобы удалить строку из группы, нажмите кнопку **меню параметров** для этой строки, а затем выберите **Удалить**.  

   4.  Чтобы выбрать группу, нажмите кнопку **меню параметров** для этой группы, а затем выберите **Выбрать группу**.  

   5.  Чтобы добавить предложение условия в группу, нажмите кнопку **меню параметров** для этой группы, выберите **Добавить предложение**, а затем выберите поле, реляционный оператор запроса и значение.  

   6.  Чтобы отменить выбор ранее выбранной группы, нажмите кнопку **меню параметров** для этой группы, а затем выберите **Отменить выбор группы**.  

   7.  Чтобы отменить группирование группы, нажмите кнопку **меню параметров** для этой группы, а затем выберите **Разгруппировать группу**.  

   8.  Чтобы изменить группу **Группа И** на группу **Группа ИЛИ** или группу **Группа ИЛИ** на группу **Группа И**, нажмите кнопку **меню параметров** для этой группы, а затем выберите **Изменить на ИЛИ** или **Изменить на И**.  

   > [!TIP]
   > - Чтобы очистить все условия и начать заново, на панели инструментов фильтра выберите **Очистить**, а затем выберите **Подтвердить**.  
   > - Чтобы удалить строку, нажмите кнопку **меню параметров** для этой строки, а затем выберите **Удалить**.  
  
4. Когда все будет готово, выберите **Сохранить фильтр по умолчанию**.
