# <a name="edit-the-default-filter-of-a-report"></a>Изменение фильтра отчета по умолчанию

Если отчет является SQL Server Reporting Services отчетом, включает предварительную фильтрацию и имеет фильтр по умолчанию, можно изменить фильтр по умолчанию, чтобы отобразить данные, которые должны отображаться в отчете. Этот фильтр используется каждый раз, когда любой пользователь запускает отчет.

1. В области навигации слева выберите область отчеты.
2. Выберите отчет и на панели коммбар выберите **изменить фильтр по умолчанию**.

     > [!div class="mx-imgBorder"]
     > ![Изменить фильтр отчета по умолчанию](media/edit_filter.png "Изменить фильтр отчета по умолчанию")
  
3. Измените условия фильтра.  
  
   Критерии группируются по типам записей, которые можно использовать в фильтре, например в учетных **записях** или **контактах**.  
  
   ### <a name="to-edit-an-existing-row"></a>Изменение существующей строки
   1. Выберите реляционный оператор запроса и выберите оператор или выберите подчеркнутое значение и введите новое значение.  
  
   2. Выберите реляционный оператор запроса и выберите оператор.  
  
   Чтобы добавить строку условий, выполните следующие действия.  

   1.  Выберите **выбрать**и укажите поле для фильтрации.  

   2.  Выберите реляционный оператор запроса и выберите оператор.  

   3.  Выберите **ввести значение**и введите значение для фильтрации. Для некоторых значений можно нажать кнопку **выбрать или изменить значения для этого поля** кнопка с многоточием ![](media/ellipsis-button.png ", чтобы открыть") диалоговое окно **Выбор значений** и выбрать нужное значение.  

   ### <a name="to-group-criteria"></a>Для группирования критериев
   Необходимо выбрать две или более строк для одного типа записей. Однако строки со значениями полей из различных типов записей, таких как типы записей **учетных записей** и **контактов** , не могут быть сгруппированы.  

   1.  Для каждой строки, которую необходимо сгруппировать, в режиме подробных сведений нажмите кнопку **меню Параметры** для этой строки, а затем выберите команду **выбрать строку**.  

   2.  На панели инструментов Фильтр выберите **группы и** или **Группа или**.  

   3.  Чтобы удалить строку из группы, нажмите кнопку **меню Параметры** для этой строки, а затем выберите **Удалить**.  

   4.  Чтобы выбрать группу, нажмите кнопку **меню Параметры** для этой группы, а затем выберите **команду выбрать группу**.  

   5.  Чтобы добавить условие отбора к группе, нажмите кнопку **меню Параметры** для этой группы, выберите пункт **Добавить предложение**, а затем выберите поле, реляционный оператор запроса и значение.  

   6.  Чтобы отменить выбор ранее выбранной группы, нажмите кнопку **меню Параметры** для этой группы, а затем выберите отменить **Выбор группы**.  

   7.  Чтобы разгруппировать группу, нажмите кнопку **меню Параметры** для этой группы, а затем выберите команду разгруппировать.  

   8.  Чтобы изменить **группу и** группу на группу **или** группу, а также группу **или** группу для **группы и** группы, нажмите кнопку **меню Параметры** для этой группы, а затем выберите команду **изменить на или** или **изменить на и**.  

   > [!TIP]
   > - Чтобы очистить все условия и начать заново, на панели инструментов Фильтр выберите **очистить**и нажмите кнопку **подтвердить**.  
   > - Чтобы удалить строку, нажмите кнопку **меню Параметры** для этой строки, а затем выберите **Удалить**.  
  
4. По завершении нажмите кнопку **сохранить фильтр по умолчанию**.
