# <a name="how-do-i-find-and-run-apps"></a>Поиск и запуск приложений

Теперь стоящие перед вами задачи не обязательно целиком решать вручную. PowerApps позволяет вам быстро выполнять работу, создавая и используя приложения на основе модели или холста. Создав такое приложение, вы можете сделать его доступным другим пользователям своей организации. 

[Приложения на основе холста](/powerapps/maker/canvas-apps/getting-started) позволяют гибко контролировать доступные пользователям функции. Разработчики могут использовать шаблоны или настраивать приложения любым удобным способом. Как и следует из их названия, эти приложения дают вам готовый холст, на котором можно создавать любые возможности, необходимые пользователям вашей организации.

Поскольку они позволяют создавать практически что угодно, для готовых приложений на основе холста обычно бывает мало справочной документации. Если вам нужна по ним помощь, обратитесь к автору приложения или администратору PowerApps.

[Приложения на основе модели](/powerapps/maker/model-driven-apps/model-driven-app-overview) используют [единый интерфейс](unified-interface.md), обеспечивающий удобство и быстроту работы. Эти приложения можно запускать в веб-браузере и на популярных мобильных устройствах. Они содержат множество компонентов, включая панели мониторинга, формы, представления, диаграммы и бизнес-процессы, сочетание которых позволяет существенно упростить работу. Приложения на основе модели создаются и моделируются на платформе Microsoft Dynamics 365 Customer Engagement, поэтому их интерфейс напоминает и другие ее [приложения](/dynamics365/customer-engagement/basics/business-apps-dynamics-365).

Этот раздел содержит сведения о том, как находить и запускать приложения на основе холста и модели в различных средах, а также сведения о принципах работы в приложениях на основе модели: [Использование приложений на основе модели](use-model-driven-apps.md).


## <a name="whats-required-to-run-apps"></a>Что необходимо для запуска приложений?
- Проверьте, используете ли вы [поддерживаемую платформу устройств и браузер](../maker/canvas-apps/limits-and-config.md).
- Для запуска приложений на основе холста скачайте и установите PowerApps из [App Store](https://itunes.apple.com/app/powerapps/id1047318566?mt=8) или [Google Play](https://play.google.com/store/apps/details?id=com.microsoft.msapps).
- Запуск приложений на основе модели:
    - для запуска на телефоне скачайте и установите Dynamics 365 для телефонов из [App Store](https://itunes.apple.com/app/dynamics-crm-for-phones/id1003997947?ls=1&mt=8) или [Google Play](https://play.google.com/store/apps/details?id=com.microsoft.crm.crmphone); 
    - для запуска на планшете скачайте и установите Dynamics 365 для планшетов из [App Store](https://itunes.apple.com/app/microsoft-dynamics-crm/id678800460?mt=8), [Google Play](https://play.google.com/store/apps/details?id=com.microsoft.crm.crmtablet) или [Microsoft Store](https://www.microsoft.com/store/p/microsoft-dynamics-365/9nblggh4rfqp).
