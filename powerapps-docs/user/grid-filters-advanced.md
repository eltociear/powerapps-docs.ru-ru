# <a name="edit-or-create-personal-views-using-advanced-grid-filters"></a>Создание личных представлений с использованием дополнительных фильтров сетки 

С помощью дополнительных параметров фильтрации можно создать личное представление, чтобы просматривать важные для вас записи. Дополнительные параметры фильтрации позволяют создавать разнообразные представления — от простых до сложных. Они также позволяют добавлять условия группы и вложенные условия в фильтры.


> [!NOTE]
> Дополнительные параметры фильтрации доступны только в версиях на английском языке.

При создании и сохранении личного представления оно отображается в списке личных представлений в разделе **Мои представления**.

> [!div class="mx-imgBorder"]
> ![Личные представления](media/my_peronsal_view.png "Личные представления")


## <a name="see-the-current-view-definition"></a>Просмотр определения текущего представления

Чтобы увидеть, какие фильтры применены к текущему представлению, выберите представление, а затем **Фильтр** ![значок фильтра](media/commandbar_filter_icon.png "Значок фильтра"). Откроется окно построителя выражений с определением представления по умолчанию.

> [!div class="mx-imgBorder"] 
> ![Определение текущего представления](media/current_view_def.gif "На этом рисунке показано, как просмотреть фильтры для представления")

## <a name="add-conditions-to-filters"></a>Добавление условий к фильтрам

1. Чтобы изменить текущее представление и добавить дополнительные фильтры, выберите представление, а затем выберите **Фильтр** ![значок фильтра](media/commandbar_filter_icon.png "Значок фильтра").
2. На экране **Дополнительные фильтры** в построителе выражений добавьте условия к фильтрам. Дополнительные сведения о добавлении условий см. в этом [разделе](https://docs.microsoft.com/powerapps/maker/model-driven-apps/create-edit-view-filters#add-conditions-to-a-filter).
3. Когда все будет готово, нажмите кнопку **Применить**. 

   > [!div class="mx-imgBorder"] 
   > ![Добавление фильтров](media/add_filters.gif "На этом рисунке показано, как добавить фильтры с помощью построителя выражений")

### <a name="add-grouped-or-nested-conditions"></a>Добавление условий группы или вложенных условий

Чтобы просматривать более детализированные данные, можно добавить в фильтры условия группы или вложенные условия. Дополнительные сведения см. в разделе [Добавление условия группы в фильтр](https://docs.microsoft.com/powerapps/maker/model-driven-apps/create-edit-view-filters#add-a-group-condition-to-a-filter).

   > [!div class="mx-imgBorder"] 
   > ![Добавление условия группы или вложенного условия](media/group_condition.gif "На этом рисунке показано, как добавить условия группы или вложенные условия")

### <a name="clear-filters"></a>Очистка фильтров

Чтобы очистить примененные фильтры и вернуться к исходному определению, щелкните **Очистить фильтр** ![значок очистки фильтра](media/clear_filter_icon.png "Значок очистки фильтра").

### <a name="save-your-personal-view"></a>Сохранение личного представления

Звездочка рядом с именем представления указывает, что представление не сохранено. 

   > [!div class="mx-imgBorder"] 
   > ![Несохраненное представление](media/unsaved_view.png "Несохраненное представление")

1. Чтобы сохранить личное представление, на панели команд выберите меню **Дополнительно** ![значок меню "Дополнительно"](media/commandbar_more_icon.png "Значок меню "Дополнительно""). 
2. Выберите элементы **Создать представление** > **Save filter as new view** (Сохранить фильтр как новое представление).
3. В диалоговом окне **Save as new view** (Сохранить как новое представление) введите имя и описание представления, а затем щелкните**Сохранить**.
4. Представление появится в списке личных представлений в разделе **Мои представления**.

   > [!div class="mx-imgBorder"] 
   > ![Сохранение личного представления](media/save_personal_view.gif "На этом рисунке показано, как сохранить личное представление")
