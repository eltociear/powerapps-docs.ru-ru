# <a name="create-edit-or-save-an-advanced-find-search"></a>Создание, изменение и сохранение поиска с расширенным поиском

Найдите нужные записи в приложении с помощью команды Расширенный поиск. Кроме того, с помощью расширенного поиска можно подготовить данные для экспорта в Office Excel, чтобы анализировать, суммировать или объединять данные, а также создавать сводные таблицы для просмотра данных с разных точек зрения.  
  
1. На панели команд нажмите кнопку **Расширенный поиск** .
  
2. Укажите, что искать.  
  
   1.  В списке **Искать** выберите тип записи.  
  
   2.  Щелкните **выбрать** , чтобы определить критерий поиска: поле (например, имя учетной записи или город), оператор запроса отношения (часть выражения, например "is равно" или "Contains", который определяет, как заданный атрибут должен сравниваться со значением). искомые значения (например, "Сиэтл" или "Электронная почта").  
  
       Можно выбрать поля из типа текущей записи или из связанных записей. Например, у учетной записи может быть много связанных записей контактов.  
  
       В нижней части списка **выбора** в связанном разделе  отображаются связанные типы записей. Например, большинство типов записей имеют связанный тип записи Notes. Чтобы выбрать поля из связанного типа записи, выберите поле и отобразится новая ссылка **SELECT** .  



3. Укажите столбцы, которые необходимо включить в результаты поиска.  
  
   1.  Выберите **изменить столбцы**, а затем выберите **Добавить столбцы**.  
  
   2.  Выберите тип записи, включающий столбцы, которые необходимо добавить.  
  
   3.  Выберите столбцы, которые необходимо добавить, а затем нажмите кнопку **ОК**.  
  
4. Укажите порядок сортировки.  
  
   1.  Выберите **изменить столбцы**.  
  
   2.  Выберите **настроить сортировку**.  
  
   3.  Укажите столбец для сортировки, укажите порядок сортировки и нажмите кнопку **ОК**.  
  
   4.  Выберите **ОК**.  
  
5. Выберите **результаты**.
