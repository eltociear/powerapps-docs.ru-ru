# <a name="preview-insert-an-email-template"></a>Предварительная версия: Вставка шаблона сообщения электронной почты

Вставка шаблона сообщения электронной почты — это функция раннего доступа. Вы можете включить эти функции в своей среде. Это позволит вам протестировать эти функции, а затем принять их в своих средах. Сведения о том, как включить эти функции, см. в разделе [Согласие на обновления первой волны выпуска 2020](https://docs.microsoft.com/power-platform/admin/opt-in-early-access-updates).

Чтобы быстро создавать и отправлять сообщения электронной почты, можно использовать шаблон сообщения электронной почты (предварительно отформатированное письмо). Вы можете вставить шаблон при создании сообщения электронной почты, выбрав **Вставить шаблон** на панели команд. Список доступных шаблонов отображается в окне **Шаблоны сообщений электронной почты**. В разделе **Недавно использованные** отображаются четыре недавно использовавшихся шаблона. В разделе **Все шаблоны** отображается список всех готовых шаблонов электронной почты (глобальных и зависящих от сущностей) в алфавитном порядке. Глобальные шаблоны отображаются с типом "Пользователь". Если вы создали пользовательский шаблон сообщения электронной почты, он также будет доступен здесь. Дополнительные сведения о создании пользовательского шаблона сообщения электронной почты см. в разделе [Создание шаблонов для электронной почты](https://docs.microsoft.com/power-platform/admin/create-templates-email).

Чтобы просмотреть шаблоны определенного языка, выберите язык в списке **Язык**. Можно либо найти шаблон, либо просмотреть список и выбрать его. При выборе шаблона сообщения в правой части окна отображается предварительный просмотр. В предварительном просмотре отображается содержимое, чтобы можно было выбрать шаблон, который лучше всего соответствует вашим потребностям. После вставки шаблона сообщения электронной почты можно изменить его содержимое, а затем отправить.

> [!NOTE]
> Поиск не поддерживает регулярные выражения и работает только с именем шаблона.

**Вставка шаблона сообщения электронной почты**

1.  В редакторе электронной почты выберите **Вставить шаблон** на панели команд.

     > [!div class="mx-imgBorder"]
     > ![Кнопка "Вставить шаблон"](media/insert-email-template-button.png "Кнопка "Вставить шаблон"") 

    Откроется окно **Шаблоны сообщений электронной почты**.

2.  Чтобы просмотреть шаблоны другого языкового стандарта, выберите язык в списке **Язык**. Шаблоны загружаются в соответствии с выбранным языком.    

3.  Найдите нужный шаблон. Выберите шаблон и просмотрите содержимое шаблона.

4.  При необходимости можно нажать кнопку со стрелкой вниз на имени шаблона, чтобы просмотреть описание его содержимого.

5.  Выберите **Применить шаблон**, чтобы вставить содержимое в сообщение электронной почты.

     > [!div class="mx-imgBorder"]
     > ![Окно "Шаблоны сообщений электронной почты"](media/email-templates-window.png "Окно "Шаблоны сообщений электронной почты"")

При попытке вставить шаблон сообщения электронной почты на устройстве с меньшим размером экрана вы увидите только возможность поиска и выбора шаблона.

> [!div class="mx-imgBorder"]
> ![Поиск шаблона](media/search-template.png "Поиск шаблона")
