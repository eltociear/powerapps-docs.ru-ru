# <a name="preview-insert-an-email-template"></a>Предварительный просмотр: вставка шаблона сообщения электронной почты

Вставка шаблона электронной почты — это функция раннего доступа. Вы можете включить эти функции в своей среде заранее. Это позволит вам протестировать эти функции и затем принять их в своих средах. Сведения о том, как включить эти функции, см. в статье [согласие на обновление 2020 выпуска 1](https://docs.microsoft.com/power-platform/admin/opt-in-early-access-updates).

Чтобы быстро создавать и отправлять сообщения электронной почты, можно использовать шаблон сообщения электронной почты (предварительно отформатированное письмо). Вы можете вставить шаблон при создании сообщения электронной почты, выбрав команду **Вставить шаблон** на панели команд. Список доступных шаблонов отображается в окне **шаблоны электронной почты** . В разделе **недавно использованных** отображаются четыре недавно использовавшиеся шаблоны. В разделе **все шаблоны** отображается список всех готовых шаблонов электронной почты (глобальных и зависящих от сущностей) в алфавитном порядке. Глобальные шаблоны отображаются как тип пользователь. Если вы создали пользовательский шаблон сообщения электронной почты, он также будет доступен здесь. Сведения о создании пользовательского шаблона электронной почты см. в разделе [Создание шаблонов для электронной почты](https://docs.microsoft.com/power-platform/admin/create-templates-email).

Чтобы просмотреть шаблоны определенного языка, выберите язык в списке **язык** . Можно либо найти шаблон, либо просмотреть список и выбрать его. При выборе шаблона сообщения в правой части окна отображается предварительный просмотр. В предварительной версии отображается содержимое, чтобы можно было выбрать шаблон, который наилучшим образом соответствует вашим потребностям. После вставки шаблона сообщения можно изменить его содержимое, а затем отправить сообщение электронной почты.

> [!NOTE]
> Поиск не поддерживает регулярные выражения и работает только с именем шаблона.

**Вставка шаблона сообщения электронной почты**

1.  В редакторе электронной почты выберите команду **Вставить шаблон** на панели команд.

     > [!div class="mx-imgBorder"]
     > ![Кнопка "вставить шаблон"](media/insert-email-template-button.png "Кнопка "вставить шаблон"") 

    Откроется окно **шаблоны электронной почты** .

2.  Чтобы просмотреть шаблоны другого языкового стандарта, выберите язык в списке **язык** . Шаблоны загружаются в соответствии с выбранным языком.    

3.  Найдите нужный шаблон. Выберите шаблон и просмотрите содержимое шаблона.

4.  При необходимости можно нажать кнопку со стрелкой вниз на имени шаблона, чтобы просмотреть описание его содержимого.

5.  Щелкните **Применить шаблон** , чтобы вставить содержимое в сообщение электронной почты.

     > [!div class="mx-imgBorder"]
     > ![Окно шаблонов сообщений электронной почты](media/email-templates-window.png "Окно шаблонов сообщений электронной почты")

При попытке вставить шаблон сообщения электронной почты на устройстве с меньшим размером экрана вы увидите только возможность поиска и выбора шаблона.

> [!div class="mx-imgBorder"]
> ![Поиск шаблона](media/search-template.png "Поиск шаблона")
