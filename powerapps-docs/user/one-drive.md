# <a name="use-onedrive-for-business"></a>Использование OneDrive для бизнеса 

С помощью OneDrive для бизнеса можно создавать личные документы в приложениях Common Data Service и управлять ими. Дополнительные сведения:  [Что такое OneDrive для бизнеса?](https://support.office.com/article/What-is-OneDrive-for-Business-187f90af-056f-47c0-9656-cc0ddca7fdc2)


Чтобы можно было использовать OneDrive для бизнеса, системный администратор должен включить эту возможность. Дополнительные сведения:

-   [Обращение к администратору или в службу поддержки](find-admin.md)  

-   [Включение OneDrive для бизнеса](https://docs.microsoft.com/power-platform/admin/enable-onedrive-for-business)  


## <a name="the-first-time-you-view-your-documents"></a>При первом просмотре документов  

1. Откройте запись и перейдите в представление **Сетка, связанная с документом**. Например, откройте запись контакта.

2.  В открытой записи перейдите на вкладку **Связанные** и выберите **Документы**.

     > [!div class="mx-imgBorder"]
     > ![Открытие вкладки "Документы" в записи](media/onedrive_nav.png "Открытие вкладки "Документы" в записи")

3.  Выберите **Расположение документа** > **OneDrive**.

     > [!div class="mx-imgBorder"]
     > ![Открытие вкладки "Документы" и выбор OneDrive](media/onedrive_menu.png "Открытие вкладки "Документы" и выбор OneDrive")

4. После включения OneDrive для бизнеса при переходе на вкладку **Документы** для просмотра документов в Common Data Service и отправки файлов в OneDrive, а также при попытке создать документ или папку будет открываться представленное ниже диалоговое окно.  

    > [!div class="mx-imgBorder"]
    > ![Изменение папки OneDrive](media/setup_onedrive.png "Изменение папки OneDrive")  

5. Выберите **Изменить расположение папки**, чтобы выбрать новое расположение для хранения документов OneDrive, или нажмите кнопку **Продолжить**, чтобы принять расположение папки по умолчанию.

  
## <a name="view-existing-onedrive-documents"></a>Просмотр существующих документов OneDrive 
 
1. Откройте запись и перейдите в представление **Сетка, связанная с документом**. Например, откройте запись контакта.

2. В открытой записи перейдите на вкладку **Связанные** и выберите **Документы**.
 
    > [!div class="mx-imgBorder"]
    > ![Открытие вкладки "Документы" в записи](media/onedrive_nav.png "Открытие вкладки "Документы" в записи")
 
3. Выберите **Расположение документа**, чтобы отфильтровать список документов.

    > [!div class="mx-imgBorder"]
    > ![Открытие расположения документа](media/onedrive_doc_location.png "Открытие расположения документа")

4.  Выберите расположение, как описано в таблице ниже.  

   |    Расположение документа      |  Описание                                   |
   |---------------------------|------------------------------------------------|
   |      OneDrive             | Документы, хранящиеся в OneDrive для бизнеса      |
   | Документы на сайте по умолчанию | Документы, хранящиеся на сайте SharePoint по умолчанию  |
   | Мне предоставлен доступ            | Связанные с этой записью документы, к которым вам предоставили доступ другие пользователи<!--note from editor: Edit okay? I haven't seen an "app record" defined.-->    |
   |  Все расположения            |     Все расположения документов, связанные с этой записью     |

5. После выбора расположения вы увидите сохраненные в нем документы.

## <a name="create-a-new-document-and-save-it-to-onedrive"></a>Создание документа и его сохранение в OneDrive

1. Откройте запись и перейдите в представление **Сетка, связанная с документом**. Например, откройте запись контакта.

2. В открытой записи перейдите на вкладку **Связанные** и выберите **Документы**.
 
    > [!div class="mx-imgBorder"]
    > ![Открытие вкладки "Документы" в записи](media/onedrive_nav.png "Открытие вкладки "Документы" в записи")

2. Выберите **Расположение документа** и измените расположение на **OneDrive**.

3. Выберите **Создать**, а затем выберите тип документа, например PowerPoint или Word. 

    > [!div class="mx-imgBorder"]
    > ![Создание документа](media/onedrive_new_doc.png "Создание документа")

4. Введите имя документа и нажмите кнопку **Сохранить**.  


## <a name="things-to-consider"></a>Полезная информация 

Учитывайте приведенные ниже особенности, касающиеся использования OneDrive для бизнеса в Common Data Service.

- Папки хранилища OneDrive создаются на текущем языке пользователя Common Data Service. При смене языка папки начнут создаваться на новом языке. Старые папки останутся на прежнем языке.  

- После предоставления общего доступа к документам в OneDrive они могут становиться доступными для других пользователей с задержкой.
