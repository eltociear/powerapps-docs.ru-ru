# <a name="take-notes-by-using-onenote"></a>Создание заметок с помощью OneNote 

С помощью OneNote можно добавлять и просматривать заметки, идеи, планы и результаты исследований в представлении **Сетка, связанная с документом** записи в Common Data Service.

Чтобы можно было использовать OneNote, системный администратор должен включить эту возможность. Дополнительные сведения:

-   [Обращение к администратору или в службу поддержки](find-admin.md)  

-   [Настройка интеграции с OneNote](https://docs.microsoft.com/power-platform/admin/set-up-onenote-integration-in-dynamics-365)  

## <a name="start-a-new-onenote-notebook"></a>Создание записной книжки OneNote

1. Откройте запись и перейдите в представление **Сетка, связанная с документом**. Например, откройте запись контакта.

2. В открытой записи перейдите на вкладку **Связанные** и выберите **Документы**.
 
    > [!div class="mx-imgBorder"]
    > ![Открытие вкладки "Документы" в записи](media/onedrive_nav.png "Открытие вкладки "Документы" в записи")

3. Выберите **Расположение документа**, а затем выберите место для сохранения записной книжки. Дополнительные сведения см. в разделе [Просмотр существующих документов OneDrive](one-drive.md#view-existing-onedrive-documents).

4. Щелкните **Создать**, а затем выберите **OneNote**. 

    > [!div class="mx-imgBorder"]
    > ![Создание записной книжки OneNote](media/onenote.png "Создание записной книжки OneNote")

5. Введите имя записной книжки и нажмите кнопку **Сохранить**. Будет создана и открыта записная книжка.

## <a name="open-an-existing-onenote-notebook"></a>Открытие существующей записной книжки OneNote

1. Откройте запись и перейдите в представление **Сетка, связанная с документом**. Например, откройте запись контакта.

2. В открытой записи перейдите на вкладку **Связанные** и выберите **Документы**.
 
    > [!div class="mx-imgBorder"]
    > ![Открытие вкладки "Документы" в записи](media/onedrive_nav.png "Открытие вкладки "Документы" в записи")

3. Выберите **Расположение документа**, а затем выберите место хранения записной книжки. Например, выберите **OneDrive**, если записная книжка хранится в OneDrive. Дополнительные сведения: [Просмотр существующих документов OneDrive](one-drive.md#view-existing-onedrive-documents)

4. Выберите записную книжку, чтобы начать добавлять заметки.
 
    > [!div class="mx-imgBorder"]
    > ![Открытие существующей записной книжки OneNote](media/existing_onenote.png "Открытие существующей записной книжки OneNote")
