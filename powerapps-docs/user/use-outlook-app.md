# <a name="use-dynamics-365-app-for-outlook"></a>Использование приложения Dynamics 365 для Outlook

Используйте приложение Dynamics 365 для Outlook для управления данными Dynamics 365 с помощью Outlook на рабочем столе, Интернете или телефоне. При установке приложения Dynamics 365 для Outlook в зависимости от установленной версии приложения вы увидите Common Data Service область или окно рядом с выбранным сообщением электронной почты Outlook или при создании сообщения электронной почты или настройке собрания. книж.


   > [!div class="mx-imgBorder"] 
   > ![Приложение Dynamics 365 для Outlook](media/outlookapp.png "Приложение Dynamics 365 для Outlook")

## <a name="what-dynamics-365-app-for-outlook-offers"></a>Какое приложение Dynamics 365 для Outlook предлагает?

Приложение Dynamics 365 для Outlook позволяет выполнять следующие действия:  
  
- Просмотр сведений о Common Data Service контактах и учетных записях во время работы в Outlook. Эти сведения можно просмотреть в контексте сообщения электронной почты, собрания или встречи. Например, просмотрите номера телефонов, названия компаний, последние и последующие действия и последние записи. 
  
- Связывание сообщений электронной почты, собраний и встреч с одной записью одним щелчком мыши. Например, свяжите сообщение электронной почты с определенной учетной записью или контактом. Приложение Dynamics 365 для Outlook также поддерживает пользовательские сущности.  
  
- Непосредственное открытие записей для поиска или ввода более подробных сведений.  
  
- Добавьте действие телефонный звонок, задачу или встречу.  
  
- Создание новой записи для любой сущности (тип записи).  
  
- Следите за контактами Outlook в Common Data Service.  

Сведения об использовании приложения Dynamics 365 для Outlook см. в статье [пользовательское приложение dynamics 365 для Outlook](https://docs.microsoft.com/dynamics365/customer-engagement/outlook-app/dynamics-365-app-outlook-user-s-guide).
