# <a name="use-dynamics-365-app-for-outlook"></a>Использование приложения Dynamics 365 для Outlook

Используйте приложение Dynamics 365 для Outlook для управления данными Dynamics 365 с помощью Outlook на настольном компьютере, в браузере или на телефоне. При установке приложения Dynamics 365 для Outlook, в зависимости от установленной версии приложения, вы увидите область или окно Common Data Service рядом с выбранным сообщением электронной почты Outlook, а также при создании сообщения электронной почты или настройке собрания.


   > [!div class="mx-imgBorder"] 
   > ![Приложение Dynamics 365 для Outlook](media/outlookapp.png "Приложение Dynamics 365 для Outlook")

## <a name="what-dynamics-365-app-for-outlook-offers"></a>Что есть в приложении Dynamics 365 для Outlook?

Приложение Dynamics 365 для Outlook имеет следующие возможности.  
  
- Просмотр сведений о контактах и учетных записях Common Data Service во время работы в Outlook. Эти сведения можно просмотреть в контексте сообщения электронной почты, собрания или встречи. Например, вы можете просмотреть номера телефонов, названия компаний, последние и последующие действия или последние записи. 
  
- Связывание сообщений электронной почты, собраний и встреч с записью одним щелчком мыши. Например, вы можете связать сообщение электронной почты с определенной учетной записью или контактом. Приложение Dynamics 365 для Outlook также поддерживает пользовательские сущности.  
  
- Прямое открытие записей для поиска или ввода более подробных сведений.  
  
- Добавьте действие — телефонный звонок, задачу или встречу.  
  
- Создание новой записи для любой сущности (тип записи).  
  
- Отслеживание контактов Outlook в службе Common Data Service.  

Сведения об использовании приложения Dynamics 365 для Outlook см. в разделе [Руководство пользователя приложения Dynamics 365 для Outlook](https://docs.microsoft.com/dynamics365/customer-engagement/outlook-app/dynamics-365-app-outlook-user-s-guide).
