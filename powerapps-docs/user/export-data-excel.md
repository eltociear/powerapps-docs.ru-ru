# <a name="export-data-to-excel"></a>Экспорт данных в Excel

Вы хотите анализировать данные и преобразовывать их в элементы, которые помогут вам увеличить объем продаж? Теперь это можно сделать при экспорте данных в Excel или Excel Online. Кроме того, вы можете анализировать большие наборы данных, так как можно экспортировать до 100 000 строк.
  
Можно экспортировать статические или динамические листы, которые затем можно импортировать обратно в приложение. Если требуются более сложные функции, можно экспортировать динамическую сводную таблицу, что упрощает организацию и сведение данных.  
  
Данные можно экспортировать в стандартный файл Excel, который можно использовать на любом устройстве, например на телефоне, планшете или настольном компьютере. Данные экспортируются в том же формате, который вы видите в приложении. Текст останется текстом, числа — числами, а даты — датами. Однако при экспорте данных из приложения в Excel формат некоторых ячеек может измениться. В следующей таблице представлены сведения о том, как будут отображаться данные в приложениях и как изменится формат ячеек при экспорте данных в Excel.  
  
## <a name="cell-format-when-data-is-exported-from-model-driven-apps"></a>Формат ячеек при экспорте данных из приложений на основе модели
  
| Формат данных в приложениях на основе модели |                                            Формат ячеек в Excel                                             |
|----------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------|
|            Текст, символ тикера, телефон, набор параметров и поиск            |                                                       Отображается как текст, набор параметров становится раскрывающимся списком                                                       |
|                                 Электронная почта, URL-адрес                                 |                                                                        Отображается как общее                                                                         |
|                                   число;                                   |                                                             Отображается как число без разделителя групп                                                             |
|                                  Валюта                                  |                                                         Отображается как число и не включает знак доллара ($)                                                         |
|                          Только дата, дата и время                          |                                                                       Отображается только как дата                                                                        |
|                       Вычисляемые и сведенные поля                        | Можно редактировать в Excel, но нельзя импортировать обратно в Power Apps |
|                               Защищенные поля                               | Можно редактировать в Excel, но нельзя импортировать обратно в Power Apps |

При экспорте данных в Excel (XLSX-формат) с последующим добавлением или изменением столбцов импортировать данные обратно в Dynamics 365 нельзя. Эта возможность не поддерживается для формата файла XLSX.  
  
Если вы используете Excel 2010, то при экспорте данных из области учетных записей может появиться следующее сообщение об ошибке: 
 
`The file is corrupt and cannot be opened.`  
  
Сообщение об ошибке возникает из-за настройки в Excel. Чтобы устранить эту проблему, выполните следующие действия:  
  
1. Откройте Excel 2010.  
  
2. Щелкните **Файл** > **Параметры**.  
  
3. Перейдите в **Центр управления безопасностью** > **Параметры центра управления безопасностью**.  
  
4. Выберите **Защищенное представление**, а затем снимите флажки для первых двух параметров.  
  
5. Нажмите **OK** и закройте диалоговое окно **Параметры**.
