# <a name="export-data-to-excel"></a>Экспорт данных в Excel

Нужно ли анализировать данные и преобразовывать их в элементы, которые помогут вам сократить объем продаж? Теперь это можно сделать при экспорте данных в Excel или Excel Online. Кроме того, анализ больших наборов данных не является проблемой, так как можно экспортировать до 100 000 строк.
  
Можно экспортировать статические листы или динамические листы, которые можно импортировать обратно в приложение. Если требуются более сложные функции, можно экспортировать динамическую сводную таблицу, что упрощает организацию и сведение данных.  
  
Данные можно экспортировать в стандартный файл Excel, который можно использовать на любом устройстве, например на телефоне, планшете или настольном компьютере. Данные экспортируются в том же формате, который вы видите в приложении. Текст останется в тексте, числа будут оставаться числами, а даты останутся датами. Однако при экспорте данных из приложения в Excel некоторые форматы ячеек могут измениться. В следующей таблице представлены сведения о том, как увидеть данные в приложениях и изменить формат ячейки при экспорте данных в Excel.

| Формат данных в приложениях, управляемых моделью |                                            Формат ячейки в Excel                                             |
|----------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------|
|            Текст, символ Tick, Телефон, набор параметров и поиск            |                                                       Отображается как текст и набор параметров становятся раскрывающимся списком                                                       |
|                                 Электронная почта, URL-адрес                                 |                                                                        Показано как общее                                                                         |
|                                   Номер                                   |                                                             Показывает число без разделителя групп                                                             |
|                                  Валюта                                  |                                                         Отображается как число и не включает знак доллара ($)                                                         |
|                          Только дата, Дата и время                          |                                                                       Отображается только как Дата                                                                        |
|                       Вычисляемые и сведенные поля                        | Можно редактировать в Excel, но нельзя импортировать обратно в Power Apps |
|                               Защищенные поля                               | Можно редактировать в Excel, но нельзя импортировать обратно в Power Apps |

При экспорте данных в формате Excel (XLSX-формат) с последующим добавлением или изменением столбцов импортировать данные обратно в Dynamics 365 нельзя. Это не поддерживается для формата файла XLSX.  
  
Если вы используете Excel 2010, то при экспорте данных из области учетных записей может появиться следующее сообщение об ошибке: 
 
`The file is corrupt and cannot be opened.`  
  
Сообщение об ошибке возникает из-за настройки в Excel. Чтобы устранить эту проблему, сделайте следующее:  
  
1. Откройте Excel 2010.  
  
2. Перейдите в раздел **параметры** > **файлов** .  
  
3. Перейдите в **центр управления безопасностью** > **Параметры центра управления безопасностью**.  
  
4. Выберите **защищенное представление** и снимите флажки для первых двух параметров.  
  
5. Нажмите кнопку **ОК** , а затем закройте диалоговое окно **Параметры** .
