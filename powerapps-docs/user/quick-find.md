# <a name="using-quick-find-to-search-for-records"></a>Использование быстрого поиска для поиска записей

## <a name="single-entity-quick-find"></a>Быстрый поиск по одной сущности

Быстрый поиск по одной сущности используется для поиска записей только одного типа. Этот параметр поиска доступен в представлении. 

   > [!div class="mx-imgBorder"]
   > ![Быстрый поиск по одной сущности](media/single-quick-find-search-box.png "Поле быстрого поиска по одной сущности") 

## <a name="multiple-entity-quick-find-categorized-search"></a>Быстрый поиск с несколькими сущностями (поиск по категориям)

Быстрый поиск по нескольким сущностям также называется поиском по категориям. 

1.  Чтобы начать поиск по категориям, в верхней панели навигации выберите **Поиск**.  

     > [!div class="mx-imgBorder"]
     > ![Кнопка глобального поиска](media/global-search-button.png "Глобальный поиск")
  
2.  Введите поисковый запрос в поле поиска, а затем выберите **Поиск**. Поиск по категориям возвращает результаты, сгруппированные по типам сущностей, таким как учетные записи или контакты.

     > [!div class="mx-imgBorder"]
     > ![Результаты поиска по категориям](media/categorized-search-results.png "Страница результатов поиска по категориям") 

С помощью поиска по категориям можно выполнять поиск записей, начинающихся с определенного слова, или использовать подстановочный знак.
  
- **Начинается с**: результаты включают записи, которые начинаются с определенного слова. Например, если вы хотите найти "Горнолыжная база", введите **гор** в поле поиска. Если ввести **база**, запись не будет отображаться.  
  
- **Подстановочный знак**: например, *база или *баз\*. 

  > [!NOTE]
  >  Использование подстановочного знака в начале поискового запроса (по одной или нескольким сущностям) может привести к снижению производительности.
  
## <a name="filter-categorized-search-results"></a>Фильтрация результатов поиска по категориям 
  
-   Чтобы отфильтровать результаты по типу записи, выберите тип записи из списка **Фильтровать по**. 

    > [!div class="mx-imgBorder"]
    > ![Фильтрация результатов поиска по категориям](media/filter-categorized-search-results.png "Фильтрация результатов поиска по категориям")  

  
-   Чтобы найти все типы записей, выберите **Нет** в списке **Фильтровать по**.
