# <a name="using-quick-find-to-search-for-records"></a>Использование быстрого поиска для поиска записей

## <a name="single-entity-quick-find"></a>Быстрый поиск по одной сущности

Быстрый поиск с одной сущностью используется для поиска записей только одного типа. Этот параметр поиска доступен в представлении. 

   > [!div class="mx-imgBorder"]
   > ![Быстрый поиск по одной сущности](media/single-quick-find-search-box.png "Поле поиска с одним объектом для быстрого поиска") 

## <a name="multiple-entity-quick-find-categorized-search"></a>Быстрый поиск с несколькими сущностями (Поиск по категориям)

Быстрый поиск с несколькими сущностями также называется поиском по категориям. 

1.  Чтобы начать поиск по категориям, на верхней панели навигации выберите **Поиск**.  

     > [!div class="mx-imgBorder"]
     > ![Кнопка глобального поиска](media/global-search-button.png "Глобальный поиск")
  
2.  Введите искомые слова в поле поиска и нажмите кнопку **Поиск**. Поиск по категориям возвращает результаты, сгруппированные по типам сущностей, таким как учетные записи или контакты.

     > [!div class="mx-imgBorder"]
     > ![Результаты поиска по категориям](media/categorized-search-results.png "Страница «результаты поиска по категориям»") 

С помощью поиска по категориям можно выполнять поиск записей, начинающихся с определенного слова, или использовать подстановочный знак.
  
- **Начинается с**: результаты включают записи, которые начинаются с определенного слова. Например, если вы хотите выполнить поиск по слову "Alpine Ski House", введите **Ахметом** в поле поиска. Если ввести **Ski**, запись не отобразится.  
  
- **Подстановочный знак**: например, * Ski или * Ski\*. 

  > [!NOTE]
  >  Использование подстановочного знака в начале поискового запроса с одним или несколькими сущностями может привести к снижению производительности.
  
## <a name="filter-categorized-search-results"></a>Фильтровать результаты поиска по категориям 
  
-   Чтобы отфильтровать результаты по типу записей, выберите тип записи из списка **Фильтр** . 

    > [!div class="mx-imgBorder"]
    > ![Фильтрация результатов поиска по категориям](media/filter-categorized-search-results.png "Фильтрация результатов поиска по категориям")  

  
-   Чтобы найти все типы записей, выберите **нет** в списке **Фильтр с** .
