# <a name="preview-send-email-using-the-enhanced-email-experience"></a>Предварительная версия: отправка электронной почты с помощью расширенного интерфейса электронной почты

Отправка электронной почты с помощью улучшенной функции электронной почты — это функция раннего доступа. Вы можете включить эти функции в своей среде заранее. Это позволит вам протестировать эти функции и затем принять их в своих средах. Сведения о том, как включить эти функции, см. в статье [согласие на обновление 2020 выпуска 1](https://docs.microsoft.com/power-platform/admin/opt-in-early-access-updates).

Усовершенствованный интерфейс электронной почты в приложениях на основе моделей позволяет создавать сообщения электронной почты, не оставляя записи, над которыми они работают. Благодаря расширенной работе с электронной почтой вы можете:

- Переход к разным страницам без потери содержимого электронной почты.
- Сократите окно электронной почты, чтобы вернуться к записям, над которыми вы работали.
- Разверните всплывающее окно редактора электронной почты, чтобы просмотреть дополнительные параметры электронной почты.
- Одновременно откройте три всплывающих окна для создания сообщений электронной почты.
- Поиск и применение предопределенного шаблона к отправляемому сообщению электронной почты.
- Вставка вложений в сообщение электронной почты.


> [!IMPORTANT]
> - Системные администраторы должны включить расширенные возможности работы с электронной почтой, прежде чем их можно будет использовать.
> - [!INCLUDE[cc_preview_features_definition](../includes/cc-preview-features-definition.md)]  
> - [!INCLUDE[cc_preview_features_expect_changes](../includes/cc-preview-features-expect-changes.md)]
> - [!INCLUDE[cc-preview-features-no-ms-support](../includes/cc-preview-features-no-ms-support.md)]

Создание сообщения электронной почты с помощью расширенного интерфейса:

1. В разделе **временной шкалы** записей, таких как учетная запись или контакт, выберите **+** а затем в разделе **действия**выберите **адрес электронной почты**.

   Откроется всплывающее окно нового сообщения электронной почты. 

   > [!div class="mx-imgBorder"]
   > ![Всплывающее окно "расширенная электронная почта"](media/enhanced-email-pop-up.png "Всплывающее окно "расширенная электронная почта"")

   Поля **от** и **до** автоматически заполняются на основе пользователя, учетной записи и контакта исходной записи.

2. Напишите электронное письмо с нуля или выберите **Вставить шаблон** , чтобы найти и применить шаблон. Дополнительные сведения о вставке шаблона сообщения электронной почты см. в разделе [Вставка шаблона сообщения электронной почты](insert-email-template.md).

3. Выберите **Присоединить файл** , если требуется добавить вложения.

4. Выберите пункт **Вставить подпись** для поиска и добавьте подпись.

5. Когда все будет готово, нажмите кнопку **Отправить**. 

> [!IMPORTANT]
> - Улучшенные возможности электронной почты доступны только для действий электронной почты, созданных в разделе **временной шкалы** приложения, управляемого моделью. 
> - Всплывающее окно "расширенная электронная почта" открывается только в том случае, если высота и ширина экрана имеют размер не менее 400 x 650 пикселей или выше. Если это так, то вместо расширенного интерфейса электронной почты будет предпринята стандартная форма.
