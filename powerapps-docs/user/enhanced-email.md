# <a name="preview-send-email-using-the-enhanced-email-experience"></a>Предварительная версия: Отправка сообщения с помощью улучшенного интерфейса электронной почты

Отправка сообщения с помощью улучшенного интерфейса электронной почты — это функция раннего доступа. Вы можете включить эти функции в своей среде. Это позволит вам протестировать эти функции, а затем принять их в своих средах. Сведения о том, как включить эти функции, см. в разделе [Согласие на обновления первой волны выпуска 2020](https://docs.microsoft.com/power-platform/admin/opt-in-early-access-updates).

Усовершенствованный интерфейс электронной почты в приложениях на основе модели позволяет создавать сообщения электронной почты, находясь в записи, с которой вы работаете. В улучшенном интерфейсе электронной почты вам доступны следующие задачи.

- Переход к разным страницам без потери содержимого электронной почты.
- Свертывание окна электронной почты, чтобы вернуться к записям, над которыми вы работали.
- Развертывание всплывающего окна редактора электронной почты, чтобы просмотреть дополнительные параметры электронной почты.
- Одновременное открытие трех всплывающих окон для создания сообщений электронной почты.
- Поиск и применение предопределенного шаблона к создаваемому сообщению электронной почты.
- Вставка вложений в сообщение электронной почты.


> [!IMPORTANT]
> - Системные администраторы должны включить расширенный интерфейс электронной почты, прежде чем его можно будет использовать.

> - [!INCLUDE[cc_preview_features_expect_changes](../includes/cc-preview-features-expect-changes.md)]

Создание сообщения электронной почты с помощью улучшенного интерфейса.

1. В разделе **Временная шкала** таких записей, как "Учетная запись" или "Контакт", выберите **+** , а затем в разделе **Действия** выберите **Электронная почта**.

   Откроется всплывающее окно нового сообщения электронной почты. 

   > [!div class="mx-imgBorder"]
   > ![Всплывающее окно расширенного интерфейса электронной почты](media/enhanced-email-pop-up.png "Всплывающее окно расширенного интерфейса электронной почты")

   Поля **От кого** и **Кому** заполняются автоматически на основе пользователя, учетной записи и контакта исходной записи.

2. Напишите сообщение с нуля или выберите команду **Вставить шаблон**, чтобы найти и применить шаблон. Дополнительные сведения о вставке шаблона сообщения электронной почты см. в разделе [Вставка шаблона сообщения электронной почты](insert-email-template.md).

3. Выберите **Вложить файл**, если требуется добавить вложения.

4. Выберите **Вставить подпись**, чтобы найти и добавить подпись.

5. После завершения выберите команду **Отправить**. 

> [!IMPORTANT]
> - Расширенный интерфейс электронной почты доступен только для действий электронной почты, созданных в разделе **Временная шкала** приложения на основе модели. 
> - Всплывающее окно расширенного интерфейса электронной почты открывается только в том случае, если высота и ширина экрана составляют не менее 400 x 650 пикселей. Если разрешение меньше, вы перейдете в стандартную форму вместо расширенного интерфейса электронной почты.
