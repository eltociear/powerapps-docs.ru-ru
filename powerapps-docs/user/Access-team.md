# <a name="share-records-using-access-team"></a>Совместное использование записей с помощью группы доступа

Группа доступа предоставляет доступ к записям с помощью их совместного использования. Члены группы доступа пользуются привилегиями в соответствии с их личными ролями безопасности и ролью в командах, членами которых они являются. 

> [!NOTE]
> Ваш администратор должен настроить шаблон группы доступа, прежде чем вы сможете совместно использовать записи с помощью группы доступа. Дополнительные сведения см в статье [О шаблонах рабочих групп](https://docs.microsoft.com/previous-versions/dynamicscrm-2016/admins-customizers-dynamics-365/mt812239(v%3dcrm.8)). 

1. Чтобы предоставить пользователю доступ к записи, выберите тип записи на карте сайта. Например, **Учетные записи**.
2. Откройте в списке записей ту, к которой вы желаете предоставить доступ пользователю.

  > [!div class="mx-imgBorder"]
  > ![My active accounts](media/AccessTeam1.png "Мои активные учетные записи")

3. В разделе **Access Team Members** (Члены группы доступа) выберите **Больше команд** (**…**) > **Добавить пользователя**.

  > [!div class="mx-imgBorder"]
  > ![Add user to Access Team](media/AccessTeam2.png "Добавить пользователя в группу доступа")

 4. В поле поиска введите имя пользователя для поиска пользователя, а затем выберите **Добавить**.
  
  > [!div class="mx-imgBorder"]
  > ![Найти пользователя](media/AccessTeam3.png "Найти пользователя")  
  
 
## <a name="remove-a-user-from-access-teams"></a>Удаление пользователя из группы доступа

 Вы можете закрыть доступ пользователя к записи так же легко, как и предоставить его.
 
1.  Откройте запись, доступ к которой вы желаете закрыть для пользователя.
2.  Во вложенной сетке **Access Team Members** (Члены группы доступа) выберите **Удалить пользователя**.

  > [!div class="mx-imgBorder"]
  > ![Remove user from Access Team](media/AccessTeam4.png "Удалить пользователя с группы доступа")
