# <a name="work-with-reports"></a>Работа с отчетами

Отчеты помогают отслеживать ход выполнения ваших бизнес-задач, показывая ваш прогресс на пути к цели. Вы также можете отслеживать тенденции, которые могут дать преимущество над конкурентами.  

Дополнительные сведения об организации и создании отчетов см. в следующих статьях. [Настройка и организация отчетов](https://docs.microsoft.com/powerapps/maker/model-driven-apps/add-reporting-to-app).
  
## <a name="run-a-report"></a>Запуск отчета  
  
1. На панели навигации слева выберите область отчетов. 
2. Выберите нужный отчет > **Запустить отчет**.  
  
   > [!NOTE]
   >  В диалоговом окне **Средство просмотра отчетов** можно оставить условия поиска как есть или изменить их при необходимости.  
   
   > [!div class="mx-imgBorder"]
   > ![Запуск отчета](media/report-run.png "Запуск отчета")
 
  
## <a name="share-a-report-with-other-users-or-teams"></a>Предоставление общего доступа к отчету другим пользователям или командам    

1. На панели навигации слева выберите область отчетов.  
2. В списке отчетов выберите отчет, к которому необходимо предоставить общий доступ.  
3. На панели команд выберите **Поделиться**.

   > [!div class="mx-imgBorder"]
   > ![Предоставление общего доступа к отчету](media/report-share.png "Предоставление общего доступа к отчету")
  
4. В диалоговом окне **Общий доступ к отчету** выберите **Добавить пользователя/команду**.    
5. В диалоговом окне **Поиск записей** найдите записи пользователей или команды, которым требуется предоставить общий доступ к отчету, и установите флажок рядом с записью.

   > [!div class="mx-imgBorder"]
   > ![Выбор пользователя для предоставления общего доступа к отчету](media/report-share1.png "Выбор пользователя для предоставления общего доступа к отчету")

6. Нажмите **Выбрать**, чтобы добавить запись пользователя или команды в поле **Выбранные записи**, а затем нажмите **Добавить**.

   > [!div class="mx-imgBorder"]
   > ![Добавление пользователя для предоставления общего доступа к отчету](media/report-share2.png "Добавление пользователя для предоставления общего доступа к отчету")
  
7. В диалоговом окне **Общий доступ к отчету** выберите тип доступа. Доступны следующие разрешения: чтение, запись, удаление, добавление, назначение или предоставление общего доступа. Это приведет к добавлению записи пользователя или команды в поле **Выбранные записи**.

   > [!div class="mx-imgBorder"]
   > ![Выбор общего доступа](media/report-share3.png "Выбор общего доступа")
  

## <a name="share-a-report-with-your-organization-for-admins"></a>Предоставление общего доступа к отчету для организации (для администраторов)
 Если отчет будет полезен для всех пользователей, сделайте его доступным для организации.  

1. На панели навигации слева выберите область отчетов.  
2. В списке отчетов выберите отчет, к которому необходимо предоставить общий доступ.  
3. На панели команд выберите **Изменить**.  
4. В меню **Действия** выберите **Сделать отчет доступным для организации**.  
  
   > [!div class="mx-imgBorder"]
   > ![Предоставление доступа к отчету для организации](media/report-share4.png "Предоставление доступа к отчету для организации")

## <a name="download-a-report"></a>Загрузка отчета

1. На панели навигации слева выберите область отчетов. 
2. В списке отчетов выберите отчет, к которому необходимо предоставить общий доступ.  
3. На панели команд выберите **Изменить**.  
4. В меню **Действия** выберите **Загрузить отчет**.  
RDL-файл содержит fetchXML, на котором основан отчет.
5. Откройте отчет после завершения загрузки.
