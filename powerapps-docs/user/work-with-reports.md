# <a name="work-with-reports"></a>Работа с отчетами

Отчеты помогают отслеживать ход выполнения ваших бизнес-задач, помогая вам понять, как именно это делается. Вы также можете отслеживать тенденции, которые могут дать преимущество вашим конкурентам.  

Дополнительные сведения о организации и создании отчетов см. в следующих статьях: [Настройка и организация отчетов](https://docs.microsoft.com/powerapps/maker/model-driven-apps/add-reporting-to-app).
  
## <a name="run-a-report"></a>Запуск отчета  
  
1. В области навигации слева выберите область отчеты. 
2. Выберите отчет, который нужно > **запустить отчет**.  
  
   > [!NOTE]
   >  В диалоговом окне **средство просмотра отчетов** можно оставить критерий поиска как есть или изменить его при необходимости.  
   
   > [!div class="mx-imgBorder"]
   > ![Запуск отчета](media/report-run.png "Запуск отчета")
 
  
## <a name="share-a-report-with-other-users-or-teams"></a>Предоставление общего доступа к отчету другим пользователям или командам    

1. В области навигации слева выберите область отчеты.  
2. В списке отчетов выберите отчет, который необходимо поделиться.  
3. На панели команд выберите Share ( **общий доступ**).

   > [!div class="mx-imgBorder"]
   > ![предоставление общего доступа к отчету](media/report-share.png "предоставление общего доступа к отчету")
  
4. В диалоговом окне **общий доступ** к отчету выберите **Добавить пользователя или рабочую группу**.    
5. В диалоговом окне **Поиск записей** найдите пользователей или запись команды, которым требуется предоставить общий доступ к отчету, и установите флажок рядом с записью.

   > [!div class="mx-imgBorder"]
   > ![Выбор пользователя для предоставления общего доступа] к отчету (media/report-share1.png "Выберите пользователя для предоставления общего доступа") к отчету

6. Нажмите кнопку **выбрать** , чтобы добавить запись пользователя или команды в поле **Выбранные записи** , а затем нажмите кнопку **добавить**.

   > [!div class="mx-imgBorder"]
   > ![Добавление пользователя для предоставления общего доступа] к отчету (media/report-share2.png "Добавить пользователя в отчет общего доступа")
  
7. В диалоговом окне **Общий отчет** выберите нужный тип доступа к общему ресурсу. Доступны следующие разрешения: Чтение, запись, удаление, добавление, назначение или совместное использование. Это приведет к добавлению записи пользователя или команды в поле **Выбранные записи** .

   > [!div class="mx-imgBorder"]
   > ![Выбор общего доступа](media/report-share3.png "Выбор общего доступа")
  

## <a name="share-a-report-with-your-organization-for-admins"></a>Предоставление общего доступа к отчету организации (для администраторов)
 Если отчет будет полезен для всех пользователей, сделайте его доступным для Организации.  

1. В области навигации слева выберите область отчеты.  
2. В списке отчетов выберите отчет, который необходимо поделиться.  
3. На панели команд выберите пункт **изменить**.  
4. В меню **действия** выберите команду **сделать отчет доступным для Организации**.  
  
   > [!div class="mx-imgBorder"]
   > ![Общий доступ к отчету с помощью Организации](media/report-share4.png "Общий доступ к отчету с помощью Организации")

## <a name="download-a-report"></a>Скачать отчет

1. В области навигации слева выберите область отчеты. 
2. В списке отчетов выберите отчет, который необходимо поделиться.  
3. На панели команд выберите пункт **изменить**.  
4. В меню **действия** выберите **скачать отчет**.  
RDL-файл содержит fetchXML, на котором основан отчет.
5. Откройте отчет после завершения скачивания.
