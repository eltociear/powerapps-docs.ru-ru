# <a name="user-guide-for-model-driven-apps-running-on-the-power-apps-mobile-app"></a>Руководство пользователя по приложениям на основе моделей, выполняющихся в мобильном приложении Power Apps

[!INCLUDE [cc-beta-prerelease-disclaimer](../includes/cc-beta-prerelease-disclaimer.md)]

Используйте мобильное приложение Power Apps для запуска приложений на основе моделей на мобильном устройстве. Дополнительные сведения о том, как установить приложение и приступить к работе, см. в разделе [Запуск приложений на основе холста и приложений на основе моделей на мобильном устройстве](run-canvas-and-model-apps-on-mobile.md).

> [!IMPORTANT]
> Приложения на основе моделей для Dynamics 365 Sales, Dynamics 365 Customer Service и Dynamics 365 Field Service<!--For sure this list doesn't include Dynamics 365 Marketing, and Dynamics 365 Project Service Automation? That's the list of model-driven apps according to the Dynamics Style Guide.--> не запускаются в мобильном приложении Power Apps. Вместо этого вы используете приложение Dynamics 365 для телефонов и планшетов. Дополнительные сведения: [Руководство пользователя по Dynamics 365 для телефонов и планшетов](https://docs.microsoft.com/dynamics365/mobile-app/dynamics-365-phones-tablets-users-guide).

## <a name="home-screen"></a>Начальный экран 

Перемещаться в мобильном приложении Power Apps можно с легкостью. На следующей иллюстрации показаны основные элементы навигации начального экрана. 

![Элементы управления навигацией, развернутое представление](media/home_screen_iphone.png "Элементы управления навигацией, развернутое представление")

Условные обозначения

1. **Карта сайта**. Откройте меню и перемещайтесь между приложениями, обращайтесь к избранному и недавно использованным записям, параметрам доступа и другим разделам.
2. **Поиск**. Позволяет вести поиск записей приложений в Common Data Service.
3. **Быстрое создание**. Позволяет создать запись, чтобы быстро ввести практически любой тип информации в систему.
4. **Помощник по связям.** С помощью помощника можно отслеживать ежедневные действия и коммуникации. Помощник предоставляет актуальные и практические сведения на картах аналитики, которые отображаются на экране приложения.

## <a name="site-map"></a>Карта сайта 

На начальном экране выберите карту сайта ![значок карты сайта](media/pa_mobile_sitemap_icon.png "Значок карты сайта"), чтобы получить доступ к сущностям, избранному или наиболее часто используемым записям, другим приложениям и параметрам.

 
   > [!div class="mx-imgBorder"]
   > ![Экран "Карта сайта"](media/go_to_sitemap_iphone.gif "На этом рисунке показано, как перейти к экрану "Карта сайта"")
   
 *Обновите страницу, чтобы перезапустить действие GIF*.

На следующей иллюстрации показаны основные элементы навигации карты сайта. 

![Экран "Карта сайта"](media/site_map_iphone.png "Экран "Карта сайта"")

Условные обозначения

1. **Выбор приложений**. Откройте это меню, чтобы закрыть приложение и переключиться на другое приложение.
2. **Начальный экран**. Выберите этот параметр, чтобы вернуться на начальный экран.
3. **Profile** (Профиль). Перейдите на экран профиля, чтобы выйти из приложения или перенастроить его. 
4. **Последние записи**. Позволяет просмотреть список недавно использованных записей. 
5. **Закрепленные записи**. Позволяет просмотреть и открыть избранные (закрепленные) записи. 
6. **Навигатор по сущностям**. В этой области отображается список сущностей, доступных в приложении.
7. **Справка**. Дополнительные сведения о том, как использовать мобильное приложение Power Apps.
8. **Автономное состояние**. Работа с данными в автономном режиме, даже если у вас нет доступа к Интернету. Дополнительные сведения: [Используете мобильное устройство автономно?](https://docs.microsoft.com/dynamics365/mobile-app/work-in-offline-mode)
9. **Параметры**: доступ к параметрам.

## <a name="pin-favorite-records"></a>Закрепление избранных записей

Списки **Закрепленные** и **Последние** обеспечивают быстрый доступ к записям, которые недавно использовались или закреплены в избранном. Используйте список **Последние** для закрепления избранных записей.  

1. На карте веб-сайта ![Значок карты сайта](media/pa_mobile_sitemap_icon.png "Значок карты сайта")выберите **Последние** ![Значок последней записи](media/pa_mobile_recent_icon.png "Значок "Последние записи"").

2. На экране **Последние** щелкните значок кнопки рядом с записью, чтобы добавить ее в избранное (закрепленные записи).

3. Чтобы просмотреть новые закрепленные записи, выберите ![значок "Назад"](media/mobile_go_back_icon.png "Значок "Назад""), а затем выберите **Закрепленные** ![Значок закрепленного избранного](media/mobile_pinned_favs_icon.png "Значок "Закрепленные в избранном"").


   > [!div class="mx-imgBorder"]
   > ![Закрепление записи в избранном](media/pin_favs.gif "На этом рисунке показано, как закрепить избранные записи")
   
*Обновите страницу, чтобы перезапустить действие GIF*.

### <a name="unpin-a-record"></a>Открепление записи

1. На карте сайта ![Значок карты сайта](media/pa_mobile_sitemap_icon.png "Значок карты сайта") выберите **Закрепленные** ![значок "Закрепленные в избранном"](media/mobile_pinned_favs_icon.png "Значок "Закрепленные в избранном"").

2. Щелкните значок "Удалить закрепленный элемент" ![значок "Удалить закрепленный элемент"](media/pa_mobile_remove_pin_icon.png "Значок открепления") рядом с записью, чтобы удалить ее из избранного (закрепленных записей).


   > [!div class="mx-imgBorder"]
   > ![Открепление записи](media/unpin_favs.gif "На этом рисунке показано, как открепить запись")
   
*Обновите страницу, чтобы перезапустить действие GIF*.

## <a name="add-a-record-quickly"></a>Быстрое добавление записи

1. На начальном экране выберите **Создать** ![кнопка "Создать запись"](media/create-record-button.png "Кнопка Создать запись").
2. Заполните поля и нажмите кнопку **Сохранить**.
3. После создания записи можно ее просмотреть. 

   > [!div class="mx-imgBorder"]
   > ![Создание записи](media/pamobile_add_record.gif "На этом рисунке показано, как создать новую запись")

*Обновите страницу, чтобы перезапустить действие GIF*.

-  Чтобы сохранить и открыть созданную запись, выберите **Дополнительно** ![Значок "Другие команды"](media/pa_mobile_more_commands_icon.png "Значок "Дополнительные команды""), а затем выберите **Сохранить и открыть**.

- Чтобы сохранить и создать другую запись, щелкните **Дополнительно** ![Значок "Другие команды"](media/pa_mobile_more_commands_icon.png "Значок "Дополнительные команды""), а затем выберите **Сохранить и создать новую**.

   > [!div class="mx-imgBorder"]
   > ![Создание записи](media/pa_mobile_save_create_new.gif "На этом рисунке показано, как сохранить и открыть или создать запись")

*Обновите страницу, чтобы перезапустить действие GIF*.
