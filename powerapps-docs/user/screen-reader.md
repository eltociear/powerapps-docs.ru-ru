# <a name="use-a-screen-reader"></a>Использование средства чтения с экрана 


Программы чтения с экрана делают приложения, управляемые моделями, доступными людям, которые не имеют какого-либо зрения или могут требовать дополнительной поддержки для временного сценария, например глаза выносливости. Поддерживаются часто используемые средства чтения с экрана, такие как экранный диктор, JAWS и NVDA. 

- [Дополнительные сведения о работе с экранным диктором Майкрософт](https://support.microsoft.com/help/22798)
- [Дополнительные сведения о работе с JAWS](https://www.freedomscientific.com/Products/Blindness/JawsDocumentation)


- [Дополнительные сведения о работе с NVDA](https://www.nvaccess.org/get-help/)


## <a name="basic-tasks-using-a-screen-reader"></a>Основные задачи с использованием средства чтения с экрана 

### <a name="open-an-app"></a>Запуск приложения

1.  На панели навигации с помощью клавиши **Tab** перейдите в раскрывающийся список приложение и нажмите клавишу **Ввод** , чтобы открыть карту узла.
2.  Нажимайте клавишу **Tab** , пока не услышите имя приложения, которое нужно открыть, например «Sales» (продажи). Нажмите клавишу **Ввод** , чтобы открыть приложение.

### <a name="use-scan-mode-in-narrator"></a>Использование режима сканирования в экранном дикторе
Режим сканирования можно использовать для быстрого перехода к приложениям с помощью клавиш со стрелками и общих сочетаний клавиш. Быстрый переход к заголовкам, ссылкам, ориентирам, полям форм, элементам управления и таблицам в этом режиме. Включите или отключите режим сканирования, нажав клавиши **Caps Lock + пробел**. Дополнительные сведения: [Использование режима Scan](https://support.microsoft.com/help/22809/windows-10-narrator-using-scan-mode)

### <a name="find-your-way-around-the-app"></a>Найдите свой способ вокруг приложения

#### <a name="navigation-bar"></a>Панель навигации
При открытии приложения в левой части отображается вертикальная черта с значками подобласти. Можно использовать клавишу **Tab** для перемещения по этим значкам до тех пор, пока не появится имя нужной подобласти, например "учетные записи", или можно использовать элемент управления картой узла. Например, нажмите клавишу **Tab** , пока не услышите "учетные записи", а затем нажмите клавишу **Ввод** , чтобы открыть представление учетные записи.

#### <a name="grids"></a>Сетки
Средства чтения с экрана пропускают сетки более надежно и согласованно и объявляют заголовки строк и столбцов, а также расположение в сетке. При первом открытии сетки позиция табуляции по умолчанию — это селектор представления. 

При вводе ячейки в сетке за пределами сетки экранный диктор объявляет имя таблицы, число строк и столбцов, а также позицию курсора в таблице.

Если курсор находится внутри заголовка таблицы, можно быстро переходить между заголовками с помощью клавиши **Tab** или **SHIFT + TAB**. Экранный диктор будет объявлять имя каждого заголовка по мере ввода ячейки заголовка. Он также объявляет о типе ячейки (например, "заголовок столбца"), расположении столбца (например, "столбец 1 из 6") и о том, отсортирован или выбран столбец. Если нажать клавишу **Ввод** в заголовке таблицы, таблица будет отсортирована по этому столбцу. Экранный диктор объявляет порядок сортировки, и вы можете нажать клавишу **Ввод** еще раз, чтобы изменить порядок.

При перемещении из последнего столбца в таблице курсор перемещается во вторую строку сетки, и с этого момента для перехода между ячейками, не являющимися заголовками, необходимо использовать клавиши со стрелками вверх и вниз. Если вы снова нажмете клавишу **Tab** , курсор перейдет к следующему интерактивному элементу, обычно к списку фильтра таблицы. При перемещении между ячейками строк, отличных от заголовков, Экранный диктор объявляет имя столбца, расположение столбца и текст в ячейке.

#### <a name="forms"></a>Формы
Для перемещения по форме с помощью экранного диктора доступны несколько режимов навигации с наиболее распространенными режимами: ориентиры, заголовки и поля формы. Чтобы изменить режим навигации, нажмите клавишу **Caps Lock + стрелка вверх**. Удерживайте клавишу Caps Lock, удерживая нажатой клавишу Стрелка вверх, чтобы циклически прокручивать режимы до тех пор, пока не будет слышен нужный режим. Затем используйте клавишу Caps Lock + клавиши со стрелками влево и вправо для перемещения по различным элементам. Например, если вы хотите переходить к полю Last Name (фамилия) в разделе Контактные данные контакта, выполните следующие действия.

1.  Нажмите и удерживайте клавишу **Caps Lock** и нажимайте клавишу **стрелка вверх** , пока вы не услышите «ориентиры».
2.  Нажмите и удерживайте клавишу **Caps Lock** и нажимайте клавишу **со стрелкой вправо** до тех пор, пока не будет слышно «контактные данные».
3.  Измените режим, нажав и удерживая клавишу **Caps Lock** и нажимая клавишу **стрелка вверх** до тех пор, пока не будет слышно «поля формы».
4.  Перейдите к полю Last Name (фамилия), используя клавишу Caps Lock + клавиши со стрелками влево и вправо, пока не услышите поле Last Name (фамилия). Экранный диктор также объявляет тип элемента управления, значение, состояние и любые специальные инструкции для поля.

Можно также использовать клавишу TAB для быстрого перехода к интерактивным элементам в форме. Некоторые поля формы имеют значок, который будет выполнять действие по умолчанию при нажатии клавиш CTRL + ВВОД. Например, поле формы электронной почты может иметь значок конверта, который открывает редактор электронной почты. 

#### <a name="dashboardscharts"></a>Панели мониторинга и диаграммы
Вы можете перемещаться по диаграммам панели мониторинга с помощью клавиши TAB и клавиша Caps Lock + стрелка. Нажмите клавишу **Tab** , чтобы быстро перейти к интерактивным элементам и использовать клавишу Caps Lock + стрелка для навигации по неинтерактивным элементам, таким как заголовки, ориентиры и элементы.


> [!NOTE]
> Необходимо установить Последнее обновление [Windows 10](https://www.microsoft.com/enable/products/windows10/default.aspx) , чтобы иметь доступ ко всем специальным возможностям для диаграмм.

#### <a name="interactive-dashboard-streams"></a>Интерактивные потоки панели мониторинга
Можно использовать клавиши **Tab** или **SHIFT + TAB** для перемещения между интерактивными потоками панелей мониторинга, например, найденными на панели мониторинга учетных записей, или просто изменить режим навигации до тех пор, пока не пройдет «заголовки», а затем быстро перейти с помощью клавиши **Tab** . потоки панели мониторинга.

Для перемещения по каждому элементу потока панели мониторинга используйте клавиши со стрелками вверх и вниз. Экранный диктор будет считывать тип элемента управления и заголовок элемента управления.

#### <a name="business-process-flows"></a>Потоки бизнес-процессов
Можно перемещаться по потоку бизнес-процесса, например, который находится в верхней части формы интереса, нажимая клавишу **Tab** для перемещения вперед, а **SHIFT + TAB** — для перемещения назад между сущностями. Чтобы отобразить дополнительные сущности в потоке, используйте клавишу **Ввод** на **перемещении влево** или **вверх на правую** кнопку. Программа экранного диктора считывает тип сущности, этап, состояние, заголовок, номер элемента из общего числа элементов и указывает, выбран ли он в данный момент.

#### <a name="dialog-boxes"></a>Диалоговые окна

Когда открывается диалоговое окно, Экранный диктор объявляет название. Для диалоговых окон с полями ввода Кнопка **Закрыть** имеет фокус по умолчанию, что позволяет закрыть диалоговое окно, нажав клавишу **Ввод**. Для диалоговых окон, требующих действий пользователя, фокус находится на главной кнопке действия, такой как **Удалить** или **ОК**.

Перемещаться по элементам управления можно с помощью клавиши **Tab** . Курсор будет циклически проходить по каждому элементу в диалоговом окне, и вы можете нажать клавишу **ESC** , чтобы закрыть его.
