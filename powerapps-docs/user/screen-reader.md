# <a name="use-a-screen-reader"></a>Использование средства чтения с экрана 


Средства чтения с экрана делают приложения на основе модели доступными людям со слабым зрением, слепым или тем, кому временно требуется помощь, например при зрительном утомлении. Поддерживаются популярные средства чтения с экрана, такие как Экранный диктор, JAWS и NVDA. 

- [Узнайте больше о работе с Экранным диктором Майкрософт](https://support.microsoft.com/help/22798)
- [Узнайте больше о работе с JAWS](https://www.freedomscientific.com/Products/Blindness/JawsDocumentation)


- [Узнайте больше о работе с NVDA](https://www.nvaccess.org/get-help/)


## <a name="basic-tasks-using-a-screen-reader"></a>Основные задачи при использовании средства чтения с экрана 

### <a name="open-an-app"></a>Запуск приложения

1.  На панели навигации перейдите на вкладку с помощью клавиши **TAB**, чтобы открыть раскрывающийся список приложения, и нажмите **ВВОД**, чтобы открыть карту сайта.
2.  Нажимайте клавишу **TAB**, пока не услышите имя приложения, которое нужно открыть, например Sales. Нажмите **ВВОД**, чтобы открыть приложение.

### <a name="use-scan-mode-in-narrator"></a>Использование режима сканирования в Экранном дикторе
Режим сканирования можно использовать для быстрой навигации по приложениям с помощью клавиш со стрелками и стандартных сочетаний клавиш. Вы сможете быстро переходить к заголовкам, ссылкам, ориентирам, полям форм, элементам управления и таблицам в этом режиме. Включите или отключите режим сканирования, нажав клавиши **CAPS LOCK + ПРОБЕЛ**. Дополнительные сведения: [Использование режима сканирования](https://support.microsoft.com/help/22809/windows-10-narrator-using-scan-mode)

### <a name="find-your-way-around-the-app"></a>Навигация в приложении

#### <a name="navigation-bar"></a>Панель навигации
При открытии приложения в левой части отображается вертикальная черта со значками подобластей. Нажимайте клавишу **TAB** для перемещения по этим значкам до тех пор, пока не услышите имя нужной подобласти, например "Учетные записи", или используйте элемент управления картой сайта. Например, нажимайте клавишу **TAB**, пока не услышите "Учетные записи", а затем нажмите клавишу **ВВОД**, чтобы открыть представление "Учетные записи".

#### <a name="grids"></a>Сетки
Средства чтения с экрана перемещаются по сеткам более надежно и согласованно и объявляют заголовки строк и столбцов, а также расположение в сетке. При первом открытии сетки по умолчанию фокус находится на селекторе представления. 

При вводе ячейки в сетке из-за пределов сетки Экранный диктор объявляет имя таблицы, число строк и столбцов, а также позицию курсора в таблице.

Если курсор находится в пределах заголовка таблицы, вы можете быстро переходить между заголовками с помощью клавиши **TAB** или **SHIFT + TAB**. Экранный диктор будет объявлять имя каждого заголовка по мере ввода данных в ячейку заголовка. Он также объявляет тип ячейки (например, "Заголовок столбца"), расположение столбца (например, "столбец 1 из 6") и сведения о том, отсортирован или выбран этот столбец. При нажатии клавиши **ВВОД** в заголовке таблицы таблица будет отсортирована по этому столбцу. Экранный диктор объявляет порядок сортировки, и вы можете нажать клавишу **ВВОД** еще раз, чтобы изменить порядок.

Из последнего столбца в таблице курсор перемещается во вторую строку сетки, и с этого момента для перехода между ячейками, не являющимися заголовками, необходимо использовать клавиши со стрелками вверх и вниз. Если вместо этого нажать клавишу **TAB** еще раз, курсор переместится к следующему интерактивному элементу, обычно к списку фильтров таблицы. При перемещении между ячейками строк, отличных от заголовков, Экранный диктор объявляет имя столбца, расположение столбца и текст в ячейке.

#### <a name="forms"></a>Формы
Для перемещения по форме с помощью Экранного диктора доступно несколько режимов навигации с наиболее распространенными вариантами: ориентиры, заголовки и поля формы. Чтобы изменить режим навигации, нажмите клавиши **CAPS LOCK + стрелка вверх**. Удерживайте клавишу CAPS LOCK, удерживая нажатой клавишу стрелки вверх, чтобы циклически прокручивать режимы до тех пор, пока не услышите нужный. Затем нажимайте CAPS LOCK и стрелки влево и вправо для перемещения по различным элементам. Например, если вы хотите перейти к полю "Фамилия" в разделе "Контактная информация", выполните следующие действия.

1.  Нажмите и удерживайте клавишу **CAPS LOCK** и нажимайте клавишу **стрелка вверх**, пока не услышите "Ориентиры".
2.  Нажмите и удерживайте клавишу **CAPS LOCK** и нажимайте клавишу **стрелка вправо**, пока не услышите "Контактная информация".
3.  Измените режим, нажав и удерживая клавишу **CAPS LOCK** и нажимая клавишу **стрелка вверх**, пока не услышите "Поля формы".
4.  Перейдите к полю "Фамилия", используя клавишу CAPS LOCK и клавиши со стрелками влево и вправо, пока не услышите "Фамилия". Экранный диктор также объявляет тип элемента управления, значение, состояние и любые специальные инструкции для поля.

Можно также использовать клавишу TAB для быстрого перехода к интерактивным элементам в форме. Некоторые поля формы имеют значок, который будет выполнять действие по умолчанию при нажатии клавиш CTRL + ВВОД. Например, поле формы электронной почты может иметь значок конверта, который открывает редактор электронной почты. 

#### <a name="dashboardscharts"></a>Панели мониторинга и диаграммы
Вы можете перемещаться по диаграммам панели мониторинга с помощью клавиши TAB и клавиш CAPS LOCK + стрелки. Нажмите клавишу **TAB**, чтобы быстро перейти к интерактивным элементам, и используйте клавиши CAPS LOCK + стрелки для навигации по неинтерактивным элементам, таким как заголовки, ориентиры и элементы.


> [!NOTE]
> Необходимо установить последнюю версию обновления [Windows 10](https://www.microsoft.com/enable/products/windows10/default.aspx), чтобы иметь доступ ко всем специальным возможностям для диаграмм.

#### <a name="interactive-dashboard-streams"></a>Интерактивные потоки панели мониторинга
Используйте клавишу **TAB** или **SHIFT + TAB**, чтобы перемещаться между интерактивными потоками панелей мониторинга, такими как на панели мониторинга "Учетные записи", или просто измените режим навигации на "Заголовки", а затем нажимайте клавишу **TAB**, чтобы быстро перемещаться между потоками панели мониторинга.

Для перемещения по каждому элементу потока панели мониторинга используйте клавиши со стрелками вверх и вниз. Экранный диктор будет считывать тип и заголовок элемента управления.

#### <a name="business-process-flows"></a>Потоки бизнес-процессов
Можно перемещаться по потоку бизнес-процесса, как, например, в верхней части формы "Потенциальный клиент", нажимая клавишу **TAB** для перемещения вперед и **SHIFT + TAB** для перемещения назад между сущностями. Нажмите клавишу **ВВОД** на кнопке **Перейти влево** или **Перейти вправо**, чтобы отобразить дополнительные сущности в потоке процесса. Экранный диктор считывает тип сущности, этап, состояние, заголовок, номер элемента из общего числа элементов и указывает, выбран ли он в данный момент.

#### <a name="dialog-boxes"></a>Диалоговые окна

Когда открывается диалоговое окно, Экранный диктор объявляет заголовок. Для диалоговых окон с полями ввода кнопка **Закрыть** имеет фокус по умолчанию, что позволяет закрыть диалоговое окно, нажав клавишу **ВВОД**. Для диалоговых окон, требующих действия пользователя, фокус находится на главной кнопке действия, например **Удалить** или **ОК**.

Вы можете перемещаться по элементам управления с помощью клавиши **TAB**. Курсор будет циклически проходить по каждому элементу в диалоговом окне, и вы можете нажать клавишу **ESC**, чтобы закрыть его.
