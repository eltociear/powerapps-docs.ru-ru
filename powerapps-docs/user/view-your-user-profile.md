# <a name="view-your-user-profile"></a>Просмотр профиля пользователя  


В вашем профиле пользователя отображаются полезные сведения для всей Организации. Например, ваши контактные данные, ваша организация и роль безопасности. В зависимости от роли безопасности вы можете вносить изменения в профиль пользователя.  
  
1. В правом верхнем углу экрана выберите **Параметры** ![параметры персонализации "пользователь-профиль-параметры".](media/user-profile-settings-button.gif) >   
 
2. Прокрутите вниз до нижнего края диалогового окна **Задание личных параметров** и выберите **Просмотреть сведения о пользователе**.  
  
3. Выберите различные области, например " **Сводка**", " **сведения**" или " **Администрирование** ", чтобы просмотреть сведения о вашем профиле.
