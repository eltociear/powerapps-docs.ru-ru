# <a name="view-your-user-profile"></a>Просмотр профиля пользователя  


В вашем профиле пользователя для всей организации отображаются полезные сведения, например ваши контактные данные, ваша организация и роль безопасности. В зависимости от роли безопасности вы можете вносить изменения в свой профиль пользователя.  
  
1. В правом верхнем углу экрана выберите **Параметры** ![пользовательский профиль-параметры-кнопка](media/user-profile-settings-button.gif) > **Параметры персонализации**.  
 
2. Прокрутите вниз до самой нижней части диалогового окна **Задание личных параметров**, а затем выберите команду **Просмотреть сведения о пользователе**.  
  
3. Чтобы просмотреть сведения в профиле, выбирайте различные области, такие как **Сводка**, **Сведения** или **Администрирование**.
