# <a name="add-an-appointment-email-phone-call-note-or-task-activity-to-the-timeline"></a>Добавление встречи, электронной почты, телефонного звонка, заметок или задачи во временной шкале 


Добавьте **Действия** на стену **Временная шкала**, чтобы отслеживать все ваши сообщения с клиентом или связь. Например, вы можете делать заметки, добавлять публикации, добавлять задачи, отправлять сообщения электронной почты, добавлять сведения о телефонных звонках или назначать встречи. Система автоматически отметит время каждого действия и покажет, кто его создал. Вы и другие пользователи вашей группы можете просматривать действия, чтобы увидеть историю во время работы с клиентом. 

- Действия, которые вы добавляете из записи, отображаются на стене **Временная шкала** записи. 
- Если в действии задано поле **В отношении**, то действие отображается в записи, с которой оно связано. 
- Вы также можете выбрать панель фильтра для фильтрации действий по типу записи и дате. 
- При создании нового действия вы получите уведомление **О том, что пропустили** на стене **Временная шкала**.
- Сообщение электронной почты с прикрепленным изображением будет отображаться в тексте сообщения электронной почты.

  > [!div class="mx-imgBorder"]
  > ![Представление действий в Power Apps на временной шкале](media/TimelineViewOfActivity.png "Представление действий в Power Apps на временной шкале")


Условные обозначения

  1. Поиск в записях
  2. Создание заметок
  3. Добавление сведений и действий
  4. Фильтр
  5. Дополнительные команды
  6. Состояние действия
  7. Значки действий
  8. Дата и время
 
## <a name="add-an-activity-from-the-nav-bar"></a>Добавление действия из панели навигации
 
Самый быстрый способ добавить действие — использовать ярлык на навигационной панели, а затем связать его с записью. Например, вы можете создать операцию телефонного вызова и затем связать ее с контактом в системе, используя поле **В отношении**.

1. На верхней панели навигации последовательно выберите **Создать** ![кнопка "Создать запись"](media/create-record-button.png "Кнопка "Создать запись"") > **Действия** и выберите тип действия, который вы хотите добавить.

   > [!div class="mx-imgBorder"]
   > ![Ярлык для добавления действия в Power Apps](media/add_new_activity_from_nav.gif "Ярлык для добавления действия в Power Apps")  
 
3. Введите требуемые данные. Используйте поле **В отношении**, чтобы связать действие с записью.

4. Когда все будет готово, щелкните **Сохранить и закрыть** или **Сохранить и создать**. 

## <a name="add-an-activity-from-within-a-record"></a>Добавление действия из записи

Можно также открыть запись, а затем добавить в нее действие. 

   > [!div class="mx-imgBorder"]
   > ![Ярлык для добавления действия в Power Apps](media/add_new_activity_from_record.gif "Ярлык для добавления действия в Power Apps") 


### <a name="add-a-phone-call"></a>Добавление телефонного звонка  
  
1. Откройте запись, к которой вы желаете добавить действие. Например, откройте запись контакта.
  
2. В разделе **Временная шкала** выберите **Добавить информацию и действия** ![Добавить действия](media/add-activity-button.png "Кнопка "Добавить действия"") > **Телефонный звонок**. 


   > [!div class="mx-imgBorder"]
   > ![Добавление действия с телефоном в Power Apps](media/addphonecall.png "Добавление действия с телефоном в Power Apps")
  
3. Заполните **Тему** вызова.

     В области **Описание** предоставьте сведения о разговоре с клиентом. 
  
     Поле **Вызов** автоматически заполняется записью, к которой вы добавили действие телефонного звонка. При необходимости можно выбрать другую запись.  
  
4. По умолчанию для направления установлено значение **Исходящий**. Вы можете изменить его на **Входящий**, выбрав **Исходящий**.
  
5. Чтобы сохранить действие телефонного звонка после заполнения формы, выберите **Сохранить и закрыть**.  
  
### <a name="add-a-task"></a>Добавление задачи  
  
1. Откройте запись, к которой вы желаете добавить действие. Например, откройте запись контакта.
  
2. В разделе **Временная шкала** выберите **Добавить информацию и действия** ![Добавить действия](media/add-activity-button.png "Кнопка "Добавить действия"") > **Задача**.
  
3. Для текущего пользователя установлено поле **Владелец** по умолчанию. Если вы хотите переназначить задачу, выберите значок поиска, а затем выберите другого пользователя или группу.  
  
4. Чтобы сохранить введенные сведения о задаче, выберите **Сохранить и закрыть**. 
  
### <a name="add-an-email"></a>Добавление адреса электронной почты  

Чтобы добавить действие электронной почты в запись, сначала необходимо сохранить запись, в которую вы добавляете действие.  
  
1. Откройте запись, к которой вы желаете добавить действие. Например, откройте запись контакта.
  
2. В разделе **Временная шкала** выберите **Добавить информацию и действия** ![Добавить действия](media/add-activity-button.png "Кнопка "Добавить действия"") > **Электронная почта**. 

3. Заполните тему сообщения электронной почты и используйте свободное место для написания сообщения.
  
4. Чтобы добавить вложение в сообщение электронной почты, сохраните сообщение. Затем на панели команд выберите **Вложить файл** > **Выбрать файл** а затем выберите файл, который нужно вложить в сообщение электронной почты.

   > [!NOTE]
   > Сообщение электронной почты с прикрепленным изображением будет отображаться в тексте сообщения электронной почты.
  
5. Чтобы использовать шаблон для текста электронного письма, в командной строке щелкните **Вставить шаблон**, а затем выберите шаблон. Дополнительные сведения о вставке шаблона сообщения электронной почты см. в разделе [Вставка шаблона сообщения электронной почты](insert-email-template.md). 
  
6. Завершив составление сообщения электронной почты, на панели команд щелкните **Отправить**. 


#### <a name="list-emails-in-a-conversation-view"></a>Вывод списка сообщений электронной почты в представлении беседы

1. Чтобы вывести список сообщений электронной почты в представлении беседы, перейдите в раздел **Параметры** > **Параметры персонализации**.

   > [!div class="mx-imgBorder"]
   > ![Настройка личных параметров](media/emailsettings1.png "Настройка дополнительных параметров")

2. на вкладке **Электронная почта**, а затем выберите **Показывать сообщение электронной почты как беседу на временной шкале**. Дополнительные сведения о личных параметрах см. в разделе [Настройка личных параметров](https://docs.microsoft.com/powerapps/user/set-personal-options#email-tab-options). После включения можно открыть любую форму с временной шкалой, и сообщения электронной почты будут сгруппированы в беседы, причем последнее сообщение будет отображаться сверху.
   
   > [!div class="mx-imgBorder"]
   > ![Настройка личных параметров для электронной почты](media/emailsettings2.png "Настройка личных параметров для электронной почты")

  
### <a name="add-an-appointment"></a>Добавление встречи  

Чтобы добавить действие встречи в запись, сначала необходимо сохранить запись, в которую вы добавляете это действие.  

> [!NOTE]
> Повторяющиеся встречи не поддерживаются в приложении Dynamics 365 для Outlook, Dynamics 365 для телефонов и при запуске веб-клиента приложений на основе модели в веб-браузере мобильного телефона.
  
1. Откройте запись, к которой вы желаете добавить действие. Например, откройте запись контакта.
  
2.  В разделе **Временная шкала** выберите **Добавить информацию и действия** ![Добавить действия](media/add-activity-button.png "Кнопка "Добавить действия"") > **Встреча**.  
  
3. Заполните **тему** встречи и укажите значения в полях **Время начала** и **Время окончания**.
  
4. Чтобы сохранить введенные сведения о встрече, выберите **Сохранить и закрыть**.

### <a name="add-notes"></a>Добавление заметок

Вы также можете легко добавлять заметки в области действия.
  
1. Откройте запись, к которой вы желаете добавить действие. Например, откройте запись контакта.
  
2. В разделе **Временная шкала** выберите область **Ввести заметку**.

3. Добавьте заголовок заметки и сведения о заметке.

4. Чтобы добавить вложения к заметке, щелкните **Добавить вложение**.

3. Чтобы сохранить заметку, нажмите кнопку **Добавить заметку**.

   > [!div class="mx-imgBorder"]
   > ![Добавление примечания](media/addnote.png "Добавление примечания")

> [!NOTE]
> Вы можете также добавить заметку, выбрав **Добавить информацию и действия** ![Добавить действия](media/add-activity-button.png "Кнопка "Добавить действия"") > **Заметка**.

#### <a name="edit-or-delete-a-note"></a>Изменение или удаление заметки

- Чтобы изменить или удалить созданную заметку, выберите ее, а затем щелкните **Изменить эту заметку** или **Удалить заметку**. 

  > [!div class="mx-imgBorder"]
  > ![Обновление примечания](media/addnote2.png "Обновление примечания")

### <a name="add-a-post"></a>Добавление публикации 

1. Откройте запись, к которой вы желаете добавить публикацию. Например, откройте запись контакта.

2. В разделе **Временная шкала** выберите **Добавить информацию и действия** ![Добавить действия](media/add-activity-button.png "Кнопка "Добавить действия"") > **Публикация**. 

3. Введите сведения о публикации в текстовом поле.

4. Чтобы сохранить публикацию, нажмите кнопку **Добавить**.

   > [!div class="mx-imgBorder"]
   > ![Обновление публикации](media/post.png "Добавление публикации")
  
  После сохранения публикация будет отображаться в верхней части стены временной шкалы.
  
## <a name="refresh-the-timeline"></a>Обновление временной шкалы 

Вы можете обновить стену временной шкалы для просмотра новой информации.

В разделе **Временная шкала** нажмите ![кнопку "Дополнительно"](media/MoreButton.png "Кнопка "Дополнительно""), а затем выберите **Обновить временную шкалу**.

  > [!div class="mx-imgBorder"]
  > ![Обновление временной шкалы ](media/refresh.png "Обновление временной шкалы")


## <a name="use-the-filter-pane"></a>Использование области фильтра

Быстрая фильтрация действий, заметок или сообщений на стене временной шкалы по типу записи, виду деятельности и дате, с помощью области фильтра. Одновременно можно выбрать несколько фильтров и параметров фильтра. Вы можете фильтровать и просматривать дату выполнения действия, дату изменения или действия по состоянию.

- В разделе **Временная шкала** щелкните **Открыть область фильтра** и выберите способ фильтрации действий.

  > [!div class="mx-imgBorder"]
  > ![Область фильтра на временной шкале](media/timeline-filter2.png "Область фильтра на временной шкале") ![Область фильтра на временной шкале](media/timeline-filter5.png "Область фильтра на временной шкале")
