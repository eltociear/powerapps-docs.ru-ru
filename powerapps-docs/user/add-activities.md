# <a name="add-an-appointment-email-phone-call-note-or-task-activity-to-the-timeline"></a>Добавление встречи, электронной почты, телефонного звонка, заметок или задачи во временной шкале 

[!INCLUDE [cc-beta-prerelease-disclaimer](../includes/cc-beta-prerelease-disclaimer.md)]

Добавьте **Действия** на стену **Временная шкала**, чтобы отслеживать все ваши сообщения с клиентом или связь. Например, вы можете делать заметки, добавлять публикации, добавлять задачи, отправлять сообщения электронной почты, добавлять сведения о телефонных звонках или назначать встречи. Система автоматически отметит время каждого действия и покажет, кто его создал. Вы и другие пользователи вашей группы можете просматривать действия, чтобы увидеть историю во время работы с клиентом. 

- Действия, которые вы добавляете из записи, отображаются на стене **Временная шкала** записи. 
- Если в действии задано поле **В отношении**, то действие отображается в записи, с которой оно связано. 
- Вы также можете выбрать панель фильтра для фильтрации действий по типу записи и дате. 
- При создании нового действия вы получите уведомление **О том, что пропустили** на стене **Временная шкала**.
- Сообщение электронной почты с прикрепленным изображением будет отображаться в тексте сообщения электронной почты.

  > [!div class="mx-imgBorder"]
  > ![Представление временной шкалы действий в PowerApps](media/TimelineViewOfActivity.png "Представление временной шкалы действий в PowerApps")  
 
## <a name="add-an-activity-from-the-nav-bar"></a>Добавление действия из панели навигации
 
Самый быстрый способ добавить действие — использовать ярлык на навигационной панели, а затем связать его с записью. Например, вы можете создать операцию телефонного вызова и затем связать ее с контактом в системе, используя поле **В отношении**.

1. На панели навигации выберите **знак "плюс"** ![кнопку "Создать запись"](media/create-record-button.png "кнопку \"Создать запись\""), а затем выберите **Действия**. 

   > [!div class="mx-imgBorder"]
   > ![Ярлык для добавления действия в PowerApps](media/QuickCreate.png "Ярлык для добавления действия в PowerApps")  
 
2. Выберите тип действия, который вы хотите добавить.

3. Введите требуемые данные. Используйте поле **В отношении**, чтобы связать действие с записью.

4. После завершения выберите **Сохранить**.

 
## <a name="add-a-phone-call"></a>Добавление телефонного звонка  
  
1. Откройте запись, к которой вы желаете добавить действие. Например, запись контакта.
  
2. На стене **Временная шкала** выберите **знак "плюс"**  > **Телефонный звонок**. 


   > [!div class="mx-imgBorder"]
   > ![Добавление действия звонка в PowerApps](media/addphonecall.png "Добавление действия звонка в PowerApps")
  
3. Заполните **Тему** вызова.

     В области **Заметки** предоставьте сведения о разговоре с клиентом. 
  
     Поле **Вызов** автоматически заполняется записью, к которой вы добавили действие телефонного звонка. При необходимости можно выбрать другую запись.  
  
4. По умолчанию для направления установлено значение **Исходящий**. Вы можете изменить его на **Входящий**, выбрав **Исходящий**. 
  
5. Чтобы сохранить действие после заполнения формы, выберите **Сохранить**.  
  
## <a name="add-a-task"></a>Добавление задачи  
  
1. Откройте запись, к которой вы желаете добавить действие. Например, запись контакта.
  
2. На стене **Временная шкала** выберите **знак "плюс"**  > **Задача**.
  
3. Для текущего пользователя установлено поле **Владелец** по умолчанию. Если вы хотите переназначить задачу, выберите значок поиска, а затем выберите другого пользователя или группу.  
  
4. Чтобы сохранить действие после заполнения формы, выберите **Сохранить**. 
  
## <a name="add-an-email"></a>Добавление адреса электронной почты  

Чтобы добавить действие электронной почты в запись, сначала необходимо сохранить запись, в которую вы добавляете действие.  
  
1. Откройте запись, к которой вы желаете добавить действие. Например, запись контакта.
  
2. На стене **Временная шкала** выберите **знак "плюс"**  > **Электронная почта**. 

3. Заполните тему сообщения электронной почты и используйте свободное место для написания сообщения.
  
4. Чтобы добавить вложение в сообщение электронной почты, сохраните сообщение. Затем в разделе **Вложения** выберите **+** , чтобы добавить вложение.  
  
5. Чтобы использовать шаблон для текста электронного письма, в командной строке щелкните **Вставить шаблон**, а затем выберите шаблон.   
  
6. После заполнения формы, выберите **Отправить**. 


    > [!NOTE]
    > Чтобы вывести список сообщений электронной почты в представлении беседы, выберите **Параметры** > Настройка**персонализации** > **Электронная почта** , а затем **на временной шкале щелкните Показать сообщение электронной почты**. Дополнительные сведения о личных параметрах см. в разделе [Настройка личных](set-personal-options.md)параметров.

   > [!div class="mx-imgBorder"]
   > ![Настройка личных параметров](media/emailsettings1.png "Настройка личных параметров")
   
    > [!div class="mx-imgBorder"]
    > ![Задание адреса электронной почты для личных параметров](media/emailsettings2.png "Настройка личных параметров для электронной почты")

  
## <a name="add-an-appointment"></a>Добавление встречи  

Чтобы добавить действие встречи в запись, сначала необходимо сохранить запись, в которую вы добавляете действие.  
  
1. Откройте запись, к которой вы желаете добавить действие. Например, запись контакта.
  
2. На стене **Временная шкала**, выберите **знак "плюс"**  > **Встреча**.  
  
3. Используйте подсказки, чтобы заполнить необходимые сведения.
  
4. Чтобы сохранить встречу после заполнения формы, выберите **Сохранить**.

## <a name="add-notes"></a>Добавление заметок

Вы также можете легко добавлять заметки в области действия.
  
1. Откройте запись, к которой вы желаете добавить действие. Например, запись контакта.
  
2. На стене **Временная шкала**, начните вводить свои заметки. Чтобы добавить любые вложения в заметку используйте **эту кнопку**.

3. Чтобы сохранить заметку после заполнения формы, выберите **Добавить заметку**.


> [!NOTE]
> Вы можете также добавить примечание с помощью **знака "плюс"** в верхней части стены **Временная шкала**.

   > [!div class="mx-imgBorder"]
   > ![Добавить заметку](media/addnote.png "Добавить заметку")

После добавления заметки вы можете ее удалить или отредактировать.


> [!div class="mx-imgBorder"]
> ![Обновить заметку](media/addnote2.png "Обновить заметку")

## <a name="add-a-post"></a>Добавление публикации 

1. Откройте запись, к которой вы желаете добавить публикацию. Например, запись контакта.

2. На стене **Временная шкала**, выберите **знак "плюс"**  > **Публикация**. 

3. Введите публикацию в текстовом поле 

4. Чтобы сохранить публикацию после заполнения формы, выберите **Добавить**.

> [!div class="mx-imgBorder"]
> ![Обновить публикацию](media/post.png "Добавить публикацию")
  
  После сохранения публикация будет отображаться в верхней части стены временной шкалы.

## <a name="use-the-filter-pane"></a>Использование области фильтра

Быстрая фильтрация действий, заметок или сообщений на стене временной шкалы по типу записи, виду деятельности и дате, с помощью области фильтра. Одновременно можно выбрать несколько фильтров и параметров фильтра. Вы можете фильтровать и просматривать дату выполнения действия, дату изменения или состояние действия.

- На стене **временной шкалы** выберите **Открыть панель фильтра** значок воронки.

> [!div class="mx-imgBorder"]
> ![Область фильтра на временной шкале](media/filterpane.png "Область фильтра на временной шкале")
