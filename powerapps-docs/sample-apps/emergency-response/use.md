# <a name="use-the-hospital-emergency-response-app"></a>Используйте приложение Hospital Emergency Response

Персонал больницы сталкивается с проблемой увеличения числа пациентов при управлении цепочкой поставок во время чрезвычайной ситуации. С помощью мобильного приложения Hospital Emergency Response работники, работающие непосредственно с клиентами, могут быстро просматривать и добавлять данные о вентиляторах, персонале, ожидаемых выписках и пациентах, связанных с COVID-19.

## <a name="prerequisites"></a>Предварительные условия

Чтобы начать работу с приложением, вам необходимо скачать Power Apps Mobile на вашем устройстве с помощью магазина приложений устройства.

- **Download** [**Power Apps Mobile**](https://powerapps.microsoft.com/downloads)
    - Для устройств **Apple** с iOS, таких как iPhone и iPad, используйте [**Apple App Store**](https://aka.ms/powerappsios)
    - Для устройств на **Android** используйте [**Google Play**](https://aka.ms/powerappsandroid)
- Убедитесь, что ваша организация развернула и настроила приложение Hospital Emergency Response, как описано в [Разверните и настройте приложение](deploy-configure.md).

После установки Power Apps Mobile, откройте приложение с вашего устройства и войдите в систему Azure Active Directory с учетная запись вашей компании. После входа в систему вы можете просмотреть все приложения, которыми вы поделились в вашей организации. Дополнительные сведения см. в [вход в мобильное устройство Power Apps](https://docs.microsoft.com/powerapps/user/run-app-client#open-power-apps-and-sign-in).

## <a name="hospital-emergency-response-app"></a>Приложение Hospital Emergency Response

![Приложение Hospital Emergency Response](media/use/app-launcher.png)

Мобильное приложение Hospital Emergency Response имеет модульную структуру с различными приложениями в зависимости от вашей роли. Откройте мобильное приложение Hospital Emergency Response из Power Apps Mobile, выберите свой **Больничная система**, **Регион, помещение** и выберите **Далее** чтобы начать работу.

> [!NOTE]
> Когда вы запускаете мобильное приложение Hospital Emergency Response или любой из его компонентов *первый раз*, вас спросят о вашем согласии разрешить приложению доступ к профилю ваших *пользователей Office 365* и вашему *Расположению*. Вы должны выбрать **Разрешить**, прежде чем вы сможете начать использовать выбранное приложение. Дополнительные сведения см. [предоставить согласие](https://docs.microsoft.com/powerapps/user/run-app-client#give-consent).

## <a name="app-components"></a>Компоненты приложения

![Компоненты мобильного приложения Hospital Emergency Response](media/use/app-components.png)

Образец приложения для решения Hospital Emergency Response состоит из нескольких приложений для улучшения взаимодействия с пользователем. В зависимости от вашей роли, вы можете увидеть один или несколько компонентов в **Hospital Emergency Response**.

- **Персонал и оборудование**
    <br> Соберите статус RN и критического оборудования в зависимости от расположения в этом помещении.

- **Медикаменты**
    <br> Отслеживайте ключевые медикаменты для более эффективного отслеживания, управления и прогнозирования запасов. 

- **Кадровые потребности**
    <br> Сбор запросов на персонал по подразделениям, ролям и срочности.

- **статистика COVID-19**
    <br> Соберите информацию о том, сколько пациентов находится под наблюдением в отношении COVID-19 и сколько имеет положительный результат теста.

- **Планирование выписки**
    <br> Соберите статус и проекции по выпискам пациентов.

## <a name="staff--equipment"></a>Персонал и оборудование

![Персонал и оборудование](media/use/staff-equipment.png)

Предоставьте сведения о запасах с конкретными расположениями для зарегистрированных медсестер, пациентов и оборудования. Список областей состоит из всех расположений, определенных для помещения, выбранного в **Приложении Hospital Emergency Response**. Выберите расположение из доступных вариантов для обновления других полей.

После выбора области введите необходимые значения для полей, чтобы сохранить записи в базе данных решения. Вам не нужно вводить значения для каждого поля на экране. Введите номер для поля, которое необходимо сохранить в базе данных решения.

Например, если вам нужно добавить число зарегистрированных медсестер, запрошенное как 3, введите 3 в поле **Зарегистрированные дежурные медсестры — по запросу** и выберите **послать**. Если вам также необходимо обновить используемые вентиляторы до 6, введите 3 в поле **Зарегистрированные дежурные медсестры — по запросу**, затем введите 6 в **Вентиляторы** в **Используемое оборудование** и выберите **послать**.

Выбрать **назад** левом верхнем углу, если вы хотите вернуться к **Приложению Hospital Emergency Response** без отправки каких-либо изменений. Кнопка **Отправить** отправляет введенные вами значения.

После того, как вы отправите данные, у вас есть возможность вернуться в приложение **Персонал и оборудование** для создания другой записи, используя кнопка **Отслеживать другой**. Выбрать **главная страница**, чтобы вернуться к **приложение Hospital Emergency Response**.

### <a name="fields-and-description"></a>Поля и описание

| **Имя параметра**               | **Описание**                                                                                   |
|-------------------------------|---------------------------------------------------------------------------------------------------|
| Место                      | Имя и тип помещения, отделения или любого другого специального расположения в выбранном помещении. |
| Количество пациентов            | Текущее общее количество пациентов в выбранном расположение.                                        |
| **Зарегистрированные дежурные медсестры** |                                                                                                   |
| *Партнеры*                    | Количество зарегистрированных партнеров-медсестер в выбранном расположение.                             |
| *Запрошено*                   | Количество запрошенных зарегистрированных медсестер для выбранного расположения.                                  |
| *Назначенные*                    | Количество зарегистрированных медсестер, назначенных для выбранного расположения.                                    |
| *Неназначенные*                  | Количество зарегистрированных медсестер, не назначенных для любой задачи в выбранном расположении.                    |
| **Используемое оборудование**          |                                                                                                   |
| *Вентиляторы*                 | Количество вентиляторов, используемых в выбранном расположение.                                            |
| *колпаки PAPR*                  | Количество используемых колпаков фильтрующих респираторов с принудительной подачей воздуха в выбранном расположение.                 |
| *ремни PAPR*                  | Количество используемых ремней фильтрующих респираторов с принудительной подачей воздуха в выбранном расположении.                 |
| *Зарядные устройства PAPR*               | Количество используемых зарядных устройств фильтрующих респираторов с принудительной подачей воздуха в выбранном расположении.              |


## <a name="supplies"></a>Медикаменты

![Медикаменты](media/use/supplies.png)

Просмотрите запас медикаментов с помощью приложения **Медикаменты**. Вы можете обновить количество компонентов поставки во всем запасе помещения и ежедневную скорость потребления из этого приложения.

> [!NOTE]
> Введите значения в оба поля, **На складе** и **Использовано за последние 24 часа**, прежде чем выбрать **Отправить**.

Выбрать **назад** в левом верхнем углу, если вы хотите вернуться к **Приложению Hospital Emergency Response** без отправки каких-либо изменений. Кнопка **Отправить** отправляет введенные вами значения. Выбрать **главная страница**, чтобы вернуться к **приложение Hospital Emergency Response** после отправки.

### <a name="fields-and-description"></a>Поля и описание

Список элементов приложения медикаментов может отличаться в зависимости от требований вашей организации. См. ресурсы вашей организации для описания названий поставок.

ИТ-администраторы могут добавлять или обновлять список элементов приложения медикаментов, используя приложение на основе модели для Power Apps. Дополнительные сведения см. в [руководстве по настройке](deploy-configure.md).

> [!NOTE]
> Значения элемента запаса медикаментов должны быть в числовом формате.

## <a name="staffing-needs"></a>Кадровые потребности

![Кадровые потребности](media/use/staffing-needs.png)

Собирает запросы кадрового пула для выбранного помещения. Прежде чем вы сможете подать запрос о кадровом пуле для помещения, убедитесь, что поля помеченные как *обязательные* (*) заполнены.

Выбрать **назад** левом верхнем углу, если вы хотите вернуться к **Приложению Hospital Emergency Response** без отправки каких-либо изменений. Кнопка **Отправить** отправляет введенные вами значения. Выбрать **главная страница**, чтобы вернуться к **приложение Hospital Emergency Response** после отправки.

### <a name="fields-and-description"></a>Поля и описание

| **Имя поля**           | **Описание**                                                                            |
|--------------------------|--------------------------------------------------------------------------------------------|
| Отдел               | Название подразделения, подавшего заявку на персонал. Это поле является *обязательным*.             |
| Расположение подразделения      | Расположение подразделения.                                                                |
| Тип запроса             | Тип запроса на персонал, такого как клинический и неклинический. Это поле является *обязательным.* |
| Требуемая роль              | Роль запрашиваемого персонала, такого как сиделка или дипломированная медсестра.                          |
| Требуется сейчас или в следующую смену | Выберите смену для запрашиваемого персонала, текущая смена или предстоящая смена.                |
| Сколько                 | Сколько требуется ресурсов, в числовом формате.                |
| Подробно                  | Опишите дополнительные сведения или комментарии для запроса кадрового пула.                        |

## <a name="covid-19-stats"></a>статистика COVID-19

![Статистика COVID-19](media/use/covid19-stats.png)

Отправить конкретные сведения по COVID-19, используя приложение **статистика COVID-19**. Вы можете обновить количество пациентов с конкретными расположениями, находящихся под наблюдением, и пациентов с положительным результатом теста.

Вы также можете добавить другое расположение, используя кнопку **Добавить другое расположение** для отправки статистики для более чем одного расположения.

Выбрать **назад** левом верхнем углу, если вы хотите вернуться к **Приложению Hospital Emergency Response** без отправки каких-либо изменений. Кнопка **Отправить** отправляет введенные вами значения.

После того, как вы отправите данные, у вас есть возможность вернуться в приложение **статистика COVID-19** для создания другой записи, используя кнопку **Отслеживать другой**. Выбрать **главная страница**, чтобы вернуться к **приложение Hospital Emergency Response**.

### <a name="fields-and-description"></a>Поля и описание

| **Имя поля**  | **Описание**                                                                                    |
|-----------------|----------------------------------------------------------------------------------------------------|
| Место        | Имя и тип помещения, отделения или любого другого специального расположения в выбранном помещении.  |
| Пациенты под наблюдением (PUI)            | Количество пациентов под наблюдением.                                                            |
| Положительный        | Число пациентов с положительным результатом теста на COVID-19.                                                         |

## <a name="discharge-planning"></a>Планирование выписки

Предоставьте информацию о выписке и статусе пациента с общим количеством, используя приложение **Планирование выписки**. Вы можете обновить сведения о выписках за последние 24 часа, текущие препятствия при выписке и разбор для препятствий.

Выбрать **назад** левом верхнем углу, если вы хотите вернуться к **Приложению Hospital Emergency Response** без отправки каких-либо изменений. Кнопка **Отправить** отправляет введенные вами значения. Выбрать **главная страница**, чтобы вернуться к **приложение Hospital Emergency Response** после отправки.

### <a name="fields-and-description"></a>Поля и описание

| **Имя поля**            | **Описание**                                                    |
|---------------------------|--------------------------------------------------------------------|
| Авторизация             | Количество пациентов в процессе авторизации.                   |
| Медицинское оборудование длительного пользования | Количество пациентов, использующих медицинское оборудование длительного пользования.            |
| Попечительство              | Количество пациентов под опекой.                             |
| Услуги на дому и в сообществе | Количество пациентов, пользующихся услугами на дому или в сообществе.               |
| Размещение                 | Количество мест размещения необходимо.                                       |
| Квалифицированная учреждение сестринского ухода  | Количество квалифицированных учреждений сестринского ухода.                              |
| **Выписки пациентов**            |                                                                    |
| Последние 24 ч                 | Число пациентов, которых ожидали выписать за последние 24 часа.  |
| Вероятно в следующие 24 часа          | Число пациентов, выписанных за последние 24 часа.                    |

## <a name="other-options"></a>Другие параметры

В этом разделе описываются другие действия, которые вы можете выполнять с помощью компонентов мобильного приложения Hospital Emergency Response.

### <a name="end-shift---sign-out"></a>Окончание смены — выход

Вы можете выйти из приложения, используя значок профиля в левом верхнем углу экрана.  

![Выход](media/use/sign-out.png)

Выберите кнопку **Конец смены**, чтобы завершить сеанс и выйти.

> [!NOTE]
> *Конец смены* может быть недоступна, если ваш IT Администратор отключил совместное использование устройства.

### <a name="provide-feedback"></a>Отправить отзыв

Вы можете поделиться своим отзывом с помощью параметра **Отправить отзыв** из любого компонента мобильного приложения Emergency Response. Чтобы поделиться отзывом, выберите свой профиль в верхнем левом углу, а затем выберите кнопка **Отправить отзыв**:

![Отправить отзыв](media/use/give-feedback.png)

Когда вы выбираете **Отправить отзыв**, у вас есть возможность поделиться похвалой, идеей или сообщить о проблеме с приложением.

### <a name="switch-facility"></a>Сменить помещение

![Сменить помещение](media/use/switch-facility.png)

Переключите помещение в любое время, выбрав название помещения в правой верхней части экрана. После того, как вы выберете название расположения, вы попадете на экран **Приложения Hospital Emergency Response**, где вы можете выбрать другую больницу, регион или помещение.
