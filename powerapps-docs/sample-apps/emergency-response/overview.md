# <a name="hospital-emergency-response---power-platform-sample-solution"></a>Служба экстренной медицинской помощи — пример решения Power Platform

Пример решения для службы экстренной медицинской помощи позволяет медицинским учреждениям собирать актуальные данные о доступных койках, запасах, пациентах с COVID-19, персонале и пациентах, ожидающих выписку. Решение также предоставляет панели мониторинга, которые обобщают ключевые данные и аналитические сведения и позволяют принимать обоснованные решения, увеличивая эффективность развертывания и использования ресурсов.

> [!div class="mx-imgBorder"] 
> ![Приложение для службы экстренной медицинской помощи](media/conf-ermerg-response-solution-overview.png)

Основные компоненты решения для службы экстренной медицинской помощи:

- **Мобильное приложение для персонала, работающего напрямую с пациентами** — персонал, напрямую работающий с пациентами, например медсестры и практикующие врачи, могут использовать мобильное приложение для ввода необходимой информации и ее быстрого просмотра.
- **Веб-приложение для ИТ-администраторов больницы** — администраторы больницы могут использовать это приложение для добавления системных данных, необходимых для работы решения, и управления ими.
- **Панели мониторинга для руководителей, принимающих решения относительно медицинского обслуживания** — используйте панели мониторинга для быстрого просмотра важных данных и метрик, которые помогут в эффективном принятии решений.
